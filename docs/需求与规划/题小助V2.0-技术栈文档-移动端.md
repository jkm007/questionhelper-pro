## 三、网络与存储

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| HTTP 客户端 | uni.request 封装 | 统一请求拦截、Token 注入、错误处理 |
| 状态管理 | Pinia | 跨端状态管理，Composition API setup 风格 |
| 路由拦截 | uni.addInterceptor | 页面权限拦截，未登录跳转登录页 |
| 本地存储 | uni.setStorageSync | Token 缓存、用户配置持久化 |
| 分享 | uni.share | 微信/小程序分享 |

**API 请求封装要点**：
- 基础 URL 来自 `VITE_API_BASE_URL`（默认 `/api/v1`）
- 自动注入 Bearer Token
- 响应码 `'00000'` 为成功
- 响应码 `'A0003'`/`'A0004'` 为令牌过期，跳转登录页
- 请求失败统一 Toast 提示

## 四、平台差异处理

使用 `#ifdef` 条件编译处理平台差异代码：

| 功能 | H5 | 微信小程序 | App |
|------|-----|-----------|-----|
| 登录 | 账号密码 / 手机号 | 微信一键登录 | 手机号 + 第三方登录 |
| 分享 | 复制链接 | 小程序分享 | 系统分享 |
| 支付 | H5 支付 | 小程序支付 | App 支付 |
| 推送 | 不支持 | 订阅消息 | 推送通知 |
| 文件选择 | input[type=file] | uni.chooseMessageFile | uni.chooseFile |
| 扫码 | 不支持 | uni.scanCode | uni.scanCode |
| 图片预览 | uni.previewImage | uni.previewImage | uni.previewImage |

## 七、移动端特有能力

### 7.1 离线支持

| 功能 | 说明 |
|------|------|
| 答题草稿 | 考试/练习中的答案本地缓存，断网后恢复 |
| 题目缓存 | 已加载的题目本地缓存，减少重复请求 |
| 离线提示 | 网络断开时提示用户，不影响已缓存功能 |

### 7.2 性能优化

| 优化项 | 方案 |
|------|------|
| 分包加载 | 按模块分包，首屏只加载核心包 |
| 图片懒加载 | 列表图片懒加载，减少首屏资源 |
| 骨架屏 | 页面加载时显示骨架屏，提升体验 |
| 接口缓存 | 频繁请求的接口结果缓存（分类、配置等） |

### 7.3 小程序特有

| 功能 | 说明 |
|------|------|
| 小程序登录 | wx.login() 获取 code，后端换取 openid |
| 订阅消息 | 考试提醒、成绩通知等订阅消息 |
| 分享 | 页面分享到聊天/朋友圈 |
| 生成小程序码 | 班级邀请、考试入口的小程序码 |

## 八、构建与发布

### 8.1 开发命令

```bash
# H5 开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# App 开发
npm run dev:app
```

### 8.2 生产构建

```bash
# H5 构建
npm run build:h5

# 微信小程序构建
npm run build:mp-weixin

# App 构建
npm run build:app
```

### 8.3 环境配置

```typescript
// .env.development
VITE_API_BASE_URL=http://localhost:8080/api/v1

// .env.production
VITE_API_BASE_URL=https://api.questionhelper.com/api/v1
```

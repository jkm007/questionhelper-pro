## 四、数据库

| 数据库 | 版本 | 用途 | 说明 |
|-------|------|------|------|
| MySQL | 8.0 | 主数据库 | 核心业务数据存储，支持 JSON、窗口函数 |
| GORM | 1.25+ | ORM 层 | 模型定义、自动迁移、关联查询、事务 |

**ORM 规范**：
- 表名：复数蛇形命名（如 `users`、`exam_records`）
- 字段：必含 `id`、`created_at`、`updated_at`、`deleted_at`
- 软删除：统一使用 `deleted_at` 字段
- 外键：应用层维护，不使用数据库外键约束

| 数据结构 | 使用场景 |

| Hash | 用户会话、考试答题缓存 |

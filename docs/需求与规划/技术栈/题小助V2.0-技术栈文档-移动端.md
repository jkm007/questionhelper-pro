## 一、基础框架

| 组件 | 技术选型 | 版本 | 说明 |
|------|---------|------|------|
| 跨端框架 | uni-app | 3.x | Vue 3 + TypeScript，一套代码多端运行 |
| 框架 | Vue 3 | 3.4+ | Composition API、`<script setup>` 语法 |
| 类型系统 | TypeScript | 5.x | 类型安全，提升代码质量 |
| 构建工具 | Vite | 5.x | 快速开发构建、HMR 热更新 |
| 包管理 | npm | - | 依赖管理 |

**目标平台**：
- H5（移动端浏览器）
- 微信小程序
- App（iOS / Android，通过 uni-app 原生编译）

## 三、核心依赖

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| HTTP 客户端 | uni.request 封装 | 统一请求拦截、Token 注入、错误处理 |
| 状态管理 | Pinia | 跨端状态管理，Composition API setup 风格 |
| 路由拦截 | uni.addInterceptor | 页面权限拦截，未登录跳转登录页 |
| WebSocket | uni.connectSocket | 实时通知推送（考试提醒、成绩发布等） |
| 本地存储 | uni.setStorageSync | Token 缓存、用户配置持久化、离线答题草稿 |
| 分享 | uni.share | 微信/小程序分享 |

## 四、平台差异

使用 `#ifdef` 条件编译处理平台差异代码：

| 功能 | H5 | 微信小程序 | App |
|------|-----|-----------|-----|
| 登录 | 账号密码 / 手机号 | 微信一键登录 | 手机号 + 第三方登录 |
| 分享 | 复制链接 | 小程序分享 | 系统分享 |
| 支付 | H5 支付 | 小程序支付 | App 支付 |
| 推送 | 不支持 | 订阅消息 | 推送通知 |
| 文件选择 | input[type=file] | uni.chooseMessageFile | uni.chooseFile |
| 扫码 | 不支持 | uni.scanCode | uni.scanCode |

## 五、性能优化方案

| 优化项 | 方案 |
|------|------|
| 分包加载 | 按模块分包，首屏只加载核心包 |
| 图片懒加载 | 列表图片懒加载，减少首屏资源 |
| 骨架屏 | 页面加载时显示骨架屏，提升体验 |
| 接口缓存 | 频繁请求的接口结果缓存（分类、配置等） |
| 离线缓存 | 答题草稿、已加载题目本地缓存，断网后恢复 |

## 六、构建与环境

### 6.1 开发命令

```bash
npm run dev:h5          # H5 开发服务器
npm run dev:mp-weixin   # 微信小程序开发服务器
```

### 6.2 生产构建

```bash
npm run build:h5        # H5 生产构建
npm run build:mp-weixin # 微信小程序生产构建
```

### 6.3 环境配置

```typescript
// .env.development
VITE_API_BASE_URL=http://localhost:8080/api/v1

// .env.production
VITE_API_BASE_URL=https://api.questionhelper.com/api/v1
```

### 6.4 小程序特有

| 能力 | 说明 |
|------|------|
| 小程序登录 | wx.login() 获取 code，后端换取 openid |
| 订阅消息 | 考试提醒、成绩通知等订阅消息 |
| 分享 | 页面分享到聊天/朋友圈 |
| 小程序码 | 班级邀请、考试入口的小程序码生成 |

## 五、开发规范

### 5.1 包管理与 Node 要求

- 包管理器：**pnpm**（强制使用，不使用 npm/yarn）
- Node 版本：`^20.19.0 || >=22.12.0`

### 5.2 样式规范

- SCSS + UnoCSS
- Prettier：100 字符宽度、2 空格缩进、双引号、分号、尾逗号

### 5.3 代码规范

- ESLint：模板中组件名使用 PascalCase
- Vue 块顺序：template → script → style
- 提交信息：`pnpm commit` 交互式提交，强制 Conventional Commits

### 5.4 常用命令

```bash
pnpm dev              # Vite 开发服务器
pnpm build            # 类型检查 + 生产构建
pnpm build-only       # 生产构建（跳过类型检查）
pnpm type-check       # vue-tsc --noEmit
pnpm lint             # ESLint + Prettier + Stylelint
```

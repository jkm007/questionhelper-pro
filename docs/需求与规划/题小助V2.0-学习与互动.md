## 六、刷题练习系统

### 6.1 资源访问规则

**公共资源访问**：
- 所有登录用户可浏览公共分类树和公共题目
- 所有用户可使用公共题目进行练习和考试

**私有资源访问**：
- 创作者/教师可浏览自己创建的私有分类和题目
- 创作者/教师可使用私有题目进行练习

**班级资源访问**：
- 班级成员可浏览班级内的分类和题目
- 班级成员可使用班级题目进行练习和考试

**练习与错题本关联**：
- 练习答错的题目自动收录到错题本（来源标记为 practice）
- 错题重练模式从错题本中选题练习
- 详见 **七、错题本系统 7.1 错题收集**

> 关联章节：[三、题库管理系统](题小助V2.0-题库与考试.md#三)（题目来源）、[四、考试管理系统 4.3](题小助V2.0-题库与考试.md#四)（考试答题）、**七、错题本系统**（错题管理）

### 6.2 分类浏览

| 题目搜索 | 按关键词搜索题目内容 |

### 6.3 练习系统

#### 6.3.1 练习模式

| 模式 | 说明 | 出题方式 |
|------|------|---------|
| 随机练习 | 按条件随机出题 | 按分类/知识点/难度/标签/数量随机抽取 |
| 顺序练习 | 按顺序逐题练习 | 按题目编号顺序 |
| 错题重练 | 从错题本中选题练习 | 从错题本中选取 |
| 收藏练习 | 从收藏中选题练习 | 从收藏夹中选取 |
| 模拟考试 | 模拟真实考试环境 | 限时、计分，模拟考试流程 |
| 闯关模式 | 逐关挑战 | 按难度递进出题 |
| 计时模式 | 限时答题 | 限定时间内尽可能多答题 |

**出题规则**：
- 父分类包含子分类：选择父分类时自动包含子分类题目
- 资源来源：练习时可选择公共资源、私有资源或班级资源
- 题目数量自定义：自定义每次练习的题目数量
- 题目难度自适应：根据用户水平自动调整题目难度

#### 6.3.2 练习功能

| 功能点 | 功能描述 |
|-------|---------|
| 即时评判 | 选择题点击选项即时判断对错，显示解析 |
| 自动保存 | 每答一题自动保存进度 |
| 练习暂停/继续 | 暂停练习，下次继续 |
| 练习记录 | 记录每次练习的答题结果，通过 session_id 聚合同一批次 |
| 错题自动加入 | 答错的题目自动加入错题本 |

**错题收录规则**：
- 客观题答错：自动加入错题本，记录来源为 practice
- 主观题：用户自判后标记为错的，自动加入错题本
- 详见 **七、错题本系统 7.1 错题收集**

#### 6.3.3 练习管理

| 功能点 | 功能描述 |
|-------|---------|
| 练习统计 | 统计练习时长、正确率、完成题数等 |
| 练习历史 | 查看历史练习记录 |
| 练习计划 | 制定练习计划，按计划推送题目 |
| 每日练习 | 每日推荐一定数量的题目 |
| 练习目标 | 设置练习目标（如：每天30题） |
| 练习打卡 | 练习打卡记录，连续打卡奖励 |
| 练习排行榜 | 练习排行榜（按题数、正确率等） |
| 练习提醒 | 定时提醒用户进行练习 |
| 练习报告 | 生成详细的练习报告（含分析和建议） |

#### 6.3.4 练习数据结构

**session_id 设计**：
- UUID，同一次连续练习共享，便于按批次查询和统计
- 练习暂停时保存 session_id，继续时恢复
- 建议对超过 90 天的练习记录定期归档

**练习记录字段**：
- session_id：练习会话 ID
- user_id：用户 ID
- question_id：题目 ID
- user_answer：用户答案
- is_correct：是否正确
- answer_time：答题时间
- source：来源（practice/exam）

## 七、错题本系统

### 7.1 错题收集

| 功能点 | 功能描述 |
|-------|---------|
| 自动收集 | 考试交卷和练习答错时自动写入错题本 |
| 来源记录 | 记录错题来源（exam / practice）及来源记录 ID |
| 累计次数 | 记录累计答错次数 |
| 手动添加 | 用户可手动添加错题 |

**错题来源说明**：

| 来源 | 触发时机 | 关联章节 |
|------|---------|---------|
| exam（考试） | 考试交卷后客观题答错自动收录 | [四、考试管理系统 4.3.1](题小助V2.0-题库与考试.md#四) |
| practice（练习） | 练习模式答错自动收录 | **六、刷题练习系统 6.1** |

**错题数据关联**：
- 错题记录关联题目 ID，原题目删除后保留快照（显示"题目已删除"）
- 错题记录关联考试/练习记录 ID，支持追溯原始答题场景
- 详见 [三、题库管理系统 3.1.3](题小助V2.0-题库与考试.md#三)（题目删除影响）
| 错题分类 | 错题自动分类（按知识点、题型等） |
| 错题标签 | 给错题打标签（如：粗心、不会、混淆） |
| 错题备注 | 对错题添加备注/笔记 |
| 错题图片 | 支持添加错题图片（如手写解题过程） |
| 错题语音 | 支持添加语音备注 |

### 7.2 错题管理

#### 7.2.1 错题操作

| 功能点 | 功能描述 |
|-------|---------|
| 错题列表 | 查看错题列表，支持按分类筛选 |
| 逐题重做 | 对错题重新作答 |
| 自动移除 | 连续答对 N 次后自动从错题本移除（correct_streak 达到阈值） |
| 手动移除 | 用户手动移除错题 |
| 错题收藏 | 收藏重点错题 |
| 批量操作 | 批量移除、批量收藏等 |

**自动移除规则**：
- 连续答对 3 次自动移除（可配置）
- 正确次数记录在 correct_streak 字段
- 重做答错时重置 correct_streak

#### 7.2.2 错题查询与导出

| 功能点 | 功能描述 |
|-------|---------|
| 错题搜索 | 搜索错题内容 |
| 错题排序 | 按时间、错误次数、分类等排序 |
| 错题筛选 | 按来源、时间范围、错误次数等筛选 |
| 错题导出 | 导出错题为Word/PDF |
| 错题打印 | 打印错题（可选是否包含解析） |
| 错题分享 | 分享错题给其他用户 |
| 错题导入 | 从其他来源导入错题 |
| 错题同步 | 多设备同步错题 |

### 7.3 错题分析

#### 7.3.1 统计分析

| 功能点 | 功能描述 |
|-------|---------|
| 错题趋势 | 查看近 7/30/90 天错题趋势 |
| 薄弱知识点 | 各分类错误率排行，定位薄弱知识点 |
| 错题统计 | 统计错题数量、分布等 |
| 错题对比 | 对比不同时间段的错题情况 |
| 错题排行榜 | 错题数量排行榜（可选） |
| 错题热力图 | 错题时间分布热力图 |

#### 7.3.2 智能推荐

| 功能点 | 功能描述 |
|-------|---------|
| 智能推荐 | 根据错题推荐同类题目强化练习 |
| 错题预测 | 预测可能再次犯错的题目 |
| 学习建议 | 根据错题给出学习建议 |
| 错题报告 | 生成详细的错题分析报告 |
| 错题知识图谱 | 错题知识点关联图谱 |

**推荐规则**：
- 根据错题的知识点推荐同类题目
- 根据错题的难度推荐相近难度题目
- 根据错误频率优先推荐高频错题

| 评论对象 | 说明 | 关联章节 |

| 功能点 | 功能描述 |

### 9.2 创作者功能

#### 9.2.1 私有资源管理

| 功能点 | 功能描述 |
|-------|---------|
| 创建私有题目 | 创作者创建私有题目，仅自己可见，不影响公共题库 |

**私有资源规则**：
- 私有资源仅创建者可见，不影响公共题库
- 私有资源不能提交为公共资源（需通过审核）
- 详见 [三、题库管理系统 3.1.1 资源类型与权限](题小助V2.0-题库与考试.md#三)

6. 详见 [三、题库管理系统 3.5 内容审核](题小助V2.0-题库与考试.md#三)

### 14.1 文件上传

| 功能点 | 功能描述 |
|-------|---------|
| 图片上传 | 支持 jpg/png/gif/webp 格式，单文件最大 5MB |
| 文档上传 | 支持 xlsx/docx/pdf 格式，单文件最大 20MB |
| 音频上传 | 支持 mp3/wav/m4a 格式，单文件最大 50MB |
| 视频上传 | 支持 mp4/webm 格式，单文件最大 200MB |
| 头像上传 | 自动裁剪为正方形，最大 2MB |
| 封面上传 | 班级封面、试卷封面，最大 5MB |
| 批量上传 | 支持多文件同时上传，最多 20 个文件 |
| 拖拽上传 | 支持拖拽文件到上传区域 |
| 上传进度 | 显示上传进度条和上传速度 |

**上传限制规则**：
- 文件名长度不超过 200 字符
- 文件名自动重命名（UUID + 原始扩展名），防止冲突和路径注入
- 图片自动压缩（超过 1920px 宽度时等比缩放）
- 上传前校验文件 MIME 类型，拒绝不匹配的文件
- 同一用户每日上传文件数限制 500 个

### 14.2 文件存储

| 功能点 | 功能描述 |
|-------|---------|
| 存储配置 | 支持本地存储、OSS（阿里云/腾讯云/MinIO）等存储后端 |
| 目录结构 | 按业务类型分目录：avatar/、question/、exam/、class/、temp/ |
| 文件访问 | 公开文件直接访问，私有文件签名 URL（有效期 1 小时） |
| CDN 加速 | 支持配置 CDN 域名加速静态资源访问 |
| 缩略图 | 图片自动生成缩略图（100x100、300x300、800x800） |
| 文件去重 | 基于文件 MD5 去重，相同文件只存储一份 |

### 14.3 文件管理

| 功能点 | 功能描述 |
|-------|---------|
| 文件列表 | 管理员查看所有上传文件，支持按类型、大小、时间筛选 |
| 文件详情 | 查看文件信息：名称、大小、类型、上传者、上传时间、引用次数 |
| 文件删除 | 删除文件（检查引用计数，有引用时提示确认） |
| 孤立文件清理 | 定时清理未被引用的临时文件（temp 目录下超过 24 小时的文件） |
| 存储统计 | 统计存储使用量：总量、各类型占比、增长趋势 |
| 文件搜索 | 按文件名、上传者、类型搜索文件 |
| 批量删除 | 批量删除选中的文件 |

### 14.4 文件引用与安全

| 功能点 | 功能描述 |
|-------|---------|
| 引用计数 | 记录每个文件被引用的次数，防止误删正在使用的文件 |
| 防盗链 | 配置 Referer 白名单，防止外部站点盗链 |
| 病毒扫描 | 上传文件自动进行病毒扫描（可选） |
| 敏感内容检测 | 图片上传自动进行敏感内容检测（可选） |
| 水印功能 | 图片可配置自动添加水印（文字/图片水印） |
| 访问日志 | 记录文件访问日志（访问时间、IP、文件路径） |

**文件生命周期**：
```
上传 → 临时存储（temp/）→ 业务关联 → 正式存储
                                    ↓
                              解除引用 → 标记为孤立
                                    ↓
                              定时清理（24小时后）
```

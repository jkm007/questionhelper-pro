## 二、分层架构说明

| 层级 | 目录 | 职责 |
|------|------|------|
| Views | views/ | 页面视图（按模块划分，每个子目录下 index.vue） |
| Components | components/ | 公共可复用组件（通用 UI 组件） |
| API | api/ | 后端接口封装（扁平文件，每个领域一个 .ts） |
| Store | stores/ | 状态管理（Pinia，扁平文件结构） |
| Utils | utils/ | 工具函数（请求封装、Token 管理） |
| Composables | composables/ | 组合式函数（待开发） |
| Router | router/ | 路由配置 + 权限守卫 |
| Styles | styles/ | 样式文件 |
| Layout | layouts/ | 布局组件（BaseLayout） |
| Directive | directives/ | 自定义指令（权限指令） |
| Types | types/ | TypeScript 类型声明 |
| Plugins | plugins/ | 插件（待开发） |

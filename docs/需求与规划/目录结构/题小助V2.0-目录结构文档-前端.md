## 二、分层架构说明

| 层级 | 目录 | 职责 |
|------|------|------|
| Views | views/ | 页面视图（按模块划分） |
| Components | components/ | 公共可复用组件 |
| API | api/ | 后端接口封装 |
| Store | store/ | 状态管理（Pinia） |
| Utils | utils/ | 工具函数 |
| Hooks | hooks/composables/ | 组合式函数 |
| Router | router/ | 路由配置 |
| Styles | styles/ | 样式文件 |
| Layout | layout/ | 布局组件 |
| Directive | directive/ | 自定义指令 |

## 二、分层架构说明

| 层级 | 目录 | 职责 |
|------|------|------|
| Pages | pages/ | 页面（按功能模块划分） |
| Components | components/ | 公共组件 |
| API | api/ | 接口封装 |
| Store | store/ | 状态管理（Pinia，模块化） |
| Hooks | hooks/ | 自定义 Hooks |
| Utils | utils/ | 工具函数 |
| Static | static/ | 静态资源 |

## 二、分层架构说明

| 层级 | 目录 | 职责 |
|------|------|------|
| Controller | internal/controller/ | 接收请求、参数校验、调用 Service、返回响应 |
| Service | internal/service/ | 核心业务逻辑、事务管理 |
| Repository | internal/repository/ | 数据库操作、数据访问 |
| Model | internal/model/ | 数据模型定义（GORM 模型） |
| DTO | internal/dto/ | 请求/响应数据结构体 |
| Middleware | internal/middleware/ | HTTP 中间件（认证、权限、限流、日志等） |
| Router | internal/router/ | 路由定义（Gin 路由组，按领域分文件） |
| Task | internal/task/ | 定时任务（清理、提醒、过期处理） |
| Job | internal/job/ | 异步任务（导入导出、通知推送等） |
| WS | internal/ws/ | WebSocket 实时通信（Hub/Client 模式） |
| PKG | pkg/ | 公共工具包（可对外暴露） |

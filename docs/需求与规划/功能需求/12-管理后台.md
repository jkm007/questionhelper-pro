## 十二、管理后台功能

### 12.1 仪表盘

#### 12.1.1 数据概览

| 功能点 | 功能描述 |
|-------|---------|
| 核心指标 | 总用户数、总题目数、总考试数、今日活跃等核心指标卡片 |
| 趋势图表 | 用户增长趋势、答题量趋势、正确率趋势 |
| 数据对比 | 不同时间段数据对比（本周/本月/自定义） |

#### 12.1.2 快捷操作

| 功能点 | 功能描述 |
|-------|---------|
| 快速入口 | 新增题目、创建考试、用户管理等快捷入口 |
| 最近活动 | 最近操作记录 |
| 数据预警 | 数据异常预警（如突然大量注册、错误率飙升） |

#### 12.1.3 数据导出与订阅

| 功能点 | 功能描述 |
|-------|---------|
| 数据导出 | 导出仪表盘数据为 Excel/PDF |
| 数据订阅 | 订阅定期数据报告（邮件发送） |
| 数据分享 | 分享数据报告链接 |

**数据订阅 API**：

| API | 方法 | 说明 |
|-----|------|------|
| /api/v1/admin/subscriptions | GET | 我的数据订阅列表 |
| /api/v1/admin/subscriptions | POST | 创建数据订阅 |
| /api/v1/admin/subscriptions/:id | PUT | 修改订阅 |
| /api/v1/admin/subscriptions/:id | DELETE | 删除订阅 |

**订阅配置**：
- 报告类型：日报/周报/月报
- 发送时间：可配置
- 发送方式：邮件
- 包含内容：核心指标、趋势图、异常预警

### 12.2 菜单权限控制

**菜单结构**：
```
├── 首页 (Dashboard)
├── 用户管理
│   ├── 用户列表 (user:list:view)
│   └── 班级创作者管理 (class:creator:view)
├── 题库管理
│   ├── 公共题目 (question:public:view)
│   ├── 私有题目 (question:private:view)
│   ├── 班级题目 (question:class:view)
│   ├── 题目分类 (category:list:view)
│   ├── 知识点管理 (knowledge:list:view)
│   ├── 批量导入 (question:import:view)
│   └── 批量导出 (question:export:view)
├── 考试管理
│   ├── 试卷列表 (exam:paper:view)
│   ├── 组卷管理 (exam:composer:view)
│   ├── 考试发布 (exam:publish:view)
│   ├── 考试监控 (exam:monitor:view)
│   └── 成绩管理 (score:list:view)
├── 班级管理
│   ├── 班级列表 (class:list:view)
│   ├── 班级详情 (class:detail:view)
│   ├── 班级创作者审批 (class:creator:approve)
│   └── 班级资源管理 (class:resource:view)
├── 数据统计
│   ├── 数据概览 (statistics:overview:view)
│   ├── 用户统计 (statistics:user:view)
│   ├── 答题统计 (statistics:answer:view)
│   ├── 成绩分析 (statistics:score:view)
│   └── 系统监控 (statistics:system:view)
├── 系统设置
│   ├── 基础设置 (system:settings:view)
│   ├── 安全设置 (system:security:view)
│   ├── 角色管理 (role:list:view)
│   ├── 权限管理 (permission:list:view)
│   ├── 操作审计 (system:audit:view)
│   ├── 日志管理 (system:logs:view)
│   ├── 通知设置 (system:notification:view)
│   └── 数据备份 (system:backup:view)
└── 个人中心
    ├── 个人信息 (profile:info:view)
    ├── 修改密码 (profile:password:view)
    └── 我的消息 (profile:message:view)
```

**说明**：以上菜单结构为管理后台（管理员/教师/创作者视角）。学生角色通过移动端使用功能，不访问管理后台，因此学生端功能（答题、错题本、收藏、练习等）不在管理后台菜单树中。学生端菜单详见 **十三、移动端 13.6 Tab 栏导航**。

**动态菜单管理功能**：

> 菜单通过后端接口动态管理，前端根据用户角色动态加载菜单

| 功能点 | 功能描述 |
|-------|---------|
| 菜单列表 | 查看所有菜单，支持树形展示（无限层级） |
| 新增菜单 | 创建菜单，设置菜单名称、路由路径、图标、父菜单、排序、类型（目录/菜单/按钮） |
| 编辑菜单 | 修改菜单信息 |
| 删除菜单 | 删除菜单（有子菜单时禁止删除） |
| 菜单状态 | 启用/禁用菜单，禁用后不显示 |
| 菜单图标 | 支持 Element Plus 图标或自定义 SVG 图标 |
| 菜单类型 | 目录（父级分组）、菜单（可点击页面）、按钮（页面内操作权限） |
| 菜单缓存 | 菜单数据缓存到 Redis，修改后自动刷新缓存 |

**菜单数据结构**：

| 字段 | 说明 |
|------|------|
| id | 菜单 ID |
| parent_id | 父菜单 ID（NULL=顶级） |
| name | 菜单名称（路由 name） |
| type | 菜单类型：1=目录 / 2=菜单 / 3=按钮 |
| path | 路由路径（如 /user/list） |
| component | 前端组件路径（如 system/user/index），目录类型填 Layout |
| redirect | 重定向路径 |
| title | 菜单显示标题 |
| icon | 菜单图标（Element Plus 图标名） |
| hidden | 是否隐藏（false=显示 true=隐藏） |
| permission | 权限标识（三级编码，如 user:list:view） |
| sort | 排序权重（升序） |
| status | 状态（0=禁用 1=启用） |

**动态菜单加载流程**：

> 详见 [技术实现文档 §5.2 动态菜单加载](../../技术实现文档/12-管理后台.md#52-动态菜单加载)

**菜单权限管理**：

> 详细权限管理功能见 **11.4 权限管理**

**按钮权限管理功能**：

> 按钮权限挂在菜单下，通过 permission 标识控制页面内操作

| 功能点 | 功能描述 |
|-------|---------|
| 按钮列表 | 查看菜单下的按钮权限列表 |
| 新增按钮 | 创建按钮权限，设置名称、权限标识（如 user:create）、所属菜单 |
| 编辑按钮 | 修改按钮权限信息 |
| 删除按钮 | 删除按钮权限 |
| 按钮状态 | 启用/禁用按钮权限 |

**按钮权限数据结构**：

| 字段 | 说明 |
|------|------|
| id | 按钮 ID |
| menu_id | 所属菜单 ID |
| name | 按钮名称（如"新增"、"编辑"、"删除"） |
| permission | 权限标识（如 user:create、user:edit、user:delete） |
| status | 状态（0=禁用 1=启用） |

**按钮权限使用示例**：

> 详见 [技术实现文档 §5.3 按钮权限控制](../../技术实现文档/12-管理后台.md#53-按钮权限控制)

**权限标识命名规范**：`模块:子模块:操作`（三级编码）

| 模块 | 示例 |
|------|------|
| 用户 | `user:list:view`, `user:list:create`, `user:list:edit`, `user:list:delete` |
| 题目 | `question:public:view`, `question:private:view`, `question:class:view`, `question:public:create` |
| 考试 | `exam:paper:view`, `exam:paper:create`, `exam:publish:view`, `exam:monitor:view` |
| 班级 | `class:list:view`, `class:list:create`, `class:member:manage`, `class:creator:approve` |
| 系统 | `system:settings:view`, `system:log:view`, `system:backup:view` |

注意：与 **11.4 权限管理** 中的编码规范保持一致（`模块:子模块:操作`）。

### 12.3 通用组件

#### 12.3.1 列表与表单

| 功能点 | 功能描述 |
|-------|---------|
| 分页组件 | 统一分页参数：page + page_size，响应包含 list/total/page/page_size |
| 搜索筛选 | 关键词搜索 + 多条件筛选组合 |
| 批量操作 | 列表页支持批量选择 + 批量操作 |
| 表单验证 | 前端表单校验 + 后端参数校验（go-playground/validator） |
| 树形选择器 | 分类选择统一使用树形选择器组件 |

#### 12.3.2 富文本与文件

| 功能点 | 功能描述 |
|-------|---------|
| 富文本编辑器 | 题目内容支持图片、公式、代码等富文本 |
| 文件上传 | Excel 导入、头像上传、封面上传 |
| 数据导出 | 导出数据为 Excel/CSV |
| 数据导入 | 导入数据 |
| 打印功能 | 打印页面/数据 |

#### 12.3.3 界面与交互

| 功能点 | 功能描述 |
|-------|---------|
| 资源类型切换 | 创作者/教师可在公共/私有/班级资源间切换查看 |
| 审批流程 | 班级创作者申请审批流程组件 |
| 主题切换 | 切换管理后台主题（亮色/暗色） |
| 多语言支持 | 多语言切换 |
| 快捷键 | 键盘快捷键支持 |
| 消息通知 | 站内消息通知 |

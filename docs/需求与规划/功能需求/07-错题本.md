## 七、错题本系统

### 7.1 错题收集

| 功能点 | 功能描述 |
|-------|---------|
| 自动收集 | 考试交卷和练习答错时自动写入错题本 |
| 来源记录 | 记录错题来源（exam / practice）及来源记录 ID |
| 累计次数 | 记录累计答错次数 |
| 手动添加 | 用户可手动添加错题 |

**错题来源说明**：

| 来源 | 触发时机 | 关联章节 |
|------|---------|---------|
| exam（考试） | 考试交卷后客观题答错自动收录 | **四、考试管理系统 4.3.1** |
| practice（练习） | 练习模式答错自动收录 | **六、刷题练习系统 6.1** |

**错题数据关联**：
- 错题记录关联题目 ID，原题目删除后保留快照（显示"题目已删除"）
- 错题记录关联考试/练习记录 ID，支持追溯原始答题场景
- 详见 **三、题库管理系统 3.1.3**（题目删除影响）
| 错题分类 | 错题自动分类（按知识点、题型等） |
| 错题标签 | 给错题打标签（如：粗心、不会、混淆） |
| 错题备注 | 对错题添加备注/笔记 |
| 错题图片 | 支持添加错题图片（如手写解题过程） |
| 错题语音 | 支持添加语音备注 |

### 7.2 错题管理

#### 7.2.1 错题操作

| 功能点 | 功能描述 |
|-------|---------|
| 错题列表 | 查看错题列表，支持按分类筛选 |
| 逐题重做 | 对错题重新作答 |
| 自动移除 | 连续答对 N 次后自动从错题本移除（correct_streak 达到阈值） |
| 手动移除 | 用户手动移除错题 |
| 错题收藏 | 收藏重点错题 |
| 批量操作 | 批量移除、批量收藏等 |

**自动移除规则**：
- 连续答对 3 次自动移除（可配置）
- 正确次数记录在 correct_streak 字段
- 重做答错时重置 correct_streak

#### 7.2.2 错题查询与导出

| 功能点 | 功能描述 |
|-------|---------|
| 错题搜索 | 搜索错题内容 |
| 错题排序 | 按时间、错误次数、分类等排序 |
| 错题筛选 | 按来源、时间范围、错误次数等筛选 |
| 错题导出 | 导出错题为Word/PDF |
| 错题打印 | 打印错题（可选是否包含解析） |
| 错题分享 | 分享错题给其他用户 |
| 错题导入 | 从其他来源导入错题 |
| 错题同步 | 多设备同步错题 |

### 7.3 错题分析

#### 7.3.1 统计分析

| 功能点 | 功能描述 |
|-------|---------|
| 错题趋势 | 查看近 7/30/90 天错题趋势 |
| 薄弱知识点 | 各分类错误率排行，定位薄弱知识点 |
| 错题统计 | 统计错题数量、分布等 |
| 错题对比 | 对比不同时间段的错题情况 |
| 错题排行榜 | 错题数量排行榜（可选） |
| 错题热力图 | 错题时间分布热力图 |

#### 7.3.2 智能推荐

| 功能点 | 功能描述 |
|-------|---------|
| 智能推荐 | 根据错题推荐同类题目强化练习 |
| 错题预测 | 预测可能再次犯错的题目 |
| 学习建议 | 根据错题给出学习建议 |
| 错题报告 | 生成详细的错题分析报告 |
| 错题知识图谱 | 错题知识点关联图谱 |

**推荐规则**：
- 根据错题的知识点推荐同类题目
- 根据错题的难度推荐相近难度题目
- 根据错误频率优先推荐高频错题

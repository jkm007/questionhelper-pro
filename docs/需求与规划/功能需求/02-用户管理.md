## 二、用户管理系统

### 2.1 用户信息管理

#### 2.1.1 基本信息

| 功能点 | 功能描述 |
|-------|---------|
| 个人信息查看 | 查看头像、昵称、真实姓名、性别、邮箱、手机号、简介 |
| 个人信息修改 | 修改昵称、头像、性别、生日、简介等 |
| 头像上传 | 上传头像，支持jpg/png格式，大小不超过2MB，自动裁剪为正方形 |
| 角色信息 | 查看当前用户拥有的角色列表 |
| 班级创作者状态 | 查看在各班级的班级创作者状态（已申请/已通过/已驳回） |

**信息修改规则**：
- 昵称：2-20个字符，支持中英文、数字、下划线，敏感词过滤
- 简介：最多200个字符
- 用户名：注册后不可修改
- 真实姓名：实名认证后不可修改（需联系管理员）

#### 2.1.2 账号安全

| 功能点 | 功能描述 |
|-------|---------|
| 修改密码 | 输入旧密码验证后设置新密码 |
| 绑定邮箱 | 绑定或更换邮箱，需验证新邮箱 |
| 绑定手机号 | 绑定或更换手机号，需验证新手机号（已实名认证的需验证原手机号） |
| 第三方账号管理 | 查看已绑定的第三方账号，支持绑定/解绑 |
| 登录日志 | 查看最近登录记录（时间、IP、设备），支持查看异常登录 |

**密码修改规则**：
- 需输入旧密码验证
- 新密码不能与最近 3 次历史密码相同
- 密码修改成功后，所有设备强制退出，需重新登录
- 修改成功后发送站内消息 + 邮箱通知

**绑定规则**：
- 邮箱：绑定或更换需验证新邮箱验证码
- 手机号：绑定或更换需验证新手机号（已实名认证的需验证原手机号）
- 第三方账号：详见 **1.9 第三方账号绑定**

#### 2.1.3 隐私设置

| 设置项 | 可选值 | 说明 |
|-------|-------|------|
| 个人主页可见性 | 所有人/仅班级成员/仅自己 | 控制其他用户是否能访问个人主页 |
| 真实姓名可见性 | 所有人/仅班级成员/仅自己 | 控制真实姓名是否对外显示 |
| 手机号可见性 | 仅自己 | 手机号默认不对外显示 |
| 邮箱可见性 | 所有人/仅班级成员/仅自己 | 控制邮箱是否对外显示 |
| 学习统计可见性 | 所有人/仅班级成员/仅自己 | 控制学习数据是否对外显示 |
| 班级信息可见性 | 所有人/仅班级成员/仅自己 | 控制加入的班级是否对外显示 |

**隐私设置规则**：
- 默认值：手机号仅自己，其他为所有人
- 班级成员可查看：加入同一班级的成员可查看
- 仅自己：只有本人可查看

#### 2.1.4 隐私设置 API

| API | 方法 | 说明 |
|-----|------|------|
| /api/v1/user/privacy | GET | 获取当前用户隐私设置 |
| /api/v1/user/privacy | PUT | 更新隐私设置 |

**请求参数**：各隐私设置项的键值对，如 `{"profile_visible": "classmates", "name_visible": "all"}`

### 2.2 实名认证

| 功能点 | 功能描述 |
|-------|---------|
| 绑定手机号 | 绑定手机号，接收短信验证码验证（如未绑定） |
| 填写实名信息 | 填写真实姓名、身份证号 |
| 认证提交 | 提交认证，系统自动验证身份信息格式 |
| 认证状态查看 | 查看认证状态（未认证/已认证） |

**认证规则**：
- 申请创作者/教师角色前必须完成实名认证
- 实名认证通过手机号验证码+身份信息格式验证，无需人工审核
- 认证通过后姓名和身份证号不可修改，如需修改需联系管理员
- 一个身份证号只能认证一个账号
- 一个手机号只能绑定一个账号
- 绑定的手机号即为账号安全手机，用于找回密码和登录

**身份证校验规则**：
- 必须为18位身份证号码
- 校验前17位为数字，第18位为数字或X
- 校验位算法验证（ISO 7064:1983.MOD 11-2）
- 校验出生日期合法性
- 校验地址码合法性

**敏感数据保护要求**：
- 身份证号存储时使用 AES 加密，密钥独立管理
- 日志中身份证号脱敏显示（如 110***********1234）
- API 响应中身份证号脱敏显示
- 查询接口不支持按身份证号精确搜索

### 2.3 角色申请

**班级创作者身份说明**：班级创作者不是独立的系统角色，而是教师角色在班级维度的子权限。班级创作者权限通过班级成员申请 → 教师审批获得，仅在申请的班级内有效。详见 **五、班级管理系统 5.3 班级创作者管理**。

#### 2.3.1 申请成为创作者

| 功能点 | 功能描述 |
|-------|---------|
| 申请入口 | 用户可在个人中心申请成为创作者 |
| 申请条件 | 必须完成实名认证 |
| 申请信息 | 填写申请理由、擅长领域、创作经验说明 |
| 申请状态 | 查看申请状态（见下方状态机） |
| 审核结果 | 通过/驳回，驳回需填写原因 |

**申请流程**：
```
用户申请 → 填写信息 → 提交审核 → 管理员审核 → 通过 → 获得创作者角色
                              ↓
                            驳回 → 修改信息重新提交
```

#### 2.3.2 申请成为教师

| 功能点 | 功能描述 |
|-------|---------|
| 申请入口 | 用户可在个人中心申请成为教师 |
| 申请条件 | 必须完成实名认证 |
| 申请信息 | 填写申请理由、教学经验、所属机构/学校 |
| 申请状态 | 查看申请状态（见下方状态机） |
| 审核结果 | 通过/驳回，驳回需填写原因 |

**申请流程**：
```
用户申请 → 填写信息 → 提交审核 → 管理员审核 → 通过 → 获得教师角色
                              ↓
                            驳回 → 修改信息重新提交
```

#### 2.3.3 其他角色获取方式

以下角色不支持用户自主申请，由管理员手动分配：

| 角色 | 分配方式 | 说明 |
|------|---------|------|
| 内容管理员 | 管理员手动分配 | 管理公共题库内容 |
| 用户管理员 | 管理员手动分配 | 管理用户账号 |
| 数据分析师 | 管理员手动分配 | 查看统计数据 |
| 运营人员 | 管理员手动分配 | 管理公告和活动 |

#### 2.3.4 申请状态机

**状态定义**：

| 状态 | 说明 |
|------|------|
| pending（待审核） | 已提交申请，等待管理员审核 |
| approved（已通过） | 审核通过，已获得角色 |
| rejected（已驳回） | 审核驳回，可修改后重新提交 |
| revoked（已撤销） | 管理员撤销角色，需重新申请 |

**状态流转**：
```
未申请 → pending（提交申请）
pending → approved（审核通过）
pending → rejected（审核驳回）
approved → revoked（管理员撤销角色）
rejected → pending（重新提交申请）
revoked → pending（重新申请）
```

**状态流转规则**：
- 同一角色同时只能有一个 pending 状态的申请
- 已通过的申请不可重复申请（如需重新申请需先被撤销）
- 驳回后可立即重新提交（无冷却期）
- 撤销后 30 天内不可重新申请（防止滥用）

#### 2.3.5 审核规则与资源影响

**审核规则**：
- 创作者/教师申请由管理员/超级管理员审核
- 审核通过后立即生效，用户获得对应角色
- 审核驳回需填写驳回原因，用户可查看
- 用户可同时申请创作者和教师角色
- 申请期间可正常使用学生功能

**角色撤销后资源归属**：

| 角色 | 撤销后资源处理 |
|------|--------------|
| 创作者 | 私有题目保留，仅创建者可见；已发布的公共题目不受影响 |
| 教师 | 班级保留（可指定新教师）；私有题目保留；班级考试/作业保留 |
| 班级创作者 | 班级资源保留，但不可再创建新资源；已有资源不可编辑 |

**统一规则**：角色撤销后，已创建的私有资源保留但仅创建者可见；已发布的公共资源不受影响；班级资源保留归属班级。撤销后用户不可再创建新资源，已有资源不可编辑但可查看。

**用户禁用/注销后资源影响**：

| 场景 | 资源处理 |
|------|---------|
| 用户被禁用 | 用户不可登录；创建的私有题目不可见；班级资源保留可见；公共题目不受影响 |
| 用户账号注销 | 私有题目 30 天后物理删除；班级资源保留（归属班级）；公共题目保留（归属系统）；错题本数据删除 |

> 关联章节：**三、题库管理系统 3.1.1**（资源类型与权限）、**五、班级管理系统 5.3**（班级创作者管理）

### 2.4 管理端用户管理

#### 2.4.1 用户管理功能

| 功能点 | 功能描述 |
|-------|---------|
| 用户列表 | 分页查看用户，支持按角色、状态、关键字（用户名/邮箱/手机号）筛选，支持按注册时间范围筛选 |
| 用户详情 | 查看用户详细信息：基本信息、实名认证信息、角色列表、登录日志、角色申请记录、第三方账号绑定情况 |
| 创建用户 | 管理员手动创建用户，分配角色 |
| 编辑用户 | 修改用户资料、状态 |
| 删除用户 | 管理员删除用户（软删除，deleted_at 标记） |
| 用户状态 | 启用/禁用用户，禁用后不可登录 |
| 分配角色 | 为用户分配一个或多个角色，角色全量替换 |
| 重置密码 | 管理员重置用户密码，生成随机密码并强制用户下次登录修改 |
| 用户导出 | 按筛选条件导出用户列表为Excel |
| 第三方账号管理 | 查看用户绑定的第三方账号，支持管理员解绑 |

**批量操作**：

| 操作 | 说明 |
|------|------|
| 批量启用 | 批量启用选中的用户 |
| 批量禁用 | 批量禁用选中的用户 |
| 批量删除 | 批量软删除选中的用户 |
| 批量分配角色 | 批量为选中用户分配角色 |

#### 2.4.2 用户标签管理

| 功能点 | 功能描述 |
|-------|---------|
| 标签列表 | 查看所有标签，含标签名称、颜色、用户数 |
| 创建标签 | 创建标签，设置名称、颜色、描述 |
| 编辑标签 | 修改标签信息 |
| 删除标签 | 删除标签（已关联用户的标签需先解绑） |
| 打标签 | 为用户添加/移除标签，支持批量操作 |
| 按标签筛选 | 用户列表支持按标签筛选 |

**预置标签**：

| 标签 | 颜色 | 说明 |
|------|------|------|
| VIP用户 | 金色 | 付费或特殊贡献用户 |
| 活跃用户 | 绿色 | 近30天登录≥20天 |
| 新用户 | 蓝色 | 注册≤7天 |
| 待激活 | 灰色 | 注册后未完成首次登录 |
| 重点用户 | 红色 | 需要重点关注的用户 |

#### 2.4.3 角色申请审核

| 审核类型 | 审核人 | 说明 |
|---------|--------|------|
| 创作者申请审核 | 管理员/超级管理员 | 审核用户提交的创作者角色申请（通过/驳回） |
| 教师申请审核 | 管理员/超级管理员 | 审核用户提交的教师角色申请（通过/驳回） |
| 班级创作者管理 | 管理员/超级管理员/教师 | 查看所有班级创作者申请，支持管理员审批 |

**审核规则**：
- 审核通过后立即生效，用户获得对应角色
- 审核驳回需填写驳回原因，用户可查看
- 用户可同时申请创作者和教师角色
- 详见 **2.3 角色申请**

#### 2.4.4 用户删除与注销

| 对比项 | 用户删除（管理员） | 账号注销（用户主动） |
|-------|-------------------|---------------------|
| 触发方 | 管理员操作 | 用户主动申请 |
| 硬删除 | 软删除（deleted_at 标记） | 标记为已注销，30天后硬删除 |
| 数据保留 | 数据保留，可恢复 | 30天冷静期后数据永久删除 |
| 登录限制 | 立即不可登录 | 立即不可登录 |
| 恢复 | 管理员可恢复 | 30天内可联系客服撤销 |

**删除规则**：
- 管理员删除用户为软删除，数据保留可恢复
- 账号注销为用户主动申请，有 30 天冷静期
- 详见 **1.6 账号注销**

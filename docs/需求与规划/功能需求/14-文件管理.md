## 十四、文件管理

**文件使用场景**：

| 业务场景 | 文件类型 | 关联章节 |
|---------|---------|---------|
| 题目附件 | 图片、音频、视频 | **三、题库管理系统 3.1.2** |
| 试卷导出 | Word、PDF | **四、考试管理系统 4.1.2** |
| 班级封面 | 图片 | **五、班级管理系统 5.1** |
| 用户头像 | 图片 | **二、用户管理系统 2.1.1** |
| Excel 导入/导出 | Excel | **三、题库管理系统 3.2** |
| 错题导出 | Word、PDF | **七、错题本系统 7.2.2** |

### 14.1 文件上传

| 功能点 | 功能描述 |
|-------|---------|
| 图片上传 | 支持 jpg/png/gif/webp 格式，单文件最大 5MB |
| 文档上传 | 支持 xlsx/docx/pdf 格式，单文件最大 20MB |
| 音频上传 | 支持 mp3/wav/m4a 格式，单文件最大 50MB |
| 视频上传 | 支持 mp4/webm 格式，单文件最大 200MB |
| 头像上传 | 自动裁剪为正方形，最大 2MB |
| 封面上传 | 班级封面、试卷封面，最大 5MB |
| 批量上传 | 支持多文件同时上传，最多 20 个文件 |
| 拖拽上传 | 支持拖拽文件到上传区域 |
| 上传进度 | 显示上传进度条和上传速度 |

**上传限制规则**：
- 文件名长度不超过 200 字符
- 文件名自动重命名（UUID + 原始扩展名），防止冲突和路径注入
- 图片自动压缩（超过 1920px 宽度时等比缩放）
- 上传前校验文件 MIME 类型，拒绝不匹配的文件
- **文件类型校验增强**：
  - 后端必须校验文件 Magic Number（文件头字节），不仅依赖扩展名和 MIME 类型
  - 图片文件尝试解码验证（调用 image.Decode）
  - 禁止双扩展名攻击（如 .jpg.exe）
  - 文件扩展名白名单校验
- **文件名校验规则**：
  - 文件名长度不超过 255 字符
  - 禁止特殊字符：`< > : " / \ | ? *`
  - 禁止双扩展名（如 report.jpg.exe）
  - 上传后重命名为 UUID + 原始扩展名
  - 保留原始文件名用于下载显示
- 同一用户每日上传文件数限制 500 个

**文件大小限制**（全局统一）：

| 文件类型 | 单文件限制 | 允许格式 |
|---------|-----------|---------|
| 图片 | 5MB | jpg, png, gif, webp |
| 文档 | 20MB | pdf, doc, docx, xls, xlsx, ppt, pptx |
| 音频 | 50MB | mp3, wav |
| 视频 | 200MB | mp4 |
| 压缩包 | 50MB | zip, rar |

**关联说明**：题目附件大小限制与上述全局限制一致，详见 **三、题库管理系统 3.1**。

### 14.2 文件存储

| 功能点 | 功能描述 |
|-------|---------|
| 存储配置 | 支持本地存储、OSS（阿里云/腾讯云/MinIO）等存储后端 |
| 目录结构 | 按业务类型分目录：avatar/、question/、exam/、class/、temp/ |
| 文件访问 | 公开文件直接访问，私有文件签名 URL（有效期 1 小时） |
| CDN 加速 | 支持配置 CDN 域名加速静态资源访问 |
| 缩略图 | 图片自动生成缩略图（100x100、300x300、800x800） |
| 文件去重 | 基于文件 MD5 去重，相同文件只存储一份 |

### 14.3 文件管理

| 功能点 | 功能描述 |
|-------|---------|
| 文件列表 | 管理员查看所有上传文件，支持按类型、大小、时间筛选 |
| 文件详情 | 查看文件信息：名称、大小、类型、上传者、上传时间、引用次数 |
| 文件删除 | 删除文件（检查引用计数，有引用时提示确认） |
| 孤立文件清理 | 定时清理未被引用的临时文件（temp 目录下超过 24 小时的文件） |
| 存储统计 | 统计存储使用量：总量、各类型占比、增长趋势 |
| 文件搜索 | 按文件名、上传者、类型搜索文件 |
| 批量删除 | 批量删除选中的文件 |

### 14.4 文件引用与安全

| 功能点 | 功能描述 |
|-------|---------|
| 引用计数 | 记录每个文件被引用的次数，防止误删正在使用的文件 |
| 防盗链 | 配置 Referer 白名单，防止外部站点盗链 |
| 病毒扫描 | 上传文件自动进行病毒扫描（强制） |
| 敏感内容检测 | 图片上传自动进行敏感内容检测（可选） |
| 水印功能 | 图片可配置自动添加水印（文字/图片水印） |
| 访问日志 | 记录文件访问日志（访问时间、IP、文件路径） |

**病毒扫描**：所有上传文件必须经过病毒扫描，扫描失败时拒绝上传。扫描引擎：ClamAV 或商业杀毒 SDK。

**水印安全要求**：水印内容包含用户 ID 和时间戳，使用不可见数字水印技术，普通裁剪和截图无法去除。

**签名 URL 有效期说明**：默认有效期 1 小时。考试场景中，如考试时长超过 1 小时，签名 URL 需支持自动续期（通过前端在过期前重新获取）。

**文件生命周期**：
```
上传 → 临时存储（temp/）→ 业务关联 → 正式存储
                                    ↓
                              解除引用 → 标记为孤立
                                    ↓
                              定时清理（24小时后）
```

**文件清理与软删除规则**：
- 题目软删除时，附件保留不清理（文件引用计数不变）
- 题目物理删除后，附件引用计数减 1
- 引用计数为 0 的文件标记为孤立文件
- 定时任务每日清理超过 7 天的孤立文件
- 软删除的文件保留 30 天后自动清理

**缩略图与签名 URL**：

| API | 方法 | 说明 |
|-----|------|------|
| /api/v1/files/:id/thumbnail | GET | 获取图片缩略图（支持 w/h 参数） |
| /api/v1/files/:id/signed-url | GET | 获取私有文件签名 URL（有效期 1 小时） |

**缩略图规则**：
- 仅图片文件支持缩略图
- 预设尺寸：small (100x100)、medium (300x300)、large (800x800)
- 缩略图缓存 7 天

**签名 URL 规则**：
- 私有文件通过签名 URL 访问，有效期 1 小时
- 签名包含文件 ID、过期时间、用户 ID
- 签名 URL 不可转发使用

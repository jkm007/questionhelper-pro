## 九、内容创作系统（V2.0 新增）

### 9.1 创作者入驻

| 功能点 | 功能描述 |
|-------|---------|
| 申请入驻 | 用户申请成为创作者，填写申请理由和资质说明 |
| 资质审核 | 管理员审核创作者资质 |
| 创作者认证 | 认证标识展示 |
| 创作者主页 | 创作者个人主页展示 |
| 创作者等级 | 根据贡献度设置创作者等级 |
| 创作者协议 | 创作者协议/规则说明 |
| 创作者资质 | 创作者资质展示 |
| 创作者作品集 | 创作者作品集展示 |

**创作者等级体系**：

| 等级 | 条件 | 权益 |
|------|------|------|
| 初级创作者 | 发布 ≥ 10 道公共题目 | 基础创作权限 |
| 中级创作者 | 发布 ≥ 50 道公共题目，平均正确率 40%-80% | 优先审核、创作者标识 |
| 高级创作者 | 发布 ≥ 200 道公共题目，被使用 ≥ 1000 次 | 专属推荐、数据报告 |
| 特邀创作者 | 管理员手动设置 | 全部权益、免审核 |

**升降级规则**：
- 每月初自动评估，满足条件自动升级
- 连续 3 个月不满足当前等级条件，降一级
- 降级后已发布内容不受影响

### 9.2 创作者功能

#### 9.2.1 私有资源管理

| 功能点 | 功能描述 |
|-------|---------|
| 创建私有题目 | 创作者创建私有题目，仅自己可见，不影响公共题库 |
| 创建私有分类 | 创作者创建私有分类，仅自己使用 |
| 创建私有知识点 | 创作者创建私有知识点，关联私有题目 |
| 创建私有试卷 | 创作者使用私有题目或公共题目创建私有试卷 |
| 使用公共资源 | 创作者可浏览和使用公共题库、公共分类、公共知识点 |
| 内容管理 | 管理自己的私有内容（只能看到自己创建的） |

**私有资源规则**：
- 私有资源仅创建者可见，不影响公共题库
- 私有资源不能提交为公共资源（需通过审核）
- 详见 **三、题库管理系统 3.1.1 资源类型与权限**

**私有资源管理 API**：

| API | 方法 | 说明 |
|-----|------|------|
| /api/v1/my/questions | GET | 我创建的私有题目列表 |
| /api/v1/my/papers | GET | 我创建的私有试卷列表 |
| /api/v1/my/categories | GET | 我创建的私有分类列表 |
| /api/v1/my/knowledge | GET | 我创建的私有知识点列表 |
| /api/v1/my/statistics | GET | 我的内容使用统计 |

#### 9.2.2 内容操作

| 功能点 | 功能描述 |
|-------|---------|
| 内容编辑 | 编辑已创建的内容 |
| 内容删除 | 删除已创建的内容 |
| 内容复制 | 复制已有内容 |
| 内容导出 | 导出内容为Word/PDF |
| 内容分享 | 分享内容给其他用户 |
| 内容版本管理 | 管理内容的版本历史 |
| 内容标签 | 给内容打标签 |
| 内容收藏 | 收藏常用内容 |
| 内容预览 | 预览内容 |
| 内容搜索 | 搜索内容 |
| 内容排序 | 按时间、热度等排序 |

**版本管理规则**：
- 每次保存自动生成新版本（版本号递增：v1、v2、v3...）
- 支持查看任意历史版本
- 支持版本差异对比（高亮显示修改内容）
- 支持回滚到任意历史版本
- 最多保留 50 个历史版本

#### 9.2.3 提交审核

| 功能点 | 功能描述 |
|-------|---------|
| 提交公共审核 | 创作者可将私有题目/试卷提交为公共内容，需审核 |
| 数据统计 | 查看自己内容的使用数据（被练习次数、正确率等） |

**提交审核流程**：
1. 创作者选择要提交的私有内容
2. 填写提交说明
3. 内容管理员/管理员审核
4. 审核通过 → 内容变为公共内容
5. 审核驳回 → 内容退回为私有状态，创作者可修改后重新提交
6. 详见 **三、题库管理系统 3.5 内容审核**

### 9.3 教师功能（比创作者多班级管理）

| 功能点 | 功能描述 |
|-------|---------|
| 创建班级 | 教师可创建班级，邀请成员加入 |
| 班级资源管理 | 教师可为班级创建题目、分类、知识点，仅班级成员可见 |
| 班级创作者管理 | 教师可审批班级成员的班级创作者申请 |
| 班级考试 | 教师可为班级发布考试，选择公共题目或班级题目 |
| 班级成绩管理 | 教师可查看班级成员的考试成绩和学习进度 |
| 使用公共资源 | 教师可浏览和使用公共题库、公共分类、公共知识点 |
| 私有资源管理 | 教师可创建私有题目、分类、知识点、试卷 |
| 班级作业管理 | 布置和管理班级作业 |
| 班级公告管理 | 发布和管理班级公告 |
| 班级讨论管理 | 管理班级讨论区 |
| 班级成员管理 | 管理班级成员 |
| 班级数据分析 | 班级数据分析 |
| 班级报告 | 班级报告生成 |

### 9.4 班级创作者功能

> 详细管理功能见 **5.3 班级创作者管理**

| 功能点 | 功能描述 |
|-------|---------|
| 申请成为班级创作者 | 班级成员可申请成为班级创作者，填写申请理由 |
| 创建班级题目 | 班级创作者可在班级内创建题目，仅班级成员可见 |
| 创建班级分类 | 班级创作者可在班级内创建分类，仅班级内使用 |
| 创建班级知识点 | 班级创作者可在班级内创建知识点，关联班级题目 |
| 使用公共题目 | 班级创作者可使用公共题目创建班级考试 |
| 使用班级题目 | 班级创作者可使用班级内其他创作者的题目 |
| 班级资源管理 | 管理班级资源 |
| 班级资源导出 | 导出班级资源 |
| 班级资源统计 | 班级资源统计 |

### 9.5 内容审核

> 创作者/教师提交内容审核的详细流程和权限见 **三、题库管理系统 3.5 内容审核**

**内容创作系统特有的审核规则**：
- 创作者/教师可将私有题目/试卷提交为公共内容
- 审核通过后，内容变为公共内容，所有用户可见
- 审核驳回后，内容退回为私有状态，创作者可修改后重新提交
- 审核结果通过站内消息通知创作者

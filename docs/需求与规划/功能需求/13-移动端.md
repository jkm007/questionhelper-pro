## 十三、移动端（用户端）功能

**移动端功能与后端模块对应关系**：

| 移动端页面 | 对应后端模块 | 关联章节 |
|-----------|-------------|---------|
| 首页 | 数据统计、推荐系统 | 十、数据统计系统 |
| 题库页面 | 题库管理、分类管理 | 三、题库管理系统 |
| 考试页面 | 考试管理、在线答题 | 四、考试管理系统 |
| 班级页面 | 班级管理、班级资源 | 五、班级管理系统 |
| 练习页面 | 刷题练习 | 六、刷题练习系统 |
| 错题本 | 错题本系统 | 七、错题本系统 |
| 个人中心 | 用户管理、角色申请 | 二、用户管理系统 |

### 13.1 首页

| 功能点 | 功能描述 |
|-------|---------|
| 数据概览 | 总题量、今日练习数、正确率、连续打卡天数 |
| 快捷入口 | 每日练习、模拟考试、错题本、我的班级 |
| 热门分类 | 横向滚动展示热门公共分类 |
| 推荐题目 | 推荐公共题目列表 |
| 即将开始的考试 | 展示即将开始的考试（公共考试 + 班级考试） |
| 学习计划 | 显示今日学习计划 |
| 每日打卡 | 每日打卡入口 |
| 消息通知 | 消息通知入口 |
| 搜索入口 | 搜索入口 |

### 13.2 题库页面

| 功能点 | 功能描述 |
|-------|---------|
| 分类树浏览 | 按公共分类树层级浏览题目 |
| 题目列表 | 分页展示公共题目列表 |
| 题目详情 | 查看题目内容、选项、解析 |
| 搜索题目 | 搜索框搜索公共题目 |
| 开始练习 | 进入练习模式 |

**创作者/教师额外功能**：

| 功能点 | 功能描述 |
|-------|---------|
| 我的题目 | 查看自己创建的私有题目 |
| 创建题目 | 在移动端创建私有题目 |
| 我的分类 | 查看自己创建的私有分类 |
| 提交审核 | 将私有题目提交为公共题目 |

**用户互动功能**：

| 功能点 | 功能描述 |
|-------|---------|
| 题目收藏 | 收藏题目 |
| 题目分享 | 分享题目 |
| 题目笔记 | 添加题目笔记 |
| 题目纠错 | 报告题目错误 |

### 13.3 考试页面

| 功能点 | 功能描述 |
|-------|---------|
| 考试列表 | 查看可参加的考试列表（公共考试 + 班级考试） |
| 考试详情 | 查看考试信息、已考次数、题型分布 |
| 开始考试 | 进入答题页面 |
| 答题页面 | 完整答题功能（见 4.3 在线答题） |
| 考试结果 | 查看成绩、逐题对比、解析 |

**考试辅助功能**：

| 功能点 | 功能描述 |
|-------|---------|
| 考试提醒 | 考试开始前提醒通知 |
| 考试历史 | 查看历史考试记录 |
| 成绩排名 | 查看考试成绩排名 |

### 13.4 班级页面（教师/班级成员）

| 功能点 | 功能描述 |
|-------|---------|
| 我的班级 | 查看我加入/创建的班级 |
| 班级详情 | 查看班级信息、成员列表、班级题目、班级考试 |
| 班级题目 | 浏览班级内的题目（公共题目 + 班级题目） |
| 班级考试 | 查看班级考试列表，参加班级考试 |
| 班级成员 | 查看班级成员列表 |
| 申请班级创作者 | 班级成员可申请成为班级创作者 |

**教师额外功能**：
| 功能点 | 功能描述 |
|-------|---------|
| 创建班级 | 在移动端创建班级 |
| 邀请成员 | 生成邀请码邀请成员 |
| 审批申请 | 审批班级创作者申请 |

**教师班级管理**：

| 功能点 | 功能描述 |
|-------|---------|
| 布置作业 | 为班级布置作业 |
| 发布考试 | 为班级发布考试 |
| 班级公告 | 查看和发布班级公告 |
| 班级讨论 | 班级讨论区 |
| 班级通知 | 班级通知管理 |

### 13.5 个人中心

#### 13.5.1 个人信息

| 功能点 | 功能描述 |
|-------|---------|
| 个人信息 | 查看和修改个人信息 |
| 实名认证 | 绑定手机号、填写真实姓名和身份证号完成认证 |
| 设置 | 修改密码、清除缓存、关于我们 |

#### 13.5.2 角色申请

| 功能点 | 功能描述 |
|-------|---------|
| 申请创作者 | 申请成为创作者，查看申请状态 |
| 申请教师 | 申请成为教师，查看申请状态 |
| 班级创作者状态 | 查看在各班级的班级创作者申请状态 |

#### 13.5.3 学习与练习

| 功能点 | 功能描述 |
|-------|---------|
| 考试历史 | 查看所有已参加的考试及成绩 |
| 错题本 | 查看错题、重做错题、错题分析 |
| 学习统计 | 总练习数、正确率趋势、各分类正确率 |
| 收藏管理 | 收藏的题目列表 |
| 学习计划 | 学习计划管理 |
| 学习报告 | 学习报告查看 |

#### 13.5.4 我的内容

| 功能点 | 功能描述 |
|-------|---------|
| 我的班级 | 查看我加入的班级 |
| 我的创作 | 创作者/教师查看自己创建的内容 |
| 消息中心 | 站内消息通知 |

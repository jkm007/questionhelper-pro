## 五、班级管理系统

### 5.1 班级创建与管理

| 功能点 | 功能描述 |
|-------|---------|
| 创建班级 | 教师可创建班级，设置班级名称、描述、封面图片、最大成员数（默认 500） |
| 班级列表 | 查看班级列表，支持按关键词、角色（teacher/student）筛选 |
| 班级详情 | 查看班级名称、描述、封面、邀请码、成员数、教师信息 |
| 编辑班级 | 修改班级名称、描述、封面、最大成员数 |
| 删除班级 | 删除班级（见下方删除保护规则） |
| 生成邀请码 | 创建班级时自动生成唯一邀请码，支持重新生成 |

**班级删除保护规则**：

| 场景 | 规则 |
|------|------|
| 班级无成员 | 允许删除 |
| 班级有成员 | 禁止删除，提示"请先移除所有成员" |
| 班级有进行中的考试 | 禁止删除，提示"考试进行中，无法删除班级" |
| 班级有考试记录 | 允许软删除，考试记录保留 |
| 班级有班级资源 | 允许软删除，班级资源保留（归档） |

**班级转让规则**：
- 教师可将班级转让给其他教师
- 转让后原教师变为普通成员
- 转让需确认，不可撤销

> 关联章节：**五、班级管理系统 5.2**（成员管理）、**五、班级管理系统 5.4**（班级资源管理）
| 班级搜索 | 按关键词搜索班级 |
| 班级归档 | 归档已结束的班级（只读，保留历史数据） |
| 班级置顶 | 置顶重要班级，优先显示 |
| 班级分类/标签 | 给班级打标签便于分类管理 |
| 班级二维码 | 生成班级二维码，方便扫码加入 |
| 班级有效期 | 设置班级有效期，到期自动归档 |
| 班级模板 | 班级创建模板，快速创建相似班级 |

### 5.2 成员管理

| 功能点 | 功能描述 |
|-------|---------|
| 邀请成员 | 教师通过邀请码或链接邀请成员加入班级 |
| 加入班级 | 成员通过邀请码加入班级，校验邀请码有效性和班级人数上限 |
| 成员列表 | 分页查看班级成员，支持按角色筛选 |
| 移除成员 | 教师可从班级移除成员 |
| 成员角色设置 | 设置班级成员角色（teacher/student） |
| 退出班级 | 成员主动退出班级 |
| 批量导入成员 | 通过Excel批量导入成员（学号、姓名、手机号） |
| 成员信息导出 | 导出成员列表为Excel |
| 成员禁用 | 临时禁用成员权限（不影响其他班级） |
| 成员分组 | 成员分组管理（如：学习小组） |
| 成员邀请链接 | 生成邀请链接，一键分享 |
| 成员审核 | 加入班级需要教师审核 |
| 成员统计 | 统计成员活跃度、参与度等 |

**加入逻辑**：
- 校验邀请码有效
- 校验班级成员未满
- 校验用户不在该班级中

**成员退出/移除后资源影响**：

| 场景 | 班级题目 | 班级考试记录 | 班级创作者权限 |
|------|---------|-------------|--------------|
| 成员主动退出 | 保留（归属班级） | 保留（个人记录不可见） | 自动撤销 |
| 教师移除成员 | 保留（归属班级） | 保留（个人记录不可见） | 自动撤销 |
| 成员重新加入 | 恢复可见 | 恢复可见 | 需重新申请 |

> 关联章节：**五、班级管理系统 5.3**（班级创作者管理）、**五、班级管理系统 5.4**（班级资源管理）

### 5.3 班级创作者管理

#### 5.3.1 班级创作者功能

| 功能点 | 功能描述 |
|-------|---------|
| 申请成为班级创作者 | 班级成员可申请成为班级创作者，填写申请理由 |
| 查看申请列表 | 教师可查看班级创作者申请列表，含申请人信息、申请理由、申请时间 |
| 教师审批 | 教师可审批班级创作者申请（通过/驳回），驳回需填写原因 |
| 管理员审批 | 管理员/超级管理员可审批所有班级的创作者申请 |
| 撤销班级创作者 | 教师可撤销成员的班级创作者权限 |
| 查看班级创作者列表 | 教师可查看班级内所有班级创作者 |
| 申请历史 | 查看历史申请记录（含已通过/已驳回） |
| 创作者权限配置 | 自定义班级创作者的权限范围（如：是否可创建考试） |
| 创作者统计 | 统计班级创作者的创建题目数、活跃度等 |
| 创作者等级 | 根据贡献度设置创作者等级 |

#### 5.3.2 申请流程

```
成员申请 → 教师审批 → 通过 → 成为班级创作者
                    ↓
                  驳回 → 退回申请（可重新申请）
```

**申请规则**：
- 班级成员均可申请成为班级创作者
- 一个用户可在多个班级申请成为班级创作者
- 申请时需填写申请理由
- 审批通过后立即生效

#### 5.3.3 权限与审批

**班级创作者权限**：

| 权限项 | 班级创作者 | 普通班级成员 |
|-------|-----------|-------------|
| 查看班级题目 | ✓ | ✓ |
| 创建班级题目 | ✓ | - |
| 创建班级分类 | ✓ | - |
| 创建班级知识点 | ✓ | - |
| 使用公共题目 | ✓ | ✓ |
| 使用班级题目 | ✓ | ✓ |
| 参加班级考试 | ✓ | ✓ |
| 查看班级成绩 | ✓ | ✓ |

**权限说明**：
- 班级创作者权限仅在申请的班级内有效
- 班级创作者创建的资源仅班级成员可见
- 班级创作者不影响公共题库
- 教师可随时撤销班级创作者权限

**审批规则**：
- 教师只能审批自己班级的班级创作者申请
- 管理员/超级管理员可审批所有班级的班级创作者申请
- 审批通过后立即生效，成员获得班级创作者权限
- 审批驳回需填写驳回原因，申请人可查看

### 5.4 班级资源管理

#### 5.4.1 资源功能

| 功能点 | 功能描述 |
|-------|---------|
| 班级题库 | 教师/班级创作者可为班级创建题目，仅班级成员可见 |
| 班级分类 | 教师/班级创作者可为班级创建分类，仅班级内使用 |
| 班级知识点 | 教师/班级创作者可为班级创建知识点，关联班级题目 |
| 使用公共资源 | 教师/班级创作者可使用公共题库的题目创建班级考试 |
| 使用班级资源 | 教师/班级创作者可使用班级内其他创作者的题目 |
| 资源查看 | 班级成员可查看班级内的所有题目、分类、知识点 |
| 资源统计 | 教师可查看班级资源统计（题目数、分类数、知识点数） |
| 资源导入 | 从公共题库导入题目到班级 |
| 资源导出 | 导出班级资源（题目、分类、知识点） |
| 资源审核 | 教师创建的资源直接可见；班级创作者创建的资源可配置是否需要教师审核（默认不需要） |
| 资源搜索 | 在班级资源库中搜索题目 |
| 资源标签 | 给资源打标签便于分类 |

#### 5.4.2 资源创建规则

**创建权限**：

| 角色 | 可创建资源 | 资源可见范围 |
|------|-----------|-------------|
| 教师 | 班级题目/分类/知识点 | 班级成员 |
| 班级创作者 | 班级题目/分类/知识点 | 班级成员 |
| 普通班级成员 | - | - |

**审核规则**：
- 教师创建的资源：直接生效，无需审核
- 班级创作者创建的资源：可配置是否需要教师审核（默认不需要）
- 需要审核时：班级创作者提交 → 教师审核 → 通过/驳回

**资源使用规则**：
- 班级资源仅班级成员可见，不影响公共题库
- 班级资源不能提交为公共资源
- 教师/班级创作者可使用公共题目创建班级考试

### 5.5 班级功能

#### 5.5.1 作业管理

| 功能点 | 功能描述 |
|-------|---------|
| 布置作业 | 教师为班级布置作业（关联试卷），设置标题、截止时间、说明 |
| 作业批改 | 教师批改作业并给出反馈和评语 |
| 作业统计 | 统计作业完成率、正确率、提交时间等 |
| 作业提醒 | 作业截止前提醒未提交的学生 |
| 作业模板 | 创建作业模板便于复用 |
| 学生互评 | 支持学生之间互相评价作业 |

**作业规则**：
- 作业关联试卷，学生需在截止时间前完成
- 教师可查看作业完成情况和统计
- 作业成绩计入班级成绩统计

#### 5.5.2 考试管理

| 功能点 | 功能描述 |
|-------|---------|
| 班级考试 | 教师发布班级考试，可选择公共题目或班级题目 |
| 查看成绩 | 查看班级整体成绩 |
| 成绩分析 | 班级成绩统计分析，支持班级间对比 |
| 班级排名 | 班级内成绩排名（可配置是否显示） |
| 成绩预警 | 成绩低于阈值时自动提醒学生 |
| 成绩导出 | 导出班级考试成绩为 Excel |

**班级考试规则**：
- 教师可选择公共题目或班级题目创建班级试卷
- 班级考试仅班级成员可见可考
- 班级考试成绩仅班级成员和教师可见
- 教师可查看班级考试成绩统计和分析
- 详见 **四、考试管理系统 4.2**

#### 5.5.3 班级互动

| 功能点 | 功能描述 |
|-------|---------|
| 班级公告 | 教师可发布班级公告 |
| 班级讨论 | 班级成员可在班级内讨论 |
| 班级通知 | 班级相关通知推送（站内信） |
| 班级文件 | 存储和管理班级共享文件 |
| 班级日志 | 记录班级操作日志（成员变动、考试发布等） |

#### 5.5.4 学习管理

| 功能点 | 功能描述 |
|-------|---------|
| 学习进度 | 查看学生学习进度 |
| 学习报告 | 生成学生学习报告（成绩、进度、参与度） |
| 学习计划 | 制定学习计划并追踪完成情况 |
| 考勤管理 | 记录学生出勤情况（签到/签退） |

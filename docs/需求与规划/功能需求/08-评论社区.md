## 八、评论/社区系统

### 8.1 题目评论

**评论对象说明**：

| 评论对象 | 说明 | 关联章节 |
|---------|------|---------|
| 题目 | 对题目解析发表评论或提问 | **三、题库管理系统 3.1** |
| 考试 | 对考试发表评论（可选功能） | **四、考试管理系统 4.2** |

#### 8.1.1 评论功能

| 功能点 | 功能描述 |
|-------|---------|
| 评论列表 | 每道题下方有讨论区，获取评论列表（含回复），支持按点赞数排序 |
| 发表评论 | 用户可对题目解析发表评论或提问 |
| 嵌套回复 | 支持嵌套回复（楼中楼），最多 2 层 |
| 编辑评论 | 编辑自己的评论 |
| 删除评论 | 删除自己的评论（软删除） |
| 官方解答 | 管理员回答标记为"官方解答"置顶 |
| 点赞/取消点赞 | 用户可给评论点赞，防重复点赞（唯一索引） |
| 举报评论 | 用户可举报不当评论，填写举报原因 |

**官方解答标记规则**：管理员/教师的评论可标记为"官方解答"，标记后置顶显示并带有官方标识。举报管理：被举报的评论进入待审核状态，管理员可处理举报（维持/删除/警告用户）。

**评论状态**：approved（已通过）/ pending（待审核）/ reported（已举报）

#### 8.1.2 评论增强

| 功能点 | 功能描述 |
|-------|---------|
| 评论搜索 | 搜索评论内容 |
| 评论排序 | 按时间、点赞数等排序 |
| 评论筛选 | 按状态、时间等筛选 |
| 评论图片 | 支持在评论中添加图片 |
| 评论@功能 | 支持@其他用户 |
| 评论表情 | 支持表情包 |
| 评论置顶 | 置顶重要评论 |
| 评论精选 | 标记精选评论 |
| 评论举报管理 | 管理举报的评论（查看、处理、记录） |

**评论内容安全**：
- 评论内容必须经过敏感词过滤（使用敏感词中间件）
- 评论中的 HTML 内容必须经过净化处理（禁止 script、事件属性等）
- 评论支持举报，举报后进入待审核状态

### 8.2 评论管理（管理端）

| 功能点 | 功能描述 |
|-------|---------|
| 全部评论列表 | 查看全部评论，可按状态筛选 |
| 审核评论 | 通过/拒绝/标记举报 |
| 删除评论 | 管理员删除评论 |
| 评论统计 | 统计评论数量、活跃度等 |
| 批量操作 | 批量审核、批量删除 |
| 评论导出 | 导出评论数据 |
| 评论黑名单 | 禁止特定用户评论 |
| 评论举报处理 | 处理举报的评论（警告、禁言、删除） |
| 评论审核规则 | 设置评论审核规则（如：敏感词过滤） |

### 8.3 通知系统

#### 8.3.1 通知功能

| 功能点 | 功能描述 |
|-------|---------|
| 通知列表 | 分页查看我的通知，支持按已读/未读筛选、按类型筛选 |
| 未读数量 | 获取未读通知数量，支持按类型获取 |
| 标记已读 | 标记单条通知已读 / 批量标记已读 / 全部标记已读 |
| 通知删除 | 删除单条通知 / 批量删除 |
| 通知过期 | 默认 90 天过期，定时清理 |
| 通知设置 | 设置通知偏好（接收哪些通知） |
| 通知免打扰 | 设置免打扰时间段 |

**通知偏好 API**：

| API | 方法 | 说明 |
|-----|------|------|
| /api/v1/notification/preferences | GET | 获取通知偏好设置 |
| /api/v1/notification/preferences | PUT | 更新通知偏好 |

**偏好设置项**：
- 各通知类型的开关（comment_reply、exam_graded、system 等）
- 免打扰时间段（开始时间、结束时间）
- 通知方式偏好（仅站内消息 / 站内消息+邮箱）

#### 8.3.2 通知管理（管理端）

| 功能点 | 功能描述 |
|-------|---------|
| 通知推送 | 推送通知到手机/微信 |
| 通知模板 | 通知模板管理 |
| 通知统计 | 统计通知发送情况 |
| 通知撤回 | 撤回已发送的通知 |
| 通知定时发送 | 定时发送通知 |
| 通知群发 | 群发通知给指定用户 |

#### 8.3.3 通知类型与触发

**通知类型**：

| 类型标识 | 类型名称 | 说明 |
|---------|---------|------|
| comment_reply | 评论回复 | 有人回复我的评论 |
| exam_graded | 考试评分 | 主观题阅卷完成 |
| system | 系统通知 | 系统公告、活动通知等 |
| security_login | 登录通知 | 新设备登录、异地登录提醒 |
| security_password | 密码变更 | 密码修改成功通知 |
| security_bind | 账号绑定 | 第三方账号绑定/解绑通知 |
| security_logout | 账号注销 | 账号注销申请/撤销通知 |
| role_apply | 角色申请 | 角色申请审核结果通知 |
| class_creator | 班级创作者 | 班级创作者申请审核结果通知 |
| exam_remind | 考试提醒 | 即将开始的考试提醒 |
| homework | 作业通知 | 教师布置作业通知 |

**通知触发场景**：

| 触发事件 | 通知类型 | 通知方式 |
|---------|---------|---------|
| 有人回复我的评论 | comment_reply | 站内消息 |
| 主观题阅卷完成 | exam_graded | 站内消息 |
| 系统公告 | system | 站内消息 |
| 新设备首次登录 | security_login | 站内消息 + 邮箱 |
| 密码修改成功 | security_password | 站内消息 + 邮箱 |
| 第三方账号绑定/解绑 | security_bind | 站内消息 |
| 账号注销申请 | security_logout | 站内消息 + 邮箱 |
| 角色申请审核结果 | role_apply | 站内消息 |
| 班级创作者审核结果 | class_creator | 站内消息 |
| 考试即将开始（提前1小时） | exam_remind | 站内消息 |
| 教师布置作业 | homework | 站内消息 |
| 内容审核结果 | audit_result | 站内消息 |
| 作业截止提醒（提前2小时） | homework_deadline | 站内消息 |
| 成绩复核结果 | review_result | 站内消息 |
| 成就达成 | achievement | 站内消息 |
| 班级公告 | class_announcement | 站内消息 |

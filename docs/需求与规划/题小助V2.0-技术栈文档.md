## 技术栈总览

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 后端语言 | Go 1.21+ | 高并发、编译型语言 |
| Web 框架 | Gin 1.9+ | 轻量级 HTTP 框架 |
| ORM | GORM 1.25+ | 数据库操作、自动迁移 |
| 数据库 | MySQL 8.0 | 核心业务数据 |
| 缓存 | Redis 7.x | Token 管理、会话、延迟队列 |
| 消息队列 | Redis Stream / Go Channel | 异步任务处理 |
| 文件存储 | MinIO / 阿里云 OSS | 附件、头像、封面 |
| 管理后台 | Vue 3 + Element Plus | vue3-element-admin 模板 |
| 移动端 | uni-app (Vue 3 + TypeScript) | H5 / 微信小程序 / App |
| 容器化 | Docker + Docker Compose | 开发环境标准化 |
| 反向代理 | Nginx | 负载均衡、SSL 终止 |
| 监控 | Prometheus + Grafana | 系统监控、告警 |

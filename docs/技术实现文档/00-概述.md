## 后端分层架构

```
Router (路由层)
  │  定义 URL 路径与 HTTP 方法的映射
  │  文件: internal/router/*.go
  │
  ▼
Middleware (中间件层)
  │  JWT 认证、RBAC 权限、操作日志、敏感词过滤、限流、CORS
  │  文件: internal/middleware/*.go
  │
  ▼
Controller (控制器层)
  │  解析请求参数、调用 Service、返回响应
  │  文件: internal/controller/{domain}/*.go
  │
  ▼
Service (业务逻辑层)
  │  核心业务逻辑、事务管理、DTO 转换
  │  文件: internal/service/{domain}/*.go
  │
  ▼
Repository (数据访问层)
  │  GORM 查询、数据库操作
  │  文件: internal/repository/{domain}/*.go
  │
  ▼
Model (模型层)
  │  GORM 模型定义、数据库表映射
  │  文件: internal/model/*.go
```

**横切关注点**：
- **DTO** (`internal/dto/`)：请求/响应结构体，含验证标签
- **pkg/**：共享工具包（config、database、cache、jwt、response、errors 等）

## 前端分层架构

```
Views (页面层)
  │  按领域划分的页面组件
  │  文件: src/views/{domain}/index.vue
  │
  ▼
Components (组件层)
  │  共享组件（CURD、Upload、TreeSelect 等）
  │  文件: src/components/{ComponentName}/
  │
  ▼
API (接口层)
  │  按领域划分的 API 模块（Axios 实例）
  │  文件: src/api/{domain}.ts
  │
  ▼
Store (状态层)
  │  Pinia 状态管理（user、permission、app、settings 等）
  │  文件: src/stores/{store}.ts
  │
  ▼
Utils (工具层)
  │  request.ts（Axios 封装）、auth.ts（Token 管理）
  │  文件: src/utils/
  │
  ▼
Router (路由层)
  │  静态 + 动态路由、权限守卫
  │  文件: src/router/
```

## 通用约定

### API 响应格式

```json
{
  "code": "00000",
  "msg": "success",
  "data": {}
}
```

### 分页参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| page | int | 1 | 页码 |
| page_size | int | 10 | 每页条数，最大 100 |

### 认证方式

- **请求头**：`Authorization: Bearer <access_token>`
- **access_token 有效期**：7200s（2 小时）
- **refresh_token 有效期**：604800s（7 天）

### 数据库规范

- 表名：复数蛇形命名（如 `users`、`questions`）
- 必含字段：`id`、`created_at`、`updated_at`、`deleted_at`
- 软删除：通过 `deleted_at` 字段
- 布尔字段：`is_` 前缀

### 错误码规范

| 错误码 | 说明 |
|--------|------|
| A0001 | 用户端通用错误 |
| A0002 | 参数错误 |
| A0003 | Token 过期 |
| A0004 | Token 无效 |
| A201 | 用户名或密码错误 |
| A202 | 用户被锁定 |
| A203 | 验证码错误 |
| B0001 | 业务通用错误 |
| B0002 | 资源不存在 |
| B0003 | 无权操作 |
| S0001 | 系统通用错误 |
| S0002 | 内部错误 |

# 04-考试管理 技术实现文档

## 1. 模块概述

考试管理模块是题小助平台的核心业务模块，负责试卷组卷、考试发布与管理、在线答题、自动阅卷、成绩管理、考试监控等完整流程。该模块深度依赖题库管理模块提供的题目数据，同时与班级管理、错题本、通知系统等模块紧密集成。

### 1.1 功能边界

| 子模块 | 职责范围 |
|--------|---------|
| 试卷管理 | 手动组卷、智能组卷、模板复用、试卷预览/复制/导出、试卷共享 |
| 考试管理 | 考试创建/编辑/发布/删除、状态机管理、暂停/恢复/延期、考试密码 |
| 在线答题 | 开始考试、答题过程、自动保存、断线续考、超时交卷、手动交卷 |
| 评分系统 | 客观题自动阅卷（精确匹配/关键词匹配）、主观题人工阅卷、部分得分 |
| 成绩管理 | 成绩查询、成绩复核、成绩排名、成绩导出 |
| 考试监控 | 实时监控（WebSocket）、异常行为检测、分数分布、提交率统计 |
| 防作弊 | 三级防作弊（无/基础/严格）、切屏检测、IP 异常检测、复制粘贴拦截 |
| 考试提醒 | 开始前提醒、结束前提醒、多渠道通知 |

### 1.2 资源可见性模型

考试管理模块支持三种资源可见性，与题库管理模块保持一致：

| 可见性 | 值 | 创建者角色 | 可见范围 |
|--------|---|-----------|---------|
| 公共 (public) | 1 | 内容管理员/管理员 | 所有用户 |
| 私有 (private) | 2 | 创作者/教师 | 仅创建者 |
| 班级 (class) | 3 | 教师/班级创作者 | 班级成员 |

### 1.3 考试状态机

考试实体支持五种状态，状态流转规则如下：

| 状态 | 值 | 说明 |
|------|---|------|
| draft (草稿) | 0 | 考试已创建但未发布 |
| published (已发布) | 1 | 考试已发布，等待开始时间 |
| ongoing (进行中) | 2 | 考试正在进行 |
| paused (已暂停) | 3 | 紧急暂停，已开始考生可继续，新考生无法进入 |
| ended (已结束) | 4 | 考试已结束 |

状态流转路径：

```
draft ──发布──> published ──到达开始时间/手动开始──> ongoing
                                                │
                                    ┌──暂停──> paused ──恢复──> ongoing
                                    │
                                    └──结束──> ended
                                    │
                                    └──paused ──强制结束──> ended
```

### 3.2 依赖组装方式

考试管理模块采用与题库管理一致的**包级函数**模式，无依赖注入框架：

- Controller 层：结构体方法，通过 `NewXxxController()` 返回空结构体实例
- Service 层：包级函数，直接通过导入包互相调用
- Repository 层：包级函数，通过全局 `database.DB` 访问数据库
- 缓存层：通过 `pkg/cache` 包级函数操作 Redis

```go
// Controller 调用 Service 示例
func (ctrl *ExamController) StartExam(c *gin.Context) {
    userID := middleware.GetUserID(c)
    examID := parseUintParam(c, "id")
    var req dto.StartExamRequest
    if err := c.ShouldBindJSON(&req); err != nil {
        response.Error(c, http.StatusBadRequest, "参数错误")
        return
    }
    result, err := exam.StartExam(userID, examID, &req)
    if err != nil {
        response.Error(c, http.StatusInternalServerError, err.Error())
        return
    }
    response.Success(c, result)
}
```

### 5.4 软删除策略

`papers`、`exams`、`exam_records`、`answer_records`、`exam_notices` 使用 GORM 软删除（`deleted_at` 字段）。所有查询自动过滤已删除记录。删除保护规则：

- 试卷：被未开始的考试引用时禁止删除；被进行中的考试引用时禁止删除；被已结束的考试引用时允许软删除
- 考试：有答题中记录时禁止删除；仅有已结束记录时允许软删除
- 考试记录和答题记录：支持软删除，保留历史数据

### 7.1 组卷模式

系统支持三种组卷模式：手动组卷、智能组卷、模板复用。

#### 7.1.1 手动组卷

手动组卷允许管理员从题库中逐题选择组成试卷。

```go
// service/exam/paper_service.go
func CreatePaperManual(creatorID uint, req *dto.CreatePaperManualRequest) (*dto.PaperInfo, error) {
    // 1. 创建试卷记录
    paper := &model.Paper{
        Title:      req.Title,
        Description: req.Description,
        Type:       1, // 手动组卷
        Status:     0, // 草稿
        Visibility: req.Visibility,
        ClassID:    req.ClassID,
        CreatorID:  creatorID,
        IsTemplate: req.IsTemplate,
    }

    // 2. 计算总分和题目数量
    var totalScore float64
    for _, item := range req.Questions {
        totalScore += item.Score
    }
    paper.TotalScore = totalScore
    paper.TotalCount = len(req.Questions)

    // 3. 事务写入试卷和关联记录
    tx := database.DB.Begin()
    if err := tx.Create(paper).Error; err != nil {
        tx.Rollback()
        return nil, fmt.Errorf("创建试卷失败: %w", err)
    }

    // 4. 批量创建试卷-题目关联，同时写入快照
    for i, item := range req.Questions {
        // 查询题目完整内容用于快照
        question, err := questionRepo.FindByID(item.QuestionID)
        if err != nil {
            tx.Rollback()
            return nil, fmt.Errorf("题目 %d 不存在", item.QuestionID)
        }

        snapshot, _ := json.Marshal(question) // 序列化题目完整快照

        pq := &model.PaperQuestion{
            PaperID:    paper.ID,
            QuestionID: item.QuestionID,
            Score:      item.Score,
            Sort:       i + 1,
            Snapshot:   string(snapshot),
        }
        if err := tx.Create(pq).Error; err != nil {
            tx.Rollback()
            return nil, fmt.Errorf("关联题目失败: %w", err)
        }
    }

    tx.Commit()
    return toPaperInfo(paper), nil
}
```

#### 7.1.2 智能组卷

智能组卷按规则自动从题库中抽取题目，支持按题型、分类、难度、数量、分值等条件抽题。

```go
// service/exam/paper_service.go
func CreatePaperSmart(creatorID uint, req *dto.CreatePaperSmartRequest) (*dto.PaperInfo, error) {
    // 1. 创建试卷记录
    paper := &model.Paper{
        Title:      req.Title,
        Description: req.Description,
        Type:       2, // 智能组卷
        Status:     0,
        Visibility: req.Visibility,
        ClassID:    req.ClassID,
        CreatorID:  creatorID,
    }

    tx := database.DB.Begin()

    // 2. 维护已抽题 ID 集合，防止重复抽取
    excludes := make(map[uint]bool)
    var totalScore float64
    totalCount := 0

    // 3. 遍历每条规则
    for i, rule := range req.Rules {
        // 计算分类子树 IDs（当前分类 + 全部后代分类）
        var categoryIDs []uint
        if rule.CategoryID != nil {
            categoryIDs = getCategoryDescendantIDs(*rule.CategoryID)
            categoryIDs = append(categoryIDs, *rule.CategoryID)
        }

        // 构建排除条件
        excludeIDs := make([]uint, 0, len(excludes))
        for id := range excludes {
            excludeIDs = append(excludeIDs, id)
        }

        // 随机抽取题目
        questions, err := questionRepo.RandomSelect(&questionRepo.RandomSelectParams{
            Type:         rule.Type,
            CategoryIDs:  categoryIDs,
            Difficulty:   rule.Difficulty,
            ExcludeIDs:   excludeIDs,
            Limit:        rule.Count,
        })
        if err != nil {
            tx.Rollback()
            return nil, fmt.Errorf("抽取题目失败: %w", err)
        }

        // 4. 校验抽题数量是否满足
        if len(questions) < rule.Count {
            tx.Rollback()
            return nil, fmt.Errorf("规则 %d: 题目数量不足，需要 %d 道，仅有 %d 道",
                i+1, rule.Count, len(questions))
        }

        // 5. 创建关联记录，写入快照
        for j, q := range questions {
            excludes[q.ID] = true

            snapshot, _ := json.Marshal(q)
            pq := &model.PaperQuestion{
                PaperID:    paper.ID,
                QuestionID: q.ID,
                Score:      rule.Score,
                Sort:       totalCount + j + 1,
                Snapshot:   string(snapshot),
            }
            if err := tx.Create(pq).Error; err != nil {
                tx.Rollback()
                return nil, fmt.Errorf("关联题目失败: %w", err)
            }

            totalScore += rule.Score
        }
        totalCount += len(questions)
    }

    // 6. 更新试卷总分和总数
    paper.TotalScore = totalScore
    paper.TotalCount = totalCount

    if err := tx.Save(paper).Error; err != nil {
        tx.Rollback()
        return nil, fmt.Errorf("更新试卷失败: %w", err)
    }

    tx.Commit()
    return toPaperInfo(paper), nil
}
```

#### Step 1: 开始考试

```go
// service/exam/record_service.go
func StartExam(userID, examID uint, req *dto.StartExamRequest) (*dto.StartExamResponse, error) {
    // 1.1 查询考试信息
    exam, err := examRepo.FindByID(examID)
    if err != nil {
        return nil, errors.New("考试不存在")
    }

    // 1.2 校验考试状态
    if exam.Status == 0 {
        return nil, errors.New("考试尚未发布")
    }
    if exam.Status == 4 {
        return nil, errors.New("考试已结束")
    }
    if exam.Status == 3 { // paused
        return nil, errors.New("考试已暂停，暂无法进入")
    }

    // 1.3 校验开放时间
    now := time.Now()
    if now.Before(exam.StartTime) {
        return nil, errors.New("考试尚未开始")
    }
    if now.After(exam.EndTime) {
        return nil, errors.New("考试已结束")
    }

    // 1.4 校验可考次数
    if exam.MaxAttempts > 0 {
        var attemptCount int64
        database.DB.Model(&model.ExamRecord{}).
            Where("exam_id = ? AND user_id = ? AND deleted_at IS NULL", examID, userID).
            Count(&attemptCount)
        if int(attemptCount) >= exam.MaxAttempts {
            return nil, errors.New("已达到最大考试次数")
        }
    }

    // 1.5 密码验证（如设置了密码）
    if exam.ExamPassword != "" {
        if req.Password == "" {
            return nil, errors.New("请输入考试密码")
        }
        if !verifyExamPassword(exam.ExamPassword, req.Password) {
            return nil, errors.New("考试密码错误")
        }
    }

    // 1.6 查询试卷题目及快照
    paperQuestions, err := paperRepo.FindQuestionsByPaperID(exam.PaperID)
    if err != nil || len(paperQuestions) == 0 {
        return nil, errors.New("试卷无题目")
    }

    // 1.7 题目乱序处理
    if exam.Shuffle {
        shuffleQuestions(paperQuestions)
    }

    // 1.8 创建考试记录
    record := &model.ExamRecord{
        ExamID:    examID,
        UserID:    userID,
        Status:    0, // 进行中
        SubmitType: 1, // 手动
        StartTime: now,
        IP:        getClientIP(c),
    }
    if err := examRepo.CreateRecord(record); err != nil {
        return nil, fmt.Errorf("创建考试记录失败: %w", err)
    }

    // Step 2: 初始化 Redis 缓存
    sessionKey := fmt.Sprintf("tixuan:exam:session:%d:%d", userID, examID)
    answersKey := fmt.Sprintf("tixuan:exam:answers:%d:%d", userID, examID)
    scoresKey := fmt.Sprintf("tixuan:exam:scores:%d:%d", userID, examID)

    ttl := time.Duration(exam.Duration) * time.Minute

    // 缓存题目顺序和会话状态
    sessionData := map[string]interface{}{
        "record_id":  record.ID,
        "exam_id":    examID,
        "user_id":    userID,
        "start_time": now.Unix(),
        "duration":   exam.Duration,
        "shuffle":    exam.Shuffle,
    }
    cache.Set(sessionKey, sessionData, ttl)

    // 初始化答案缓存（Hash）
    cache.HSet(answersKey, "__init__", "1")
    cache.Expire(answersKey, ttl)

    // 缓存每题分值
    for _, pq := range paperQuestions {
        cache.HSet(scoresKey, fmt.Sprintf("%d", pq.QuestionID), pq.Score)
    }
    cache.Expire(scoresKey, ttl)

    // Step 5: 写入延迟队列（超时自动交卷）
    expireAt := now.Add(ttl).Unix()
    cache.ZAdd("tixuan:exam:delay", redis.Z{
        Score:  float64(expireAt),
        Member: fmt.Sprintf("%d:%d:%d", userID, examID, record.ID),
    })

    // 1.9 构建响应
    questions := buildQuestionList(paperQuestions, exam.Shuffle)
    return &dto.StartExamResponse{
        RecordID:     record.ID,
        Duration:     exam.Duration,
        TotalScore:   exam.TotalScore,
        TotalCount:   len(paperQuestions),
        Questions:    questions,
        StartTime:    now.Format(time.RFC3339),
        EndTime:      now.Add(ttl).Format(time.RFC3339),
        AntiCheat:    exam.AntiCheat,
        ShowAnswer:   exam.ShowAnswer,
    }, nil
}
```

#### Step 3: 答题过程 -- 保存答案

```go
// service/exam/record_service.go
func SaveAnswer(userID, recordID uint, req *dto.SaveAnswerRequest) error {
    // 3.1 校验考试记录状态
    record, err := examRepo.FindRecordByID(recordID)
    if err != nil || record.UserID != userID {
        return errors.New("考试记录不存在")
    }
    if record.Status != 0 {
        return errors.New("考试已提交，无法继续作答")
    }

    // 3.2 检查 Redis 会话是否有效（超时检查）
    sessionKey := fmt.Sprintf("tixuan:exam:session:%d:%d", userID, record.ExamID)
    exists, _ := cache.Exists(sessionKey)
    if !exists {
        return errors.New("考试会话已过期，请重新进入")
    }

    // 3.3 幂等写入 Redis Hash
    answersKey := fmt.Sprintf("tixuan:exam:answers:%d:%d", userID, record.ExamID)
    answerData, _ := json.Marshal(dto.AnswerItem{
        QuestionID: req.QuestionID,
        Answer:     req.Answer,
        UpdatedAt:  time.Now().Unix(),
    })
    cache.HSet(answersKey, fmt.Sprintf("%d", req.QuestionID), string(answerData))

    return nil
}
```

```go
// service/exam/record_service.go
func SubmitExam(userID, recordID uint, submitType int8) (*dto.SubmitExamResponse, error) {
    // 5.1 幂等检查（防止重复提交）
    lockKey := fmt.Sprintf("exam:submit:lock:%d", recordID)
    if !cache.SetNX(lockKey, 1, 30*time.Second) {
        return nil, errors.New("正在提交中，请勿重复操作")
    }
    defer cache.Del(lockKey)

    // 5.2 查询考试记录
    record, err := examRepo.FindRecordByID(recordID)
    if err != nil || record.UserID != userID {
        return nil, errors.New("考试记录不存在")
    }
    if record.Status != 0 {
        return nil, errors.New("考试已提交")
    }

    // 5.3 从 Redis 读取所有答案
    answersKey := fmt.Sprintf("tixuan:exam:answers:%d:%d", userID, record.ExamID)
    allAnswers, err := cache.HGetAll(answersKey)
    if err != nil {
        return nil, fmt.Errorf("读取答案失败: %w", err)
    }

    // 5.4 查询试卷题目信息（含正确答案）
    exam, _ := examRepo.FindByID(record.ExamID)
    paperQuestions, _ := paperRepo.FindQuestionsByPaperID(exam.PaperID)

    // 5.5 构建答案映射
    answerMap := make(map[uint]string)
    for field, val := range allAnswers {
        if field == "__init__" {
            continue
        }
        var item dto.AnswerItem
        json.Unmarshal([]byte(val), &item)
        answerMap[item.QuestionID] = item.Answer
    }

    // Step 6: 自动评分
    var totalScore, objScore, subjScore float64
    var answerRecords []model.AnswerRecord

    for _, pq := range paperQuestions {
        userAnswer := answerMap[pq.QuestionID]
        questionInfo := parseQuestionSnapshot(pq.Snapshot)

        score, isCorrect, needsReview := autoGrade(questionInfo, userAnswer, pq.Score)

        answerRecord := model.AnswerRecord{
            RecordID:   recordID,
            QuestionID: pq.QuestionID,
            Answer:     userAnswer,
            Score:      score,
            MaxScore:   pq.Score,
            IsCorrect:  boolToInt8(isCorrect),
            IsReviewed: !needsReview,
        }

        if needsReview {
            subjScore += score // 主观题默认 0 分，待阅卷
        } else {
            objScore += score
        }
        totalScore += score

        answerRecords = append(answerRecords, answerRecord)
    }

    // 5.6 事务写入数据库
    now := time.Now()
    durationUsed := int(now.Sub(record.StartTime).Seconds())

    tx := database.DB.Begin()

    // 批量写入答题记录
    if err := tx.CreateInBatches(answerRecords, 100).Error; err != nil {
        tx.Rollback()
        return nil, fmt.Errorf("写入答题记录失败: %w", err)
    }

    // 更新考试记录
    record.Score = totalScore
    record.ObjScore = objScore
    record.SubjScore = subjScore
    record.Status = 1 // 已提交
    record.SubmitType = submitType
    record.SubmitTime = &now
    record.DurationUsed = durationUsed
    if err := tx.Save(record).Error; err != nil {
        tx.Rollback()
        return nil, fmt.Errorf("更新考试记录失败: %w", err)
    }

    // 5.7 错题自动收录
    for _, ar := range answerRecords {
        if ar.IsCorrect == 0 { // 客观题答错
            addToWrongQuestions(userID, ar.QuestionID, record.ExamID, ar.Answer)
        }
    }

    // 5.8 更新试卷使用次数和统计
    tx.Model(&model.Paper{}).Where("id = ?", exam.PaperID).
        UpdateColumn("usage_count", gorm.Expr("usage_count + 1"))

    tx.Commit()

    // 5.9 清理 Redis 考试相关 Key
    cache.Del(fmt.Sprintf("tixuan:exam:session:%d:%d", userID, record.ExamID))
    cache.Del(fmt.Sprintf("tixuan:exam:answers:%d:%d", userID, record.ExamID))
    cache.Del(fmt.Sprintf("tixuan:exam:scores:%d:%d", userID, record.ExamID))
    // 从延迟队列移除
    cache.ZRem("tixuan:exam:delay", fmt.Sprintf("%d:%d:%d", userID, record.ExamID, recordID))

    // 5.10 构建返回结果
    return &dto.SubmitExamResponse{
        RecordID:     recordID,
        Score:        totalScore,
        ObjScore:     objScore,
        TotalScore:   exam.TotalScore,
        DurationUsed: durationUsed,
        SubmitTime:   now.Format(time.RFC3339),
    }, nil
}
```

```go
func autoGrade(question *dto.QuestionSnapshot, userAnswer string, maxScore float64) (float64, int8, bool) {
    switch question.Type {
    case 1: // 单选题
        return gradeSingleChoice(question.Answer, userAnswer, maxScore)
    case 2: // 多选题
        return gradeMultipleChoice(question.Answer, userAnswer, maxScore)
    case 3: // 判断题
        return gradeTrueFalse(question.Answer, userAnswer, maxScore)
    case 4: // 填空题
        return gradeFillBlank(question.Keywords, userAnswer, maxScore)
    case 5: // 简答题
        return 0, 2, true // 默认 0 分，标记待阅卷
    default:
        return 0, 2, true
    }
}
```

#### 7.3.2 多选题 -- 排序后比较 + 部分得分

```go
func gradeMultipleChoice(correctAnswer, userAnswer string, maxScore float64) (float64, int8, bool) {
    correctSet := parseAnswerSet(correctAnswer) // 如 "A,B,D" -> {"A": true, "B": true, "D": true}
    userSet := parseAnswerSet(userAnswer)

    // 完全匹配
    if answerSetEqual(correctSet, userSet) {
        return maxScore, 1, false
    }

    // 检查是否选了错误选项
    for option := range userSet {
        if !correctSet[option] {
            return 0, 0, false // 选了错误选项，0 分
        }
    }

    // 漏选部分得分：得分 = 题目分值 x (已选正确选项数 / 应选正确选项数)
    correctCount := 0
    for option := range userSet {
        if correctSet[option] {
            correctCount++
        }
    }
    totalCorrect := len(correctSet)
    partialScore := maxScore * float64(correctCount) / float64(totalCorrect)

    return math.Round(partialScore*100) / 100, 0, false // 部分得分，四舍五入到两位小数
}

func parseAnswerSet(answer string) map[string]bool {
    set := make(map[string]bool)
    parts := strings.Split(strings.TrimSpace(answer), ",")
    for _, p := range parts {
        set[strings.TrimSpace(p)] = true
    }
    return set
}

func answerSetEqual(a, b map[string]bool) bool {
    if len(a) != len(b) {
        return false
    }
    for k := range a {
        if !b[k] {
            return false
        }
    }
    return true
}
```

```go
func gradeTrueFalse(correctAnswer, userAnswer string, maxScore float64) (float64, int8, bool) {
    // 统一转小写后精确匹配
    correct := strings.ToLower(strings.TrimSpace(correctAnswer))
    user := strings.ToLower(strings.TrimSpace(userAnswer))

    // 支持多种表述: true/false, 对/错, 是/否, T/F, 1/0
    normalizeMap := map[string]string{
        "true": "true", "对": "true", "是": "true", "t": "true", "1": "true",
        "false": "false", "错": "false", "否": "false", "f": "false", "0": "false",
    }

    if normalizeMap[correct] == normalizeMap[user] {
        return maxScore, 1, false
    }
    return 0, 0, false
}
```

#### 7.3.4 填空题 -- 关键词匹配

```go
func gradeFillBlank(keywords []string, userAnswer string, maxScore float64) (float64, int8, bool) {
    if len(keywords) == 0 {
        // 无关键词定义，标记待人工阅卷
        return 0, 2, true
    }

    // 去除首尾空格，忽略大小写
    userAns := strings.TrimSpace(strings.ToLower(userAnswer))

    // 检查是否包含所有关键词
    matchedCount := 0
    for _, keyword := range keywords {
        kw := strings.TrimSpace(strings.ToLower(keyword))
        if strings.Contains(userAns, kw) {
            matchedCount++
        }
    }

    if matchedCount == len(keywords) {
        return maxScore, 1, false // 全部关键词匹配，满分
    }

    if matchedCount > 0 {
        // 部分关键词匹配，按比例得分
        partialScore := maxScore * float64(matchedCount) / float64(len(keywords))
        return math.Round(partialScore*100) / 100, 0, false
    }

    return 0, 0, false // 无关键词匹配，0 分
}
```

```go
func BatchReview(reviewerID uint, req *dto.BatchReviewRequest) (*dto.BatchResult, error) {
```

```go
// internal/task/exam_timeout.go
type ExamTimeoutTask struct{}

func (t *ExamTimeoutTask) Run() {
    now := float64(time.Now().Unix())

    // 查询所有已到期的考试记录
```

```go
    examRepo.UpdateStatus(examID, 3) // paused

// 恢复考试时恢复 Redis session TTL
func ResumeExam(examID uint) error {
    // 1. 更新考试状态
    examRepo.UpdateStatus(examID, 2) // ongoing

    // 2. 恢复所有进行中考生的 session TTL
    records, _ := examRepo.FindActiveRecords(examID)

    for _, record := range records {
        pauseKey := fmt.Sprintf("tixuan:exam:pause:%d:%d", record.UserID, examID)

        if remaining > 0 {
            // 恢复 session TTL

            cache.Expire(sessionKey, time.Duration(remaining)*time.Second)

            cache.ZAdd("tixuan:exam:delay", redis.Z{
                Score:  float64(expireAt),
                Member: fmt.Sprintf("%d:%d:%d", record.UserID, examID, record.ID),
            })
        }

        cache.Del(pauseKey)
    }

    return nil
}
```

### 7.5 防作弊

防作弊分为三个级别，逐级增强。

#### 7.5.1 防作弊级别定义

| 级别 | 名称 | 措施 |
|------|------|------|
| 0 | 无 | 不启用防作弊 |
| 1 | 基础 | 题目顺序打乱 |

| 类型 | 说明 | 检测方式 |
|------|------|---------|
| `switch_screen` | 切屏/切换窗口 | 前端 Visibility API + focus/blur 事件 |

| `copy_paste` | 复制粘贴 | 前端拦截 copy/paste/cut 事件 |
| `tab_change` | 切换标签页 | 前端 Visibility API |
| `dev_tools` | 打开开发者工具 | 前端检测 window 对象变化 |

```go
// IP 异常检测
func CheckIPChange(userID, examID uint, currentIP string) bool {
    sessionKey := fmt.Sprintf("tixuan:exam:session:%d:%d", userID, examID)
    sessionData, err := cache.Get(sessionKey)
    if err != nil {
        return false

    var session map[string]interface{}
    json.Unmarshal([]byte(sessionData.(string)), &session)

    originalIP, ok := session["ip"].(string)
    if !ok {
        // 首次记录 IP
        session["ip"] = currentIP
        cache.Set(sessionKey, session, 0)
        return false
    }

    return originalIP != currentIP
}
```

```go
func collectMonitorData(examID uint) *MonitorData {
    data := &MonitorData{ExamID: examID}

    // 统计各状态人数
    database.DB.Model(&model.ExamRecord{}).
        Where("exam_id = ? AND deleted_at IS NULL", examID).
        Select("status, count(*) as count").
        Group("status").Scan(&data.StatusCounts)

    // 计算分数统计
    database.DB.Model(&model.ExamRecord{}).
        Where("exam_id = ? AND status IN (1,2) AND deleted_at IS NULL", examID).
        Select("avg(score) as avg_score, max(score) as max_score, min(score) as min_score").
        Scan(&data)

    // 分数段分布
    segments := []struct{ min, max float64 }{
        {0, 59}, {60, 69}, {70, 79}, {80, 89}, {90, 100},
    }
    for _, seg := range segments {
        var count int64
        database.DB.Model(&model.ExamRecord{}).
            Where("exam_id = ? AND score >= ? AND score < ? AND deleted_at IS NULL",
                examID, seg.min, seg.max+1).
            Count(&count)

            Range: fmt.Sprintf("%.0f-%.0f", seg.min, seg.max),
            Count: int(count),
        })
    }

    // 最近异常记录
    warningRepo.FindRecentByExamID(examID, 10, &data.RecentWarnings)
```

#### 7.6.3 考试分析报告

```go
// service/exam/analysis_service.go
type ExamAnalysis struct {
    DifficultyIndex  float64     `json:"difficulty_index"`  // 难度指数
    Discrimination   float64     `json:"discrimination"`    // 区分度
    Reliability      float64     `json:"reliability"`       // 信度 (Cronbach's alpha)
    QuestionAnalysis []QAnalysis `json:"question_analysis"` // 逐题分析
}

type QAnalysis struct {
    QuestionID     uint    `json:"question_id"`
    CorrectRate    float64 `json:"correct_rate"`    // 正确率
    AvgScore       float64 `json:"avg_score"`       // 平均分
    Difficulty     float64 `json:"difficulty"`      // 难度指数
    Discrimination float64 `json:"discrimination"`  // 区分度
}

func GenerateAnalysis(examID uint) (*ExamAnalysis, error) {
    // 查询所有已批改的记录
    var records []model.ExamRecord
    database.DB.Where("exam_id = ? AND status IN (1,2) AND deleted_at IS NULL", examID).
        Find(&records)

    if len(records) == 0 {
        return nil, errors.New("暂无考试数据")
    }

    // 按总分排序，取前 27% 和后 27% 用于计算区分度
    sort.Slice(records, func(i, j int) bool {
        return records[i].Score > records[j].Score
    })
    n := len(records)
    highGroup := records[:n*27/100]
    lowGroup := records[n*73/100:]

    analysis := &ExamAnalysis{}

    // 计算难度指数和区分度
    exam, _ := examRepo.FindByID(examID)
    questions, _ := paperRepo.FindQuestionsByPaperID(exam.PaperID)

    for _, pq := range questions {
        var answers []model.AnswerRecord
        database.DB.Where("question_id = ? AND record_id IN (?)",
            pq.QuestionID, getRecordIDs(records)).Find(&answers)

        correctCount := 0
        var totalScore float64
        for _, a := range answers {
            if a.IsCorrect == 1 {
                correctCount++
            }
            totalScore += a.Score
        }

        qa := QAnalysis{
            QuestionID:  pq.QuestionID,
            CorrectRate: float64(correctCount) / float64(len(answers)),
            AvgScore:    totalScore / float64(len(answers)),
        }

        // 难度指数 = 1 - (答对人数 / 总人数)
        qa.Difficulty = 1 - qa.CorrectRate

        // 区分度 = (高分组正确率 - 低分组正确率)
        highCorrect := countCorrect(highGroup, pq.QuestionID)
        lowCorrect := countCorrect(lowGroup, pq.QuestionID)
        qa.Discrimination = float64(highCorrect-len(highGroup)) / float64(len(highGroup)) -
            float64(lowCorrect-len(lowGroup)) / float64(len(lowGroup))

        analysis.QuestionAnalysis = append(analysis.QuestionAnalysis, qa)
    }

    // 信度 (Cronbach's alpha)
    analysis.Reliability = calculateCronbachAlpha(records, questions)

    return analysis, nil
}
```

```go
// service/exam/score_service.go
func SubmitScoreReview(userID, recordID uint, req *dto.ScoreReviewRequest) error {
    record, err := examRepo.FindRecordByID(recordID)
    if err != nil || record.UserID != userID {
        return errors.New("考试记录不存在")
    }

    // 校验：成绩公布后 7 天内可申请
    if record.SubmitTime != nil && time.Since(*record.SubmitTime) > 7*24*time.Hour {
        return errors.New("已超过复核申请期限（7天）")
    }

    // 检查是否已有复核记录
    existingReview, _ := reviewRepo.FindByRecordID(recordID)
    if existingReview != nil && existingReview.Status == 0 {
        return errors.New("已有待审核的复核申请")
    }
    // 二次申诉检查
    if existingReview != nil && existingReview.Status != 0 && existingReview.AppealCount >= 1 {
        return errors.New("已用完申诉次数")
    }

    review := &model.ScoreReview{
        RecordID:      recordID,
        ExamID:        record.ExamID,
        UserID:        userID,
        Reason:        req.Reason,
        Status:        0, // 待审核
        OriginalScore: record.Score,
    }

    if err := reviewRepo.Create(review); err != nil {
        return fmt.Errorf("提交复核申请失败: %w", err)
    }

    // 通知教师审核
    notificationService.SendScoreReviewNotification(record.ExamID, review.ID)

    return nil
}

func HandleScoreReview(reviewerID, reviewID uint, req *dto.HandleReviewRequest) error {

    if err := reviewRepo.Update(review); err != nil {
        return fmt.Errorf("更新复核记录失败: %w", err)
    }

    // 通知学生
    notificationService.SendScoreReviewResult(review.UserID, reviewID, req.Status)

    return nil
}
```

```go
    headers := []string{"序号", "姓名", "学号", "客观题得分", "主观题得分",
        "总分", "状态", "用时(秒)", "提交时间"}
    for i, h := range headers {
        cell, _ := excelize.CoordinatesToCellName(i+1, 1)
        f.SetCellValue(sheet, cell, h)
    }

    // 写入数据
    for i, record := range records {
        row := i + 2
        f.SetCellValue(sheet, cellName(1, row), i+1)
        f.SetCellValue(sheet, cellName(2, row), record.User.Name)
        f.SetCellValue(sheet, cellName(3, row), record.User.StudentID)
        f.SetCellValue(sheet, cellName(4, row), record.ObjScore)
        f.SetCellValue(sheet, cellName(5, row), record.SubjScore)
        f.SetCellValue(sheet, cellName(6, row), record.Score)
        f.SetCellValue(sheet, cellName(7, row), statusText(record.Status))
        f.SetCellValue(sheet, cellName(8, row), record.DurationUsed)
        f.SetCellValue(sheet, cellName(9, row), record.SubmitTime.Format("2006-01-02 15:04:05"))
    }

    // 3. 保存文件
    filePath := fmt.Sprintf("exports/scores_%d_%d.xlsx", m.ExamID, time.Now().Unix())
    f.SaveAs(filePath)

    // 4. 发送下载通知
    downloadURL := generateDownloadURL(filePath)
    notificationService.Send(m.UserID, "成绩导出完成",
        fmt.Sprintf("考试成绩已导出，点击下载: %s", downloadURL))

    return nil
}
```

## 10. 安全考虑

### 10.1 输入验证

| 验证点 | 实现方式 |
|--------|---------|
| DTO 验证 | 使用 `binding` 标签进行结构化验证 |
| 考试时长 | 最小 5 分钟，最大 480 分钟 |
| 总分范围 | DECIMAL(5,2)，最大 999.99 |
| 及格分 | 不得大于总分 |
| 可考次数 | 0 = 不限，最大 999 |
| 答案长度 | 简答题最长 2000 字符 |
| 分页参数 | PageSize 最大 100 |

### 10.2 考试安全

| 安全措施 | 说明 |
|---------|------|
| 服务端倒计时 | 以服务端时间为准，Redis TTL 控制，防止客户端篡改 |
| 提交幂等锁 | Redis SETNX 防止重复提交 |
| 答案加密存储 | Redis 中答案 JSON 序列化存储，不存储明文敏感信息 |
| 考试密码 | 哈希存储（bcrypt），3 次错误锁定 30 分钟 |
| 防暴力破解 | 开始考试接口限流 5 次/分钟 |
| 权限隔离 | 班级考试仅班级成员可见，私有考试仅创建者可见 |

### 10.3 SQL 注入防护

- 使用 GORM 参数化查询，禁止字符串拼接
- 所有用户输入通过 `ShouldBindJSON` / `ShouldBindQuery` 绑定
- 分页参数限制最大值
- 排序字段白名单校验

### 10.4 权限控制

| 操作 | 权限要求 |
|------|---------|
| 查看考试列表 | 已登录用户 |
| 创建考试 | admin, content_admin, teacher, creator 角色 |
| 发布/暂停/结束考试 | 创建者本人 |
| 参加考试 | 已登录用户（需满足可见性条件） |
| 查看成绩 | 考生本人（自己的成绩）；管理员/教师（班级内成绩） |
| 主观题阅卷 | admin, content_admin, teacher（自己班级的考试） |
| 成绩导出 | admin, content_admin, teacher（自己班级的考试） |

### 10.5 删除保护

- 试卷被未开始的考试引用时禁止删除
- 试卷被进行中的考试引用时禁止删除
- 考试有进行中的答题记录时禁止删除
- 考试已结束且有答题记录时允许软删除，答题记录保留

### 10.6 Redis 安全

- Redis 实例启用 ACL 认证
- 考试相关 Key 使用独立 DB（如 DB 2）
- 答案数据写入 Redis 时进行 JSON 序列化，不存储明文敏感信息
- 定期清理过期的考试会话数据

## 11. 性能考虑

### 11.1 数据库优化

| 优化点 | 实现方式 |
|--------|---------|
| 索引优化 | exam_id, user_id, status, paper_id 建立复合索引 |
| 分页查询 | 使用 OFFSET/LIMIT，避免全表扫描 |
| 预加载 | 使用 GORM Preload 加载关联数据（Paper, User） |
| 批量插入 | 答题记录使用 `CreateInBatches`（每批 100 条） |
| 软删除索引 | deleted_at 字段建立索引 |
| 查询优化 | 成绩列表使用覆盖索引，避免回表 |

### 11.2 缓存策略

| 缓存项 | 缓存时间 | 更新策略 |
|--------|---------|---------|
| 答题会话 | 考试时长 | 开始考试时创建，交卷时删除 |
| 答案暂存 | 考试时长 | 每次答题时更新，交卷时批量写入 MySQL 后删除 |
| 监控数据 | 10 秒 | WebSocket 推送时定时刷新 |
| 考试详情 | 5 分钟 | 更新考试时主动失效 |
| 排名数据 | 考试结束后 1 小时 | 考试结束后生成 |

### 11.3 高并发场景

| 场景 | 挑战 | 解决方案 |
|------|------|---------|
| 大规模同时开考 | 数百人同时开始考试 | Redis 缓存题目快照，减少数据库查询 |
| 答题高频保存 | 每人每 30 秒自动保存 | Redis Hash 幂等写入，批量写入 MySQL |
| 超时集中交卷 | 考试结束时大量交卷 | 异步任务队列处理，Redis 锁防止重复提交 |
| 监控实时推送 | 多管理员同时监控 | WebSocket Hub 广播，缓存聚合数据 |
| 成绩查询 | 考试结束后大量查询 | 直接查 MySQL（有索引），不缓存单条记录 |

### 11.4 大规模考试优化

- 试卷题目快照缓存到 Redis，减少开始考试时的数据库查询
- 答案批量写入使用事务 + `CreateInBatches`
- 超时交卷使用延迟队列 + 批量处理，避免逐条检查
- 成绩导出使用异步任务，避免阻塞 HTTP 请求
- 排名生成在考试结束后一次性完成，不实时更新

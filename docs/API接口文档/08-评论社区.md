# 08 评论社区 API

## 枚举值速查

### 评论目标类型

| 值 | 类型 |
|----|------|
| 1 | 题目 |
| 2 | 考试 |
| 3 | 班级 |

### 评论状态

| 值 | 状态 | 说明 |
|----|------|------|
| 0 | 待审核 | 新评论默认状态 |
| 1 | 已通过 | 正常显示 |
| 2 | 已隐藏 | 管理员隐藏 |

### 通知类型

| 值 | 类型 |
|----|------|
| 1 | 系统 |
| 2 | 考试 |
| 3 | 作业 |
| 4 | 班级 |
| 5 | 评论 |

### 通知渠道

| 值 | 渠道 |
|----|------|
| `app` | 应用内通知 |
| `email` | 邮件 |
| `sms` | 短信 |
| `wechat` | 微信 |

### 举报原因类型

| 值 | 类型 |
|----|------|
| `spam` | 垃圾信息 |
| `abuse` | 辱骂/骚扰 |
| `infringement` | 侵权 |
| `other` | 其他 |

### 审核规则类型

| 值 | 类型 |
|----|------|
| `keyword` | 关键词匹配 |
| `regex` | 正则表达式 |
| `length` | 长度限制 |
| `repeat` | 重复检测 |

### 审核规则动作

| 值 | 动作 | 说明 |
|----|------|------|
| 1 | 自动通过 | 命中规则后自动通过 |
| 2 | 自动拒绝 | 命中规则后自动隐藏 |
| 3 | 标记待审 | 命中规则后标记人工审核 |
| 4 | 丢弃 | 命中规则后直接丢弃 |

### 群发通知目标类型

| 值 | 类型 | 说明 |
|----|------|------|
| `all` | 全部用户 | 所有用户 |
| `role` | 按角色 | 需指定 `target_ids`（角色 ID 列表） |
| `class` | 按班级 | 需指定 `target_ids`（班级 ID 列表） |
| `group` | 按用户组 | 需指定 `target_ids`（用户组 ID 列表） |

#### GET `/api/v1/comments`

分页获取指定目标的评论列表，支持树形嵌套（子评论在 `children` 字段中）。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码 |
| `page_size` | int | 否 | 10 | 每页数量，最大 100 |
| `target_type` | int8 | 是 | - | 评论目标类型: 1=题目, 2=考试, 3=班级 |
| `target_id` | uint | 是 | - | 目标 ID |
| `status` | int8 | 否 | - | 评论状态筛选 |
| `keyword` | string | 否 | - | 搜索关键词 |

```
GET /api/v1/comments?target_type=1&target_id=42&page=1&page_size=20
```

#### POST `/api/v1/comments`

发表一条评论，支持回复（指定 `parent_id`）和 @用户。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `target_type` | int8 | 是 | 评论目标类型: 1=题目, 2=考试, 3=班级 |
| `target_id` | uint | 是 | 目标 ID |
| `content` | string | 是 | 评论内容，最大 500 字符 |
| `parent_id` | uint | 否 | 父评论 ID（回复时填写） |
| `images` | string | 否 | 评论图片 URL（多个以逗号分隔） |
| `mentions` | string | 否 | @的用户 ID（多个以逗号分隔） |
| `reply_to_user_id` | uint | 否 | 回复目标用户 ID |

**请求示例**

```json
{
  "target_type": 1,
  "target_id": 42,
  "content": "这道题考查的是二次函数的对称性，@学生02 你觉得呢？",
  "parent_id": null,
  "images": "",
  "mentions": "11",
  "reply_to_user_id": null
}
```

**回复评论示例**

```json
{
  "target_type": 1,
  "target_id": 42,
  "content": "对的，对称轴公式是 x=-b/2a",
  "parent_id": 1,
  "images": "",
  "mentions": "",
  "reply_to_user_id": 10
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "评论成功"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误（字段缺失、内容超长、target_type 值非法等） |
| `B0001` | 500 | 服务器内部错误（含被加入黑名单时的拦截） |

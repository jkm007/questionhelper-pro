## 基础操作

### GET `/api/v1/wrong`

分页获取当前用户的错题列表，支持按掌握状态和来源筛选。

**请求参数**（Query）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | int | 否 | 页码，默认 1 |
| `page_size` | int | 否 | 每页条数，默认 10，最大 100 |
| `mastered` | boolean | 否 | 是否已掌握: true=已掌握, false=未掌握 |
| `source` | int | 否 | 错题来源 |

**请求示例**

```
GET /api/v1/wrong?page=1&page_size=20&mastered=false
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "user_id": 100,
        "question_id": 5001,
        "question": {
          "id": 5001,
          "title": "以下哪个是Go语言的并发原语？",
          "type": 1,
          "difficulty": 2,
          "category_id": 10,
          "category_name": "Go语言基础"
        },
        "source": 1,
        "source_id": 2001,
        "wrong_count": 3,
        "last_answer": "B",
        "mastered": false,
        "correct_streak": 0,
        "last_review_at": "2026-05-28T10:00:00Z",
        "next_review_at": "2026-06-01T10:00:00Z",
        "review_count": 2,
        "is_favorite": true,
        "tags": [
          {"id": 1, "name": "重点", "color": "#FF0000", "icon": "star", "sort": 0, "count": 5}
        ],
        "created_at": "2026-05-20T08:30:00Z",
        "updated_at": "2026-05-28T10:00:00Z"
      }
    ],
    "total": 58,
    "page": 1,
    "page_size": 20
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `list` | array | 错题列表 |
| `list[].id` | int | 错题记录 ID |
| `list[].user_id` | int | 用户 ID |
| `list[].question_id` | int | 关联题目 ID |
| `list[].question` | object | 题目信息 |
| `list[].question.id` | int | 题目 ID |
| `list[].question.title` | string | 题目标题 |
| `list[].question.type` | int | 题目类型: 1=单选, 2=多选, 3=判断, 4=填空, 5=简答 |
| `list[].question.difficulty` | int | 难度: 1=简单, 2=中等, 3=困难 |
| `list[].question.category_id` | int | 分类 ID |
| `list[].question.category_name` | string | 分类名称 |
| `list[].source` | int | 错题来源 |
| `list[].source_id` | int | 来源 ID（如考试 ID、练习 ID） |
| `list[].wrong_count` | int | 累计错误次数 |
| `list[].last_answer` | string | 最近一次作答答案 |
| `list[].mastered` | boolean | 是否已掌握 |
| `list[].correct_streak` | int | 连续正确次数 |
| `list[].last_review_at` | string | 最近复习时间 (ISO 8601) |
| `list[].next_review_at` | string | 下次复习时间 (ISO 8601) |
| `list[].review_count` | int | 复习总次数 |
| `list[].is_favorite` | boolean | 是否已收藏 |
| `list[].tags` | array | 标签列表 |
| `list[].tags[].id` | int | 标签 ID |
| `list[].tags[].name` | string | 标签名称 |
| `list[].tags[].color` | string | 标签颜色 |
| `list[].tags[].icon` | string | 标签图标 |
| `list[].tags[].sort` | int | 排序值 |
| `list[].tags[].count` | int | 该标签下错题数量 |
| `list[].created_at` | string | 创建时间 (ISO 8601) |
| `list[].updated_at` | string | 更新时间 (ISO 8601) |
| `total` | int | 总记录数 |
| `page` | int | 当前页码 |
| `page_size` | int | 每页条数 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0200` | 401 | 未登录或令牌无效 |
| `B0001` | 500 | 服务器内部错误 |

### POST `/api/v1/wrong/:id/review`

提交错题复习答案，系统自动判断对错并更新错题状态。

**请求参数**（Path + JSON Body）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | int | 是 | （Path）错题记录 ID |
| `answer` | string | 是 | 作答内容 |

**请求示例**

```json
{
  "answer": "A"
}
```

**响应示例**（回答正确）

```json
{
  "code": "00000",
  "msg": "回答正确！",
  "data": {
    "is_correct": true
  }
}
```

**响应示例**（回答错误）

```json
{
  "code": "00000",
  "msg": "回答错误",
  "data": {
    "is_correct": false
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `is_correct` | boolean | 是否回答正确 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 参数错误或无效的错题 ID |
| `A0200` | 401 | 未登录或令牌无效 |
| `B0001` | 500 | 服务器内部错误 |

### GET `/api/v1/wrong/search`

根据关键词和多种条件搜索错题，支持排序。

**请求参数**（Query）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | int | 否 | 页码，默认 1 |
| `page_size` | int | 否 | 每页条数，默认 10，最大 100 |
| `keyword` | string | 否 | 搜索关键词，模糊匹配题目标题 |
| `category_id` | int | 否 | 分类 ID |
| `difficulty` | int | 否 | 难度: 1=简单, 2=中等, 3=困难 |
| `type` | int | 否 | 题目类型: 1=单选, 2=多选, 3=判断, 4=填空, 5=简答 |
| `source` | int | 否 | 错题来源 |
| `mastered` | boolean | 否 | 是否已掌握 |
| `is_favorite` | boolean | 否 | 是否已收藏 |
| `tag_id` | int | 否 | 标签 ID |
| `start_date` | string | 否 | 开始日期 (YYYY-MM-DD) |
| `end_date` | string | 否 | 结束日期 (YYYY-MM-DD) |

## 错题分析

### GET `/api/v1/wrong/analysis`

获取当前用户的错题综合统计分析数据，包括总量、已掌握数、按分类/题型/难度的分布。

**请求参数**

无。

**请求示例**

```
GET /api/v1/wrong/analysis
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "total_count": 58,
    "mastered_count": 20,
    "by_category": [
      {"category_id": 10, "category_name": "Go语言基础", "count": 15},
      {"category_id": 11, "category_name": "数据结构", "count": 12}
    ],
    "by_type": [
      {"type": 1, "count": 25},
      {"type": 2, "count": 18},
      {"type": 3, "count": 15}
    ],
    "by_difficulty": [
      {"difficulty": 1, "count": 10},
      {"difficulty": 2, "count": 30},
      {"difficulty": 3, "count": 18}
    ]
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `total_count` | int | 错题总数 |
| `mastered_count` | int | 已掌握数量 |
| `by_category` | array | 按分类统计 |
| `by_category[].category_id` | int | 分类 ID |
| `by_category[].category_name` | string | 分类名称 |
| `by_category[].count` | int | 该分类下错题数量 |
| `by_type` | array | 按题型统计 |
| `by_type[].type` | int | 题目类型: 1=单选, 2=多选, 3=判断, 4=填空, 5=简答 |
| `by_type[].count` | int | 该题型下错题数量 |
| `by_difficulty` | array | 按难度统计 |
| `by_difficulty[].difficulty` | int | 难度: 1=简单, 2=中等, 3=困难 |
| `by_difficulty[].count` | int | 该难度下错题数量 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0200` | 401 | 未登录或令牌无效 |
| `B0001` | 500 | 服务器内部错误 |

### GET `/api/v1/wrong/analysis/trend`

获取错题趋势分析数据，按指定时间区间和粒度统计每日新增错题数量。

**请求参数**（Query）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `start_date` | string | 是 | 开始日期 (YYYY-MM-DD) |
| `end_date` | string | 是 | 结束日期 (YYYY-MM-DD) |
| `interval` | string | 否 | 统计粒度: `day`(默认), `week`, `month` |

**请求示例**

```
GET /api/v1/wrong/analysis/trend?start_date=2026-05-01&end_date=2026-06-01&interval=day
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": [
    {"date": "2026-05-01", "count": 3},
    {"date": "2026-05-02", "count": 5},
    {"date": "2026-05-03", "count": 2}
  ]
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `date` | string | 日期 (YYYY-MM-DD) |
| `count` | int | 该日期新增错题数量 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 参数错误（start_date 或 end_date 缺失） |
| `A0200` | 401 | 未登录或令牌无效 |
| `B0001` | 500 | 服务器内部错误 |

### GET `/api/v1/wrong/analysis/accuracy`

获取当前用户的错题正确率分析数据，包括总体正确率以及按难度和题型的细分正确率。

**请求参数**

无。

**请求示例**

```
GET /api/v1/wrong/analysis/accuracy
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "total_reviews": 120,
    "correct_reviews": 85,
    "accuracy_rate": 0.7083,
    "by_difficulty": [
      {"difficulty": 1, "total_count": 30, "correct_count": 25, "accuracy_rate": 0.8333},
      {"difficulty": 2, "total_count": 60, "correct_count": 42, "accuracy_rate": 0.7000},
      {"difficulty": 3, "total_count": 30, "correct_count": 18, "accuracy_rate": 0.6000}
    ],
    "by_type": [
      {"type": 1, "total_count": 50, "correct_count": 38, "accuracy_rate": 0.7600},
      {"type": 2, "total_count": 40, "correct_count": 28, "accuracy_rate": 0.7000},
      {"type": 3, "total_count": 30, "correct_count": 19, "accuracy_rate": 0.6333}
    ]
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `total_reviews` | int | 复习总次数 |
| `correct_reviews` | int | 回答正确次数 |
| `accuracy_rate` | float | 总体正确率（0~1） |
| `by_difficulty` | array | 按难度细分 |
| `by_difficulty[].difficulty` | int | 难度: 1=简单, 2=中等, 3=困难 |
| `by_difficulty[].total_count` | int | 该难度复习总次数 |
| `by_difficulty[].correct_count` | int | 该难度回答正确次数 |

## 错题导出

### POST `/api/v1/wrong/export`

根据条件导出错题为 CSV 文件。可通过 `ids` 指定具体错题，也可通过筛选条件导出符合条件的全部错题。

**请求参数**（JSON Body）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `ids` | array[int] | 否 | 指定错题 ID 列表（指定后忽略筛选条件） |
| `format` | string | 是 | 导出格式: `csv`, `excel` |
| `keyword` | string | 否 | 搜索关键词 |
| `category_id` | int | 否 | 分类 ID |
| `difficulty` | int | 否 | 难度: 1=简单, 2=中等, 3=困难 |
| `mastered` | boolean | 否 | 是否已掌握 |

**请求示例**

```json
{
  "format": "csv",
  "category_id": 10,
  "difficulty": 2
}
```

**响应**

返回文件流，响应头：
- `Content-Type: text/csv; charset=utf-8`
- `Content-Disposition: attachment; filename=wrong_questions_20260601120000.csv`

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 参数错误（format 缺失或不是 csv/excel） |
| `A0200` | 401 | 未登录或令牌无效 |
| `B0001` | 500 | 服务器内部错误 |

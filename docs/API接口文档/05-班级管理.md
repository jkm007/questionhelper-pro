# 05 班级管理 API

## 用户端接口 (需登录)

### 基本操作

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/class` | 班级列表 |
| GET | `/api/v1/class/:id` | 班级详情 |
| POST | `/api/v1/class` | 创建班级 |
| PUT | `/api/v1/class/:id` | 更新班级信息 |
| DELETE | `/api/v1/class/:id` | 删除班级 |
| POST | `/api/v1/class/:id/join` | 加入班级 |
| POST | `/api/v1/class/:id/leave` | 退出班级 |
| GET | `/api/v1/class/search` | 按关键词搜索班级 |
| GET | `/api/v1/class/:id/qrcode` | 生成班级二维码 |

---

#### GET `/api/v1/class`

分页获取当前用户加入的班级列表。

**认证**: 需要 JWT Bearer Token

**查询参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码 |
| `page_size` | int | 否 | 10 | 每页数量 |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "name": "高三一班",
        "description": "高三理科重点班",
        "cover": "https://...",
        "code": "ABC123",
        "creator_id": 1,
        "creator_name": "张老师",
        "member_count": 45,
        "status": 1,
        "created_at": "2026-01-01T00:00:00Z"
      }
    ],
    "total": 5,
    "page": 1,
    "page_size": 10
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `list[].id` | int | 班级 ID |
| `list[].name` | string | 班级名称 |
| `list[].description` | string | 班级描述 |
| `list[].cover` | string | 封面图 URL |
| `list[].code` | string | 班级邀请码 |
| `list[].creator_id` | int | 创建者 ID |
| `list[].creator_name` | string | 创建者名称 |
| `list[].member_count` | int | 成员数量 |
| `list[].status` | int | 状态: 0=禁用, 1=正常, 2=已归档 |
| `list[].created_at` | string | 创建时间 |

# 01 用户认证 API

## 公共接口 (无需认证)

### 登录注册

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/auth/login` | 用户登录 |
| POST | `/api/v1/auth/register` | 用户注册 |

#### POST `/api/v1/auth/login`

用户通过用户名/邮箱/手机号和密码登录，支持图形验证码验证。密码错误 3 次后需填写验证码，连续错误 5 次锁定 15 分钟。

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `username` | string | 是 | 用户名/邮箱/手机号 |
| `password` | string | 是 | 密码 |
| `captchaId` | string | 否 | 图形验证码 ID（密码错误 3 次后必填） |
| `captchaCode` | string | 否 | 图形验证码（密码错误 3 次后必填） |
| `deviceId` | string | 否 | 设备唯一标识 |
| `deviceInfo` | string | 否 | 设备信息 |
| `rememberMe` | boolean | 否 | 记住我，Refresh Token 延长至 30 天 |

**请求示例**

```json
{
  "username": "admin",
  "password": "admin123",
  "captchaId": "xxx",
  "captchaCode": "xxxx",
  "rememberMe": false
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "accessToken": "eyJ...",
    "refreshToken": "eyJ...",
    "tokenType": "Bearer",
    "expiresIn": 7200,
    "user": {
      "id": 1,
      "username": "admin",
      "nickname": "管理员",
      "email": "admin@example.com",
      "phone": "13800138000",
      "avatar": "https://...",
      "gender": 1,
      "birthday": null,
      "bio": "",
      "status": 1,
      "is_real": true,
      "roles": [
        {"id": 1, "name": "管理员", "code": "admin", "description": "系统管理员"}
      ],
      "created_at": "2026-01-01T00:00:00Z"
    }
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `accessToken` | string | 访问令牌，有效期 2 小时 |
| `refreshToken` | string | 刷新令牌，有效期 7 天（rememberMe 时 30 天） |
| `tokenType` | string | 令牌类型，固定 `Bearer` |
| `expiresIn` | int | access_token 过期时间（秒） |
| `user` | object | 用户信息 |
| `user.id` | int | 用户 ID |
| `user.username` | string | 用户名 |
| `user.nickname` | string | 昵称 |
| `user.email` | string | 邮箱 |
| `user.phone` | string | 手机号 |
| `user.avatar` | string | 头像 URL |
| `user.gender` | int | 性别: 0=未知, 1=男, 2=女 |
| `user.birthday` | string | 生日 (ISO 8601) |
| `user.bio` | string | 个人简介 |
| `user.status` | int | 状态: 0=禁用, 1=正常, 2=注销中 |
| `user.is_real` | boolean | 是否已实名认证 |
| `user.roles` | array | 角色列表 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `A0200` | 401 | 用户名或密码错误 |
| `A0204` | 403 | 账号已禁用或注销中 |
| `A0204` | 403 | 账号已锁定（密码错误 5 次） |
| `A0205` | 401 | 验证码错误 |
| `A0429` | 429 | 请求过于频繁 |

#### POST `/api/v1/auth/register`

新用户注册，注册成功后自动登录并返回令牌。用户名 4-20 位，密码 6-20 位需包含字母和数字，邮箱必填。

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `username` | string | 是 | 用户名，4-20 字符 |
| `password` | string | 是 | 密码，6-20 位，需包含字母+数字 |
| `confirmPassword` | string | 是 | 确认密码，需与 password 一致 |
| `nickname` | string | 否 | 昵称，2-20 字符 |
| `phone` | string | 否 | 手机号（选填） |
| `email` | string | 是 | 邮箱（必填） |
| `emailCode` | string | 否 | 邮箱验证码 |
| `captchaId` | string | 是 | 图形验证码 ID |
| `captchaCode` | string | 是 | 图形验证码 |
| `agreement` | boolean | 是 | 是否同意用户协议 |
| `source` | string | 否 | 注册来源: h5/miniapp/app/web |

**请求示例**

```json
{
  "username": "student01",
  "password": "pass123456",
  "confirmPassword": "pass123456",
  "nickname": "学生01",
  "email": "student@example.com",
  "emailCode": "123456",
  "captchaId": "xxx",
  "captchaCode": "xxxx",
  "agreement": true,
  "source": "web"
}
```

**响应示例**（同登录响应）

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `A0202` | 409 | 用户名或邮箱已被注册 |
| `A0205` | 400 | 验证码错误 |
| `A0206` | 400 | 验证码已过期 |

**请求示例**

  "role_id": 2
}

**响应示例**（同登录响应，返回新角色对应的 token）

**错误代码**

#### GET `/api/v1/auth/captcha`

获取图形验证码，支持 digit/letter/math 三种类型。

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | 否 | 验证码类型: `digit`(数字，默认)、`letter`(字母)、`math`(算术) |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "captchaId": "a1b2c3d4",
    "captchaBase64": "data:image/png;base64,iVBOR...",
    "captchaType": "digit"
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `captchaId` | string | 验证码 ID，验证时需传回 |
| `captchaBase64` | string | 验证码图片的 Base64 编码 |
| `captchaType` | string | 验证码类型 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `B0001` | 500 | 生成验证码失败 |

---

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|

**请求示例**

```json
{
  "account": "student@example.com",
  "captchaId": "xxx",
  "captchaCode": "xxxx"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "重置密码请求已提交，请查看手机或邮箱"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `A0205` | 400 | 验证码错误 |
| `B0001` | 500 | 发送失败 |

---

#### POST `/api/v1/auth/password/reset`

### OAuth 第三方登录

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/auth/oauth/:provider/url` | 获取 OAuth 授权 URL |
| POST | `/api/v1/auth/oauth/:provider` | OAuth 登录（通过授权码） |
| GET | `/api/v1/auth/oauth/:provider/callback` | OAuth 回调重定向 |

`provider` 可选值：`wechat`、`github`、`google`

---

#### GET `/api/v1/auth/oauth/:provider/url`

获取指定第三方平台的 OAuth 授权跳转 URL。

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `provider` | string | 是 | 第三方平台: `wechat`、`github`、`google` |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "url": "https://github.com/login/oauth/authorize?client_id=xxx&redirect_uri=xxx"
  }
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 不支持的平台 |
| `B0001` | 500 | 生成授权 URL 失败 |

---

#### POST `/api/v1/auth/oauth/:provider`

通过第三方平台授权码登录。未注册用户将自动注册。

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `provider` | string | 是 | 第三方平台 |

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | OAuth 授权码 |
| `state` | string | 否 | 状态参数（防 CSRF） |

**请求示例**

```json
{
  "code": "github_auth_code_xxx",
  "state": "random_state"
}
```

**响应示例**（同登录响应）

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 不支持的平台或 state 验证失败 |
| `A0204` | 403 | 账号已禁用或注销中 |
| `B0001` | 500 | OAuth 登录失败 |

---

### OAuth 绑定

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/user/oauth/status` | 获取 OAuth 绑定状态 |
| POST | `/api/v1/user/oauth/bind/:provider` | 绑定 OAuth 账号 |
| DELETE | `/api/v1/user/oauth/unbind/:provider` | 解绑 OAuth 账号 |

---

#### GET `/api/v1/user/oauth/status`

获取当前用户已绑定的第三方账号状态。

**认证**: 需要 JWT Bearer Token

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "wechat": {"bound": true, "nickname": "微信用户", "boundAt": "2026-01-01T00:00:00Z"},
    "github": {"bound": false},
    "google": {"bound": false}
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `wechat` | object | 微信绑定状态 |
| `github` | object | GitHub 绑定状态 |
| `google` | object | Google 绑定状态 |
| `*.bound` | boolean | 是否已绑定 |
| `*.nickname` | string | 第三方平台昵称（已绑定时） |
| `*.boundAt` | string | 绑定时间（已绑定时） |

---

#### POST `/api/v1/user/oauth/bind/:provider`

已认证用户绑定第三方平台账号。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `provider` | string | 是 | 第三方平台: `wechat`、`github`、`google` |

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | OAuth 授权码 |
| `state` | string | 否 | 状态参数 |

**请求示例**

```json
{
  "code": "github_auth_code_xxx",
  "state": "random_state"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "绑定成功"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 不支持的平台 |
| `A0202` | 409 | 当前用户已绑定该平台 |
| `A0202` | 409 | 该第三方账号已被其他用户绑定 |
| `B0001` | 500 | 绑定失败 |

---

#### DELETE `/api/v1/user/oauth/unbind/:provider`

已认证用户解绑第三方平台账号。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `provider` | string | 是 | 第三方平台 |

**响应示例**

```json
{

}
```

**错误代码**

### 账号注销

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/user/account/deactivate` | 申请注销账号 |
| POST | `/api/v1/user/account/deactivate/confirm` | 确认注销 |
| POST | `/api/v1/user/account/cancel-deactivate` | 取消注销 |
| GET | `/api/v1/user/account/export` | 导出个人数据 |

---

#### POST `/api/v1/user/account/deactivate`

已认证用户申请注销账号，需要验证密码。验证通过后发送确认验证码。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `password` | string | 是 | 当前账号密码 |
| `reason` | string | 否 | 注销原因 |

**请求示例**

```json
{
  "password": "mypassword123",
  "reason": "不再使用"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "verifyType": "sms",
    "expireIn": 300
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `verifyType` | string | 验证方式: sms/email |
| `expireIn` | int | 验证码有效期（秒） |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `A0201` | 404 | 用户不存在 |
| `A0203` | 400 | 密码错误 |
| `A0207` | 403 | 注销取消后 30 天内不可再次申请 |

---

#### POST `/api/v1/user/account/deactivate/confirm`

通过验证码确认注销账号。确认后账号进入"注销中"状态，30 天后永久删除。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | 确认验证码 |

**请求示例**

```json
{
  "code": "123456"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "注销申请已提交，30天后将永久删除账号"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `A0206` | 400 | 验证码错误或已过期 |

---

#### POST `/api/v1/user/account/cancel-deactivate`

取消已提交的账号注销申请。

**认证**: 需要 JWT Bearer Token

**响应示例**

```json
{
  "code": "00000",
  "msg": "已取消注销"
}
```

**错误代码**

|--------|-------------|------|
| `A0201` | 404 | 用户不存在 |
| `A0001` | 400 | 当前状态不允许取消 |

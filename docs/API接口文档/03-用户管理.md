# 03 用户管理 API (管理员)

所有接口均需管理员权限（JWT + admin 角色），基础路径：`/api/v1/admin`

---

## 用户管理

### 接口总览

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/admin/users` | 用户列表 |
| GET | `/api/v1/admin/users/:id` | 用户详情 |
| POST | `/api/v1/admin/users` | 创建用户 |
| PUT | `/api/v1/admin/users/:id` | 更新用户 |
| DELETE | `/api/v1/admin/users/:id` | 删除用户 |
| PUT | `/api/v1/admin/users/:id/status` | 更新用户状态 |
| POST | `/api/v1/admin/users/:id/reset-password` | 重置用户密码 |
| PUT | `/api/v1/admin/users/:id/roles` | 分配用户角色 |
| POST | `/api/v1/admin/users/batch-status` | 批量更新状态 |
| POST | `/api/v1/admin/users/batch-delete` | 批量删除用户 |
| POST | `/api/v1/admin/users/batch-roles` | 批量分配角色 |
| GET | `/api/v1/admin/users/export` | 导出用户 |

### GET /api/v1/admin/users -- 用户列表

**查询参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | int | 否 | 页码，默认 1 |
| `page_size` | int | 否 | 每页条数，默认 10，最大 100 |
| `keyword` | string | 否 | 搜索关键词（模糊匹配用户名/昵称/手机号/邮箱） |
| `status` | int | 否 | 用户状态筛选：0=禁用, 1=正常 |
| `role_id` | uint | 否 | 按角色 ID 筛选 |
| `role_code` | string | 否 | 按角色编码筛选 |
| `tag_id` | uint | 否 | 按标签筛选 |
| `start_date` | datetime | 否 | 注册开始时间 |
| `end_date` | datetime | 否 | 注册结束时间 |

        "birthday": "2000-01-01T00:00:00Z",
        "bio": "系统管理员",
        "status": 1,
        "is_real": true,
        "roles": [
          {
            "id": 1,

            "description": "系统管理员"
          }
        ],
        "created_at": "2026-01-01T00:00:00Z"

### POST /api/v1/admin/users -- 创建用户

**请求体**

```json
{
  "username": "teacher01",
  "password": "Pass123",
  "nickname": "张老师",
  "phone": "13900139000",
  "email": "teacher01@example.com",
  "role_ids": [2, 3]
}
```

| 字段 | 类型 | 必填 | 校验规则 | 说明 |
|------|------|------|----------|------|
| `username` | string | 是 | 2-20 字符 | 用户名 |
| `password` | string | 是 | 6-20 字符 | 密码 |
| `nickname` | string | 否 | 最大 50 字符 | 昵称 |
| `phone` | string | 否 | - | 手机号 |
| `email` | string | 否 | 邮箱格式 | 邮箱 |
| `role_ids` | uint[] | 否 | - | 角色 ID 列表 |

**响应**

```json
{
  "code": "00000",
  "msg": "创建成功",
  "data": null
}
```

| `id` | uint | 按钮权限 ID |

```json

}
```

| 字段 | 类型 | 必填 | 校验规则 | 说明 |

| `permission_code` | string | 否 | 最大 100 字符 | 权限编码 |
| `permission_name` | string | 否 | 最大 100 字符 | 权限名称 |
| `sort` | int | 否 | - | 排序值 |
| `status` | bool | 否 | - | 启用状态 |
| `remark` | string | 否 | 最大 500 字符 | 备注 |

**响应**

```json
{
  "code": "00000",
  "msg": "更新成功",
  "data": null
}
```

## 数据模型参考

### 用户状态

| 值 | 状态 | 说明 |
|----|------|------|
| 0 | 禁用 | 账号被禁用，不可登录 |
| 1 | 正常 | 正常可用 |
| 2 | 注销中 | 用户申请注销，等待处理 |

### 用户性别

| 值 | 性别 |
|----|------|
| 0 | 未知 |
| 1 | 男 |
| 2 | 女 |

### 菜单类型

| 值 | 类型 | 说明 |
|----|------|------|
| 1 | 目录 | 用于路由分组 |
| 2 | 菜单 | 可访问的页面路由 |
| 3 | 按钮 | 页面内操作权限 |

### 角色模型字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | uint | 角色 ID |
| `name` | string | 角色名称 |
| `code` | string | 角色编码（唯一） |
| `description` | string | 角色描述 |
| `is_default` | bool | 是否默认角色 |
| `is_system` | bool | 是否系统角色（不可删除） |
| `sort` | int | 排序值 |
| `status` | int | 状态：0=禁用, 1=正常 |

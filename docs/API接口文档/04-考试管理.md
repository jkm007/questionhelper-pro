# 04 考试管理 API

## 用户端接口 (需登录)

### 考试列表与详情

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/exam` | 考试列表 |
| GET | `/api/v1/exam/:id` | 考试详情 |
| GET | `/api/v1/exam/history` | 考试历史 |
| GET | `/api/v1/exams/upcoming` | 即将开始的考试列表 |

---

#### GET `/api/v1/exam`

获取当前学生可参加的考试列表，支持分页。

**认证**: 需要 JWT Bearer Token

**查询参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码，从 1 开始 |
| `page_size` | int | 否 | 10 | 每页条数，最大 100 |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "title": "期中考试",
        "description": "2026年春季期中考试",
        "paper_id": 10,
        "class_id": 5,
        "start_time": "2026-06-10T09:00:00Z",
        "end_time": "2026-06-10T11:00:00Z",
        "duration": 120,
        "total_score": 100,
        "pass_score": 60,
        "max_attempts": 1,
        "shuffle": true,
        "show_answer": 1,
        "anti_cheat": 1,
        "status": 1,
        "creator_id": 1
      }
    ],
    "total": 25,
    "page": 1,
    "page_size": 10
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `list[].id` | int | 考试 ID |
| `list[].title` | string | 考试标题 |
| `list[].description` | string | 考试描述 |
| `list[].paper_id` | int | 关联试卷 ID |
| `list[].class_id` | int | 关联班级 ID，null 表示不限班级 |
| `list[].start_time` | string | 开始时间 (ISO 8601) |
| `list[].end_time` | string | 结束时间 (ISO 8601) |
| `list[].duration` | int | 考试时长（分钟） |
| `list[].total_score` | number | 总分 |
| `list[].pass_score` | number | 及格分 |
| `list[].max_attempts` | int | 最大答题次数 |
| `list[].shuffle` | boolean | 是否打乱题目顺序 |
| `list[].show_answer` | int | 答案显示策略: 0=不显示, 1=交卷后显示, 2=批改后显示 |
| `list[].anti_cheat` | int | 防作弊等级: 0=关闭, 1=基础, 2=严格 |
| `list[].status` | int | 考试状态: 0=草稿, 1=已发布, 2=进行中, 3=已结束 |
| `list[].creator_id` | int | 创建者 ID |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `B0001` | 500 | 服务器内部错误 |

#### POST `/api/v1/exam/:id/start`

学生开始指定考试，系统创建考试记录并返回答题所需信息。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | int | 是 | 考试 ID |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "id": 1,
    "exam_id": 1,
    "user_id": 100,
    "score": 0,
    "status": 0,
    "start_time": "2026-06-10T09:00:00Z",
    "submit_time": null
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 考试记录 ID（后续答题、提交均使用此 ID） |
| `exam_id` | int | 考试 ID |
| `user_id` | int | 用户 ID |
| `score` | number | 得分（初始为 0） |
| `status` | int | 状态: 0=答题中 |
| `start_time` | string | 开始答题时间 |
| `submit_time` | string | 提交时间（初始为 null） |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 无效的考试 ID |
| `A0501` | 404 | 考试不存在 |
| `A0502` | 400 | 考试尚未开始 |
| `A0503` | 400 | 考试已结束 |
| `A0504` | 400 | 已达到最大答题次数 |
| `B0001` | 500 | 服务器内部错误 |

#### POST `/api/v1/exam/:recordId/submit`

学生提交考试答卷。提交后自动阅卷（客观题）并计算成绩。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `recordId` | int | 是 | 考试记录 ID（由开始考试接口返回） |

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `answers` | array | 是 | 答案列表 |
| `answers[].question_id` | int | 是 | 题目 ID |
| `answers[].answer` | string | 是 | 学生答案（单选/判断填选项标识如"A"，多选填"A,B,C"，填空/简答填文本） |

**请求示例**

```json
{
  "answers": [
    {"question_id": 1, "answer": "A"},
    {"question_id": 2, "answer": "A,B,C"},
    {"question_id": 3, "answer": "true"},
    {"question_id": 4, "answer": "光合作用"}
  ]
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "提交成功"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误（缺少答案或格式不正确） |
| `A0504` | 400 | 该考试记录已提交 |
| `B0001` | 500 | 服务器内部错误 |

#### GET `/api/v1/exam/:id/standard-answers`

获取指定考试的标准答案（需考试已结束且管理员允许查看答案）。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | int | 是 | 考试 ID |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "exam_id": 1,
    "questions": [
      {
        "question_id": 1,
        "title": "CPU的中文全称是什么？",
        "type": 1,
        "score": 5,
        "answer": "A",
        "analysis": "CPU = Central Processing Unit = 中央处理器",
        "options": [
          {"id": 1, "label": "A", "content": "中央处理器", "is_correct": true},
          {"id": 2, "label": "B", "content": "中央存储器", "is_correct": false}
        ],
        "user_answer": "A",
        "user_score": 5,
        "is_correct": true
      }
    ]
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `exam_id` | int | 考试 ID |
| `questions` | array | 题目答案列表 |
| `questions[].question_id` | int | 题目 ID |
| `questions[].title` | string | 题目标题 |
| `questions[].type` | int | 题目类型: 1=单选, 2=多选, 3=判断, 4=填空, 5=简答 |
| `questions[].score` | number | 该题满分 |
| `questions[].answer` | string | 标准答案 |
| `questions[].analysis` | string | 解析 |
| `questions[].options` | array | 选项列表（选择题/判断题） |
| `questions[].options[].id` | int | 选项 ID |
| `questions[].options[].label` | string | 选项标识 (A/B/C/D) |
| `questions[].options[].content` | string | 选项内容 |
| `questions[].options[].is_correct` | boolean | 是否正确选项 |
| `questions[].user_answer` | string | 学生答案 |
| `questions[].user_score` | number | 学生得分 |
| `questions[].is_correct` | boolean | 是否答对 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 无效的考试 ID |
| `A0501` | 404 | 考试不存在 |
| `A0503` | 400 | 考试尚未结束，无法查看答案 |
| `B0001` | 500 | 服务器内部错误 |

#### POST `/api/v1/exam-records/:recordId/save-answer`

在考试过程中保存学生对单个题目的作答（自动保存/手动保存）。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `recordId` | int | 是 | 考试记录 ID |

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `question_id` | int | 是 | 题目 ID |
| `answer` | string | 否 | 学生答案，留空表示清除答案 |
| `is_marked` | boolean | 否 | 是否标记该题 |

**请求示例**

```json
{
  "question_id": 1,
  "answer": "A",
  "is_marked": false
}
```

| 参数 | 类型 | 必填 | 说明 |

}

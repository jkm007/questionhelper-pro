# 03 题库管理 API

## 用户端接口 (需登录)

### 题目浏览

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/questions` | 题目列表 (支持分页、筛选) |
| GET | `/api/v1/questions/search` | 题目搜索 |
| GET | `/api/v1/questions/:id` | 题目详情 |
| POST | `/api/v1/questions/:id/favorite` | 收藏/取消收藏题目 |
| POST | `/api/v1/questions/:id/like` | 点赞/取消点赞题目 |
| GET | `/api/v1/questions/:id/versions` | 题目版本历史 |

---

#### GET `/api/v1/questions`

获取题目列表，支持分页和多条件筛选，带数据权限过滤（普通用户只能看到自己创建的和公开的题目）。

**认证**: 需要 JWT Bearer Token

**查询参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码，从 1 开始 |
| `page_size` | int | 否 | 10 | 每页条数，最大 100 |
| `keyword` | string | 否 | - | 搜索关键词，模糊匹配题目标题和内容 |
| `category_id` | uint | 否 | - | 分类 ID |
| `type` | int | 否 | - | 题型: 1=单选, 2=多选, 3=判断, 4=填空, 5=简答 |
| `difficulty` | int | 否 | - | 难度: 1=简单, 2=中等, 3=困难 |
| `visibility` | int | 否 | - | 可见性: 1=公开, 2=私有, 3=班级可见 |
| `status` | int | 否 | - | 状态: 0=草稿, 1=已发布, 2=已归档 |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "title": "以下哪个是 Go 语言的关键字？",
        "content": "",
        "type": 1,
        "difficulty": 1,
        "answer": "A",
        "analysis": "func 是 Go 语言中定义函数的关键字。",
        "category_id": 1,
        "category": {
          "id": 1,
          "parent_id": null,
          "name": "Go 语言基础",
          "children": []
        },
        "options": [
          {"id": 1, "label": "A", "content": "func", "is_correct": true},
          {"id": 2, "label": "B", "content": "function", "is_correct": false},
          {"id": 3, "label": "C", "content": "def", "is_correct": false},
          {"id": 4, "label": "D", "content": "method", "is_correct": false}
        ],
        "visibility": 1,
        "creator_id": 1,
        "creator_name": "管理员",
        "status": 1,
        "view_count": 120,
        "like_count": 15
      }
    ],
    "total": 50,
    "page": 1,
    "page_size": 10
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `list` | array | 题目列表 |
| `list[].id` | int | 题目 ID |
| `list[].title` | string | 题目标题 |
| `list[].content` | string | 题目内容 |
| `list[].type` | int | 题型: 1=单选, 2=多选, 3=判断, 4=填空, 5=简答 |
| `list[].difficulty` | int | 难度: 1=简单, 2=中等, 3=困难 |
| `list[].answer` | string | 答案 |
| `list[].analysis` | string | 解析 |
| `list[].category_id` | int | 分类 ID |
| `list[].category` | object | 分类信息 |
| `list[].category.id` | int | 分类 ID |
| `list[].category.parent_id` | int | 父分类 ID |
| `list[].category.name` | string | 分类名称 |
| `list[].options` | array | 选项列表（选择题适用） |
| `list[].options[].id` | int | 选项 ID |
| `list[].options[].label` | string | 选项标签 (A/B/C/D) |
| `list[].options[].content` | string | 选项内容 |
| `list[].options[].is_correct` | boolean | 是否正确答案 |
| `list[].visibility` | int | 可见性: 1=公开, 2=私有, 3=班级可见 |
| `list[].creator_id` | int | 创建者 ID |
| `list[].creator_name` | string | 创建者名称 |
| `list[].status` | int | 状态: 0=草稿, 1=已发布, 2=已归档 |
| `list[].view_count` | int | 浏览次数 |
| `list[].like_count` | int | 点赞数 |
| `total` | int | 总记录数 |
| `page` | int | 当前页码 |
| `page_size` | int | 每页条数 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 参数错误 |
| `B0001` | 500 | 服务器内部错误 |

#### POST `/api/v1/questions/:id/favorite`

收藏或取消收藏题目。通过 `action` 参数控制操作类型。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | uint | 是 | 题目 ID |

**查询参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `action` | string | 否 | 操作类型，传 `remove` 为取消收藏，不传则收藏 |

**请求体参数**（收藏时）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `folder_id` | uint | 否 | 收藏夹 ID，不传则收藏到默认收藏夹 |
| `note` | string | 否 | 收藏备注，最大 500 字符 |

**请求示例**

```json
{
  "folder_id": 1,
  "note": "重要题目，需要反复练习"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "收藏成功"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 无效的题目 ID |
| `A0401` | 404 | 题目不存在 |
| `B0001` | 500 | 服务器内部错误 |

#### GET `/api/v1/shares/:code`

通过分享码获取分享的题目信息。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | 分享码 |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "id": 1,
    "share_code": "abc123",
    "share_url": "https://example.com/share/abc123",
    "question_id": 1,
    "share_type": 1,
    "has_password": false,
    "view_count": 25,
    "status": 1,
    "expires_at": "2026-12-31T23:59:59Z",
    "created_at": "2026-06-01T00:00:00Z"
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 分享记录 ID |
| `share_code` | string | 分享码 |
| `share_url` | string | 分享链接 |
| `question_id` | int | 题目 ID |
| `share_type` | int | 分享类型: 1=链接分享, 2=密码分享 |
| `has_password` | boolean | 是否需要密码 |
| `view_count` | int | 查看次数 |
| `status` | int | 状态: 1=有效, 2=已撤销 |
| `expires_at` | string | 过期时间 (ISO 8601)，null 表示永不过期 |
| `created_at` | string | 创建时间 (ISO 8601) |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0002` | 404 | 分享码不存在或已失效 |
| `B0001` | 500 | 服务器内部错误 |

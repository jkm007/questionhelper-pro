#### GET `/api/v1/statistics/overview`

获取当前用户的仪表盘概览数据，包含用户统计、练习统计、考试统计、班级统计四个维度的汇总信息。

**认证**: 需要 JWT Bearer Token

**请求参数**

无。

**请求示例**

```
GET /api/v1/statistics/overview
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "user_stats": {
      "total_users": 1000,
      "active_users": 500,
      "new_users_today": 20,
      "new_users_week": 150
    },
    "practice_stats": {
      "total_sessions": 50000,
      "total_questions": 200000,
      "avg_accuracy": 0.756,
      "total_duration": 3600000
    },
    "exam_stats": {
      "total_exams": 200,
      "total_records": 8000,
      "avg_score": 72.5,
      "pass_rate": 0.68
    },
    "class_stats": {
      "total_classes": 50,
      "total_members": 1000,
      "avg_member_count": 20.0
    }
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `user_stats` | object | 用户统计 |
| `user_stats.total_users` | int64 | 用户总数 |
| `user_stats.active_users` | int64 | 活跃用户数 |
| `user_stats.new_users_today` | int64 | 今日新增用户 |
| `user_stats.new_users_week` | int64 | 本周新增用户 |
| `practice_stats` | object | 练习统计 |
| `practice_stats.total_sessions` | int64 | 练习总次数 |
| `practice_stats.total_questions` | int64 | 练习总题数 |
| `practice_stats.avg_accuracy` | float64 | 平均正确率 |
| `practice_stats.total_duration` | int64 | 练习总时长（秒） |
| `exam_stats` | object | 考试统计 |
| `exam_stats.total_exams` | int64 | 考试总数 |
| `exam_stats.total_records` | int64 | 考试记录总数 |
| `exam_stats.avg_score` | float64 | 平均分 |
| `exam_stats.pass_rate` | float64 | 通过率 |
| `class_stats` | object | 班级统计 |
| `class_stats.total_classes` | int64 | 班级总数 |
| `class_stats.total_members` | int64 | 成员总数 |
| `class_stats.avg_member_count` | float64 | 平均成员数 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0200` | 401 | 未授权（未提供 Token 或 Token 无效） |
| `A0230` | 401 | 访问令牌无效或过期 |
| `B0001` | 500 | 获取统计数据失败 |

#### GET `/api/v1/statistics/ranking`

获取用户排行榜数据，支持按练习、考试、积分三种类型排行，支持分页。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `type` | int | 是 | - | 排行榜类型: 1=练习, 2=考试, 3=积分 |
| `page` | int | 否 | 1 | 页码，从 1 开始 |
| `page_size` | int | 否 | 10 | 每页条数，最大 100 |

**请求示例**

```
GET /api/v1/statistics/ranking?type=1&page=1&page_size=10
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [
      {
        "rank": 1,
        "user_id": 42,
        "user_name": "张三",
        "avatar": "https://example.com/avatar/42.jpg",
        "score": 98.5,
        "count": 500
      },
      {
        "rank": 2,
        "user_id": 15,
        "user_name": "李四",
        "avatar": "https://example.com/avatar/15.jpg",
        "score": 95.0,
        "count": 420
      }
    ],
    "total": 100,
    "page": 1,
    "page_size": 10
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `list` | array | 排行榜列表 |
| `list[].rank` | int | 排名 |
| `list[].user_id` | uint | 用户 ID |
| `list[].user_name` | string | 用户名 |
| `list[].avatar` | string | 头像 URL |
| `list[].score` | float64 | 得分（正确率/考试分/积分） |
| `list[].count` | int | 数量（练习题数/考试次数） |
| `total` | int64 | 总记录数 |
| `page` | int | 当前页码 |
| `page_size` | int | 每页条数 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 参数错误（type 必填且为 1/2/3） |
| `A0200` | 401 | 未授权（未提供 Token 或 Token 无效） |
| `A0230` | 401 | 访问令牌无效或过期 |
| `B0001` | 500 | 获取排行榜失败 |

#### GET `/api/v1/statistics/question/difficulty`

获取题目难度分析数据，包含每道题的正确率、答题次数、平均耗时等信息。可按分类筛选。

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `category_id` | uint | 否 | 分类 ID，不传则查询全部 |

**请求示例**

```
GET /api/v1/statistics/question/difficulty?category_id=5
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": [
    {
      "question_id": 101,
      "title": "HTTP 状态码 200 表示什么？",
      "type": 1,
      "difficulty": 1,
      "correct_rate": 0.92,
      "answer_count": 500,
      "avg_duration": 15.5
    },
    {
      "question_id": 102,
      "title": "请简述 TCP 三次握手过程",
      "type": 5,
      "difficulty": 3,
      "correct_rate": 0.35,
      "answer_count": 200,
      "avg_duration": 120.0
    }
  ]
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `[].question_id` | uint | 题目 ID |
| `[].title` | string | 题目标题 |
| `[].type` | int8 | 题目类型: 1=单选, 2=多选, 3=判断, 4=填空, 5=简答 |
| `[].difficulty` | int8 | 难度: 1=简单, 2=中等, 3=困难 |

#### POST `/api/v1/statistics/events`

普通用户上报行为事件数据，用于用户行为分析。系统自动记录用户 ID 和客户端 IP。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `event_type` | string | 是 | 事件类型，最多 50 字符（如 `click`、`page_view`） |
| `event_name` | string | 是 | 事件名称，最多 100 字符 |
| `page` | string | 否 | 页面路径，最多 200 字符 |
| `element` | string | 否 | 触发元素，最多 200 字符 |
| `extra_data` | string | 否 | 扩展数据（JSON 字符串） |
| `session_id` | string | 否 | 会话 ID，最多 64 字符 |
| `device_type` | string | 否 | 设备类型，最多 20 字符（如 `web`、`ios`、`android`） |

**请求示例**

```json
{
  "event_type": "click",
  "event_name": "开始练习按钮点击",
  "page": "/practice/start",
  "element": "button#start-practice",
  "extra_data": "{\"category_id\": 5}",
  "session_id": "sess_abc123",
  "device_type": "web"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "上报成功"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 参数错误（event_type 或 event_name 未填，或字段超过长度限制） |
| `A0200` | 401 | 未授权（未提供 Token 或 Token 无效） |
| `A0230` | 401 | 访问令牌无效或过期 |
| `B0001` | 500 | 上报失败 |

# 13 移动端 API

本文档覆盖移动端 (UniApp H5/小程序) 常用的全部用户端接口，包括个人统计、排行榜、打卡、练习计划、班级学习计划、考试、错题本、推荐与分类等。

**认证**：所有接口均需 JWT Bearer Token，请求头携带：

```
Authorization: Bearer <access_token>
```

---

## 移动端统计

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/statistics/mobile/overview` | 获取移动端个人学习概览 |

#### GET `/api/v1/statistics/mobile/overview`

获取当前登录用户的个人学习概览数据，包括总练习数、总题量、平均正确率、考试统计、学习天数、排名等。

**认证**: 需要 JWT Bearer Token

**请求参数**

无。

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "total_practice": 128,
    "total_questions": 1560,
    "avg_accuracy": 78.5,
    "total_exams": 12,
    "avg_score": 82.3,
    "study_days": 45,
    "study_duration": 32400,
    "rank": 15
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `total_practice` | int | 总练习次数 |
| `total_questions` | int | 总练习题量 |
| `avg_accuracy` | float | 平均正确率（百分比） |
| `total_exams` | int | 参加考试次数 |
| `avg_score` | float | 考试平均分 |
| `study_days` | int | 累计学习天数 |
| `study_duration` | int | 累计学习时长（秒） |
| `rank` | int | 排名 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0200` | 401 | 未认证或 Token 无效 |
| `B0001` | 500 | 服务器内部错误 |

```
GET /api/v1/statistics/mobile/practice?days=7
```

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "today_count": 25,
    "week_count": 150,
    "month_count": 520,
    "today_accuracy": 80.0,
    "week_accuracy": 76.5,

        "date": "2026-06-01",
        "sessions": 3,
        "questions": 30,

}
```

**响应参数**

| 字段 | 类型 | 说明 |

| `today_count` | int | 今日练习题数 |
| `week_count` | int | 本周练习题数 |
| `month_count` | int | 本月练习题数 |
| `today_accuracy` | float | 今日正确率（百分比） |

#### POST `/api/v1/practice/checkin`

用户进行每日练习打卡，记录当日练习量和学习时长，获取打卡奖励和连续天数。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `question_count` | int | 否 | 当日练习题数 |
| `duration` | int | 否 | 当日学习时长（秒） |

**请求示例**

```json
{
  "question_count": 25,
  "duration": 1800
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "id": 101,
    "user_id": 1,
    "date": "2026-06-02",
    "is_checkin": true,
    "question_count": 25,
    "duration": 1800,
    "streak": 7,
    "reward": 10
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 打卡记录 ID |
| `user_id` | int | 用户 ID |
| `date` | string | 打卡日期 (YYYY-MM-DD) |
| `is_checkin` | boolean | 是否已打卡 |
| `question_count` | int | 练习题数 |
| `duration` | int | 学习时长（秒） |
| `streak` | int | 连续打卡天数 |
| `reward` | int | 打卡奖励积分 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0200` | 401 | 未认证或 Token 无效 |
| `A0001` | 400 | 今日已打卡（重复打卡） |
| `B0001` | 500 | 服务器内部错误 |

---

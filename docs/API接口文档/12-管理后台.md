# 12 管理后台 API

## 概述

管理后台模块提供操作日志和登录日志的查询、导出、清理功能。所有接口均需要管理员权限。

**基础路径**: `/api/v1/admin`

**认证方式**: 所有接口均需要 JWT Bearer Token 且用户具有 `admin` 角色。

**请求头**:
```
Authorization: Bearer <access_token>
```

**通用响应结构**:

所有接口统一返回 JSON 信封格式：

```json
{
  "code": "00000",
  "msg": "success",
  "data": {}
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `code` | string | 响应码，`"00000"` 表示成功 |
| `msg` | string | 响应消息 |
| `data` | object/array/null | 响应数据，具体结构因接口而异 |

**分页响应结构**:

分页接口在 `data` 中返回：

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [],
    "total": 0,
    "page": 1,
    "page_size": 10
  }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.list` | array | 当前页数据 |
| `data.total` | int | 总记录数 |
| `data.page` | int | 当前页码 |
| `data.page_size` | int | 每页条数 |

## 操作日志接口

### GET `/api/v1/admin/operation-logs`

获取操作日志列表，支持分页和多条件筛选。按 ID 降序排列。

**认证**: 需要 JWT Bearer Token + Admin 角色

**请求参数（Query）**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码，从 1 开始 |
| `page_size` | int | 否 | 10 | 每页条数，最大 100 |
| `user_id` | uint | 否 | - | 按用户 ID 筛选（精确匹配） |
| `module` | string | 否 | - | 按模块筛选，如 `user`、`question`、`exam` 等（精确匹配） |
| `action` | string | 否 | - | 按操作类型筛选，如 `create`、`update`、`delete` 等（精确匹配） |
| `status` | int8 | 否 | - | 按状态筛选: `1`=成功, `0`=失败 |
| `ip` | string | 否 | - | 按 IP 地址筛选（精确匹配） |
| `start_time` | string | 否 | - | 开始时间 (ISO 8601 格式, 如 `2026-01-01T00:00:00Z`)，筛选 `created_at >= start_time` |
| `end_time` | string | 否 | - | 结束时间 (ISO 8601 格式, 如 `2026-06-01T23:59:59Z`)，筛选 `created_at <= end_time` |

**请求示例**

```
GET /api/v1/admin/operation-logs?page=1&page_size=20&module=question&action=delete&status=1
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1001,
        "user_id": 1,
        "username": "admin",
        "module": "question",
        "action": "delete",
        "resource": "question",
        "resource_id": "42",
        "description": "删除题目「Go语言基础测试」",
        "ip": "192.168.1.100",
        "user_agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36",
        "status": 1,
        "error_msg": "",
        "created_at": "2026-06-01T14:30:00Z"
      }
    ],
    "total": 156,
    "page": 1,
    "page_size": 20
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.list` | array | 当前页操作日志列表 |
| `data.list[].id` | uint | 日志 ID |
| `data.list[].user_id` | uint | 操作用户 ID |
| `data.list[].username` | string | 操作用户名，最大 50 字符 |
| `data.list[].module` | string | 操作模块，如 `user`、`question`、`exam`、`class` 等，最大 50 字符 |
| `data.list[].action` | string | 操作类型，如 `create`、`update`、`delete` 等，最大 50 字符 |
| `data.list[].resource` | string | 操作资源类型，最大 50 字符 |
| `data.list[].resource_id` | string | 操作资源 ID，最大 50 字符 |
| `data.list[].description` | string | 操作描述，最大 200 字符 |
| `data.list[].ip` | string | 操作者 IP 地址，最大 50 字符 |
| `data.list[].user_agent` | string | 操作者浏览器 User-Agent，最大 500 字符 |
| `data.list[].status` | int8 | 操作状态: `1`=成功, `0`=失败 |
| `data.list[].error_msg` | string | 错误信息（失败时），最大 500 字符 |
| `data.list[].created_at` | string | 操作时间 (ISO 8601 格式) |
| `data.total` | int | 总记录数 |
| `data.page` | int | 当前页码 |
| `data.page_size` | int | 每页条数 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误（参数校验失败） |
| `A0200` | 401 | 未认证或 Token 无效（中间件拦截） |
| `A0301` | 403 | 无管理员权限（中间件拦截） |
| `A0001` | 500 | 服务器内部错误（数据库查询失败） |

### GET `/api/v1/admin/operation-logs/export`

根据条件导出操作日志为 CSV 文件。不分页，最多导出 10000 条记录。CSV 文件包含 UTF-8 BOM 头，可直接用 Excel 打开且中文不乱码。

**认证**: 需要 JWT Bearer Token + Admin 角色

**请求参数（Query）**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `user_id` | uint | 否 | 按用户 ID 筛选（精确匹配） |
| `module` | string | 否 | 按模块筛选（精确匹配） |
| `action` | string | 否 | 按操作类型筛选（精确匹配） |
| `status` | int8 | 否 | 按状态筛选: `1`=成功, `0`=失败 |
| `ip` | string | 否 | 按 IP 地址筛选（精确匹配） |
| `start_time` | string | 否 | 开始时间 (ISO 8601 格式) |
| `end_time` | string | 否 | 结束时间 (ISO 8601 格式) |

**请求示例**

```
GET /api/v1/admin/operation-logs/export?start_time=2026-05-01T00:00:00Z&end_time=2026-06-01T00:00:00Z
```

**响应**

HTTP 状态码 200，返回 CSV 文件流。

响应头：
```
Content-Type: text/csv; charset=utf-8
Content-Disposition: attachment; filename=operation_logs_20260501.csv
```

文件名格式为 `operation_logs_{start_time日期}.csv`。若未指定 `start_time`，则文件名中的日期部分为空。

CSV 表头及字段说明：

| 列名 | 说明 | 对应字段 |
|------|------|----------|
| ID | 日志 ID | `id` |
| 用户ID | 操作用户 ID | `user_id` |
| 用户名 | 操作用户名 | `username` |
| 模块 | 操作模块 | `module` |
| 操作 | 操作类型 | `action` |
| 资源 | 操作资源类型 | `resource` |
| 描述 | 操作描述 | `description` |
| IP | 操作者 IP | `ip` |
| 状态 | 操作状态（中文：成功/失败） | `status` |
| 时间 | 操作时间 (YYYY-MM-DD HH:MM:SS) | `created_at` |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误（参数校验失败） |
| `A0200` | 401 | 未认证或 Token 无效（中间件拦截） |
| `A0301` | 403 | 无管理员权限（中间件拦截） |
| `A0001` | 500 | 服务器内部错误（数据库查询或 CSV 生成失败） |

**请求示例**


## 数据库模型参考

以下为数据库层面的完整字段定义，API 响应仅暴露 DTO 中定义的字段。

### OperationLog 表 (operation_logs)

| 字段 | 类型 | 索引 | 说明 |
|------|------|------|------|
| `id` | uint | 主键 | 日志 ID |
| `user_id` | uint | 索引 | 操作用户 ID |
| `username` | varchar(50) | - | 操作用户名 |
| `module` | varchar(50) | - | 操作模块 |
| `action` | varchar(50) | NOT NULL | 操作类型 |
| `resource` | varchar(50) | - | 操作资源类型 |
| `resource_id` | varchar(50) | 索引 | 操作资源 ID |
| `description` | varchar(200) | - | 操作描述 |
| `method` | varchar(10) | - | HTTP 请求方法（数据库存储，API 不返回） |
| `path` | varchar(500) | - | 请求路径（数据库存储，API 不返回） |
| `ip` | varchar(50) | - | 操作者 IP 地址 |
| `user_agent` | varchar(500) | - | 浏览器 User-Agent |
| `status` | tinyint | - | 状态: 0=失败, 1=成功，默认 1 |
| `error_msg` | varchar(500) | - | 错误信息 |
| `latency` | bigint | - | 请求耗时（毫秒，数据库存储，API 不返回） |
| `created_at` | datetime | - | 创建时间 |

### LoginLog 表 (login_logs)

| 字段 | 类型 | 索引 | 说明 |
|------|------|------|------|
| `id` | uint | 主键 | 日志 ID |
| `user_id` | uint | 索引, 可 NULL | 用户 ID（登录失败时可能为空） |
| `username` | varchar(50) | - | 用户名 |
| `ip` | varchar(50) | - | 登录 IP 地址 |
| `location` | varchar(100) | - | IP 归属地 |
| `user_agent` | varchar(500) | - | 浏览器 User-Agent（数据库存储，API 不返回） |
| `device_type` | varchar(20) | - | 设备类型: web/ios/android/miniapp（数据库存储，API 不返回） |
| `device_name` | varchar(100) | - | 设备名称（数据库存储，API 不返回） |
| `browser` | varchar(50) | - | 浏览器信息 |
| `os` | varchar(50) | - | 操作系统 |
| `status` | tinyint | - | 状态: 0=失败, 1=成功，默认 1 |
| `msg` | varchar(200) | - | 登录结果消息 |
| `login_type` | varchar(20) | - | 登录类型: password/sms/oauth，默认 password（数据库存储，API 不返回） |
| `is_new_device` | boolean | - | 是否新设备，默认 false（数据库存储，API 不返回） |
| `created_at` | datetime | - | 创建时间 |

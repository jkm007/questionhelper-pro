# 06 刷题练习 API

## 用户端接口 (需登录)

### 基本练习

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/practice` | 练习历史 |
| GET | `/api/v1/practice/stats` | 练习统计 |
| GET | `/api/v1/practice/history` | 获取练习历史记录 |

---

#### GET `/api/v1/practice`

分页获取当前用户的练习历史记录。

**认证**: 需要 JWT Bearer Token

**查询参数**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `page` | int | 否 | 1 | 页码 |
| `page_size` | int | 否 | 10 | 每页数量 |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "user_id": 10,
        "category_id": 1,
        "total_count": 20,
        "correct_count": 15,
        "accuracy": 0.75,
        "duration": 1200,
        "status": 1,
        "created_at": "2026-06-01T12:00:00Z"
      }
    ],
    "total": 50,
    "page": 1,
    "page_size": 10
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `list[].id` | int | 练习会话 ID |
| `list[].user_id` | int | 用户 ID |
| `list[].category_id` | int | 分类 ID |
| `list[].total_count` | int | 总题数 |
| `list[].correct_count` | int | 正确题数 |
| `list[].accuracy` | float | 正确率 |
| `list[].duration` | int | 练习时长（秒） |
| `list[].status` | int | 状态: 0=进行中, 1=已完成 |
| `list[].created_at` | string | 创建时间 |

#### GET `/api/v1/practice/mock/:id/detail`

根据 ID 获取模拟考试详细信息，包含答题记录。

**认证**: 需要 JWT Bearer Token

**路径参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | uint | 是 | 模拟考试 ID |

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "session": {
      "id": 1,
      "user_id": 10,
      "config_id": 1,
      "config_name": "每日模拟",
      "category_id": 1,
      "total_count": 50,
      "total_score": 100,
      "pass_score": 60,
      "duration": 3600,
      "status": 1,
      "started_at": "2026-06-01T10:00:00Z",
      "submitted_at": "2026-06-01T11:00:00Z"
    },
    "records": [
      {
        "id": 1,
        "question_id": 101,
        "answer": "A",
        "correct": "A",
        "is_correct": true,
        "score": 2.0,
        "duration": 30
      }
    ],
    "score": 85.5,
    "is_passed": true,
    "rank": 5
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `session` | object | 考试会话信息 |
| `records` | array | 答题记录列表 |
| `records[].question_id` | int | 题目 ID |
| `records[].answer` | string | 用户答案 |
| `records[].correct` | string | 正确答案 |
| `records[].is_correct` | boolean | 是否正确 |
| `records[].score` | float | 得分 |
| `records[].duration` | int | 答题用时（秒） |
| `score` | float | 总得分 |
| `is_passed` | boolean | 是否及格 |
| `rank` | int | 排名 |

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 无效的模拟考试 ID |
| `B0001` | 500 | 查询失败 |

#### POST `/api/v1/practice/plans`

创建一个新的练习计划。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | string | 是 | 计划名称，最大 200 字符 |
| `description` | string | 否 | 计划描述，最大 500 字符 |
| `plan_type` | int | 是 | 计划类型: 1=按题量, 2=按时长, 3=混合 |
| `category_id` | uint | 否 | 分类 ID |
| `question_type` | int | 否 | 题目类型: 1-5 |
| `difficulty` | int | 否 | 难度: 1-3 |
| `daily_count` | int | 否 | 每日题量，1-200 |
| `daily_duration` | int | 否 | 每日时长（分钟），1-480 |
| `start_date` | string | 是 | 开始日期 (YYYY-MM-DD) |
| `end_date` | string | 否 | 结束日期 |
| `total_target` | int | 否 | 总目标题量 |

**请求示例**

```json
{
  "name": "每日刷题计划",
  "description": "每天刷20道题",
  "plan_type": 1,
  "category_id": 1,
  "daily_count": 20,
  "start_date": "2026-06-01",
  "end_date": "2026-07-01",
  "total_target": 600
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "id": 1,
    "name": "每日刷题计划",
    "plan_type": 1,
    "status": 1
  }
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `B0001` | 500 | 创建失败 |

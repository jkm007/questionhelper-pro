# 02 用户管理 API

## 用户端接口 (需登录)

### 用户资料

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/user/profile` | 获取当前用户资料 |
| GET | `/api/v1/users/me` | 获取当前用户资料 (兼容路径) |
| PUT | `/api/v1/user/profile` | 更新个人资料 |
| POST | `/api/v1/user/avatar` | 上传头像 |

---

#### GET `/api/v1/user/profile`

获取当前登录用户的个人资料信息。

**认证**: 需要 JWT Bearer Token

**响应示例**

```json
{
  "code": "00000",
  "msg": "success",
  "data": {
    "id": 1,
    "username": "student01",
    "nickname": "学生01",
    "email": "student@example.com",
    "phone": "13800138000",
    "avatar": "https://...",
    "gender": 1,
    "birthday": "2000-01-01T00:00:00Z",
    "bio": "热爱学习",
    "status": 1,
    "is_real": true,
    "roles": [
      {"id": 1, "name": "学生", "code": "student", "description": "普通学生"}
    ],
    "created_at": "2026-01-01T00:00:00Z"
  }
}
```

**响应参数**

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 用户 ID |
| `username` | string | 用户名 |
| `nickname` | string | 昵称 |
| `email` | string | 邮箱 |
| `phone` | string | 手机号 |
| `avatar` | string | 头像 URL |
| `gender` | int | 性别: 0=未知, 1=男, 2=女 |
| `birthday` | string | 生日 (ISO 8601) |
| `bio` | string | 个人简介 |
| `status` | int | 状态: 0=禁用, 1=正常, 2=注销中 |
| `is_real` | boolean | 是否已实名认证 |
| `roles` | array | 角色列表 |
| `roles[].id` | int | 角色 ID |
| `roles[].name` | string | 角色名称 |
| `roles[].code` | string | 角色编码 |
| `roles[].description` | string | 角色描述 |
| `created_at` | string | 注册时间 |

---

#### PUT `/api/v1/user/profile`

更新当前用户的个人资料。

**认证**: 需要 JWT Bearer Token

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `nickname` | string | 否 | 昵称，最大 50 字符 |
| `email` | string | 否 | 邮箱 |
| `gender` | int | 否 | 性别: 0=未知, 1=男, 2=女 |
| `birthday` | string | 否 | 生日 (ISO 8601) |
| `bio` | string | 否 | 个人简介，最大 500 字符 |

**请求示例**

```json
{
  "nickname": "新昵称",
  "gender": 1,
  "bio": "热爱编程和学习"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "更新成功"
}
```

**错误代码**

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `B0001` | 500 | 更新失败 |

---

#### POST `/api/v1/user/avatar`

上传并更新用户头像。

**认证**: 需要 JWT Bearer Token

**请求格式**: `multipart/form-data`

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file` | file | 是 | 头像文件（支持 jpg/png/gif，建议 200x200） |

**响应示例**

```json
{
  "code": "00000",
  "msg": "上传成功",
  "data": {
    "avatar": "https://..."
  }
}
```

#### POST `/api/v1/user/bind-phone`

**请求参数**

| 参数 | 类型 | 必填 | 说明 |

| `phone` | string | 是 | 手机号 |
| `code` | string | 是 | 短信验证码，6 位 |

**请求示例**

```json
{
  "phone": "13800138000",
  "code": "123456"
}
```

**响应示例**

```json
{
  "code": "00000",
  "msg": "绑定成功"
}
```

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `A0001` | 400 | 请求参数错误 |
| `A0202` | 409 | 手机号已被其他用户绑定 |
| `B0001` | 500 | 绑定失败 |


## 全局设计规范

### 技术栈

- **框架**：UniApp（Vue 3 + TypeScript，`<script setup>` 语法）
- **单位**：`rpx`（响应式像素，750rpx = 屏幕宽度）
- **状态管理**：Pinia（user / exam / practice 三个 store）
- **HTTP**：封装 `uni.request`，Bearer Token 认证，Token 过期自动跳转登录

### 两套主色方案

| 模块 | 主色 | 渐变 | 页面背景 |
|------|------|------|---------|
| 首页 / 登录 / 题库 / 班级 / 个人中心 / 错题 / 评论 / 通知 | `#4A90D9` 或 `#1677ff` | 无 | `#f5f6fa` 或 `#F5F7FA` |
| 练习 / 考试 | `#667eea` | `#667eea` → `#764ba2` | `#f5f5f5` |

### 通用组件

| 组件 | 用途 | 实际使用情况 |
|------|------|-------------|
| `QuestionCard` | 题目卡片（类型标签 + 难度徽章 + 标题 + 分类 + 浏览/点赞数） | 首页、题库列表、搜索结果 |
| `ExamCard` | 考试卡片（标题 + 状态 + 时间 + 人数 + 得分） | 首页热门考试 |
| `Empty` | 空状态占位（图标 + 文字 + 可选按钮） | 题库列表等 |
| `SearchBar` | 搜索栏（支持 readonly 跳转模式） | 已实现但部分页面使用内联搜索栏 |
| `ClassCard` | 班级卡片（封面 + 名称 + 创建者 + 人数） | 已实现但部分页面使用内联卡片 |
| `AnswerSheet` | 答题卡弹窗（已答/未答/标记统计 + 题号网格） | 已实现但考试答题页使用内联实现 |
| `OptionsList` | 选项列表（单选/多选 + 正确/错误高亮） | 已实现但答题页使用内联实现 |
| `Timer` | 计时器（格式化显示 + 紧急状态变色） | 已实现但答题页使用内联实现 |
| `ScoreChart` | 分数展示（数值 + 进度条 + 颜色分级） | 已实现 |
| `LoadMore` | 加载更多（加载中 / 已完成 / 点击加载） | 已实现但列表页使用内联文字 |

### 通用交互模式

- **下拉刷新 + 上拉加载**：`<scroll-view>` 的 `refresher-enabled` + `@scrolltolower`
- **确认弹窗**：`uni.showModal` 用于删除、退出等危险操作
- **导航**：`uni.navigateTo`（带查询参数）、`uni.switchTab`（Tab 页切换）
- **底部安全区**：`padding-bottom: env(safe-area-inset-bottom)`
- **空状态**：列表为空时显示图片/表情 + 描述文字
- **状态徽章**：小型彩色药丸标签，标识状态（进行中/已完成/待审核等）

### 语言说明

- **中文页面**：首页、登录注册、题库、班级、个人中心、错题本、评论、通知
- **英文页面**：练习（practice）、考试（exam）、反馈（feedback）、统计（statistics）模块的 UI 文字为英文

## 1. 首页（1页）

### 1.1 `pages/index/index` — 首页

**导航栏**：自定义导航栏，蓝色背景（`#4A90D9`），左侧标题"题小助"，右侧通知图标带未读角标。

**搜索栏**：圆角搜索框，点击跳转搜索页（readonly 模式）。

**快捷入口**：4 宫格，每个入口包含表情图标 + 文字：
- 📝 刷题练习 → `switchTab` 到练习页
- 📋 考试中心 → `switchTab` 到考试页
- 👥 班级学习 → `switchTab` 到班级页
- ❌ 我的错题 → `navigateTo` 到错题列表

**推荐题目**：标题"推荐题目" + 右侧"更多"链接，使用 `QuestionCard` 组件展示 5 道题。

**热门考试**：标题"热门考试" + 右侧"更多"链接，使用 `ExamCard` 组件展示 3 场考试。

**数据加载**：`onMounted` 时并行调用 `getQuestions`、`getExams`、`getUnreadCount`。

## 2. 登录注册（3页）

### 2.1 `pages/login/index` — 登录

**Logo 区**：应用图标（160rpx）+ 应用名"题小助"（`#4A90D9`）+ 标语"学习好帮手"。

**登录表单**：
- 用户名/手机号输入框（`#F5F7FA` 背景，16rpx 圆角）
- 密码输入框（`:password="true"`，始终隐藏，无显示/隐藏切换）
- 登录按钮（`#4A90D9` 全宽圆角按钮）

**底部链接**：注册 / 忘记密码，分隔线 + 微信登录圆形按钮（`#07C160` 背景，文字"微信"）。

**逻辑**：调用 `useUserStore().login()`，成功后 `switchTab` 到首页。

### 2.2 `pages/login/register` — 注册

**表单字段**：
- 昵称输入框
- 手机号输入框
- 短信验证码输入框 + 发送按钮（60 秒倒计时）
- 密码（带 emoji 图标显示/隐藏切换 🙈/👁️）
- 确认密码（带 emoji 图标显示/隐藏切换 🙈/👁️）
- 用户协议和隐私政策复选框

**注册按钮**：`#4A90D9` 全宽按钮，底部"已有账号？去登录"链接。

**验证**：密码最少 6 位，两次密码必须一致，必须勾选协议。

### 2.3 `pages/login/forget` — 忘记密码

**步骤指示器**：3 步横向流程（验证手机 → 重置密码 → 完成），圆点 + 连接线 + 文字。激活=`#4A90D9`，完成=`#67C23A`。

**步骤 1**：手机号 + 短信验证码（60 秒倒计时）。
**步骤 2**：新密码 + 确认密码（带显示/隐藏切换）。
**步骤 3**：成功图标（绿色圆 + 对勾）+ "去登录"按钮。

## 3. 题库（4页）

### 3.1 `pages/question/list` — 题库列表（Tab 页）

**搜索栏**：点击跳转搜索页（readonly 模式，内联实现，未使用 `SearchBar` 组件）。

**筛选栏**：横向滚动，药丸按钮：
- 全部 / 单选题 / 多选题 / 判断题（类型筛选）
- 分类（点击打开右侧分类抽屉）

**题目列表**：使用 `QuestionCard` 组件，支持下拉刷新和上拉加载。加载更多使用内联文字（"加载中..." / "没有更多了" / "上拉加载更多"）。

**分类抽屉**：右侧滑入（600rpx 宽），半透明遮罩（`rgba(0,0,0,0.5)`），树形结构（父分类 + 子分类），点击筛选。

### 3.2 `pages/question/detail` — 题目详情

**加载状态**：加载中显示"加载中..."文字（非骨架屏）。

**题目信息**：
- 类型标签（单选=蓝、多选=橙、判断=绿、填空=紫、简答=红）
- 难度徽章（简单=绿、中等=黄、困难=红）
- 标题、分类、浏览数、点赞数

**题目内容**：富文本展示。

**选项列表**：可选择的选项（单选/多选/判断题），选中高亮。

**折叠区域**：
- 答案（点击展开/收起）
- 解析（点击展开/收起）

**知识点标签**：药丸形标签（`#F5F7FA` 背景）。

**创建者信息**：头像 + 昵称。

**底部操作栏**：5 个操作按钮（表情图标 40rpx）：
- ⭐/☆ 收藏 / ❤️/🤍 点赞 / 📤 分享 / ⚠️ 纠错（跳转纠错反馈页） / 📝 错题本

### 3.3 `pages/question/search` — 搜索

**搜索头部**：输入框（自动聚焦）+ 清除按钮 + 取消链接（`#4A90D9`）。使用内联搜索栏实现。

**搜索前状态**：
- 搜索历史：药丸标签（localStorage 存储，最多 20 条）
- 热门搜索：编号列表，前 3 名红色（`#F56C6C`），热门标识基于 `item.hot` 字段

**搜索后状态**：使用 `QuestionCard` 展示结果，空状态提示。

### 3.4 `pages/question/edit` — 编辑题目

**基本信息区**：
- 题型选择：药丸按钮组（单选/多选/判断/填空/简答）
- 难度选择：药丸按钮组（简单/中等/困难）
- 分类选择器：点击弹出底部分类选择弹窗

**题目内容区**：
- 标题输入框
- 内容文本域

**选项区**（选择题类型）：
- 动态添加/删除选项
- 选项标签（A-H，48rpx 圆形，`#4A90D9` 背景）
- 正确答案切换按钮（单选=单选，多选=多选）

**答案与解析**：文本域输入。

**知识点标签**：可添加/删除的知识点药丸标签，点击弹出多选弹窗。

**提交按钮**：页面内提交按钮（非底部固定）。

## 4. 练习（3页）

> ⚠️ 练习模块 UI 文字为英文

### 4.1 `pages/practice/index` — 练习首页（Tab 页）

**渐变头部**：`#667eea` → `#764ba2`，显示"Practice"标题 + "Improve your skills"副标题。

**今日统计卡片**：浮在头部下方（负边距重叠），显示 Questions（刷题数）、Accuracy（正确率）、Duration（学习时长）。

**快捷开始卡片**：渐变背景，▶ 图标 + "Quick Start" + "Start practice with recommended settings"。

**练习模式网格**：2 列布局，7 种模式卡片：

| 模式 | 图标 | 描述 | 背景色 |
|------|------|------|--------|
| Random | 🔀 | Random questions | `#E3F2FD` |
| Sequential | 📋 | In order | `#E8F5E9` |
| Wrong Questions | ❌ | From mistakes | `#FFF3E0` |
| Favorites | ⭐ | Saved questions | `#FCE4EC` |
| Mock Exam | 📝 | Full simulation | `#E8EAF6` |
| Challenge | 🏆 | Gamified practice | `#F3E5F5` |
| Timed | ⏱ | Speed practice | `#E0F7FA` |

**最近记录**：展示最近的练习记录列表。

**API 调用**：`getTodayStats`、`getRecentRecords`。

### 4.2 `pages/practice/session` — 练习答题

**自定义导航栏**：返回按钮 + 模式名称 + 收藏切换（★/☆）。

**计时栏**：计时器 + 进度条 + 题号计数（如 "3/20"）。

**题目区域**：Swiper 横向滑动切换题目，每题包含：
- 题号 + 题型标签
- 题目内容
- 选项列表（选中=`#f0f2ff`，正确=`#e8f5e9`，错误=`#ffebee`）
- Explanation 卡片（`#fff8e1` 背景，`#ff9800` 左边框，点击"Show Answer"后出现）

**底部操作栏**：
- Previous（灰色）
- Show Answer / Hide Answer（橙色）
- Next / Submit All（渐变紫色）

### 4.3 `pages/practice/result` — 练习结果

**渐变分数区**：渐变背景，动画分数圆环（CSS 旋转），Accuracy 标签 + 励志文案：
- ≥90%：Excellent!
- ≥80%：Great job!
- ≥70%：Good work!
- ≥60%：Not bad, keep practicing!
- <60%：Keep trying, you will improve!

**统计行**：Total / Correct / Wrong / Duration。

**题目详情**：Question Details，可展开的逐题列表，每题显示正确/错误指示器、选项高亮、Explanation。

**底部栏**：Retry / Back to Practice。

## 5. 考试（5页）

> ⚠️ 考试模块 UI 文字为英文

### 5.1 `pages/exam/list` — 考试列表（Tab 页）

**渐变头部**：`#667eea` → `#764ba2`，标题"Exams" + 副标题"Manage your examinations"。

**状态标签栏**：3 个标签（Upcoming / In Progress / Completed），激活标签下方有 `60rpx` 宽的 `#667eea` 指示条。

**考试卡片**：白色圆角卡片（20rpx），内联实现（未使用 `ExamCard` 组件），包含：
- 标题 + 状态徽章（蓝=Upcoming、绿=In Progress、灰=Completed）
- 📅 日期范围 / ⏱ Duration / 👥 participants
- 底部：倒计时（Upcoming）/ "In Progress"（进行中）/ Your Score（Completed）

**交互**：下拉刷新 + 上拉加载，点击卡片跳转考试详情。

### 5.2 `pages/exam/detail` — 考试详情

**渐变头部**：`#667eea` → `#764ba2`，返回按钮 + "Exam Details"标题。

**信息卡片**：浮在头部下方（负边距），阴影 `0 4rpx 20rpx rgba(0,0,0,0.08)`，包含：
- 标题、状态、描述
- 信息网格：📅 时间 / ⏱ Duration / 📊 Total Score / 👥 Participants

**规则区域**：考试规则说明。

**参与人员**：头像列表。

**历史记录**：已完成的考试尝试列表（分数、Pass/Fail 徽章）。

**底部按钮**：Enter Exam（渐变按钮，未开始时禁用）。

### 5.3 `pages/exam/answer` — 考试答题

**自定义导航栏**：返回按钮 + 倒计时（红色 `#f44336`，`HH:MM:SS` 格式）+ 题号网格按钮。

**题号网格抽屉**：右侧滑入（60% 宽度），颜色编码：
- 当前题：`#667eea` 边框 + `#e8eaf6` 背景
- 已答题：`#4caf50` 边框 + `#e8f5e9` 背景
- 标记题：`#ff9800` 边框 + `#fff3e0` 背景
- 底部图例：Current / Answered / Flagged

**题目区域**：Swiper 横向滑动，每题包含：
- 题号 + 分值 + 标记按钮（⚑/🚩）
- 题目内容
- 选项列表（单选/多选，选中=`#f0f2ff`）

**底部答题卡**：
- 进度条（渐变填充 `#667eea` → `#764ba2`）
- "X/Y answered" 文字
- Prev / Submit / Next 按钮

**自动保存**：每 30 秒自动保存进度（`saveExamProgress` API）。
**时间到自动提交**：倒计时归零时自动提交。

### 5.4 `pages/exam/result` — 考试结果

**分数区**：Pass=绿渐变（`#4caf50` → `#81c784`），Fail=红渐变（`#f44336` → `#e57373`）。动画分数圆圈（1500ms 计数动画），Pass/Fail 图标和文字。

**统计行**：Score / Rank / Correct Rate / Duration。

**题目明细**：可展开的逐题列表，显示 Your Answer vs Correct Answer，选项高亮，Explanation 卡片（`#fff8e1` + `#ff9800` 左边框）。

**底部栏**：View Answers / Back。

### 5.5 `pages/exam/history` — 考试历史

**渐变头部**：`#667eea` → `#764ba2`。

**历史卡片**：每张卡片包含：
- 标题 + Pass/Fail 徽章
- 大号分数显示（48rpx 粗体，`#667eea`）
- 分数进度条（Pass=绿渐变，Fail=红渐变）
- 底部：📅 日期 / ⏱ Duration / 🔄 Attempt #N

## 6. 错题本（3页）

### 6.1 `pages/wrong/list` — 错题列表

**搜索栏**：输入框 + 筛选按钮。

**活跃筛选标签**：可移除的药丸标签，显示当前筛选条件。

**批量操作栏**：全选复选框 + 已选数量 + 移除按钮（`#ff4d4f`）。

**错题卡片**：每张卡片包含：
- 复选框（批量模式）
- 标题 + 类型标签（颜色编码）
- 错误次数（红色）
- 下次复习时间
- 掌握度进度条（红=`#ff4d4f` / 黄=`#faad14` / 绿=`#52c41a`，阈值：<30 红、<70 黄、≥70 绿）

**底部栏**：批量切换 / 查看分析链接。

**筛选抽屉**：右侧滑入（600rpx），包含：
- 分类选择
- 掌握程度（药丸选项：未掌握/初步掌握/基本掌握）
- 题型选择
- 重置 / 确认按钮

### 6.2 `pages/wrong/detail` — 错题详情

**加载骨架屏**（`skeleton-title` + `skeleton-line`）。

**题目内容**：类型标签 + 题目文字 + 选项（正确/错误高亮）。

**答案对比**：双卡片并排——
- 我的答案：红色卡片（`#fff1f0` 背景，`#ffccc7` 边框）
- 正确答案：绿色卡片（`#f6ffed` 背景，`#b7eb8f` 边框）
- 中间箭头指示

**解析区域**：解析文字 + 知识点药丸标签。

**错题时间线**：圆点 + 日期，展示历史错误记录。

**掌握度指示器**：
- 圆环（120rpx，`#1677ff` 边框，8rpx 宽）显示百分比
- 状态文字（未掌握 / 初步掌握 / 基本掌握 / 已完全掌握）
- 进度条（红/黄/绿）

**学习笔记**：文本域（500 字），失焦时自动保存。

**底部栏**：再做一次 / 标记已掌握。

### 6.3 `pages/wrong/analysis` — 错题分析

**汇总卡片**：3 张卡片——错题总数 / 已掌握数 / 掌握率。

**分类分布**：水平柱状图（5 种颜色循环：`#1677ff` / `#52c41a` / `#faad14` / `#722ed1` / `#ff4d4f`）。

**薄弱知识点 TOP10**：排名列表——
- 排名圆圈（第 1 名=`#ff4d4f` 红、第 2 名=`#fa8c16` 橙、第 3 名=`#faad14` 黄）

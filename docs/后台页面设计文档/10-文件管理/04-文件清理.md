## 清理类型

| 类型 | 说明 | 风险 |
|------|------|------|
| 孤立文件 | 数据库无记录的文件 | 低 |
| 过期文件 | 超过指定天数未访问 | 低 |
| 重复文件 | MD5相同的文件 | 中 |

## 清理日志表格

| 字段 | 说明 |
|------|------|
| 时间 | 清理时间 |
| 类型 | 清理类型 |
| 清理数量 | 清理的文件数量 |
| 释放空间 | 释放的存储空间 |
| 操作人 | 执行清理的用户 |
| 状态 | 成功/失败/部分成功 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 扫描 | 扫描可清理的文件 | `file:cleanup:view` |
| 清理全部 | 清理所有扫描到的文件 | `file:cleanup:clean` |
| 选择性清理 | 勾选后清理选中的文件 | `file:cleanup:clean` |
| 查看日志 | 查看历史清理记录 | `file:cleanup:view` |

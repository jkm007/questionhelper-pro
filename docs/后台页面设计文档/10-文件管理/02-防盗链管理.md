# 防盗链管理

## 基本信息

- 路由路径：`/file/hotlink`
- 权限标识：`file:hotlink:view`
- 页面类型：列表页（CURD）

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 规则名称 | string | 150 | - |
| allowed_domains | 允许域名 | string | 200 | 逗号分隔 |
| intercept_type | 拦截方式 | tag | 100 | 返回403/返回空图/重定向 |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 150 | - |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 规则名称 | input | ✅ | 2-30位 | - |
| allowed_domains | 允许域名 | textarea | ✅ | - | 每行一个域名，支持通配符 |
| intercept_type | 拦截方式 | radio | ✅ | - | 返回403/返回空图/重定向 |
| redirect_url | 重定向URL | input | - | URL格式 | 仅重定向方式需要 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 新增 | 创建防盗链规则 | `file:hotlink:create` |
| 编辑 | 编辑规则信息 | `file:hotlink:edit` |
| 删除 | 删除规则 | `file:hotlink:delete` |
| 批量删除 | 批量删除规则 | `file:hotlink:delete` |

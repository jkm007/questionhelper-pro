# 水印管理

## 基本信息

- 路由路径：`/file/watermark`
- 权限标识：`file:watermark:view`
- 页面类型：列表页 + 配置

## 水印配置项

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| enabled | 水印开关 | switch | - | 是否开启水印 |
| type | 水印类型 | radio | ✅ | 文字/图片 |
| text | 水印文字 | input | - | 仅文字类型 |
| font_size | 字体大小 | number | - | 仅文字类型 |
| font_color | 字体颜色 | color | - | 仅文字类型 |
| rotate | 旋转角度 | number | - | 度数 |
| opacity | 透明度 | slider | - | 0-100% |
| image | 水印图片 | upload | - | 仅图片类型 |
| image_scale | 图片缩放 | number | - | 仅图片类型 |
| position | 水印位置 | checkbox | - | 左上/右上/居中/左下/右下 |
| density | 水印密度 | radio | - | 单个/平铺 |
| gap_x | 水平间距 | number | - | 仅平铺模式 |
| gap_y | 垂直间距 | number | - | 仅平铺模式 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看当前水印配置 | `file:watermark:view` |
| 保存配置 | 保存水印配置 | `file:watermark:edit` |
| 预览 | 预览水印效果 | - |
| 重置 | 恢复默认配置 | `file:watermark:edit` |

- 权限标识：`profile:password`

## 表单字段

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| old_password | 原密码 | input(password) | ✅ | - | 验证当前密码 |
| new_password | 新密码 | input(password) | ✅ | 6-20位，包含字母和数字 | - |
| confirm_password | 确认密码 | input(password) | ✅ | 与新密码一致 | - |
| captcha | 验证码 | input | ✅ | 6位数字 | 短信验证码 |

## 密码强度

| 强度 | 规则 | 颜色 |
|------|------|------|
| 弱 | 纯数字或纯字母 | 红色 |
| 中 | 字母+数字 | 橙色 |
| 强 | 大小写+数字+特殊字符 | 绿色 |

## 功能点

| 功能 | 说明 |
|------|------|
| 发送验证码 | 发送短信验证码，60秒倒计时 |
| 修改密码 | 验证通过后修改密码 |
| 密码强度 | 实时显示密码强度 |

## 交互说明

1. **验证码**：点击发送后60秒倒计时
2. **密码强度**：输入时实时计算并显示
3. **修改成功**：提示"密码修改成功，请重新登录"
4. **重新登录**：修改成功后跳转登录页

# 我的消息

## 基本信息

- 路由路径：`/profile/message`
- 权限标识：`profile:message:view`
- 页面类型：列表页

## 关联接口文档

> 详见 [API接口文档/11-系统设置.md](../API接口文档/11-系统设置.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/user/notifications` | GET | 消息列表 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/user/notifications/:id/read` | PUT | 标记已读 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/user/notifications/read-all` | PUT | 全部已读 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/user/notifications/:id` | DELETE | 删除消息 | [链接](../API接口文档/11-系统设置.md) |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看消息 | 查看消息列表 | `profile:message:view` |
| 消息详情 | 查看消息详细内容 | `profile:message:view` |
| 标记已读 | 标记单条消息为已读 | `profile:message:read` |
| 全部已读 | 标记所有消息为已读 | `profile:message:read` |
| 删除消息 | 删除单条消息 | `profile:message:delete` |
| 批量删除 | 批量删除消息 | `profile:message:delete` |
| 筛选 | 按类型筛选消息 | - |

## 交互说明

1. **未读标识**：未读消息显示蓝色圆点
2. **点击消息**：打开详情弹窗，自动标记已读
3. **全部已读**：二次确认后标记所有消息为已读
4. **删除确认**：删除前二次确认
5. **实时通知**：SSE实时接收新消息

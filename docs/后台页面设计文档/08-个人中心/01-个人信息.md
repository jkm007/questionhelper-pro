# 个人信息

## 基本信息

- 路由路径：`/profile`
- 权限标识：`profile:view`
- 页面类型：表单页

| `/api/v1/user/info` | GET | 获取用户信息 | [链接](../API接口文档/02-用户管理.md) |

## 表单字段

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| avatar | 头像 | upload | - | 图片格式 | 点击更换 |
| username | 用户名 | input | - | - | 只读，不可修改 |
| nickname | 昵称 | input | ✅ | 2-20位 | - |
| phone | 手机号 | input | ✅ | 手机号格式 | 需验证 |
| email | 邮箱 | input | - | 邮箱格式 | 需验证 |
| gender | 性别 | radio | - | - | 男/女/保密 |
| birthday | 生日 | date | - | - | - |
| bio | 个人简介 | textarea | - | 最长200字 | - |

## 账号信息（只读）

| 字段 | 说明 |
|------|------|
| 用户ID | 唯一标识 |
| 角色 | 用户角色 |
| 注册时间 | 注册日期 |
| 最后登录 | 最近一次登录时间和IP |
| 登录次数 | 累计登录次数 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 修改头像 | 上传新头像 | `profile:edit` |
| 修改信息 | 编辑个人信息 | `profile:edit` |
| 修改手机号 | 验证后修改 | `profile:phone` |
| 修改邮箱 | 验证后修改 | `profile:email` |

## 交互说明

1. **头像上传**：支持裁剪，限制图片大小（2MB）
2. **手机号修改**：需验证原手机+新手机
3. **邮箱修改**：需验证新邮箱
4. **保存提示**：保存成功后提示"个人信息已更新"

# 知识点管理

## 基本信息

- 路由路径：`/question/knowledge`
- 权限标识：`knowledge:view`
- 页面类型：树形列表页

## 关联接口文档

> 详见 [API接口文档/03-题库管理.md](../API接口文档/03-题库管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/knowledge/tree` | GET | 知识点树 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/knowledge` | POST | 新增知识点 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/knowledge/:id` | PUT | 编辑知识点 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/knowledge/:id` | DELETE | 删除知识点 | [链接](../API接口文档/03-题库管理.md) |

## 页面布局

与题目分类页面布局相同，区别：
1. 标题：题库管理 > 知识点管理
2. 表格字段增加"关联题目数"

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 知识点名称 | string | 200 | 树形缩进显示 |
| code | 编码 | string | 100 | 知识点编码 |
| sort_order | 排序 | number | 80 | 可拖拽排序 |
| question_count | 关联题目数 | number | 100 | - |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 180 | - |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| parent_id | 上级知识点 | tree-select | - | - | 空则为顶级 |
| name | 知识点名称 | input | ✅ | 2-30位 | 同级唯一 |
| code | 编码 | input | ✅ | 字母数字 | 全局唯一 |
| sort_order | 排序 | number | ✅ | ≥0 | - |
| status | 状态 | switch | ✅ | - | 启用/禁用 |
| remark | 备注 | textarea | - | 最长200字 | - |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 新增顶级知识点 | 添加顶级知识点 | `knowledge:create` |
| 添加子知识点 | 在选中知识点下添加 | `knowledge:create` |
| 编辑 | 修改知识点信息 | `knowledge:edit` |
| 删除 | 删除知识点（需无子知识点和关联题目） | `knowledge:delete` |
| 拖拽排序 | 拖拽调整顺序 | `knowledge:sort` |
| 查看关联题目 | 跳转到题目列表并筛选 | `knowledge:view` |

# 私有题目

## 基本信息

- 路由路径：`/question/private`
- 权限标识：`question:private:view`
- 页面类型：列表页（CURD）

## 关联接口文档

> 详见 [API接口文档/03-题库管理.md](../API接口文档/03-题库管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/questions` | GET | 题目列表（type=private） | [链接](../API接口文档/03-题库管理.md) |
| 其他接口同公共题目 | - | - | - |

## 页面布局

与公共题目页面布局相同，区别：
1. 数据范围：仅显示当前用户创建的私有题目
2. 权限标识：`question:private:*`
3. 标题：题库管理 > 私有题目

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `question:private:view` |
| 新增 | 新增私有题目 | `question:private:create` |
| 编辑 | 编辑私有题目 | `question:private:edit` |
| 删除 | 删除私有题目 | `question:private:delete` |
| 发布到公共 | 将私有题目发布为公共题目 | `question:public:create` |
| 其他 | 同公共题目 | - |

## 特殊说明

- 私有题目仅创建者可见
- 发布到公共后，原私有题目仍保留
- 教师/创作者角色使用

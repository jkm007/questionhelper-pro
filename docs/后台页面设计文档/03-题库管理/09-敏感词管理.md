# 敏感词管理

## 基本信息

- 路由路径：`/question/sensitive`
- 权限标识：`question:sensitive:view`
- 页面类型：列表页（CURD）

|------|------|------|----------|
| `/api/v1/admin/sensitive-words` | GET | 敏感词列表 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/sensitive-words` | POST | 添加敏感词 | [链接](../API接口文档/03-题库管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索敏感词 |
| category | 分类 | select | 全部/违禁词/广告词/涉政词/其他 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| word | 敏感词 | string | 150 | 脱敏显示 |
| category | 分类 | tag | 80 | 违禁词/广告词/涉政词/其他 |
| replacement | 替换词 | string | 100 | 替换内容 |
| use_count | 使用次数 | number | 80 | 被检测到的次数 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 120 | - |

## 表单字段（添加敏感词弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| word | 敏感词 | input | ✅ | 1-20字 | - |
| category | 分类 | select | ✅ | - | 违禁词/广告词/涉政词/其他 |
| replacement | 替换词 | input | - | - | 默认为"***" |

## 批量导入弹窗

| 字段 | 说明 |
|------|------|
| 文件格式 | TXT/CSV，每行一个敏感词 |
| 分类 | 导入的敏感词分类 |
| 跳过重复 | 自动跳过已存在的敏感词 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 搜索敏感词 | `question:sensitive:view` |
| 添加 | 添加敏感词 | `question:sensitive:create` |
| 删除 | 删除敏感词 | `question:sensitive:delete` |
| 批量删除 | 批量删除敏感词 | `question:sensitive:delete` |
| 批量导入 | 从文件导入敏感词 | `question:sensitive:import` |
| 导出 | 导出敏感词列表 | `question:sensitive:export` |

## 配置项

### 数据范围

| 选项 | 说明 |
|------|------|
| 全部题目 | 所有类型的题目 |
| 公共题目 | 仅公共题目 |
| 私有题目 | 仅当前用户的私有题目 |
| 班级题目 | 仅班级题目 |

### 筛选条件

| 字段 | 类型 | 说明 |
|------|------|------|
| 题型 | select | 单选/多选/判断/填空/简答 |
| 难度 | select | 简单/中等/困难 |
| 分类 | tree-select | 树形分类选择 |
| 知识点 | tree-select | 树形知识点选择 |
| 状态 | select | 草稿/启用/禁用 |
| 创建时间 | daterange | 日期范围 |

### 导出字段

| 字段 | 默认 | 说明 |
|------|------|------|
| 题目内容 | ✅ | 题目正文 |
| 题型 | ✅ | 单选题/多选题等 |
| 选项 | ✅ | 选项列表 |
| 正确答案 | ✅ | 正确答案 |
| 解析 | ✅ | 答案解析 |
| 难度 | ✅ | 简单/中等/困难 |
| 分类 | ✅ | 所属分类 |
| 知识点 | ✅ | 关联知识点 |
| 分值 | ✅ | 默认分值 |
| 创建时间 | ❌ | - |
| 创建者 | ❌ | - |
| 使用次数 | ❌ | 被组卷使用次数 |

## 功能点

| 功能 | 说明 |
|------|------|
| 预览 | 预览前10条数据，确认导出内容 |
| 导出 | 根据配置导出数据 |
| 全选/反选 | 快速选择导出字段 |
| 重置 | 恢复默认配置 |

## 交互说明

1. **预览**：点击预览按钮，加载前10条数据
2. **导出**：点击导出按钮，调用接口下载文件
3. **大数据量**：超过1000条时提示"数据量较大，是否继续？"
4. **导出进度**：显示loading状态

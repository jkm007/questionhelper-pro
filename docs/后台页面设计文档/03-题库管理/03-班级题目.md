# 班级题目

## 基本信息

- 路由路径：`/question/class`
- 权限标识：`question:class:view`
- 页面类型：列表页（CURD）

## 关联接口文档

> 详见 [API接口文档/03-题库管理.md](../API接口文档/03-题库管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/questions` | GET | 题目列表（type=class） | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/classes` | GET | 班级列表（下拉） | [链接](../API接口文档/05-班级管理.md) |
| 其他接口同公共题目 | - | - | - |

## 页面布局

与公共题目页面布局相同，区别：
1. 数据范围：仅显示关联班级的题目
2. 权限标识：`question:class:*`
3. 标题：题库管理 > 班级题目
4. 搜索栏增加"所属班级"筛选

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索题目内容 |
| type | 题型 | select | 单选/多选/判断/填空/简答 |
| difficulty | 难度 | select | 简单/中等/困难 |
| class_id | 所属班级 | select | 关联班级 |
| category_id | 分类 | tree-select | 树形分类选择 |
| status | 状态 | select | 草稿/启用/禁用 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `question:class:view` |
| 新增 | 新增班级题目 | `question:class:create` |
| 编辑 | 编辑班级题目 | `question:class:edit` |
| 删除 | 删除班级题目 | `question:class:delete` |
| 移动到公共 | 将班级题目发布为公共 | `question:public:create` |
| 其他 | 同公共题目 | - |

## 特殊说明

- 班级题目仅班级成员可见
- 需要选择所属班级
- 班级教师/创作者管理

# 批量导入

## 基本信息

- 路由路径：`/question/import`
- 权限标识：`question:import`
- 页面类型：导入向导页

## 关联接口文档

|------|------|------|----------|
| `/api/v1/admin/questions/import` | POST | 批量导入 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/questions/import/template` | GET | 下载模板 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/questions/import/preview` | POST | 预览导入数据 | [链接](../API接口文档/03-题库管理.md) |

## 步骤说明

### 步骤一：上传文件

| 功能 | 说明 |
|------|------|
| 文件上传 | 支持拖拽或点击上传 |
| 文件格式 | .xlsx, .xls |
| 文件大小 | 最大 10MB |
| 下载模板 | 下载标准导入模板 |
| 目标分类 | 选择导入到的分类 |
| 题目状态 | 设置导入题目的初始状态 |

## Excel 模板字段

| 列 | 字段名 | 必填 | 说明 |
|----|--------|------|------|
| A | 题目内容 | ✅ | 题目正文 |
| B | 题型 | ✅ | 单选题/多选题/判断题/填空题/简答题 |
| C | 选项A | - | 单选/多选必填 |
| D | 选项B | - | 单选/多选必填 |
| E | 选项C | - | - |
| F | 选项D | - | - |
| G | 正确答案 | ✅ | 单选填A/B/C/D，多选填A,B,C |
| H | 解析 | - | 答案解析 |
| I | 难度 | ✅ | 简单/中等/困难 |
| J | 分类 | ✅ | 分类名称（支持多级用/分隔） |
| K | 分值 | - | 默认5分 |

## 功能点

| 功能 | 说明 |
|------|------|
| 下载模板 | 下载标准Excel导入模板 |
| 文件上传 | 支持拖拽和点击上传 |
| 数据预览 | 解析Excel并预览数据 |
| 错误提示 | 显示每行的错误信息 |
| 选择性导入 | 跳过错误行或仅导入有效数据 |
| 导入结果 | 显示成功/失败数量和详情 |
| 下载失败数据 | 导出失败行数据便于修正 |

## 交互说明

1. **步骤切换**：通过步骤条或按钮切换
2. **上传验证**：验证文件格式和大小
3. **预览加载**：解析Excel时显示loading
4. **导入进度**：显示导入进度条
5. **错误处理**：错误行高亮显示

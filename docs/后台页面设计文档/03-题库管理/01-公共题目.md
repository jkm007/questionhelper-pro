# 公共题目

## 基本信息

- 路由路径：`/question/public`
- 权限标识：`question:public:view`
- 页面类型：列表页（CURD）

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索题目内容 |
| type | 题型 | select | 单选/多选/判断/填空/简答 |
| difficulty | 难度 | select | 简单/中等/困难 |
| category_id | 分类 | tree-select | 树形分类选择 |
| knowledge_id | 知识点 | tree-select | 树形知识点选择 |
| status | 状态 | select | 草稿/启用/禁用 |
| created_at | 创建时间 | daterange | 日期范围 |

| 按钮 | 权限 | 说明 |
|------|------|------|

| 删除 | `question:public:delete` | 二次确认后删除 |

| - 禁用/启用 | `question:public:edit` | 切换状态 |

## 题型差异

### 单选题
```
选项列表：
┌────────────────────────────────────────┐
│ A. [选项内容________] [删除]            │
│ B. [选项内容________] [删除]            │
│ C. [选项内容________] [删除]            │
│ D. [选项内容________] [删除]            │
│ [+ 添加选项]                           │
└────────────────────────────────────────┘
正确答案：( ) A  ( ) B  ( ) C  ( ) D
```

### 多选题
```
选项列表：（同上）
正确答案：[x] A  [ ] B  [x] C  [ ] D
```

### 判断题
```
正确答案：( ) 正确  ( ) 错误
```

### 填空题
```
题目内容中使用 _____ 表示填空位置
正确答案：
填空1: [________]
填空2: [________]
```

### 简答题
```
正确答案：[富文本编辑器]
```

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `question:public:view` |
| 新增 | 跳转新增页面或弹窗 | `question:public:create` |
| 编辑 | 跳转编辑页面或弹窗 | `question:public:edit` |
| 删除 | 二次确认后删除 | `question:public:delete` |
| 批量删除 | 勾选后批量删除 | `question:public:delete` |
| 预览 | 弹窗预览题目完整内容 | `question:public:view` |
| 复制 | 复制为新题目 | `question:public:create` |
| 状态切换 | 启用/禁用/草稿 | `question:public:edit` |
| 批量审核 | 批量设置为启用 | `question:public:edit` |
| 导入 | Excel批量导入 | `question:public:import` |
| 导出 | 导出为Excel | `question:public:export` |

## 样式要点

- 题目内容列：最大宽度300px，溢出省略
- 题型标签：不同题型不同颜色
- 难度标签：简单(success)/中等(warning)/困难(danger)
- 预览弹窗：宽度700px
- 表单页：宽度800px弹窗或独立页面

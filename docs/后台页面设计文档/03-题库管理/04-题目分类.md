# 题目分类

## 基本信息

- 路由路径：`/question/category`
- 权限标识：`category:view`
- 页面类型：树形列表页

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 分类名称 | string | 200 | 树形缩进显示 |
| sort_order | 排序 | number | 80 | 可拖拽排序 |
| question_count | 题目数 | number | 80 | 该分类下题目数量 |
| status | 状态 | tag | 80 | 启用(success)/禁用(danger) |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 180 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 添加子分类 | `category:create` | 在当前分类下添加子分类 |
| 编辑 | `category:edit` | 编辑分类信息 |
| 删除 | `category:delete` | 有子分类或题目时禁止删除 |
| 上移/下移 | `category:sort` | 调整排序 |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| parent_id | 上级分类 | tree-select | - | - | 空则为根分类 |
| name | 分类名称 | input | ✅ | 2-20位 | 同级唯一 |
| sort_order | 排序 | number | ✅ | ≥0 | 数字越小越靠前 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |
| remark | 备注 | textarea | - | 最长200字 | - |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 新增根分类 | 添加顶级分类 | `category:create` |
| 添加子分类 | 在选中分类下添加 | `category:create` |
| 编辑 | 修改分类信息 | `category:edit` |
| 删除 | 删除分类（需无子分类和题目） | `category:delete` |
| 拖拽排序 | 拖拽调整顺序 | `category:sort` |
| 展开/折叠 | 展开或折叠所有节点 | - |

## 交互说明

1. **树形展示**：默认展开第一级
2. **拖拽排序**：支持同级拖拽排序
3. **删除限制**：有子分类或题目时，删除按钮禁用并提示
4. **级联选择**：新增/编辑时，上级分类使用树形选择器

## 样式要点

- 树形缩进：每级缩进24px
- 展开/折叠图标：箭头图标
- 拖拽时：显示拖拽指示线

# 题目审核

## 基本信息

- 路由路径：`/question/review`
- 权限标识：`question:review:view`
- 页面类型：列表页（审核）

## 关联接口文档

> 详见 [API接口文档/03-题库管理.md](../API接口文档/03-题库管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/question-reviews` | GET | 审核列表 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/question-reviews/:id/approve` | POST | 审核通过 | [链接](../API接口文档/03-题库管理.md) |
| `/api/v1/admin/question-reviews/:id/reject` | POST | 审核驳回 | [链接](../API接口文档/03-题库管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索题目内容 |
| type | 题型 | select | 单选/多选/判断/填空/简答 |
| submitter | 提交人 | input | 搜索提交人姓名 |
| status | 状态 | select | 全部/待审核/已通过/已驳回 |
| created_at | 提交时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| content | 题目内容 | string | 300 | 溢出省略 |
| type | 题型 | tag | 80 | - |
| difficulty | 难度 | tag | 80 | - |
| submitter | 提交人 | string | 100 | - |
| status | 状态 | tag | 80 | 待审核(warning)/已通过(success)/已驳回(danger) |
| created_at | 提交时间 | datetime | 160 | - |
| 操作 | 操作 | - | 200 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 通过 | `question:review:approve` | 审核通过 |
| 驳回 | `question:review:reject` | 审核驳回 |
| 预览 | `question:review:view` | 预览题目内容 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `question:review:view` |
| 审核通过 | 通过题目审核 | `question:review:approve` |
| 审核驳回 | 驳回题目，填写原因 | `question:review:reject` |
| 批量通过 | 批量通过审核 | `question:review:approve` |
| 批量驳回 | 批量驳回审核 | `question:review:reject` |
| 预览 | 预览题目完整内容 | `question:review:view` |

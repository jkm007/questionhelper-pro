## 6. 组件规范

### 6.1 按钮

| 类型 | 用途 | 样式 |
|------|------|------|
| Primary | 主操作（新增、提交） | 蓝色实底 |
| Success | 成功操作（启用、通过） | 绿色实底 |
| Warning | 警告操作（编辑） | 橙色实底 |
| Danger | 危险操作（删除、驳回） | 红色实底 |
| Info | 次要操作（取消、关闭） | 灰色实底 |
| Text | 文字按钮（详情、链接） | 无边框，文字色 |

**按钮尺寸**：
- large：高度 40px，用于表单提交
- default：高度 32px，常规操作
- small：高度 24px，表格内操作

### 6.2 表格

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| border | true | 显示边框 |
| stripe | true | 斑马纹 |
| size | default | default/small/large |
| highlight-current-row | true | 高亮当前行 |
| show-overflow-tooltip | true | 溢出省略提示 |

**操作列**：
- 固定在右侧（fixed="right"）
- 宽度根据按钮数量自适应
- 超过3个操作使用"更多"下拉

### 6.3 表单

| 配置项 | 规范 |
|--------|------|
| label-position | right |
| label-width | 100px |
| 表单项间距 | 18px |
| 必填标识 | 红色星号 * |
| 验证时机 | blur 触发 |

### 6.4 弹窗

| 类型 | 宽度 | 用途 |
|------|------|------|
| 小弹窗 | 400px | 确认删除、简单表单 |

| 大弹窗 | 800px | 复杂表单、详情 |
| 全屏弹窗 | 80% | 数据预览、富文本编辑 |

### 6.5 标签 Tag

| 状态 | 颜色 | 用途 |
|------|------|------|
| 启用/成功 | success | 启用状态、通过 |

## 10. 水印规范

### 10.1 水印功能

水印覆盖整个页面内容区，防止截图泄露，支持自定义配置。

**技术实现**：使用 Canvas 绘制水印，通过 MutationObserver 防止水印被删除。

### 10.2 水印配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| enabled | true | 是否开启水印 |
| text | "题小助" + 用户名 | 水印文字内容 |
| color | `rgba(0, 0, 0, 0.05)` | 水印颜色（浅灰色） |
| fontSize | 14px | 字体大小 |
| rotate | -20度 | 旋转角度 |
| gapX | 100px | 水平间距 |
| gapY | 100px | 垂直间距 |
| zIndex | 9999 | 层级（覆盖所有内容） |

### 10.3 水印样式

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│   题小助 admin        题小助 admin        题小助 admin           │
│        ↘                    ↘                    ↘              │
│                                                                  │
│                    (页面正常内容)                                │
│                                                                  │
│   题小助 admin        题小助 admin        题小助 admin           │
│        ↘                    ↘                    ↘              │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

水印特性：
- 浅灰色、半透明
- 倾斜 20 度
- 平铺满整个页面
- 不影响正常操作
- 防止通过 DevTools 删除
```

### 10.4 水印数据

水印文字格式：`{系统名称} {用户名} {日期}`

示例：`题小助 admin 2024-03-15`

### 10.5 防篡改机制

```javascript
// 使用 MutationObserver 监听 DOM 变化
// 如果水印被删除，自动重新创建
const observer = new MutationObserver((mutations) => {
  if (!document.querySelector('.watermark')) {
    createWatermark(); // 重新创建水印
  }
});

observer.observe(document.body, {
  childList: true,
  subtree: true,
});
```

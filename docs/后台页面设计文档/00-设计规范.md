# 管理后台设计规范

## 1. 色彩规范

### 1.1 主色调

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| Primary | `#409EFF` | 主按钮、链接、选中状态 |
| Success | `#67C23A` | 成功状态、启用标签 |
| Warning | `#E6A23C` | 警告状态、待处理 |
| Danger | `#F56C6C` | 错误状态、删除按钮、禁用标签 |
| Info | `#909399` | 辅助信息、禁用状态 |

### 1.2 中性色

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 主要文字 | `#303133` | 标题、重要文本 |
| 常规文字 | `#606266` | 正文内容 |
| 次要文字 | `#909399` | 辅助说明、占位符 |

| 三级边框 | `#EBEEF5` | 轻微分割 |

| 二级背景 | `#F5F7FA` | 页面背景、表格斑马纹 |

## 3. 间距规范

| lg | 16px | 区块间距、页面边距 |

### 3.2 布局间距

| 区域 | 间距 |
|------|------|
| 页面内容区 padding | 20px |
| 卡片间距 | 16px |

| 表格行高 | 48px（默认）/ 40px（紧凑） |

## 4. 圆角规范

| 名称 | 数值 | 用途 |

## 5. 阴影规范

| 名称 | 数值 | 用途 |
|------|------|------|
| 无阴影 | none | 平面元素 |

## 6. 组件规范

### 6.1 按钮

| 类型 | 用途 | 样式 |
|------|------|------|
| Primary | 主操作（新增、提交） | 蓝色实底 |
| Success | 成功操作（启用、通过） | 绿色实底 |
| Warning | 警告操作（编辑） | 橙色实底 |
| Danger | 危险操作（删除、驳回） | 红色实底 |
| Info | 次要操作（取消、关闭） | 灰色实底 |
| Text | 文字按钮（详情、链接） | 无边框，文字色 |

**按钮尺寸**：
- large：高度 40px，用于表单提交
- default：高度 32px，常规操作
- small：高度 24px，表格内操作

### 6.2 表格

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| border | true | 显示边框 |
| stripe | true | 斑马纹 |
| size | default | default/small/large |
| highlight-current-row | true | 高亮当前行 |
| show-overflow-tooltip | true | 溢出省略提示 |

**操作列**：
- 固定在右侧（fixed="right"）
- 宽度根据按钮数量自适应
- 超过3个操作使用"更多"下拉

### 6.3 表单

| 配置项 | 规范 |
|--------|------|
| label-position | right |
| label-width | 100px |
| 表单项间距 | 18px |
| 必填标识 | 红色星号 * |
| 验证时机 | blur 触发 |

### 6.4 弹窗

| 类型 | 宽度 | 用途 |
|------|------|------|
| 小弹窗 | 400px | 确认删除、简单表单 |
| 中弹窗 | 600px | 标准表单 |
| 大弹窗 | 800px | 复杂表单、详情 |
| 全屏弹窗 | 80% | 数据预览、富文本编辑 |

### 6.5 标签 Tag

| 状态 | 颜色 | 用途 |
|------|------|------|
| 启用/成功 | success | 启用状态、通过 |
| 禁用/失败 | danger | 禁用状态、驳回 |
| 待处理 | warning | 待审核、待发布 |
| 默认/草稿 | info | 草稿、默认状态 |

### 6.6 分页

```vue
<el-pagination
  v-model:current-page="page"
  v-model:page-size="pageSize"
  :page-sizes="[10, 20, 50, 100]"
  :total="total"
  layout="total, sizes, prev, pager, next, jumper"
/>
```

## 8. 交互规范

### 8.1 加载状态

| 场景 | 处理方式 |
|------|----------|
| 页面初始加载 | 骨架屏或 v-loading |
| 表格数据加载 | 表格区域 loading |
| 按钮提交 | 按钮 loading + 禁用 |
| 弹窗提交 | 弹窗 loading + 按钮禁用 |

### 8.2 空状态

```vue
<el-empty description="暂无数据">
  <el-button type="primary">新增</el-button>
</el-empty>
```

### 8.3 确认操作

| 操作类型 | 确认方式 |
|----------|----------|
| 删除 | 二次确认弹窗（"确定要删除吗？"） |
| 批量删除 | 二次确认弹窗（显示数量） |
| 状态切换 | 直接切换，失败回滚 |
| 提交表单 | 表单验证通过后提交 |

### 8.4 消息提示

| 类型 | 场景 | 示例 |
|------|------|------|
| success | 操作成功 | "新增成功"、"删除成功" |
| warning | 警告提示 | "请先选择数据" |
| error | 操作失败 | "删除失败，请重试" |
| info | 信息提示 | "已复制到剪贴板" |

## 9. 响应式规范

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| xs | < 768px | 侧边栏折叠，表格横向滚动 |
| sm | 768px - 992px | 侧边栏折叠 |
| md | 992px - 1200px | 正常布局 |
| lg | 1200px - 1920px | 正常布局 |
| xl | > 1920px | 内容区最大宽度 1920px |

## 10. 图标规范

常用图标映射：
| 功能 | 图标 |
|------|------|
| 新增 | `Plus` |
| 编辑 | `Edit` |
| 删除 | `Delete` |
| 搜索 | `Search` |
| 刷新 | `Refresh` |
| 导出 | `Download` |
| 导入 | `Upload` |
| 设置 | `Setting` |
| 返回 | `Back` |
| 更多 | `MoreFilled` |

# 班级资源管理

## 基本信息

- 路由路径：`/class/resource`
- 权限标识：`class:resource:view`
- 页面类型：列表页（资源管理）

## 关联接口文档

> 详见 [API接口文档/05-班级管理.md](../API接口文档/05-班级管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/classes/:id/resources` | GET | 资源列表 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/classes/:id/resources` | POST | 上传资源 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/classes/:id/resources/:rid` | DELETE | 删除资源 | [链接](../API接口文档/05-班级管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索资源名称 |
| type | 类型 | select | 全部/文档/图片/视频/音频/其他 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 资源名称 | string | 200 | 显示文件图标+名称 |
| type | 类型 | tag | 80 | 文档/图片/视频/音频/其他 |
| size | 大小 | string | 80 | 显示文件大小 |
| uploader | 上传者 | string | 100 | - |
| upload_time | 上传时间 | datetime | 160 | - |
| 操作 | 操作 | - | 150 | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 下载 | `class:resource:view` | 下载资源文件 |
| 删除 | `class:resource:manage` | 二次确认后删除 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 按名称/类型搜索 | `class:resource:view` |
| 上传资源 | 上传文件到班级 | `class:resource:manage` |
| 批量上传 | 支持多文件上传 | `class:resource:manage` |
| 下载 | 下载资源文件 | `class:resource:view` |
| 删除 | 删除资源文件 | `class:resource:manage` |
| 批量删除 | 批量删除资源 | `class:resource:manage` |

## 上传弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| files | 选择文件 | upload | ✅ | 支持多选，最大100MB |
| description | 资源说明 | textarea | - | 最长200字 |

## 交互说明

1. **切换班级**：下拉选择班级后刷新资源列表
2. **上传进度**：显示上传进度条
3. **文件预览**：图片类型支持预览
4. **下载**：直接下载文件

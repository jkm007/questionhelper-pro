# 班级创作者审批

## 基本信息

- 路由路径：`/class/approval`
- 权限标识：`class:creator:approve`
- 页面类型：列表页（审批）

## 关联接口文档

> 详见 [API接口文档/05-班级管理.md](../API接口文档/05-班级管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/creators/applications` | GET | 申请列表 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/creators/applications/:id/approve` | POST | 审批通过 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/creators/applications/:id/reject` | POST | 审批驳回 | [链接](../API接口文档/05-班级管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索申请人姓名/手机号 |
| status | 状态 | select | 全部/待审核/已通过/已驳回 |
| apply_time | 申请时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| applicant | 申请人 | string | 120 | - |
| phone | 手机号 | string | 120 | 脱敏显示 |
| apply_reason | 申请说明 | string | 200 | 溢出省略提示 |
| status | 状态 | tag | 100 | 待审核(warning)/已通过(success)/已驳回(danger) |
| apply_time | 申请时间 | datetime | 160 | - |
| approve_time | 审批时间 | datetime | 160 | 仅已审核状态显示 |
| approver | 审批人 | string | 100 | 仅已审核状态显示 |
| 操作 | 操作 | - | 200 | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 通过 | `class:creator:approve` | 二次确认后通过 | 状态=待审核 |
| 驳回 | `class:creator:approve` | 弹窗填写驳回原因 | 状态=待审核 |
| 查看详情 | `class:creator:view` | 查看申请详情 | 所有状态 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `class:creator:view` |
| 审批通过 | 通过创作者申请 | `class:creator:approve` |
| 审批驳回 | 驳回申请，填写原因 | `class:creator:approve` |
| 查看详情 | 查看申请详细信息 | `class:creator:view` |

## 驳回弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| reject_reason | 驳回原因 | textarea | ✅ | 最长200字 |

## 交互说明

1. **审批通过**：二次确认弹窗，确认后调用通过接口
2. **审批驳回**：弹窗填写驳回原因（必填），提交后调用驳回接口
3. **查看详情**：抽屉或弹窗显示申请详情

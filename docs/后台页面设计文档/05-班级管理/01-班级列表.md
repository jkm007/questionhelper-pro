# 班级列表

## 基本信息

- 路由路径：`/class/list`
- 权限标识：`class:list:view`
- 页面类型：列表页（CURD）

|------|------|------|----------|
| `/api/v1/admin/classes` | GET | 班级列表 | [链接](../API接口文档/05-班级管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索班级名称 |
| status | 状态 | select | 启用/禁用 |
| created_at | 创建时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 排序 | 说明 |
|--------|------|------|------|------|------|
| id | ID | number | 60 | ✅ | - |
| name | 班级名称 | string | 180 | - | 可点击跳转详情 |
| code | 班级码 | string | 100 | - | 学生加入班级使用 |
| member_count | 成员数 | number | 80 | - | 当前成员数量 |
| creator | 创建者 | string | 100 | - | 班级创建者 |
| status | 状态 | tag | 80 | - | 启用(success)/禁用(danger) |
| created_at | 创建时间 | datetime | 160 | ✅ | - |
| 操作 | 操作 | - | 250 | - | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 编辑 | `class:list:edit` | 编辑班级信息 |
| 详情 | `class:detail:view` | 跳转班级详情页 |
| 成员管理 | `class:member:manage` | 管理班级成员 |
| 删除 | `class:list:delete` | 二次确认后删除 |

## 表单字段（创建/编辑班级弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 班级名称 | input | ✅ | 2-30位 | - |
| description | 班级描述 | textarea | - | 最长500字 | - |
| cover | 封面图片 | upload | - | 图片格式 | 可选 |
| max_members | 最大成员数 | number | - | 1-500 | 默认50 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `class:list:view` |
| 创建班级 | 弹窗填写班级信息 | `class:list:create` |
| 编辑 | 编辑班级信息 | `class:list:edit` |
| 删除 | 删除班级（需无成员） | `class:list:delete` |
| 查看详情 | 跳转班级详情 | `class:detail:view` |
| 成员管理 | 跳转成员管理 | `class:member:manage` |
| 复制班级码 | 点击复制班级码 | - |

## 交互说明

1. **班级码复制**：点击班级码自动复制到剪贴板
2. **删除限制**：有成员时禁止删除，提示"请先移除所有成员"
3. **状态切换**：禁用后学生无法加入

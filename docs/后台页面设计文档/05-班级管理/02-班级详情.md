# 班级详情

## 基本信息

- 路由路径：`/class/detail/:id`
- 权限标识：`class:detail:view`
- 页面类型：详情页（多Tab）

## 关联接口文档

> 详见 [API接口文档/05-班级管理.md](../API接口文档/05-班级管理.md)

|------|------|------|----------|
| `/api/v1/admin/classes/:id` | GET | 班级详情 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/classes/:id/members` | GET | 成员列表 | [链接](../API接口文档/05-班级管理.md) |

## 班级信息卡片

| 字段 | 说明 |
|------|------|
| 封面图片 | 班级封面，可点击放大 |
| 班级名称 | 显示名称 |
| 班级码 | 学生加入使用，可复制 |
| 创建者 | 班级创建者姓名 |
| 创建时间 | 创建日期 |
| 成员数 | 当前/最大成员数 |
| 状态 | 启用/禁用 |

## Tab 1: 成员管理

### 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索姓名/学号 |
| role | 角色 | select | 全部/学生/教师 |

### 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 姓名 | string | 120 | - |
| student_no | 学号 | string | 100 | - |
| phone | 手机号 | string | 120 | 脱敏显示 |
| role | 角色 | tag | 80 | 学生/教师 |
| join_time | 加入时间 | datetime | 160 | - |
| last_active | 最后活跃 | datetime | 160 | - |
| 操作 | 操作 | - | 120 | - |

### 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 移除 | `class:member:manage` | 二次确认后移除 |

### 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 按姓名/学号搜索 | `class:member:view` |
| 添加成员 | 弹窗搜索用户添加 | `class:member:manage` |
| 移除成员 | 移除班级成员 | `class:member:manage` |
| 批量移除 | 批量移除成员 | `class:member:manage` |
| 导出成员 | 导出成员列表 | `class:member:export` |

## 交互说明

1. **Tab切换**：点击Tab切换内容区
2. **添加成员**：弹窗搜索用户，选择后添加
3. **移除确认**：移除成员前二次确认
4. **复制班级码**：点击复制到剪贴板

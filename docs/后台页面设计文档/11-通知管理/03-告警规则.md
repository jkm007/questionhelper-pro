## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 规则名称 | string | 150 | - |
| metric | 监控指标 | tag | 120 | CPU/内存/磁盘/请求量等 |
| condition | 触发条件 | string | 150 | 如 "> 80%" |
| duration | 持续时间 | string | 80 | 如 "5分钟" |
| notify_channels | 通知方式 | tag[] | 150 | 邮件/短信/站内 |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 150 | - |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 规则名称 | input | ✅ | 2-30位 | - |
| metric | 监控指标 | select | ✅ | - | CPU/内存/磁盘/请求量/错误率 |
| operator | 比较运算符 | select | ✅ | - | >/>=/</<=/== |
| threshold | 阈值 | number | ✅ | - | 触发阈值 |
| duration | 持续时间 | number | ✅ | - | 分钟数 |
| notify_channels | 通知方式 | checkbox | ✅ | 至选一个 | 邮件/短信/站内 |
| notify_users | 通知人员 | user-select | ✅ | - | 接收告警的人员 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |
| description | 描述 | textarea | - | 最长200字 | - |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 新增 | 创建告警规则 | `notification:alert-rule:create` |
| 编辑 | 编辑规则信息 | `notification:alert-rule:edit` |
| 删除 | 删除规则 | `notification:alert-rule:delete` |
| 批量删除 | 批量删除规则 | `notification:alert-rule:delete` |
| 状态切换 | 启用/禁用规则 | `notification:alert-rule:edit` |

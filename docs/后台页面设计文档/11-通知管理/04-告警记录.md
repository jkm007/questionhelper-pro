## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索告警名称 |
| level | 告警级别 | select | 严重/警告/提示 |
| status | 状态 | select | 告警中/已恢复/已处理 |
| created_at | 告警时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 告警名称 | string | 180 | - |
| level | 告警级别 | tag | 80 | 严重(danger)/警告(warning)/提示(info) |
| metric | 监控指标 | string | 120 | - |
| trigger_value | 触发值 | string | 80 | 当前值 |
| threshold | 阈值 | string | 80 | 触发阈值 |
| status | 状态 | tag | 80 | 告警中(danger)/已恢复(success)/已处理(info) |
| created_at | 告警时间 | datetime | 160 | - |
| resolved_at | 恢复时间 | datetime | 160 | - |
| 操作 | 操作 | - | 120 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 处理 | `notification:alert-record:handle` | 标记为已处理 |
| 查看详情 | `notification:alert-record:view` | 查看告警详情 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `notification:alert-record:view` |
| 处理告警 | 标记告警为已处理 | `notification:alert-record:handle` |
| 查看详情 | 查看告警详细信息 | `notification:alert-record:view` |
| 导出 | 导出告警记录 | `notification:alert-record:export` |

# 通知模板

## 基本信息

- 路由路径：`/notification/template`
- 权限标识：`notification:template:view`
- 页面类型：列表页（CURD）

|------|------|------|----------|
| `/api/v1/admin/notification-templates` | GET | 模板列表 | [链接](../API接口文档/11-系统设置.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索模板名称/编码 |
| type | 通知类型 | select | 考试/成绩/班级/系统 |
| channel | 通知渠道 | select | 站内/邮件/短信 |
| status | 状态 | select | 启用/禁用 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 模板名称 | string | 150 | - |
| code | 模板编码 | string | 120 | 唯一标识 |
| type | 通知类型 | tag | 100 | 考试/成绩/班级/系统 |
| channels | 通知渠道 | tag[] | 150 | 站内/邮件/短信 |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 200 | - |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 模板名称 | input | ✅ | 2-30位 | - |
| code | 模板编码 | input | ✅ | 字母下划线 | 唯一，不可修改 |
| type | 通知类型 | select | ✅ | - | 考试/成绩/班级/系统 |
| channels | 通知渠道 | checkbox | ✅ | 至选一个 | 站内/邮件/短信 |
| title | 消息标题 | input | ✅ | 最长50字 | 支持变量 |
| content | 消息内容 | rich-text | ✅ | - | 支持变量 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |

## 变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| {{.UserName}} | 用户名称 | 张三 |
| {{.ExamName}} | 考试名称 | 期末考试 |
| {{.ClassName}} | 班级名称 | 高三一班 |
| {{.Score}} | 考试分数 | 85 |
| {{.Time}} | 时间 | 2024-03-15 14:00 |
| {{.Link}} | 跳转链接 | /exam/123 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `notification:template:view` |
| 新增 | 创建通知模板 | `notification:template:create` |
| 编辑 | 编辑模板信息 | `notification:template:edit` |
| 删除 | 删除模板 | `notification:template:delete` |
| 预览 | 预览模板渲染效果 | `notification:template:view` |

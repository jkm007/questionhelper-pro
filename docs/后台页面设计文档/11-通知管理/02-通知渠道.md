# 通知渠道

## 基本信息

- 路由路径：`/notification/channel`
- 权限标识：`notification:channel:view`
- 页面类型：列表页（配置）

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/notification-channels` | GET | 渠道列表 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/notification-channels/:id` | PUT | 更新渠道 | [链接](../API接口文档/11-系统设置.md) |

## 渠道配置

| 渠道 | 配置项 | 说明 |
|------|--------|------|
| 站内通知 | 无 | 系统内置，无需配置 |
| 邮件通知 | SMTP服务器、端口、账号、密码 | 需要配置邮件服务 |
| 短信通知 | 服务商、AccessKey、模板 | 需要配置短信服务 |
| 微信推送 | AppID、AppSecret | 需要配置微信公众号 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看渠道配置 | `notification:channel:view` |
| 保存配置 | 保存渠道配置 | `notification:channel:edit` |
| 测试发送 | 发送测试消息 | `notification:channel:test` |
| 启用/禁用 | 开关通知渠道 | `notification:channel:edit` |

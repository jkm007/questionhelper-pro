# 资源设置

- 路由路径：`/setting/resource`
- 权限标识：`system:resource:view`
- 页面类型：配置页

|------|------|------|----------|
| `/api/v1/admin/settings/resource` | GET | 获取资源设置 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/settings/resource` | PUT | 更新资源设置 | [链接](../API接口文档/11-系统设置.md) |

## 配置项

### 资源类型

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 公共资源 | 是否开启公共资源 | 开启 |
| 私有资源 | 是否开启私有资源 | 开启 |
| 班级资源 | 是否开启班级资源 | 开启 |

### 资源审核

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 公共资源审核 | 公共资源是否需要审核 | 开启 |
| 私有资源审核 | 私有资源是否需要审核 | 关闭 |
| 班级资源审核 | 班级资源是否需要审核 | 关闭 |

### 资源分享

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 允许分享 | 是否允许分享资源 | 开启 |
| 分享方式 | 支持的分享方式 | 链接分享 |
| 分享有效期 | 分享链接的有效期 | 7天 |

### 资源存储

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 图片压缩 | 是否自动压缩图片 | 开启 |
| 压缩质量 | 图片压缩质量 | 80% |
| 最大图片尺寸 | 图片最大宽度/高度 | 1920px |
| 视频转码 | 是否转码视频 | 关闭 |
| 视频分辨率 | 转码后的分辨率 | 720p |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看当前资源配置 | `system:resource:view` |
| 保存配置 | 保存资源配置 | `system:resource:edit` |
| 重置 | 恢复默认配置 | `system:resource:edit` |

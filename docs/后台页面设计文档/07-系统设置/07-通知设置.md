# 通知设置

## 基本信息

- 路由路径：`/setting/notification`
- 权限标识：`system:notification:view`
- 页面类型：表单配置页

## 关联接口文档

> 详见 [API接口文档/11-系统设置.md](../API接口文档/11-系统设置.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/notifications/settings` | GET | 获取通知设置 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/notifications/settings` | PUT | 更新通知设置 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/notifications/templates` | GET | 通知模板列表 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/notifications/templates/:id` | PUT | 更新模板 | [链接](../API接口文档/11-系统设置.md) |

## Tab 1: 通知开关

### 通知渠道配置

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| in_app_enabled | 站内通知 | switch | 是否开启站内通知 |
| email_enabled | 邮件通知 | switch | 是否开启邮件通知 |
| email_frequency | 邮件频率 | select | 实时/每天汇总/每周汇总 |
| sms_enabled | 短信通知 | switch | 是否开启短信通知 |

### 通知类型配置

| 通知类型 | 站内 | 邮件 | 短信 |
|----------|------|------|------|
| 用户注册通知 | ✅ | ✅ | ❌ |
| 考试发布通知 | ✅ | ✅ | ✅ |
| 考试提醒通知 | ✅ | ✅ | ✅ |
| 成绩发布通知 | ✅ | ✅ | ❌ |
| 班级加入通知 | ✅ | ❌ | ❌ |
| 系统告警通知 | ✅ | ✅ | ✅ |

## Tab 2: 通知模板

### 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 模板名称 | string | 150 | - |
| code | 模板编码 | string | 150 | 唯一标识 |
| type | 通知类型 | tag | 100 | 站内/邮件/短信 |
| status | 状态 | tag | 80 | 启用/禁用 |
| 操作 | 操作 | - | 150 | 编辑 |

### 模板编辑弹窗

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| title | 标题 | input | ✅ | 模板标题 |
| content | 内容 | rich-text | ✅ | 支持变量 |
| variables | 变量说明 | textarea | - | 可用变量列表 |

### 模板变量

```
{{.UserName}}    用户名称
{{.ExamName}}    考试名称
{{.ClassName}}   班级名称
{{.Score}}       考试分数
{{.Time}}        时间
{{.Link}}        跳转链接
```

## Tab 3: 短信配置

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| sms_provider | 短信服务商 | select | ✅ | 阿里云/腾讯云 |
| sms_access_key | AccessKey | input | ✅ | - |
| sms_access_secret | AccessSecret | input(password) | ✅ | - |
| sms_sign_name | 短信签名 | input | ✅ | - |
| sms_template_code | 模板编码 | input | ✅ | - |

## Tab 4: 邮件配置

（同基础设置中的邮件配置）

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看设置 | 查看当前配置 | `system:notification:view` |
| 保存设置 | 保存配置修改 | `system:notification:edit` |
| 编辑模板 | 编辑通知模板 | `system:notification:edit` |
| 测试通知 | 发送测试通知 | `system:notification:test` |

## 交互说明

1. **模板预览**：编辑时可预览渲染效果
2. **变量插入**：点击变量自动插入到内容中
3. **测试发送**：输入测试账号发送测试通知

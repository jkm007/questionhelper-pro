# 角色申请审核

## 基本信息

- 路由路径：`/system/application`
- 权限标识：`system:application:view`
- 页面类型：列表页（审核）

## 关联接口文档

> 详见 [API接口文档/02-用户管理.md](../API接口文档/02-用户管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/applications` | GET | 申请列表 | [链接](../API接口文档/02-用户管理.md) |
| `/api/v1/admin/applications/:id` | GET | 申请详情 | [链接](../API接口文档/02-用户管理.md) |
| `/api/v1/admin/applications/:id/review` | POST | 审核申请 | [链接](../API接口文档/02-用户管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索申请人姓名 |
| type | 申请类型 | select | 全部/教师/创作者/管理员 |
| status | 状态 | select | 全部/待审核/已通过/已驳回 |
| apply_time | 申请时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| applicant | 申请人 | string | 120 | - |
| apply_type | 申请类型 | tag | 100 | 教师/创作者/管理员 |
| apply_reason | 申请说明 | string | 200 | 溢出省略 |
| status | 状态 | tag | 100 | 待审核(warning)/已通过(success)/已驳回(danger) |
| apply_time | 申请时间 | datetime | 160 | - |
| review_time | 审核时间 | datetime | 160 | - |
| reviewer | 审核人 | string | 100 | - |
| 操作 | 操作 | - | 200 | - |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 通过 | `system:application:review` | 二次确认后通过 | 状态=待审核 |
| 驳回 | `system:application:review` | 弹窗填写驳回原因 | 状态=待审核 |
| 查看详情 | `system:application:view` | 查看申请详情 | 所有状态 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `system:application:view` |
| 审核通过 | 通过角色申请 | `system:application:review` |
| 审核驳回 | 驳回申请，填写原因 | `system:application:review` |
| 查看详情 | 查看申请详细信息和附件 | `system:application:view` |

## 驳回弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| reject_reason | 驳回原因 | textarea | ✅ | 最长200字 |

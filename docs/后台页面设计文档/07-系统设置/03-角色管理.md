# 角色管理

## 基本信息

- 路由路径：`/setting/role`
- 权限标识：`role:view`
- 页面类型：列表页（CURD）

## 关联接口文档

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/roles` | GET | 角色列表 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/roles` | POST | 创建角色 | [链接](../API接口文档/11-系统设置.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索角色名称/标识 |
| status | 状态 | select | 启用/禁用 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| id | ID | number | 60 | - |
| name | 角色名称 | string | 150 | - |
| code | 角色标识 | string | 120 | 唯一标识 |
| user_count | 用户数 | number | 80 | 该角色下的用户数 |
| status | 状态 | tag | 80 | 启用(success)/禁用(danger) |
| description | 描述 | string | 200 | 溢出省略 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 250 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 编辑 | `role:edit` | 编辑角色信息 |
| 权限 | `role:permission` | 分配权限 |
| 用户 | `role:view` | 查看角色用户 |
| 删除 | `role:delete` | 有用户时禁止删除 |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 角色名称 | input | ✅ | 2-20位 | - |
| code | 角色标识 | input | ✅ | 字母数字下划线 | 唯一，不可修改 |
| description | 描述 | textarea | - | 最长200字 | - |
| status | 状态 | switch | ✅ | - | 启用/禁用 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 搜索角色 | `role:view` |
| 新增 | 创建角色 | `role:create` |
| 编辑 | 编辑角色信息 | `role:edit` |
| 删除 | 删除角色（需无用户） | `role:delete` |
| 分配权限 | 勾选权限树 | `role:permission` |
| 查看用户 | 查看角色下的用户 | `role:view` |

## 交互说明

1. **权限树**：支持全选/全不选/展开/折叠
2. **父子联动**：勾选父节点自动勾选子节点
3. **删除限制**：有用户时禁止删除
4. **系统角色**：内置角色不可删除

# 操作审计

## 基本信息

- 路由路径：`/setting/audit`
- 权限标识：`system:audit:view`
- 页面类型：列表页

## 关联接口文档

> 详见 [API接口文档/12-管理后台.md](../API接口文档/12-管理后台.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/logs/operation` | GET | 操作日志列表 | [链接](../API接口文档/12-管理后台.md) |
| `/api/v1/admin/logs/operation/:id` | GET | 日志详情 | [链接](../API接口文档/12-管理后台.md) |
| `/api/v1/admin/logs/operation/export` | GET | 导出日志 | [链接](../API接口文档/12-管理后台.md) |
| `/api/v1/admin/logs/operation/clean` | POST | 清空日志 | [链接](../API接口文档/12-管理后台.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| operator | 操作人 | input | 搜索操作人姓名 |
| module | 模块 | select | 用户管理/题库管理/考试管理等 |
| action | 操作类型 | select | 新增/编辑/删除/登录等 |
| status | 状态 | select | 成功/失败 |
| created_at | 操作时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| operator | 操作人 | string | 100 | - |
| module | 模块 | string | 100 | 操作模块 |
| action | 操作类型 | tag | 80 | 新增/编辑/删除等 |
| description | 描述 | string | 200 | 操作描述 |
| ip | IP地址 | string | 120 | - |
| status | 状态 | tag | 80 | 成功(success)/失败(danger) |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `system:audit:view` |
| 查看详情 | 查看日志详细信息 | `system:audit:view` |
| 导出 | 导出日志为Excel | `system:audit:export` |
| 清空 | 清空所有日志 | `system:audit:clean` |

## 交互说明

1. **清空确认**：清空前二次确认，提示"此操作不可恢复"
2. **详情脱敏**：敏感信息（密码、Token等）脱敏显示

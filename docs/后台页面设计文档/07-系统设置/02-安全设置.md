## 密码策略配置

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| password_min_length | 密码最小长度 | number | 默认6位 |
| password_require_uppercase | 包含大写字母 | switch | - |
| password_require_lowercase | 包含小写字母 | switch | - |
| password_require_number | 包含数字 | switch | - |
| password_require_special | 包含特殊字符 | switch | - |
| password_expire_days | 密码有效期 | number | 天数，0表示永不过期 |
| password_history_count | 密码历史检查 | number | 不允许使用最近N次密码 |

## 登录安全配置

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| login_lock_enabled | 登录失败锁定 | switch | - |
| login_lock_count | 失败次数 | number | 达到次数后锁定 |
| login_lock_duration | 锁定时间 | number | 分钟 |
| ip_whitelist_enabled | IP白名单 | switch | - |
| ip_whitelist | 白名单列表 | textarea | 每行一个IP |
| abnormal_login_detect | 异常登录检测 | switch | - |
|异地登录提醒 | 异地登录提醒 | switch | - |
| new_device提醒 | 新设备登录提醒 | switch | - |

## 接口安全配置

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| rate_limit_enabled | 接口限流 | switch | - |
| rate_limit_value | 限流值 | number | 每秒请求数 |
| rate_limit_scope | 限流范围 | radio | 全局/单用户/单IP |
| cors_allowed_origins | CORS域名 | textarea | 每行一个域名 |

## 数据安全配置

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| sensitive_encrypt | 敏感数据加密 | switch | - |
| audit_enabled | 操作审计 | switch | - |
| audit_scope | 审计范围 | select | 全部/写操作/关键操作 |
| audit_retention_days | 审计日志保留 | number | 天数 |
| backup_enabled | 自动备份 | switch | - |
| backup_frequency | 备份频率 | select | 每天/每周/每月 |
| backup_time | 备份时间 | time | - |
| backup_retention | 保留份数 | number | - |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看设置 | 查看当前配置 | `system:security:view` |
| 保存设置 | 保存配置修改 | `system:security:edit` |
| 重置 | 恢复默认配置 | `system:security:edit` |

| `/api/v1/admin/cache/stats` | GET | 缓存统计 | [链接](../API接口文档/11-系统设置.md) |

## 缓存概览卡片

| 指标 | 说明 |
|------|------|
| Redis状态 | 连接状态（正常/异常） |
| 内存使用 | Redis内存使用量 |
| Key数量 | 缓存Key总数 |
| 命中率 | 缓存命中率 |

## 缓存操作

| 操作 | 说明 | 确认 |
|------|------|------|
| 清除所有缓存 | 删除所有缓存数据 | 二次确认 |
| 清除过期缓存 | 删除已过期的缓存 | 直接执行 |
| 清除用户缓存 | 删除用户相关缓存 | 直接执行 |
| 清除菜单缓存 | 删除菜单数据缓存 | 直接执行 |
| 清除字典缓存 | 删除字典数据缓存 | 直接执行 |
| 清除配置缓存 | 删除系统配置缓存 | 直接执行 |
| 清除权限缓存 | 删除权限数据缓存 | 直接执行 |

## 热点Key表格

| 字段 | 说明 |
|------|------|
| Key | 缓存Key名称 |
| 类型 | String/Hash/List/Set |
| 大小 | 占用内存大小 |
| TTL | 过期时间（秒） |
| 访问次数 | 被访问的次数 |
| 最后访问 | 最后一次访问时间 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看统计 | 查看缓存统计数据 | `system:cache:view` |
| 清除缓存 | 清除指定类型的缓存 | `system:cache:clear` |
| 刷新统计 | 刷新缓存统计数据 | `system:cache:view` |

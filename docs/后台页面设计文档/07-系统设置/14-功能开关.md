# 功能开关

## 基本信息

- 路由路径：`/setting/feature`
- 权限标识：`system:feature:view`
- 页面类型：配置页

## 关联接口文档

> 详见 [API接口文档/11-系统设置.md](../API接口文档/11-系统设置.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/feature-flags` | GET | 功能开关列表 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/feature-flags/:id` | PUT | 更新开关 | [链接](../API接口文档/11-系统设置.md) |

## 功能分组

| 分组 | 功能项 |
|------|--------|
| 用户功能 | 注册、实名认证、邀请码 |
| 题库功能 | 审核、分享、评论、AI组卷 |
| 考试功能 | 防作弊、成绩复核、排名、自动交卷 |
| 班级功能 | 创建、审核、资源 |
| 通知功能 | 邮件、短信、站内 |
| 实验功能 | 数据导出、API、WebSocket、暗黑模式 |

## 功能开关配置

| 字段 | 说明 |
|------|------|
| 功能名称 | 功能显示名称 |
| 状态 | 开启/关闭 |
| 说明 | 功能描述说明 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看开关 | 查看所有功能开关状态 | `system:feature:view` |
| 切换开关 | 开启/关闭功能 | `system:feature:edit` |
| 保存 | 保存配置 | `system:feature:edit` |
| 重置 | 恢复默认配置 | `system:feature:edit` |

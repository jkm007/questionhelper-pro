# 实名审核

## 基本信息

- 路由路径：`/system/realname`
- 权限标识：`system:realname:view`
- 页面类型：列表页（审核）

## 关联接口文档

> 详见 [API接口文档/02-用户管理.md](../API接口文档/02-用户管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/realnames` | GET | 实名申请列表 | [链接](../API接口文档/02-用户管理.md) |
| `/api/v1/admin/realnames/:id/review` | POST | 审核实名 | [链接](../API接口文档/02-用户管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索用户名/真实姓名 |
| status | 状态 | select | 全部/待审核/已通过/已驳回 |
| apply_time | 申请时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| username | 用户名 | string | 120 | - |
| real_name | 真实姓名 | string | 100 | - |
| id_card | 身份证号 | string | 150 | 脱敏显示 |
| id_card_front | 身份证正面 | link | 80 | 点击查看大图 |
| id_card_back | 身份证背面 | link | 80 | 点击查看大图 |
| status | 状态 | tag | 100 | 待审核(warning)/已通过(success)/已驳回(danger) |
| apply_time | 申请时间 | datetime | 160 | - |
| review_time | 审核时间 | datetime | 160 | - |
| 操作 | 操作 | - | 180 | - |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 通过 | `system:realname:review` | 二次确认后通过 | 状态=待审核 |
| 驳回 | `system:realname:review` | 弹窗填写驳回原因 | 状态=待审核 |
| 查看详情 | `system:realname:view` | 查看申请详情 | 所有状态 |

## 驳回弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| reject_reason | 驳回原因 | textarea | ✅ | 最长200字 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `system:realname:view` |
| 审核通过 | 通过实名申请 | `system:realname:review` |
| 审核驳回 | 驳回申请，填写原因 | `system:realname:review` |
| 查看证件 | 查看身份证照片大图 | `system:realname:view` |
| 查看详情 | 查看申请详细信息 | `system:realname:view` |

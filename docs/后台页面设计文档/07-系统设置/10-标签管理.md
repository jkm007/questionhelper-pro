# 标签管理

## 基本信息

- 路由路径：`/system/tag`
- 权限标识：`system:tag:view`
- 页面类型：列表页（CURD）

## 关联接口文档

> 详见 [API接口文档/02-用户管理.md](../API接口文档/02-用户管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/tags` | GET | 标签列表 | [链接](../API接口文档/02-用户管理.md) |
| `/api/v1/admin/tags` | POST | 创建标签 | [链接](../API接口文档/02-用户管理.md) |
| `/api/v1/admin/tags/:id` | PUT | 编辑标签 | [链接](../API接口文档/02-用户管理.md) |
| `/api/v1/admin/tags/:id` | DELETE | 删除标签 | [链接](../API接口文档/02-用户管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索标签名称 |
| type | 类型 | select | 全部/系统/自定义 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 标签名称 | string | 150 | - |
| color | 颜色 | color | 60 | 颜色预览 |
| type | 类型 | tag | 80 | 系统/自定义 |
| use_count | 使用次数 | number | 80 | - |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 150 | - |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 标签名称 | input | ✅ | 2-20位 | - |
| color | 颜色 | color-picker | ✅ | - | 选择标签颜色 |
| type | 类型 | radio | ✅ | - | 系统/自定义 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |
| description | 描述 | textarea | - | 最长200字 | - |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `system:tag:view` |
| 新增 | 创建标签 | `system:tag:create` |
| 编辑 | 编辑标签信息 | `system:tag:edit` |
| 删除 | 删除标签 | `system:tag:delete` |
| 批量删除 | 批量删除标签 | `system:tag:delete` |

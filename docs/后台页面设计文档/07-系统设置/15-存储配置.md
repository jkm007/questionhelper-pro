## 存储方式

| 方式 | 说明 |
|------|------|
| 本地存储 | 文件存储在服务器本地磁盘 |
| MinIO | S3兼容的对象存储服务 |
| 阿里云OSS | 阿里云对象存储服务 |
| 腾讯云COS | 腾讯云对象存储服务 |

## 配置项

### 本地存储

| 字段 | 说明 |
|------|------|
| 存储路径 | 文件存储的绝对路径 |
| 访问URL | 文件访问的基础URL |
| 最大文件大小 | 单个文件最大大小（MB） |
| 允许的文件类型 | 允许上传的文件扩展名 |

### MinIO/S3

| 字段 | 说明 |
|------|------|
| Endpoint | 服务端点地址 |
| AccessKey | 访问密钥ID |
| SecretKey | 访问密钥Secret |
| Bucket | 存储桶名称 |
| Region | 区域 |
| 使用SSL | 是否使用HTTPS |
| 访问URL | 文件访问的基础URL |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看当前存储配置 | `system:storage:view` |
| 保存配置 | 保存存储配置 | `system:storage:edit` |
| 测试连接 | 测试存储服务连接 | `system:storage:test` |
| 切换方式 | 切换存储方式 | `system:storage:edit` |

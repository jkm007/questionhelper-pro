# 菜单管理

## 基本信息

- 路由路径：`/system/menu`
- 权限标识：`system:menu:view`
- 页面类型：树形列表页

## 关联接口文档

> 详见 [API接口文档/11-系统设置.md](../API接口文档/11-系统设置.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/menus/tree` | GET | 菜单树 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/menus` | POST | 创建菜单 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/menus/:id` | PUT | 编辑菜单 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/menus/:id` | DELETE | 删除菜单 | [链接](../API接口文档/11-系统设置.md) |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 菜单名称 | string | 200 | 树形缩进显示 |
| icon | 图标 | icon | 60 | 菜单图标 |
| type | 类型 | tag | 80 | 目录/菜单/按钮 |
| path | 路由路径 | string | 150 | - |
| component | 组件路径 | string | 200 | - |
| permission | 权限标识 | string | 150 | - |
| sort_order | 排序 | number | 80 | - |
| visible | 是否显示 | switch | 80 | - |
| status | 状态 | tag | 80 | 启用/禁用 |
| 操作 | 操作 | - | 200 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 添加子菜单 | `system:menu:create` | 在当前菜单下添加 |
| 编辑 | `system:menu:edit` | 编辑菜单信息 |
| 删除 | `system:menu:delete` | 有子菜单时禁止删除 |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| parent_id | 上级菜单 | tree-select | - | - | 空则为根菜单 |
| name | 菜单名称 | input | ✅ | 2-20位 | - |
| type | 菜单类型 | radio | ✅ | - | 目录/菜单/按钮 |
| icon | 菜单图标 | icon-select | - | - | 仅目录/菜单需要 |
| path | 路由路径 | input | - | - | 仅目录/菜单需要 |
| component | 组件路径 | input | - | - | 仅菜单需要 |
| permission | 权限标识 | input | - | - | 仅按钮需要 |
| sort_order | 排序 | number | ✅ | ≥0 | - |
| visible | 是否显示 | switch | ✅ | - | - |
| status | 状态 | switch | ✅ | - | 启用/禁用 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 新增根菜单 | 添加顶级菜单 | `system:menu:create` |
| 添加子菜单 | 在选中菜单下添加 | `system:menu:create` |
| 编辑 | 修改菜单信息 | `system:menu:edit` |
| 删除 | 删除菜单（需无子菜单） | `system:menu:delete` |
| 展开/折叠 | 展开或折叠所有节点 | - |

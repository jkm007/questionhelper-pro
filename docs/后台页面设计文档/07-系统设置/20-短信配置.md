# 短信配置

## 基本信息

- 路由路径：`/setting/sms`
- 权限标识：`system:sms:view`
- 页面类型：配置页

## 短信服务商

| 服务商 | 配置项 |
|--------|--------|
| 阿里云短信 | AccessKey ID、AccessKey Secret、签名 |
| 腾讯云短信 | SecretId、SecretKey、SDK AppID、签名 |

## 短信配置

| 配置项 | 说明 | 必填 |
|--------|------|------|
| 短信开关 | 是否开启短信功能 | - |
| 短信服务商 | 选择服务商 | ✅ |
| AccessKey | 访问密钥 | ✅ |
| SecretKey | 密钥Secret | ✅ |
| 签名 | 短信签名 | ✅ |
| 每分钟限制 | 每分钟最大发送数 | - |
| 每小时限制 | 每小时最大发送数 | - |
| 每日限制 | 每日最大发送数 | - |
| 手机号限制 | 同一手机号每日限制 | - |

## 短信模板

| 字段 | 说明 |
|------|------|
| 模板名称 | 模板显示名称 |
| 模板编码 | 唯一标识 |
| 模板内容 | 短信内容（含变量） |
| 模板ID | 服务商模板ID |
| 状态 | 启用/禁用 |

### 模板变量

| 变量 | 说明 |
|------|------|
| ${code} | 验证码 |
| ${expire} | 有效期（分钟） |
| ${name} | 用户名称 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看短信配置 | `system:sms:view` |
| 保存配置 | 保存短信配置 | `system:sms:edit` |
| 测试发送 | 发送测试短信 | `system:sms:test` |
| 管理模板 | 管理短信模板 | `system:sms:edit` |

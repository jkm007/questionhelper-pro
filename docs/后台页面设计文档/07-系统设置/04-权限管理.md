# 权限管理

## 基本信息

- 路由路径：`/setting/permission`
- 权限标识：`permission:view`
- 页面类型：树形列表页

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 权限名称 | string | 200 | 树形缩进显示 |
| code | 权限标识 | string | 200 | 如 user:list:view |
| type | 类型 | tag | 80 | 目录/菜单/按钮 |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 180 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 添加子权限 | `permission:create` | 在当前权限下添加 |
| 编辑 | `permission:edit` | 编辑权限信息 |
| 删除 | `permission:delete` | 有子权限时禁止删除 |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| parent_id | 上级权限 | tree-select | - | - | 空则为根权限 |
| name | 权限名称 | input | ✅ | 2-20位 | - |
| code | 权限标识 | input | ✅ | 模块:操作 | 全局唯一 |
| type | 类型 | radio | ✅ | - | 目录/菜单/按钮 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |
| description | 描述 | textarea | - | 最长200字 | - |

## 权限标识规范

```
格式：模块:子模块:操作

示例：
- user:list:view      用户列表-查看
- user:list:create    用户列表-新增
- question:public:view  公共题目-查看
- exam:paper:edit     试卷-编辑
- class:member:manage 班级成员-管理
- system:settings:edit 系统设置-编辑
```

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 新增根权限 | 添加顶级权限 | `permission:create` |
| 添加子权限 | 在选中权限下添加 | `permission:create` |
| 编辑 | 修改权限信息 | `permission:edit` |
| 删除 | 删除权限（需无子权限） | `permission:delete` |
| 展开/折叠 | 展开或折叠所有节点 | - |

## 交互说明

1. **树形展示**：默认展开第一级
2. **删除限制**：有子权限时禁止删除
3. **标识唯一**：权限标识全局唯一校验

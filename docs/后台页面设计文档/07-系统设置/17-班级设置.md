## 配置项

### 班级创建

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 允许创建的角色 | 哪些角色可以创建班级 | 管理员、教师、创作者 |
| 班级创建审核 | 创建班级是否需要审核 | 关闭 |
| 最大班级数 | 每人最多创建的班级数 | 10 |

### 班级成员

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 最大成员数 | 每个班级最多成员数 | 500 |
| 加入方式 | 支持的加入方式 | 班级码、邀请链接 |
| 成员审核 | 加入班级是否需要审核 | 关闭 |

### 班级资源

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 允许上传资源 | 是否允许上传文件 | 开启 |
| 最大文件大小 | 单个文件最大大小 | 50MB |
| 允许的文件类型 | 允许上传的文件类型 | pdf,doc等 |
| 最大存储空间 | 每个班级最大存储空间 | 1GB |

### 班级功能

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 班级公告 | 是否开启公告功能 | 开启 |
| 班级讨论 | 是否开启讨论功能 | 开启 |
| 班级作业 | 是否开启作业功能 | 开启 |
| 成绩统计 | 是否开启成绩统计 | 开启 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看当前班级配置 | `system:class:view` |
| 保存配置 | 保存班级配置 | `system:class:edit` |
| 重置 | 恢复默认配置 | `system:class:edit` |

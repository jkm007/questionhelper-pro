# 主题设置

## 基本信息

- 路由路径：`/setting/theme`
- 权限标识：`system:theme:view`
- 页面类型：配置页

## 关联接口文档

> 详见 [API接口文档/11-系统设置.md](../API接口文档/11-系统设置.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/theme-config` | GET | 获取主题配置 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/theme-config` | PUT | 更新主题配置 | [链接](../API接口文档/11-系统设置.md) |

## 主题配置项

| 配置项 | 选项 | 默认值 |
|--------|------|--------|
| 主题模式 | 亮色/暗色/跟随系统 | 亮色 |
| 主题色 | 10种预设+自定义 | 蓝色 |
| 侧边栏样式 | 深色/浅色/主题色 | 深色 |
| 布局模式 | 左侧/顶部/混合 | 左侧 |
| 标签页导航 | 显示/隐藏 | 显示 |
| 面包屑导航 | 显示/隐藏 | 显示 |
| 侧边栏Logo | 显示/隐藏 | 显示 |
| 全屏按钮 | 显示/隐藏 | 显示 |
| 搜索按钮 | 显示/隐藏 | 显示 |
| 水印 | 显示/隐藏 | 显示 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看当前主题配置 | `system:theme:view` |
| 保存配置 | 保存主题配置 | `system:theme:edit` |
| 预览 | 实时预览配置效果 | - |
| 恢复默认 | 恢复默认主题配置 | `system:theme:edit` |

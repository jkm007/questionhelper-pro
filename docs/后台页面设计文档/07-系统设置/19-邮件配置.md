# 邮件配置

## 基本信息

- 路由路径：`/setting/email`
- 权限标识：`system:email:view`
- 页面类型：配置页

## SMTP配置

| 配置项 | 说明 | 必填 |
|--------|------|------|
| 邮件开关 | 是否开启邮件功能 | - |
| SMTP服务器 | 邮件服务器地址 | ✅ |
| 端口 | SMTP端口（25/465/587） | ✅ |
| 使用SSL | 是否使用SSL加密 | - |
| 发件人邮箱 | 发件人邮箱地址 | ✅ |
| 发件人名称 | 发件人显示名称 | ✅ |
| 认证方式 | 账号密码/OAuth2.0 | ✅ |
| 账号 | 邮箱账号 | ✅ |
| 密码 | 邮箱密码 | ✅ |
| 每小时限制 | 每小时最大发送数 | - |
| 每日限制 | 每日最大发送数 | - |

## 邮件模板

| 字段 | 说明 |
|------|------|
| 模板名称 | 模板显示名称 |
| 模板编码 | 唯一标识 |
| 邮件主题 | 邮件标题 |
| 邮件内容 | HTML邮件内容 |
| 状态 | 启用/禁用 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看配置 | 查看邮件配置 | `system:email:view` |
| 保存配置 | 保存邮件配置 | `system:email:edit` |
| 测试发送 | 发送测试邮件 | `system:email:test` |
| 管理模板 | 管理邮件模板 | `system:email:edit` |

# 基础设置

## 基本信息

- 路由路径：`/setting/basic`
- 权限标识：`system:settings:view`
- 页面类型：表单配置页

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/settings/basic` | GET | 获取基础设置 | [链接](../API接口文档/11-系统设置.md) |
| `/api/v1/admin/settings/basic` | PUT | 更新基础设置 | [链接](../API接口文档/11-系统设置.md) |

## Tab 1: 网站信息

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| site_name | 网站名称 | input | ✅ | 最长50字符 |
| site_logo | 网站Logo | upload | - | 图片格式，建议200x50 |
| site_description | 网站描述 | textarea | - | 最长200字符 |
| contact_email | 联系邮箱 | input | - | 邮箱格式 |
| contact_phone | 联系电话 | input | - | 手机号或座机 |
| icp_number | 备案号 | input | - | ICP备案号 |
| copyright | 版权信息 | input | - | 最长100字符 |

## Tab 2: 注册设置

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| register_enabled | 开放注册 | switch | - | 是否允许新用户注册 |
| register_mode | 注册方式 | radio | - | 手机号/邮箱/用户名 |
| register_verify | 注册验证 | checkbox | - | 短信验证码/邮箱验证 |
| register_agreement | 用户协议 | rich-text | - | 注册协议内容 |
| register_captcha | 验证码 | switch | - | 是否开启图形验证码 |

## Tab 3: 登录设置

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| login_captcha | 登录验证码 | switch | - | 是否开启验证码 |
| login_retry_limit | 登录失败限制 | number | - | 失败次数限制 |
| login_lock_duration | 锁定时间 | number | - | 锁定时长（分钟） |
| login_token_expire | Token有效期 | number | - | access_token有效期（分钟） |
| login_refresh_expire | Refresh有效期 | number | - | refresh_token有效期（天） |
| login_single_device | 单设备登录 | switch | - | 是否限制单设备 |

## Tab 4: 文件上传

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| upload_enabled | 开启上传 | switch | - | 是否允许文件上传 |
| upload_max_size | 文件大小限制 | number | - | 最大文件大小（MB） |
| upload_allowed_types | 允许类型 | tag-input | - | 允许的文件扩展名 |
| upload_storage | 存储方式 | radio | - | 本地/MinIO/S3 |
| upload_path | 存储路径 | input | - | 本地存储路径 |

## Tab 5: 邮件配置

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| email_enabled | 开启邮件 | switch | - | 是否开启邮件功能 |
| email_host | SMTP服务器 | input | ✅ | 邮件服务器地址 |
| email_port | 端口 | number | ✅ | SMTP端口 |
| email_username | 用户名 | input | ✅ | 邮箱账号 |
| email_password | 密码 | input(password) | ✅ | 邮箱密码 |
| email_from | 发件人 | input | ✅ | 发件人地址 |
| email_ssl | SSL | switch | - | 是否使用SSL |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 查看设置 | 查看当前配置 | `system:settings:view` |
| 保存设置 | 保存配置修改 | `system:settings:edit` |
| 测试邮件 | 发送测试邮件 | `system:settings:test` |
| 重置 | 恢复默认配置 | `system:settings:edit` |

## 交互说明

1. **保存提示**：保存成功后提示"配置已更新"
2. **测试邮件**：点击后弹窗输入测试邮箱，发送测试邮件
3. **Tab切换**：切换Tab时检查是否有未保存的修改

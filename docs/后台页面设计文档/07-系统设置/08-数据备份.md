# 数据备份

## 基本信息

- 路由路径：`/setting/backup`
- 权限标识：`system:backup:view`
- 页面类型：列表页 + 配置

## Tab 1: 备份列表

### 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 备份名称 | string | 180 | - |
| type | 类型 | tag | 80 | 全量/增量 |
| size | 大小 | string | 80 | 文件大小 |
| status | 状态 | tag | 80 | 成功(success)/失败(danger)/进行中(info) |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 200 | 固定右侧 |

### 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 下载 | `system:backup:download` | 下载备份文件 | 状态=成功 |
| 恢复 | `system:backup:restore` | 恢复到该备份 | 状态=成功 |
| 删除 | `system:backup:delete` | 删除备份 | 所有状态 |

## Tab 2: 备份配置

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| auto_backup | 自动备份 | switch | - | 是否开启自动备份 |
| backup_frequency | 备份频率 | select | ✅ | 每天/每周/每月 |
| backup_time | 备份时间 | time | ✅ | 执行时间 |
| backup_type | 备份类型 | radio | ✅ | 全量/增量 |
| retention_count | 保留份数 | number | ✅ | 最多保留份数 |
| backup_path | 存储路径 | input | ✅ | 备份文件存储路径 |
| notify_enabled | 备份通知 | switch | - | 备份完成后通知 |
| notify_email | 通知邮箱 | input | - | 接收通知的邮箱 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 立即备份 | 手动触发备份 | `system:backup:create` |
| 下载 | 下载备份文件 | `system:backup:download` |
| 恢复 | 恢复到指定备份 | `system:backup:restore` |
| 删除 | 删除备份文件 | `system:backup:delete` |
| 批量删除 | 批量删除备份 | `system:backup:delete` |
| 保存配置 | 保存备份配置 | `system:backup:edit` |

## 交互说明

1. **备份确认**：立即备份前二次确认
2. **恢复确认**：恢复前二次确认，提示"将覆盖当前数据"
3. **恢复进度**：显示恢复进度条
4. **备份中**：备份进行中时显示进度

# 日志管理

## 基本信息

- 路由路径：`/setting/log`
- 权限标识：`system:logs:view`
- 页面类型：列表页（多Tab）

## 关联接口文档

> 详见 [API接口文档/12-管理后台.md](../API接口文档/12-管理后台.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/logs/login` | GET | 登录日志列表 | [链接](../API接口文档/12-管理后台.md) |
| `/api/v1/admin/logs/login/export` | GET | 导出登录日志 | [链接](../API接口文档/12-管理后台.md) |
| `/api/v1/admin/logs/login/clean` | POST | 清空登录日志 | [链接](../API接口文档/12-管理后台.md) |
| `/api/v1/admin/logs/system` | GET | 系统日志列表 | [链接](../API接口文档/12-管理后台.md) |
| `/api/v1/admin/logs/error` | GET | 错误日志列表 | [链接](../API接口文档/12-管理后台.md) |

## Tab 1: 登录日志

### 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| username | 用户名 | input | 搜索登录用户名 |
| status | 状态 | select | 全部/成功/失败 |
| login_time | 登录时间 | daterange | 日期范围 |

### 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| username | 用户名 | string | 120 | - |
| login_ip | 登录IP | string | 120 | - |
| login_location | 登录地点 | string | 120 | IP归属地 |
| browser | 浏览器 | string | 100 | 浏览器类型 |
| os | 操作系统 | string | 100 | 操作系统 |
| status | 状态 | tag | 80 | 成功(success)/失败(danger) |
| message | 提示信息 | string | 150 | 失败原因 |
| login_time | 登录时间 | datetime | 160 | - |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `system:logs:view` |
| 查看详情 | 查看日志详细信息 | `system:logs:view` |
| 导出 | 导出日志为Excel | `system:logs:export` |
| 清空 | 清空所有日志 | `system:logs:clean` |
| Tab切换 | 切换日志类型 | - |

## 交互说明

1. **Tab切换**：切换日志类型，重新加载数据
2. **清空确认**：清空前二次确认
3. **堆栈折叠**：错误日志堆栈默认折叠

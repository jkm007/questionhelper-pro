- 权限标识：无需登录

## 表单字段

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| username | 用户名 | input | ✅ | 长度 4-20 位 | 支持用户名/手机号/邮箱 |
| password | 密码 | input(password) | ✅ | 长度 6-20 位 | 显示/隐藏密码切换 |
| captcha | 验证码 | input | ✅ | 长度 4 位 | 点击图片刷新 |
| remember | 记住密码 | checkbox | - | - | 存储到 localStorage |

## 功能点

| 功能 | 说明 |
|------|------|
| 登录 | 表单验证通过后调用登录接口，成功后跳转到首页或 redirect 参数指定页面 |
| 验证码 | 页面加载时获取验证码，点击图片刷新，登录失败后自动刷新 |
| 记住密码 | 勾选后将用户名加密存储到 localStorage |
| 回车登录 | 焦点在表单内时，按 Enter 触发登录 |
| 注册跳转 | 点击"注册"跳转到注册页 |

## 交互说明

1. **加载状态**：页面加载时自动请求验证码
2. **登录中**：按钮显示 loading，禁止重复提交
3. **登录成功**：提示"登录成功"，跳转到首页
4. **登录失败**：提示错误信息，刷新验证码，清空密码
5. **Token 过期**：自动刷新 Token，失败则跳转登录页

## 样式要点

- 背景：渐变背景或背景图片
- 登录卡片：居中显示，宽度 400px，白色背景，圆角 8px，阴影
- Logo：居中显示，上方留白
- 按钮：宽度 100%，高度 40px
- 响应式：移动端卡片宽度 90%

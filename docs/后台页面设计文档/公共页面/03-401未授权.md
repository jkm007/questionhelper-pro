# 401 未授权页

## 基本信息

- 路由路径：`/401`
- 权限标识：无需登录
- 页面类型：错误提示页

## 页面布局

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│                                                                  │
│                                                                  │
│                        ┌─────────────────────┐                  │
│                        │                     │                  │
│                        │       401           │                  │
│                        │                     │                  │
│                        │   抱歉，您未登录     │                  │
│                        │   或登录已过期       │                  │
│                        │                     │                  │
│                        │   请重新登录后访问   │                  │
│                        │                     │                  │
│                        │  ┌───────────────┐  │                  │
│                        │  │   去登录      │  │                  │

```

## 功能点

| 功能 | 说明 |
|------|------|
| 去登录 | 跳转到登录页，携带当前页面路径作为 redirect 参数 |
| 返回首页 | 跳转到首页（已登录状态）或登录页（未登录状态） |

## 触发场景

1. 未登录访问需要认证的页面
2. Token 过期且刷新失败
3. 接口返回 401 状态码

## 样式要点

- 页面居中显示
- 401 数字：大号字体，120px，灰色
- 提示文字：18px，次要文字色
- 按钮：水平排列，Primary + Info
- 背景：浅灰色或渐变

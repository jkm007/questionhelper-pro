## 表单字段

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| username | 用户名 | input | ✅ | 长度 4-20 位，字母开头 | 唯一性后端验证 |
| phone | 手机号 | input | ✅ | 手机号格式 | 用于短信验证 |
| sms_code | 短信验证码 | input | ✅ | 6 位数字 | 60秒倒计时重发 |
| password | 密码 | input(password) | ✅ | 长度 6-20 位，包含字母和数字 | 密码强度提示 |
| confirm_password | 确认密码 | input(password) | ✅ | 与密码一致 | - |
| captcha | 图形验证码 | input | ✅ | 4 位字符 | 防机器人 |
| agree | 用户协议 | checkbox | ✅ | 必须勾选 | 链接到协议详情 |

## 功能点

| 功能 | 说明 |
|------|------|
| 注册 | 表单验证通过后调用注册接口，成功后跳转登录页 |
| 短信验证 | 点击"发送"按钮发送短信，60秒倒计时 |
| 用户名检查 | 输入框失焦时检查用户名是否已存在 |
| 密码强度 | 实时显示密码强度（弱/中/强） |
| 协议链接 | 点击"用户协议"弹窗显示协议内容 |

## 交互说明

1. **用户名检查**：失焦时异步检查，已存在显示红色提示
2. **短信发送**：点击后按钮禁用，显示倒计时
3. **密码强度**：根据长度和字符类型实时计算
4. **注册成功**：提示"注册成功"，自动跳转登录页
5. **注册失败**：提示具体错误信息

## 样式要点

- 与登录页风格一致
- 密码强度条：红/橙/绿三色
- 响应式：移动端卡片宽度 90%

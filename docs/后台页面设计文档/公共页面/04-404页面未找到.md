# 404 页面未找到

## 基本信息

- 路由路径：`/404`
- 权限标识：无需登录
- 页面类型：错误提示页

## 页面布局

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│                                                                  │
│                                                                  │
│                        ┌─────────────────────┐                  │
│                        │                     │                  │
│                        │       404           │                  │
│                        │                     │                  │
│                        │   抱歉，您访问的     │                  │
│                        │   页面不存在         │                  │
│                        │                     │                  │
│                        │   请检查链接是否正确  │                  │
│                        │                     │                  │
│                        │  ┌───────────────┐  │                  │
│                        │  │   返回首页    │  │                  │
│                        │  └───────────────┘  │                  │
│                        │                     │                  │
│                        │  ┌───────────────┐  │                  │
│                        │  │   返回上页    │  │                  │
│                        │  └───────────────┘  │                  │
│                        │                     │                  │
│                        └─────────────────────┘                  │
│                                                                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 功能点

| 功能 | 说明 |
|------|------|
| 返回首页 | 跳转到首页 |
| 返回上页 | 浏览器后退（history.back） |

## 触发场景

1. 访问不存在的路由路径
2. 动态路由加载失败
3. 后端菜单配置错误

## 样式要点

- 与 401 页面风格一致
- 404 数字：大号字体，120px，灰色
- 可选：添加 404 插图/SVG
- 按钮：水平排列，Primary + Info

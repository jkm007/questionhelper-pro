## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 规则名称 | string | 150 | - |
| type | 规则类型 | tag | 100 | 关键词/正则/长度 |
| match_type | 匹配方式 | tag | 80 | 包含/等于/正则匹配 |
| action | 处理动作 | tag | 100 | 自动屏蔽/人工审核/自动通过 |
| priority | 优先级 | number | 60 | 数字越小优先级越高 |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 150 | - |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 规则名称 | input | ✅ | 2-30位 | - |
| type | 规则类型 | radio | ✅ | - | 关键词/正则/长度 |
| match_type | 匹配方式 | radio | ✅ | - | 包含/等于/正则匹配 |
| keywords | 关键词 | textarea | ✅ | - | 每行一个关键词 |
| action | 处理动作 | radio | ✅ | - | 自动屏蔽/人工审核/自动通过 |
| priority | 优先级 | number | ✅ | ≥0 | 数字越小越优先 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |
| description | 描述 | textarea | - | 最长200字 | - |

| 功能 | 说明 | 权限 |

| 删除 | 删除规则 | `content:audit-rule:delete` |
| 批量删除 | 批量删除规则 | `content:audit-rule:delete` |
| 状态切换 | 启用/禁用规则 | `content:audit-rule:edit` |

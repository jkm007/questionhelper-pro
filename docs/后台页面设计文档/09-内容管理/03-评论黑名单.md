# 评论黑名单

## 基本信息

- 路由路径：`/content/blacklist`
- 权限标识：`content:blacklist:view`
- 页面类型：列表页（CURD）

## 关联接口文档

> 详见 [API接口文档/08-评论社区.md](../API接口文档/08-评论社区.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/blacklist` | GET | 黑名单列表 | [链接](../API接口文档/08-评论社区.md) |
| `/api/v1/admin/blacklist` | POST | 添加黑名单 | [链接](../API接口文档/08-评论社区.md) |
| `/api/v1/admin/blacklist/:id` | DELETE | 移除黑名单 | [链接](../API接口文档/08-评论社区.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索用户名/昵称 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| username | 用户名 | string | 120 | - |
| nickname | 昵称 | string | 100 | - |
| ban_reason | 封禁原因 | string | 200 | - |
| ban_time | 封禁时间 | datetime | 160 | - |
| unban_time | 解封时间 | datetime | 160 | 永久显示"永久" |
| created_at | 添加时间 | datetime | 160 | - |
| 操作 | 操作 | - | 120 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 移除 | `content:blacklist:delete` | 从黑名单移除 |

## 表单字段（添加用户弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| user_id | 用户 | user-select | ✅ | - | 搜索选择用户 |
| ban_reason | 封禁原因 | textarea | ✅ | 最长200字 | - |
| ban_duration | 封禁时长 | radio | ✅ | - | 7天/30天/永久 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 搜索黑名单用户 | `content:blacklist:view` |
| 添加用户 | 添加用户到黑名单 | `content:blacklist:create` |
| 移除 | 从黑名单移除 | `content:blacklist:delete` |
| 批量移除 | 批量移除黑名单 | `content:blacklist:delete` |

# 举报管理

## 基本信息

- 路由路径：`/content/report`
- 权限标识：`content:report:view`
- 页面类型：列表页（审核）

|------|------|------|----------|
| `/api/v1/admin/reports` | GET | 举报列表 | [链接](../API接口文档/08-评论社区.md) |
| `/api/v1/admin/reports/:id/handle` | POST | 处理举报 | [链接](../API接口文档/08-评论社区.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索举报内容 |
| status | 状态 | select | 全部/待处理/已处理/已驳回 |
| type | 举报类型 | select | 垃圾广告/人身攻击/色情低俗/其他 |
| created_at | 举报时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| content | 举报内容 | string | 200 | 被举报的内容摘要 |
| reporter | 举报人 | string | 100 | - |
| reported_user | 被举报人 | string | 100 | - |
| report_type | 举报类型 | tag | 100 | 垃圾广告/人身攻击/色情低俗/其他 |
| status | 状态 | tag | 80 | 待处理(warning)/已处理(success)/已驳回(info) |
| created_at | 举报时间 | datetime | 160 | - |
| 操作 | 操作 | - | 180 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 处理 | `content:report:handle` | 处理举报 |
| 查看详情 | `content:report:view` | 查看举报详情 |
| 驳回 | `content:report:handle` | 驳回举报 |

## 处理弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| action | 处理方式 | radio | ✅ | 删除内容/警告用户/封禁用户 |
| reason | 处理说明 | textarea | ✅ | 最长200字 |
| ban_days | 封禁天数 | number | - | 仅封禁用户时需要 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `content:report:view` |
| 处理举报 | 处理被举报内容 | `content:report:handle` |
| 驳回举报 | 驳回无效举报 | `content:report:handle` |
| 查看详情 | 查看举报详细信息 | `content:report:view` |

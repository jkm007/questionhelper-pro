# 评论管理

## 基本信息

- 路由路径：`/content/comment`
- 权限标识：`content:comment:view`
- 页面类型：列表页（审核）

|------|------|------|----------|
| `/api/v1/admin/comments` | GET | 评论列表 | [链接](../API接口文档/08-评论社区.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索评论内容/评论人 |
| status | 状态 | select | 全部/待审核/已通过/已屏蔽 |
| created_at | 评论时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| content | 评论内容 | string | 300 | 溢出省略 |
| user | 评论人 | string | 100 | - |
| target | 关联内容 | string | 150 | 被评论的题目/考试 |
| status | 状态 | tag | 80 | 待审核(warning)/已通过(success)/已屏蔽(danger) |
| is_pinned | 是否置顶 | tag | 60 | 置顶标签 |
| is_featured | 是否精选 | tag | 60 | 精选标签 |
| created_at | 评论时间 | datetime | 160 | - |
| 操作 | 操作 | - | 250 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 通过 | `content:comment:audit` | 审核通过 |
| 屏蔽 | `content:comment:audit` | 屏蔽评论 |
| 置顶 | `content:comment:pin` | 置顶/取消置顶 |
| 精选 | `content:comment:feature` | 精选/取消精选 |
| 删除 | `content:comment:delete` | 删除评论 |
| 查看详情 | `content:comment:view` | 查看评论详情 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `content:comment:view` |
| 批量审核 | 批量通过/屏蔽 | `content:comment:audit` |
| 置顶 | 置顶重要评论 | `content:comment:pin` |
| 精选 | 标记优质评论 | `content:comment:feature` |
| 删除 | 删除评论 | `content:comment:delete` |
| 导出 | 导出评论数据 | `content:comment:export` |

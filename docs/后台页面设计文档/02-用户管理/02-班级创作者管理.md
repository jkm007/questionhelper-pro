# 班级创作者管理

## 基本信息

- 路由路径：`/user/creator`
- 权限标识：`class:creator:view`
- 页面类型：列表页

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/creators` | GET | 创作者列表 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/creators/:id/approve` | POST | 审批通过 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/creators/:id/reject` | POST | 审批驳回 | [链接](../API接口文档/05-班级管理.md) |
| `/api/v1/admin/creators/:id/revoke` | POST | 撤销创作者资格 | [链接](../API接口文档/05-班级管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索用户名/昵称 |
| status | 状态 | select | 全部/待审核/已通过/已驳回 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| id | ID | number | 60 | - |
| username | 用户名 | string | 120 | - |
| nickname | 昵称 | string | 120 | - |
| apply_reason | 申请说明 | string | 200 | 溢出省略提示 |
| status | 状态 | tag | 100 | 待审核(warning)/已通过(success)/已驳回(danger) |
| apply_time | 申请时间 | datetime | 160 | - |
| 审批时间 | 审批时间 | datetime | 160 | 仅已审核状态显示 |
| 操作 | 操作 | - | 180 | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 通过 | `class:creator:approve` | 二次确认后通过 | 状态=待审核 |
| 驳回 | `class:creator:approve` | 弹窗填写驳回原因 | 状态=待审核 |
| 撤销 | `class:creator:revoke` | 二次确认后撤销资格 | 状态=已通过 |
| 查看详情 | `class:creator:view` | 查看申请详情 | 所有状态 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `class:creator:view` |
| 审批通过 | 通过创作者申请 | `class:creator:approve` |
| 审批驳回 | 驳回申请，填写原因 | `class:creator:approve` |
| 撤销资格 | 撤销已通过的创作者 | `class:creator:revoke` |
| 查看详情 | 查看申请详细信息 | `class:creator:view` |

## 交互说明

1. **审批通过**：二次确认弹窗，确认后调用通过接口
2. **审批驳回**：弹窗填写驳回原因（必填），提交后调用驳回接口
3. **撤销资格**：二次确认弹窗，提示"撤销后该用户将无法创建班级"
4. **查看详情**：抽屉或弹窗显示申请详情

## 驳回弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| reject_reason | 驳回原因 | textarea | ✅ | 最长200字 |

## 样式要点

- 状态标签颜色：待审核(warning)、已通过(success)、已驳回(danger)
- 操作按钮：通过(success)、驳回(danger)、撤销(warning)

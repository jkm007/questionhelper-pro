# 试卷列表

## 基本信息

- 路由路径：`/exam/paper`
- 权限标识：`exam:paper:view`
- 页面类型：列表页（CURD）

| `/api/v1/admin/papers` | GET | 试卷列表 | [链接](../API接口文档/04-考试管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索试卷名称 |
| status | 状态 | select | 草稿/已发布/已关闭 |
| created_at | 创建时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 排序 | 说明 |
|--------|------|------|------|------|------|
| id | ID | number | 60 | ✅ | - |
| name | 试卷名称 | string | 200 | - | 可点击跳转详情 |
| question_count | 题目数 | number | 80 | - | - |
| total_score | 总分 | number | 80 | - | 显示"XX分" |
| duration | 时长 | number | 80 | - | 显示"XX分钟" |
| status | 状态 | tag | 80 | - | 草稿(info)/已发布(success)/已关闭(danger) |
| use_count | 使用次数 | number | 80 | ✅ | 被考试使用次数 |
| created_at | 创建时间 | datetime | 160 | ✅ | - |
| 操作 | 操作 | - | 250 | - | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 编辑 | `exam:paper:edit` | 跳转组卷页面 | 状态=草稿 |
| 预览 | `exam:paper:view` | 弹窗预览试卷 | 所有状态 |
| 复制 | `exam:paper:create` | 复制为新试卷 | 所有状态 |
| 删除 | `exam:paper:delete` | 二次确认后删除 | 状态=草稿 |
| 查看成绩 | `exam:score:view` | 查看使用该试卷的考试成绩 | 状态=已发布 |

## 表单字段（创建试卷弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 试卷名称 | input | ✅ | 2-50位 | - |
| description | 试卷说明 | textarea | - | 最长500字 | - |
| total_score | 总分 | number | ✅ | 1-1000 | 默认100 |
| duration | 考试时长 | number | ✅ | 1-300分钟 | 默认120分钟 |
| pass_score | 及格分 | number | ✅ | ≤总分 | 默认60 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `exam:paper:view` |
| 创建试卷 | 弹窗填写基本信息，跳转组卷 | `exam:paper:create` |
| 编辑 | 跳转组卷页面编辑 | `exam:paper:edit` |
| 删除 | 二次确认后删除（已发布禁止删除） | `exam:paper:delete` |
| 复制 | 复制为新试卷 | `exam:paper:create` |
| 预览 | 弹窗预览试卷完整内容 | `exam:paper:view` |

## 样式要点

- 试卷名称：可点击，hover变色
- 状态标签：草稿(info)/已发布(success)/已关闭(danger)
- 预览弹窗：宽度800px，可滚动

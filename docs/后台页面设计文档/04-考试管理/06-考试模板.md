# 考试模板

## 基本信息

- 路由路径：`/exam/template`
- 权限标识：`exam:template:view`
- 页面类型：列表页（CURD）

## 关联接口文档

> 详见 [API接口文档/04-考试管理.md](../API接口文档/04-考试管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/exam-templates` | GET | 模板列表 | [链接](../API接口文档/04-考试管理.md) |
| `/api/v1/admin/exam-templates` | POST | 创建模板 | [链接](../API接口文档/04-考试管理.md) |
| `/api/v1/admin/exam-templates/:id` | PUT | 编辑模板 | [链接](../API接口文档/04-考试管理.md) |
| `/api/v1/admin/exam-templates/:id` | DELETE | 删除模板 | [链接](../API接口文档/04-考试管理.md) |
| `/api/v1/admin/exam-templates/:id/create` | POST | 从模板创建考试 | [链接](../API接口文档/04-考试管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索模板名称 |
| sort_by | 排序 | select | 使用次数/创建时间 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 模板名称 | string | 180 | - |
| paper_name | 试卷 | string | 120 | 关联的试卷 |
| duration | 时长 | string | 80 | 考试时长 |
| pass_score | 及格分 | string | 80 | - |
| use_count | 使用次数 | number | 80 | - |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 200 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 使用模板 | `exam:template:create` | 从模板创建考试 |
| 编辑 | `exam:template:edit` | 编辑模板信息 |
| 删除 | `exam:template:delete` | 删除模板 |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 模板名称 | input | ✅ | 2-30位 | - |
| paper_id | 试卷 | select | ✅ | - | 选择试卷 |
| duration | 考试时长 | number | ✅ | 分钟数 | - |
| pass_score | 及格分 | number | ✅ | ≤试卷总分 | - |
| max_attempts | 最大考试次数 | number | - | 1-10 | 默认1次 |
| show_answer | 考后显示答案 | switch | - | - | 默认关闭 |
| anti_cheat | 防作弊设置 | checkbox | - | - | 切屏检测等 |
| description | 模板说明 | textarea | - | 最长200字 | - |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 搜索模板 | `exam:template:view` |
| 新增 | 创建考试模板 | `exam:template:create` |
| 编辑 | 编辑模板信息 | `exam:template:edit` |
| 删除 | 删除模板 | `exam:template:delete` |
| 使用模板 | 从模板快速创建考试 | `exam:template:create` |

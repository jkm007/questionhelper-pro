# 组卷管理

## 基本信息

- 路由路径：`/exam/paper/:id/edit`
- 权限标识：`exam:paper:edit`
- 页面类型：编辑页面（组卷向导）

## 试卷信息区

| 字段 | 类型 | 说明 |
|------|------|------|
| 试卷名称 | input | 可编辑 |
| 总分 | number | 自动计算或手动设置 |
| 及格分 | number | 必须≤总分 |
| 时长 | number | 考试时长（分钟） |
| 说明 | textarea | 试卷说明 |

## 题目统计区

| 题型 | 题目数 | 分值 |
|------|--------|------|
| 单选题 | 自动统计 | 自动计算 |
| 多选题 | 自动统计 | 自动计算 |
| 判断题 | 自动统计 | 自动计算 |
| 填空题 | 自动统计 | 自动计算 |
| 简答题 | 自动统计 | 自动计算 |
| **合计** | 自动统计 | 自动计算 |

## 题目列表区

| 功能 | 说明 |
|------|------|

## 功能点

| 功能 | 说明 |
|------|------|
| 保存 | 保存试卷信息和题目配置 |
| 发布 | 发布试卷（需至少1道题目） |
| 手动选题 | 从题库选择题目添加到试卷 |
| 随机组卷 | 按规则随机抽取题目 |
| 题目排序 | 拖拽调整题目顺序 |
| 修改分值 | 直接编辑每题分值 |
| 删除题目 | 从试卷移除题目 |
| 预览 | 预览试卷完整内容 |

## 交互说明

1. **自动保存**：每30秒自动保存草稿
2. **分值联动**：修改分值后自动更新统计
3. **拖拽排序**：支持题目拖拽调整顺序
4. **发布校验**：发布前检查题目数量和总分

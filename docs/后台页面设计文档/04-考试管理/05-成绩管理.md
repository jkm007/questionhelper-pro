# 成绩管理

## 基本信息

- 路由路径：`/exam/score`
- 权限标识：`exam:score:view`
- 页面类型：列表页 + 统计分析

## 成绩统计卡片

| 卡片 | 字段 | 说明 |
|------|------|------|
| 参考人数 | total | 已交卷学生数 |
| 平均分 | avg_score | 所有已交卷学生的平均分 |
| 最高分 | max_score | 最高分数 |
| 及格率 | pass_rate | 及格人数/总人数 |

## 分数分布图

- 类型：柱状图
- X轴：分数段（0-59, 60-69, 70-79, 80-89, 90-100）
- Y轴：人数
- 颜色：不及格(红)、及格(蓝)、良好(绿)、优秀(深绿)

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索学生姓名/学号 |
| status | 状态 | select | 全部/及格/不及格 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 排序 | 说明 |
|--------|------|------|------|------|------|
| student_name | 学生姓名 | string | 120 | - | - |
| student_no | 学号 | string | 120 | - | - |
| score | 得分 | number | 80 | ✅ | 显示分数/总分 |
| status | 状态 | tag | 80 | - | 及格(success)/不及格(danger) |
| duration | 用时 | string | 80 | ✅ | 答题用时 |
| submit_time | 交卷时间 | datetime | 160 | ✅ | - |
| review_status | 复核状态 | tag | 80 | - | 未复核/已复核 |
| 操作 | 操作 | - | 180 | - | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 查看答卷 | `exam:score:view` | 查看学生答卷详情 |
| 复核 | `exam:score:review` | 对成绩进行复核 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 选择考试 | 下拉选择考试，切换成绩数据 | `exam:score:view` |
| 搜索 | 按学生姓名/学号搜索 | `exam:score:view` |
| 导出成绩 | 导出为Excel | `exam:score:export` |
| 批量复核 | 批量标记为已复核 | `exam:score:review` |
| 查看答卷 | 查看学生详细答卷 | `exam:score:view` |
| 成绩复核 | 对成绩进行复核标注 | `exam:score:review` |

## 交互说明

1. **切换考试**：下拉选择考试后，自动刷新统计数据和列表
2. **图表交互**：点击分数段柱状图，筛选该分数段的学生
3. **导出**：导出当前考试的全部成绩数据
4. **复核**：弹窗填写复核备注，标记为已复核

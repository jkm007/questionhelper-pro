# 成绩复核

## 基本信息

- 路由路径：`/exam/score-review`
- 权限标识：`exam:score-review:view`
- 页面类型：列表页（审核）

## 关联接口文档

> 详见 [API接口文档/04-考试管理.md](../API接口文档/04-考试管理.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/score-reviews` | GET | 复核申请列表 | [链接](../API接口文档/04-考试管理.md) |
| `/api/v1/admin/score-reviews/:id/handle` | POST | 处理复核 | [链接](../API接口文档/04-考试管理.md) |

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索申请人姓名 |
| exam_id | 考试 | select | 选择考试 |
| status | 状态 | select | 全部/待审核/已通过/已驳回 |
| created_at | 申请时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| applicant | 申请人 | string | 100 | - |
| exam_name | 考试名称 | string | 150 | - |
| original_score | 原始分数 | number | 80 | - |
| apply_reason | 申请原因 | string | 200 | 溢出省略 |
| status | 状态 | tag | 100 | 待审核(warning)/已通过(success)/已驳回(danger) |
| created_at | 申请时间 | datetime | 160 | - |
| review_score | 复核分数 | number | 80 | 仅已处理显示 |
| 操作 | 操作 | - | 150 | - |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 处理 | `exam:score-review:handle` | 处理复核申请 | 状态=待审核 |
| 查看详情 | `exam:score-review:view` | 查看申请详情 | 所有状态 |

## 处理弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| result | 处理结果 | radio | ✅ | 通过/驳回 |
| review_score | 复核分数 | number | - | 仅通过时需要 |
| reason | 处理说明 | textarea | ✅ | 最长200字 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `exam:score-review:view` |
| 处理复核 | 处理复核申请 | `exam:score-review:handle` |
| 查看详情 | 查看申请详情和答卷 | `exam:score-review:view` |

# 考试发布

## 基本信息

- 路由路径：`/exam/list`
- 权限标识：`exam:publish:view`
- 页面类型：列表页 + 发布向导

## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索考试名称 |
| status | 状态 | select | 未开始/进行中/已结束/已关闭 |
| exam_time | 考试时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 考试名称 | string | 180 | 可点击查看详情 |
| paper_name | 试卷 | string | 120 | 关联的试卷 |
| class_name | 班级 | string | 120 | 参考班级 |
| start_time | 开始时间 | datetime | 160 | - |
| end_time | 结束时间 | datetime | 160 | - |
| participant | 参加人数 | string | 100 | "已参/应参" |
| status | 状态 | tag | 80 | 未开始(info)/进行中(success)/已结束(warning)/已关闭(danger) |
| 操作 | 操作 | - | 250 | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 编辑 | `exam:publish:edit` | 编辑考试配置 | 状态=未开始 |
| 发布 | `exam:publish:publish` | 发布考试 | 状态=草稿 |
| 监控 | `exam:monitor:view` | 跳转考试监控 | 状态=进行中 |
| 成绩 | `exam:score:view` | 跳转成绩管理 | 状态=已结束 |
| 关闭 | `exam:publish:close` | 关闭考试 | 状态=进行中 |
| 删除 | `exam:publish:delete` | 删除考试 | 状态=未开始/草稿 |

## 表单字段（创建/编辑考试弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 考试名称 | input | ✅ | 2-50位 | - |
| paper_id | 试卷 | select | ✅ | - | 选择已发布的试卷 |
| class_ids | 参考班级 | select(multiple) | ✅ | 至少选择一个 | 多选班级 |
| start_time | 开始时间 | datetime | ✅ | 必须大于当前时间 | - |
| end_time | 结束时间 | datetime | ✅ | 必须大于开始时间 | - |
| pass_score | 及格分 | number | - | ≤试卷总分 | 默认使用试卷及格分 |
| max_attempts | 最大考试次数 | number | - | 1-10 | 默认1次 |
| show_answer | 考后显示答案 | switch | - | - | 默认关闭 |
| anti_cheat | 防作弊设置 | checkbox | - | - | 切屏检测、禁止复制等 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `exam:publish:view` |
| 创建考试 | 弹窗填写考试信息 | `exam:publish:create` |
| 编辑 | 编辑考试配置 | `exam:publish:edit` |
| 发布 | 发布考试，通知学生 | `exam:publish:publish` |
| 关闭 | 关闭进行中的考试 | `exam:publish:close` |
| 删除 | 删除未开始的考试 | `exam:publish:delete` |
| 监控 | 跳转到考试监控页面 | `exam:monitor:view` |
| 成绩 | 跳转到成绩管理页面 | `exam:score:view` |

## 交互说明

1. **时间校验**：结束时间必须大于开始时间
2. **发布确认**：发布前二次确认，提示"发布后将通知学生"
3. **关闭确认**：关闭前二次确认，提示"关闭后学生无法继续答题"
4. **状态流转**：草稿→未开始→进行中→已结束

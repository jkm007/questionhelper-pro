# 考试监控

## 基本信息

- 路由路径：`/exam/monitor/:id`
- 权限标识：`exam:monitor:view`
- 页面类型：实时监控页

## 实时统计卡片

| 卡片 | 字段 | 说明 |
|------|------|------|
| 应参人数 | total_students | 应该参加考试的学生总数 |
| 已参人数 | joined_students | 已经进入考试的学生数 |
| 答题中 | answering | 当前正在答题的学生数 |
| 已交卷 | submitted | 已经交卷的学生数 |
| 平均分 | avg_score | 已交卷学生的平均分 |
| 最高分 | max_score | 最高分数 |
| 最低分 | min_score | 最低分数 |
| 及格率 | pass_rate | 及格人数/已交卷人数 |

## 学生列表字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 学生姓名 | string | 120 | - |
| status | 状态 | tag | 80 | 答题中(success)/已交卷(info)/未参加(warning)/异常(danger) |
| progress | 答题进度 | progress | 120 | 进度条 + 百分比 |
| current_question | 当前题 | string | 80 | 当前正在作答的题目 |
| score | 得分 | number | 80 | 已交卷显示分数 |
| submit_time | 交卷时间 | datetime | 160 | - |
| abnormal | 异常次数 | number | 80 | 切屏、复制等 |
| 操作 | 操作 | - | 180 | 固定右侧 |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 踢出 | `exam:monitor:kick` | 将学生踢出考试 | 状态=答题中 |
| 强制交卷 | `exam:monitor:force-submit` | 强制学生交卷 | 状态=答题中 |
| 查看答卷 | `exam:monitor:view` | 查看学生答卷详情 | 状态=已交卷 |
| 延长时间 | `exam:monitor:extend` | 为学生延长考试时间 | 状态=答题中 |

## 异常行为列表

| 异常类型 | 图标 | 说明 |
|----------|------|------|
| 切屏 | ⚠️ | 学生切换了浏览器标签/窗口 |
| 复制粘贴 | ⚠️ | 学生进行了复制粘贴操作 |
| 长时间未操作 | ⚠️ | 超过5分钟未进行任何操作 |
| IP变更 | ⚠️ | 考试过程中IP地址发生变化 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 实时刷新 | 每10秒自动刷新数据 | `exam:monitor:view` |
| 踢出学生 | 二次确认后踢出 | `exam:monitor:kick` |
| 强制交卷 | 二次确认后强制交卷 | `exam:monitor:force-submit` |
| 延长时间 | 弹窗设置延长时间 | `exam:monitor:extend` |
| 发布公告 | 弹窗编辑公告内容 | `exam:monitor:announce` |
| 结束考试 | 二次确认后结束考试 | `exam:monitor:close` |
| 查看答卷 | 跳转到答卷详情 | `exam:monitor:view` |
| 筛选学生 | 按状态筛选学生 | - |
| 导出数据 | 导出监控数据 | `exam:monitor:export` |

## 弹窗

### 延长时间弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| minutes | 延长时间 | number | ✅ | 单位：分钟 |
| reason | 延长原因 | textarea | - | 可选 |

### 发布公告弹窗

| 字段 | 标签 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| content | 公告内容 | textarea | ✅ | 最长500字 |

## 交互说明

1. **自动刷新**：页面加载后每10秒自动刷新
2. **实时更新**：学生状态变化时实时更新列表
3. **异常高亮**：异常行为列表高亮显示
4. **操作确认**：踢出、强制交卷等操作需二次确认

## 样式要点

- 统计卡片：4列等分，间距16px
- 状态标签：答题中(success)/已交卷(info)/未参加(warning)/异常(danger)
- 进度条：绿色(>60%)/橙色(30-60%)/红色(<30%)
- 异常行为：黄色背景，警告图标

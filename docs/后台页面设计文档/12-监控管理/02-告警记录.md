## 搜索字段

| 字段名 | 标签 | 类型 | 说明 |
|--------|------|------|------|
| keyword | 关键词 | input | 搜索告警名称 |
| level | 告警级别 | select | 严重/警告/提示 |
| status | 状态 | select | 告警中/已恢复/已处理/已忽略 |
| target | 监控目标 | select | 服务器/数据库/API/Redis |
| created_at | 告警时间 | daterange | 日期范围 |

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 告警名称 | string | 180 | - |
| target | 监控目标 | tag | 100 | 服务器/数据库/API/Redis |
| level | 告警级别 | tag | 80 | 严重(danger)/警告(warning)/提示(info) |
| trigger_value | 触发值 | string | 80 | 当前值 |
| threshold | 阈值 | string | 80 | 触发阈值 |
| status | 状态 | tag | 80 | 告警中(danger)/已恢复(success)/已处理(info)/已忽略(warning) |
| created_at | 告警时间 | datetime | 160 | - |
| resolved_at | 恢复时间 | datetime | 160 | - |
| handled_at | 处理时间 | datetime | 160 | - |
| 操作 | 操作 | - | 150 | - |

## 操作列

| 按钮 | 权限 | 说明 | 显示条件 |
|------|------|------|----------|
| 处理 | `monitor:alert-record:handle` | 标记为已处理 | 状态=告警中/已恢复 |
| 忽略 | `monitor:alert-record:handle` | 忽略此告警 | 状态=告警中 |
| 查看详情 | `monitor:alert-record:view` | 查看告警详情 | 所有状态 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 搜索 | 组合条件搜索 | `monitor:alert-record:view` |
| 处理告警 | 标记告警为已处理 | `monitor:alert-record:handle` |
| 忽略告警 | 忽略无效告警 | `monitor:alert-record:handle` |
| 批量处理 | 批量处理告警 | `monitor:alert-record:handle` |
| 查看详情 | 查看告警详细信息 | `monitor:alert-record:view` |
| 导出 | 导出告警记录 | `monitor:alert-record:export` |

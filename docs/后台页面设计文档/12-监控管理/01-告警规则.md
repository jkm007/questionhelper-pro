## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 规则名称 | string | 150 | - |
| target | 监控目标 | tag | 120 | CPU/内存/磁盘/数据库/API |
| condition | 告警条件 | string | 150 | 如 "> 80%" |
| duration | 持续时间 | string | 80 | 如 "5分钟" |
| notify_channels | 通知方式 | tag[] | 150 | 邮件/短信/站内 |
| severity | 严重程度 | tag | 80 | 严重/警告/提示 |
| status | 状态 | tag | 80 | 启用/禁用 |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 150 | - |

## 表单字段（新增/编辑弹窗）

| 字段名 | 标签 | 类型 | 必填 | 验证规则 | 说明 |
|--------|------|------|------|----------|------|
| name | 规则名称 | input | ✅ | 2-30位 | - |
| target | 监控目标 | select | ✅ | - | CPU/内存/磁盘/数据库/API |
| metric | 监控指标 | select | ✅ | - | 根据目标动态显示 |
| operator | 比较运算符 | select | ✅ | - | >/>=/</<=/== |
| threshold | 阈值 | number | ✅ | - | 触发阈值 |
| duration | 持续时间 | number | ✅ | - | 分钟数 |
| severity | 严重程度 | radio | ✅ | - | 严重/警告/提示 |
| notify_channels | 通知方式 | checkbox | ✅ | 至选一个 | 邮件/短信/站内 |
| notify_users | 通知人员 | user-select | ✅ | - | 接收告警的人员 |
| status | 状态 | switch | ✅ | - | 启用/禁用 |

## 监控目标与指标

| 监控目标 | 监控指标 |
|----------|----------|
| CPU | CPU使用率、负载 |
| 内存 | 内存使用率、交换分区使用率 |
| 磁盘 | 磁盘使用率、IO等待 |
| 数据库 | 连接数、查询耗时、慢查询数 |
| API | QPS、响应时间、错误率 |
| Redis | 连接数、内存使用、命中率 |

## 功能点

| 功能 | 说明 | 权限 |
|------|------|------|
| 新增 | 创建告警规则 | `monitor:alert-rule:create` |
| 编辑 | 编辑规则信息 | `monitor:alert-rule:edit` |
| 删除 | 删除规则 | `monitor:alert-rule:delete` |
| 批量删除 | 批量删除规则 | `monitor:alert-rule:delete` |
| 状态切换 | 启用/禁用规则 | `monitor:alert-rule:edit` |

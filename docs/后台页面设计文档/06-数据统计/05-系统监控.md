# 系统监控

## 基本信息

- 路由路径：`/statistics/system`
- 权限标识：`statistics:system`
- 页面类型：监控仪表盘

## 关联接口文档

> 详见 [API接口文档/10-数据统计.md](../API接口文档/10-数据统计.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/statistics/system` | GET | 系统监控数据 | [链接](../API接口文档/10-数据统计.md) |
| `/api/v1/admin/statistics/system/health` | GET | 系统健康检查 | [链接](../API接口文档/10-数据统计.md) |

## 系统状态

| 指标 | 说明 |
|------|------|
| 系统状态 | 正常/警告/错误 |
| 运行时间 | 服务器运行时长 |
| 最后重启 | 上次重启时间 |

## 服务器资源监控

| 资源 | 指标 | 说明 |
|------|------|------|
| CPU | 使用率 | 百分比，>80%警告 |
| 内存 | 使用率 | 百分比，>85%警告 |
| 磁盘 | 使用率 | 百分比，>90%警告 |
| 网络 | 流量 | MB/s |
| 数据库 | 连接数 | 当前/最大连接数 |
| Redis | 连接数 | 当前/最大连接数 |

## 请求监控

| 图表 | 类型 | 说明 |
|------|------|------|
| QPS趋势 | 折线图 | 每秒请求数 |
| 响应时间分布 | 柱状图 | <100ms/<500ms/<1s/>1s |

## 错误监控

| 图表 | 类型 | 说明 |
|------|------|------|
| 错误率趋势 | 折线图 | 错误请求比例 |
| 错误类型分布 | 饼图 | 500/404/401等 |

## 告警列表

| 字段 | 说明 |
|------|------|
| 时间 | 告警时间 |
| 类型 | 警告/错误/信息 |
| 告警内容 | 告警详情 |
| 状态 | 告警中/已恢复 |

## 功能点

| 功能 | 说明 |
|------|------|
| 自动刷新 | 每30秒自动刷新数据 |
| 手动刷新 | 点击刷新按钮 |
| 告警配置 | 配置告警阈值 |
| 告警历史 | 查看历史告警记录 |
| 健康检查 | 查看各服务健康状态 |

# 用户分群

## 基本信息

- 路由路径：`/statistics/segment`
- 权限标识：`statistics:segment:view`
- 页面类型：列表页 + 分析

## 表格字段

| 字段名 | 列名 | 类型 | 宽度 | 说明 |
|--------|------|------|------|------|
| name | 分群名称 | string | 150 | - |
| user_count | 用户数 | number | 80 | 符合条件的用户数 |
| conditions | 规则条件 | string | 300 | 溢出省略 |
| updated_at | 更新时间 | datetime | 160 | - |
| created_at | 创建时间 | datetime | 160 | - |
| 操作 | 操作 | - | 200 | - |

## 操作列

| 按钮 | 权限 | 说明 |
|------|------|------|
| 查看用户 | `statistics:segment:view` | 查看分群用户列表 |
| 编辑 | `statistics:segment:edit` | 编辑分群规则 |
| 删除 | `statistics:segment:delete` | 删除分群 |
| 刷新 | `statistics:segment:edit` | 重新计算分群用户 |

## 表单字段（创建/编辑分群弹窗）

| 字段名 | 标签 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| name | 分群名称 | input | ✅ | 2-30位 |
| conditions | 规则条件 | condition-builder | ✅ | 可视化规则构建器 |
| auto_refresh | 自动刷新 | switch | - | 是否每日自动更新 |
| description | 描述 | textarea | - | 最长200字 |

## 规则构建器

支持的条件字段：

| 字段 | 运算符 | 值类型 |
|------|--------|--------|
| 注册时间 | 大于/小于/介于 | 日期 |
| 最后登录 | 大于/小于/介于 | 日期 |
| 登录次数 | 大于/小于/等于 | 数字 |
| 答题次数 | 大于/小于/等于 | 数字 |
| 平均分 | 大于/小于/等于 | 数字 |
| 所属班级 | 等于/不等于 | 班级选择 |
| 用户角色 | 等于/不等于 | 角色选择 |

支持的逻辑运算：且(AND)、或(OR)

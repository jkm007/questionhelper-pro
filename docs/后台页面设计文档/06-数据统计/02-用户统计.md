# 用户统计

## 基本信息

- 路由路径：`/statistics/user`
- 权限标识：`statistics:user`
- 页面类型：统计分析页

## 关联接口文档

> 详见 [API接口文档/10-数据统计.md](../API接口文档/10-数据统计.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/statistics/users` | GET | 用户统计 | [链接](../API接口文档/10-数据统计.md) |
| `/api/v1/admin/statistics/users/retention` | GET | 用户留存 | [链接](../API接口文档/10-数据统计.md) |
| `/api/v1/admin/statistics/users/export` | GET | 导出统计 | [链接](../API接口文档/10-数据统计.md) |

## 用户概览卡片

| 指标 | 字段 | 说明 |
|------|------|------|
| 总用户数 | totalUsers | 所有注册用户 |
| 新增用户 | newUsers | 选定时间范围内新增 |
| 活跃用户 | activeUsers | 选定时间范围内活跃 |
| 留存率 | retentionRate | 次日留存率 |

## 趋势图表

| 图表 | 类型 | 说明 |
|------|------|------|
| 用户增长趋势 | 折线图 | 每日新增用户数 |
| 活跃用户趋势 | 折线图 | 每日活跃用户数 |

## 留存分析表格

| 字段 | 说明 |
|------|------|
| 日期 | 日期 |
| 新增 | 当日新增用户数 |
| 次日留存 | 次日仍活跃的用户比例 |
| 3日留存 | 3日内仍活跃的用户比例 |
| 7日留存 | 7日内仍活跃的用户比例 |
| 30日留存 | 30日内仍活跃的用户比例 |

## 用户分布图表

| 图表 | 类型 | 说明 |
|------|------|------|
| 用户来源分布 | 饼图 | 直接访问/搜索引擎/推荐链接 |
| 用户活跃时段分布 | 柱状图 | 24小时活跃分布 |
| 用户角色分布 | 饼图 | 管理员/教师/学生 |
| 用户地域分布 | 地图 | 按省份分布 |

## 功能点

| 功能 | 说明 |
|------|------|
| 时间切换 | 切换时间范围 |
| 导出 | 导出统计数据 |
| 图表交互 | hover显示详情 |
| 留存分析 | 选择日期查看留存 |

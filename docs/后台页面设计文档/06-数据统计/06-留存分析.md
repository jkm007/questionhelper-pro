# 留存分析

## 基本信息

- 路由路径：`/statistics/retention`
- 权限标识：`statistics:retention:view`
- 页面类型：统计分析页

|------|------|------|----------|
| `/api/v1/admin/statistics/retention` | GET | 留存数据 | [链接](../API接口文档/10-数据统计.md) |
| `/api/v1/admin/statistics/retention/cohort` | GET | 同期群分析 | [链接](../API接口文档/10-数据统计.md) |

## 筛选条件

| 字段 | 类型 | 说明 |
|------|------|------|
| 时间范围 | select | 本周/本月/本季/本年/自定义 |
| 留存类型 | select | 日留存/周留存/月留存 |
| 用户类型 | select | 全部/新用户/老用户 |

## 留存概览卡片

| 指标 | 说明 |
|------|------|
| 次日留存 | 次日仍活跃的用户比例 |
| 3日留存 | 3日内仍活跃的用户比例 |
| 7日留存 | 7日内仍活跃的用户比例 |
| 30日留存 | 30日内仍活跃的用户比例 |

## 同期群表格

| 字段 | 说明 |
|------|------|
| 同期群 | 日期（新增用户日期） |
| 新增 | 当日新增用户数 |
| 次日 | 次日留存率 |
| 3日 | 3日留存率 |
| 7日 | 7日留存率 |
| 14日 | 14日留存率 |
| 30日 | 30日留存率 |

**单元格颜色**：根据留存率显示不同颜色深浅，越高越绿

## 功能点

| 功能 | 说明 |
|------|------|
| 时间切换 | 切换时间范围 |
| 留存类型切换 | 日/周/月留存 |
| 导出 | 导出留存数据 |
| 图表交互 | hover显示具体数值 |

# 漏斗分析

## 基本信息

- 路由路径：`/statistics/funnel`
- 权限标识：`statistics:funnel:view`
- 页面类型：统计分析页

## 漏斗配置

| 字段 | 说明 |
|------|------|
| 漏斗名称 | 预定义的漏斗模型 |
| 时间范围 | 分析的时间段 |
| 步骤定义 | 漏斗的各个步骤 |

## 预定义漏斗

| 漏斗名称 | 步骤 |
|----------|------|
| 用户考试流程 | 访问→详情→答题→完成→成绩 |
| 用户注册流程 | 访问→注册→验证→首次登录 |
| 题目创建流程 | 进入→编辑→预览→保存→发布 |

## 转化率表格

| 字段 | 说明 |
|------|------|
| 步骤 | 漏斗步骤名称 |
| 用户数 | 到达该步骤的用户数 |
| 转化率 | 相对于第一步的转化率 |
| 流失率 | 相对于上一步的流失率 |
| 流失人数 | 流失的用户数 |

## 功能点

| 功能 | 说明 |
|------|------|
| 选择漏斗 | 切换不同的漏斗模型 |
| 时间切换 | 切换分析时间范围 |
| 创建漏斗 | 自定义漏斗步骤 |
| 导出 | 导出漏斗数据 |
| 图表交互 | hover显示详情 |

## 筛选条件

| 字段 | 类型 | 说明 |
|------|------|------|
| 起始页面 | select | 选择起始页面 |
| 时间范围 | select | 本周/本月/本季/本年 |
| 用户类型 | select | 全部/新用户/老用户 |

## 路径流向图

- 类型：桑基图/流向图
- 节点：页面
- 连线：用户流向
- 宽度：用户数量

## 热门路径表格

| 字段 | 说明 |
|------|------|
| 排名 | 按用户数排序 |
| 路径 | 用户访问路径 |
| 用户数 | 经过该路径的用户数 |
| 占比 | 占总用户的百分比 |
| 平均停留 | 在该路径的平均停留时间 |

## 页面访问统计表格

| 字段 | 说明 |
|------|------|
| 页面 | 页面名称 |
| 访问量 | 页面访问次数 |
| 独立访客 | 独立用户数 |
| 平均停留 | 平均停留时间 |
| 跳出率 | 只访问该页面就离开的比例 |

## 功能点

| 功能 | 说明 |
|------|------|
| 筛选 | 切换筛选条件 |
| 路径详情 | 点击节点查看详细数据 |
| 导出 | 导出分析数据 |
| 图表交互 | hover显示详情 |

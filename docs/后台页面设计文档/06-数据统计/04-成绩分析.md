# 成绩分析

## 基本信息

- 路由路径：`/statistics/score`
- 权限标识：`statistics:score`
- 页面类型：统计分析页

## 关联接口文档

> 详见 [API接口文档/10-数据统计.md](../API接口文档/10-数据统计.md)

| 接口 | 方法 | 说明 | 文档位置 |
|------|------|------|----------|
| `/api/v1/admin/statistics/scores` | GET | 成绩统计 | [链接](../API接口文档/10-数据统计.md) |
| `/api/v1/admin/statistics/scores/exam` | GET | 考试成绩统计 | [链接](../API接口文档/10-数据统计.md) |
| `/api/v1/admin/statistics/scores/export` | GET | 导出统计 | [链接](../API接口文档/10-数据统计.md) |

## 成绩概览卡片

| 指标 | 字段 | 说明 |
|------|------|------|
| 考试人数 | totalStudents | 参加考试的学生数 |
| 平均分 | avgScore | 所有学生的平均分 |
| 及格率 | passRate | 及格人数/总人数 |
| 优秀率 | excellentRate | 优秀人数/总人数（≥90分） |

## 分数分布图

- 类型：柱状图
- X轴：分数段（0-59, 60-69, 70-79, 80-89, 90-100）
- Y轴：人数
- 颜色：不及格(红)、及格(蓝)、良好(绿)、优秀(深绿)

## 成绩排名表格

| 字段 | 说明 |
|------|------|
| 排名 | 按得分排序 |
| 学生姓名 | 学生姓名 |
| 班级 | 所属班级 |
| 得分 | 考试得分 |
| 用时 | 答题用时 |
| 正确率 | 答对题目比例 |

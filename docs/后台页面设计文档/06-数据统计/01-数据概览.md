# 数据概览

## 基本信息

- 路由路径：`/statistics/overview`
- 权限标识：`statistics:overview`
- 页面类型：统计仪表盘

## 时间范围选择

| 选项 | 说明 |
|------|------|
| 今日 | 当天数据 |
| 本周 | 本周一到今天 |
| 本月 | 本月1号到今天 |
| 本年 | 本年1月1号到今天 |
| 自定义 | 选择日期范围 |

## 核心指标卡片

| 指标 | 字段 | 说明 |
|------|------|------|
| 总用户数 | totalUsers | 平台所有用户数 |
| 总题目数 | totalQuestions | 所有题目数量 |
| 总考试数 | totalExams | 所有考试场次 |
| 总班级数 | totalClasses | 所有班级数量 |
| 今日新增用户 | todayNewUsers | 今日注册用户数 |
| 今日活跃用户 | todayActiveUsers | 今日登录用户数 |
| 今日答题次数 | todayAnswers | 今日答题总次数 |
| 今日考试场次 | todayExams | 今日进行的考试数 |

**卡片样式**：
- 数字：28px 加粗
- 变化率：绿色(↑) / 红色(↓)
- 对比：与上一周期对比

## 趋势图表

| 图表 | 类型 | 说明 |
|------|------|------|
| 用户增长趋势 | 折线图 | 每日新增用户数 |
| 答题量趋势 | 柱状图 | 每日答题次数 |
| 正确率趋势 | 折线图 | 每日平均正确率 |
| 用户角色分布 | 饼图 | 管理员/教师/学生占比 |

## 功能点

| 功能 | 说明 |
|------|------|
| 时间切换 | 切换时间范围，刷新数据 |
| 导出 | 导出统计数据为Excel |
| 刷新 | 刷新当前数据 |
| 图表交互 | hover显示具体数值 |

## 数据结构

```typescript
interface StatisticsOverview {
  totalUsers: number;
  totalQuestions: number;
  totalExams: number;
  totalClasses: number;
  todayNewUsers: number;
  todayActiveUsers: number;
  todayAnswers: number;
  todayExams: number;
  growthRates: {
    users: number;
    questions: number;
    exams: number;
    classes: number;
  };
  trends: {
    dates: string[];
    userCounts: number[];
    answerCounts: number[];
    accuracyRates: number[];
  };
  roleDistribution: {
    admin: number;
    teacher: number;
    student: number;
  };
}
```

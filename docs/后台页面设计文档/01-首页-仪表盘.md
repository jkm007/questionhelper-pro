## KPI 卡片

| 卡片 | 字段 | 图标 | 说明 |
|------|------|------|------|
| 总用户数 | totalUsers | User | 显示环比变化（本周 vs 上周） |
| 总题目数 | totalQuestions | Document | 显示环比变化 |
| 总考试数 | totalExams | EditPen | 显示环比变化 |
| 今日活跃 | todayActive | TrendCharts | 显示环比变化 |

**卡片样式**：
- 阴影卡片，hover 时加深
- 数字：28px 加粗
- 标签：14px 次要文字色
- 变化率：绿色(↑) / 红色(↓)

## 趋势图表

| 图表 | 类型 | X轴 | Y轴 | 时间切换 |
|------|------|------|------|----------|
| 用户增长趋势 | 折线图 | 日期 | 新增用户数 | 本周/本月/本年 |
| 答题量趋势 | 柱状图 | 日期 | 答题次数 | 本周/本月/本年 |

**图表配置**：
```javascript
const lineOption = {
  tooltip: { trigger: 'axis' },

    type: 'line',
    smooth: true,
    areaStyle: { opacity: 0.3 },
    data: values
  }]
};
```

## 快捷操作

| 按钮 | 跳转路径 | 图标 | 权限 |
|------|----------|------|------|
| 新增题目 | /question/public | Plus | question:public:create |
| 创建考试 | /exam/paper | EditPen | exam:paper:create |
| 用户管理 | /user/list | User | user:list:view |
| 数据导出 | - | Download | - |

## 最近活动

| 字段 | 说明 |
|------|------|
| 活动描述 | 文字描述活动内容 |
| 活动时间 | 相对时间（2分钟前、1小时前） |
| 活动图标 | 根据类型显示不同图标 |

**活动类型**：
- 用户注册
- 考试发布
- 班级新增成员
- 题目审核

## 交互说明

1. **时间切换**：点击"本周/本月/本年"切换图表数据
2. **图表交互**：hover 显示具体数值，点击可跳转详情
3. **快捷操作**：点击跳转到对应页面
4. **最近活动**：点击可跳转到相关详情页

## 数据结构

```typescript
interface DashboardStats {
  totalUsers: number;
  totalQuestions: number;
  totalExams: number;
  todayActive: number;
  userGrowthRate: number;    // 百分比
  questionGrowthRate: number;
  examGrowthRate: number;
  activeGrowthRate: number;
  trend: {
    dates: string[];
    userCounts: number[];
    answerCounts: number[];
  };
  recentActivities: Activity[];
}
```

## 样式要点

- KPI 卡片：4 列等分，间距 16px
- 图表：2 列等分，间距 16px
- 快捷操作 + 最近活动：2 列等分，间距 16px
- 响应式：小屏幕时 KPI 卡片 2 列，图表 1 列

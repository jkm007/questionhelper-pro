# 题目纠错

> **路径**：`pages/feedback/correction`  
> **参数**：`?questionId=xxx`（可选，从题目详情页跳转时携带）  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 导航栏
- 标题：**"Report Error"**
- 返回按钮

### 题目信息卡片
- 自动填充标识（`#e8f5e9` 背景，`#4caf50` 文字）
- **"Auto-filled"** 徽章
- 显示题目标题和类型

### 纠错表单

#### 正确答案
- 选择题：A-E 按钮组（药丸按钮）
- 非选择题：文本域

#### 解析说明
- 文本域（必填）
- 1000 字限制 + 字数计数器

#### 参考来源
- 输入框（可选）
- placeholder：请填写参考来源

#### 佐证图片
- 最多 5 张图片
- 160rpx 方形 + 右上角删除按钮
- ＋ 添加图片占位框

### 提交按钮
- **Submit Report**：`#667eea` 全宽按钮
- 底部提示文字：**"感谢您的反馈，我们会尽快核实"**（22rpx，`#999`）

---

## 交互逻辑

1. 从题目详情页跳转时自动填充题目信息
2. 选择/填写正确答案
3. 填写解析说明（必填）
4. 上传佐证图片（可选，最多 5 张）
5. 提交 → 调用纠错 API
6. 成功 → Toast 提示 + 返回上一页

---

# 考试反馈

> **路径**：`pages/feedback/exam`  
> **参数**：`?examId=xxx`（可选）  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 导航栏
- 标题：**"Exam Feedback"**
- 返回按钮

### 考试选择器
- 原生 `<picker>` 组件
- 显示当前选中考试名称

### 反馈类型
药丸按钮组：
- **Error in Question**（题目错误）
- **Wrong Answer**（答案错误）
- **Other**（其他）

### 描述文本域
- placeholder：请描述您的反馈...
- 500 字限制 + 字数计数器

### 图片上传
- 最多 3 张图片
- 180rpx 方形 + 右上角删除按钮
- ＋ 添加图片占位框

### 题目信息（可选）
- 题目 ID 输入框
- placeholder：相关题目 ID（可选）

### 提交按钮
- **Submit Feedback**：`#667eea` 全宽按钮

---

## 交互逻辑

1. 选择反馈类型
2. 填写描述（必填）
3. 上传图片（可选，最多 3 张）
4. 填写题目 ID（可选）
5. 提交 → 调用反馈 API
6. 成功 → Toast 提示 + 返回上一页

---

| 元素 | 样式 |

| 提交按钮 | `#667eea`，88rpx 高，44rpx 圆角 |

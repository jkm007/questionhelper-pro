# 我的反馈

> **路径**：`pages/feedback/my`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 标签栏
- **All** / **Pending** / **Resolved** / **Rejected**

### 反馈列表
每项包含：
- 反馈类型（考试反馈/题目纠错）
- 标题/描述（2 行截断）
- 状态徽章：
  - Pending（待处理）：橙色
  - Resolved（已处理）：绿色
  - Rejected（已拒绝）：红色
- 提交时间
- 处理结果（已处理/已拒绝时显示）

### 空状态
- **"暂无反馈记录"**

---

## 交互逻辑

1. 加载我的反馈列表
2. 切换标签 → 筛选状态
3. 点击反馈 → 查看详情（含处理结果）
4. 已拒绝的反馈可重新提交

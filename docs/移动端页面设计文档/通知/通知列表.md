# 通知列表

> **路径**：`pages/notification/list`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"消息通知"**
- 返回按钮

### 操作栏
- 未读数徽章（`#ff4757` 背景，白色数字）
- **"全部已读"** 链接（`#007aff`，右侧）

### 通知列表
每条通知包含：
- **类型图标**：彩色圆形背景 + 表情
  - 1-系统通知：橙色 📢
  - 2-考试通知：绿色 📋
  - 3-作业通知：蓝色 📝
  - 4-班级通知：紫色 🏫
  - 5-评论通知：粉色 💬
- **未读圆点**（`#ff4757`，16rpx，左侧）
- **标题**（28rpx，`#333`，粗体）
- **时间**（22rpx，`#999`，相对时间：Just now / Xm ago / Xh ago / Xd ago）
- **内容预览**（26rpx，`#666`，单行截断）
- **删除按钮**（右侧，滑动显示或长按菜单）

### 空状态
- 表情图标（🔔）+ **"暂无通知"**（28rpx，`#999`）

---

## 交互逻辑

1. 加载通知列表（分页）
2. 点击通知 → `navigateTo` 通知详情页 + 标记已读
3. 点击"全部已读"→ 调用 API + 更新列表
4. 左滑/长按通知 → 显示删除按钮
5. 删除 → 确认弹窗 → 调用删除 API
6. 下拉刷新 + 上拉加载

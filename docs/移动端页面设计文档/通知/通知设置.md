# 通知设置

> **路径**：`pages/notification/settings`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 按类型设置开关

| 通知类型 | 类型码 | 推送 | 声音 | 震动 |
|---------|--------|------|------|------|
| 系统通知 | 1 | ✅ | ✅ | ✅ |
| 考试通知 | 2 | ✅ | ✅ | ✅ |
| 作业通知 | 3 | ✅ | ✅ | ✅ |
| 班级通知 | 4 | ✅ | ✅ | ✅ |
| 评论通知 | 5 | ✅ | ✅ | ❌ |

- 每种类型独立控制推送/声音/震动

### 免打扰时段
- 开关：**"免打扰模式"**
- 时间范围：开始时间 ~ 结束时间
- 免打扰期间仅静音，不中断接收

### 全局设置
- **全部开启** / **全部关闭** 快捷按钮

---

## 交互逻辑

1. 加载当前通知设置
2. 切换开关 → 调用更新 API
3. 设置免打扰时段 → 时间选择器

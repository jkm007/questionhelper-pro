# 通知详情

> **路径**：`pages/notification/detail`  
> **参数**：`?id=xxx`  
> **页面背景**：`#ffffff`

---

## 页面结构

### 导航栏
- 标题：**"通知详情"**
- 返回按钮

### 加载状态
- 加载中显示加载动画

### 详情头部
- 类型图标（彩色圆形背景 + 表情）
- 标题（32rpx，`#333`，粗体）
- 时间（24rpx，`#999`）

### 分割线
- 浅灰色分割线（`#eee`）

### 内容区
- 富文本渲染（`rich-text` 组件）

### 关联资源
- 链接卡片（🔗 图标 + 标题）
- 点击跳转内部页面或 WebView

### 元数据
- 类型 + ID（22rpx，`#ccc`）

### 错误状态
- 加载失败时显示错误信息
- **重试** 按钮

### 底部按钮
- **删除通知**：红色按钮（`#fff0f0` 背景，`#ffd0d0` 边框）

---

## 交互逻辑

1. 根据 `id` 加载通知详情
2. 加载成功 → 标记已读
3. 点击关联资源 → 跳转对应页面
4. 点击删除 → 确认弹窗 → 调用删除 API + 返回列表
5. 加载失败 → 显示错误 + 重试按钮

---

| 元素 | 样式 |

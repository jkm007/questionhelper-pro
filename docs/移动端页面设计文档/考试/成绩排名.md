# 成绩排名

> **路径**：`pages/exam/ranking`  
> **参数**：`?examId=xxx`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 导航栏
- 标题：**"Ranking"**
- 返回按钮

### 我的排名卡片
- 排名（大号显示）
- 分数 + 正确率
- 超过 XX% 的考生

### 排行榜列表
每行包含：
- 排名（前 3 名颜色编码：🥇🥈🥉）
- 头像 + 昵称
- 分数（粗体）
- 用时（24rpx，`#999`）
- 当用户是自己时，背景高亮（`#e8f5e9`）

### 排名统计
- 参与人数
- 平均分
- 最高分
- 及格率

---

## 交互逻辑

1. 根据 `examId` 加载排名数据
2. 上拉加载更多
3. 当前用户始终可见（固定在顶部或列表中高亮）

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 我的排名卡片 | 渐变背景 `#667eea` → `#764ba2` |
| 排名数字 | 前 3 名 `#ff9800`，其余 `#333` |
| 当前用户行 | `#e8f5e9` 背景 |

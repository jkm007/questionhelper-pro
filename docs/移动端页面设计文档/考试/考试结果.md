# 考试结果

> **路径**：`pages/exam/result`  
> **参数**：`?recordId=xxx`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 分数区
- Pass = 绿渐变（`#4caf50` → `#81c784`）
- Fail = 红渐变（`#f44336` → `#e57373`）
- 动画分数圆圈（1500ms 计数动画）
- Pass/Fail 图标和文字

### 统计行
4 列统计：
- **Score**（得分）
- **Rank**（排名）
- **Correct Rate**（正确率）
- **Duration**（用时）

### 题目明细
- 标题：**"Question Details"**
- 可展开的逐题列表
- 每题显示：
  - Your Answer vs Correct Answer
  - 选项高亮（正确=绿，错误=红）
  - Explanation 卡片（`#fff8e1` 背景，`#ff9800` 左边框）

### 底部栏
- **View Answers** 按钮（`#667eea`）
- **Back** 按钮（灰色）

---

## 交互逻辑

1. 根据 `recordId` 加载考试结果
2. 分数动画：页面加载后 1500ms 内从 0 递增到实际分数
3. 点击题目 → 展开/收起详情
4. View Answers → 展开所有题目详情
5. Back → 返回考试列表

# 考试答题

> **路径**：`pages/exam/answer`  
> **参数**：`?examId=xxx`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 自定义导航栏
- 返回按钮（点击弹出确认退出弹窗）
- 倒计时：红色 `#f44336`，`HH:MM:SS` 格式
- 题号网格按钮（点击打开题号抽屉）

### 题号网格抽屉
- 右侧滑入（60% 宽度）
- 题号网格（5 列）
- 颜色编码：
  - 当前题：`#667eea` 边框 + `#e8eaf6` 背景
  - 已答题：`#4caf50` 边框 + `#e8f5e9` 背景
  - 标记题：`#ff9800` 边框 + `#fff3e0` 背景
  - 未答题：`#ddd` 边框 + 白色背景
- 底部图例：Current / Answered / Flagged

### 题目区域
- Swiper 横向滑动切换题目
- 每题包含：
  - 题号 + 分值 + 标记按钮（⚑/🚩）
  - 题目内容
  - 选项列表（单选/多选，选中=`#f0f2ff`）

### 底部答题卡
- 进度条（渐变填充 `#667eea` → `#764ba2`）
- **"X/Y answered"** 文字
- Prev / Submit / Next 按钮

---

## 交互逻辑

1. 进入页面开始倒计时
2. Swiper 滑动切换题目
3. 点击选项选择答案（自动保存到本地缓存）
4. 点击标记按钮 → 标记/取消标记当前题
5. 题号抽屉点击 → 跳转到指定题目
6. Submit → 确认弹窗 → 提交试卷
7. **自动保存**：每 30 秒自动保存进度（`saveExamProgress` API）
8. **答案本地缓存**：答案同步写入 `uni.setStorageSync`，网络恢复后自动同步
9. **时间到自动提交**：倒计时归零时自动提交
10. 返回按钮 → 确认弹窗（"确定退出考试？答题进度将保存"）

### 考试时间控制
- 倒计时从后端获取服务端时间戳，客户端仅做展示
- 实际超时由后端 Redis 延迟队列控制
- 客户端每分钟与服务端同步一次时间，误差超过 30 秒时校准

# 发布考试（教师端）

> **路径**：`pages/exam/publish`  
> **参数**：`?classId=xxx`（班级考试时携带）  
> **页面背景**：`#f5f6fa`  
> **权限**：教师/创作者

---

## 页面结构

### 表单字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 考试标题 | 输入框 | 必填，50 字限制 |
| 选择试卷 | 选择器 | 从已有试卷中选择 |
| 考试描述 | 文本域 | 选填 |
| 开始时间 | 日期时间选择器 | 必填 |
| 结束时间 | 日期时间选择器 | 必填 |
| 考试时长 | 数字输入 | 分钟，必填 |
| 总分 | 自动计算 | 根据试卷自动填充 |
| 参与范围 | 选择器 | 全部学生 / 指定分组 |
| 允许重考 | 开关 | 默认关闭 |
| 重考次数 | 数字输入 | 允许重考时显示 |
| 显示答案 | 选择器 | 考试结束后 / 从不显示 |
| 需要审批成绩 | 开关 | 默认关闭 |

### 提交按钮
- **发布考试**：`#667eea` 全宽按钮

---

## 交互逻辑

1. 选择试卷 → 自动填充总分
2. 设置时间 → 验证结束时间 > 开始时间
3. 提交 → 表单验证 + 调用发布 API
4. 成功 → Toast 提示 + 返回考试列表

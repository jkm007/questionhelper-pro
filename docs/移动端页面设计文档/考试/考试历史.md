# 考试历史

> **路径**：`pages/exam/history`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 渐变头部
- 背景：`#667eea` → `#764ba2`
- 返回按钮 + **"Exam History"** 标题

### 历史卡片列表
每张卡片包含：
- 标题（30rpx，`#333`，粗体）+ Pass/Fail 徽章
- 大号分数显示（48rpx 粗体，`#667eea`）
- 分数进度条（Pass=绿渐变，Fail=红渐变）
- 底部信息行：
  - 📅 日期
  - ⏱ Duration
  - 🔄 Attempt #N（第 N 次尝试）

### 分页
- 下拉刷新 + 上拉加载

---

## 交互逻辑

1. 加载考试历史列表
2. 点击卡片 → `navigateTo` 考试结果页
3. 下拉刷新 → 重新加载
4. 上拉加载更多

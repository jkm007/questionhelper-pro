# 数据统计

> **路径**：`pages/statistics/index`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 汇总卡片
4 张卡片，2 列网格布局：

| 卡片 | 渐变色 | 图标 | 数据 |
|------|--------|------|------|
| Practice | `#4a90d9` → `#357abd` | 📝 | 总练习数 |
| Exams | `#67c23a` → `#529b2e` | 📋 | 总考试数 |
| Accuracy | `#e6a23c` → `#cf9236` | 📊 | 平均正确率 |
| Duration | `#9b59b6` → `#8e44ad` | ⏱ | 总学习时长 |

- 数字：48rpx，白色，粗体
- 标签：24rpx，白色，半透明

### 每周练习趋势
- 标题：**"Weekly Practice Trend"**
- Canvas 柱状图
- 日期标签（X 轴）
- 数量标签（Y 轴）

### 分类正确率雷达图
- 标题：**"Category Accuracy"**
- Canvas 多边形雷达图
- 网格环（3-5 层）
- 分类标签（外围）
- 数据填充（半透明 `#4a90d9`）

### 考试分数趋势
- 标题：**"Exam Score Trend"**
- Canvas 面积图（折线 + 渐变填充 + 数据点）
- `#67c23a` 主色

### 排行榜
- 标题：**"Leaderboard"**
- 排名列表：
  - 排名圆圈：前 3 名 `#ff9800`，其余 `#f5f5f5`
  - 头像 + 昵称
  - 统计数据（"X practices | Y% accuracy"）
  - **"You"** 徽章（当前用户高亮，`#4a90d9` 背景）

---

## 数据加载

调用 API：
- `getStatisticsSummary` — 获取汇总数据
- `getWeeklyTrend` — 获取每周趋势
- `getCategoryAccuracy` — 获取分类正确率
- `getExamTrend` — 获取考试趋势
- `getLeaderboard` — 获取排行榜

---

## 交互逻辑

1. 进入页面并行加载所有数据
2. 图表支持触摸交互（显示数据点详情）
3. 排行榜支持上拉加载更多

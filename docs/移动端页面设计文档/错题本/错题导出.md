# 错题导出

> **路径**：`pages/wrong/export`（或作为错题列表页的弹窗功能）  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 导出设置

#### 导出范围
- **全部错题**
- **按分类**：选择分类
- **按掌握度**：未掌握 / 部分掌握 / 已掌握
- **按题型**：单选 / 多选 / 判断 / 填空 / 简答
- **按时间**：最近 7 天 / 30 天 / 自定义范围

#### 导出格式
- **Excel (.xlsx)** — 包含题目、选项、我的答案、正确答案、解析
- **CSV (.csv)** — 纯文本格式
- **PDF (.pdf)** — 排版打印友好（可选）

#### 导出内容
- 题目内容 ✅（默认）
- 选项 ✅（默认）
- 我的答案 ✅（默认）
- 正确答案 ✅（默认）
- 解析 ☑️（可选）
- 知识点 ☑️（可选）
- 错误次数 ☑️（可选）
- 复习记录 ☑️（可选）

### 预览区
- 显示前 5 条数据预览
- 导出总数显示

### 导出按钮
- **Export**：`#667eea` 全宽按钮

---

## 交互逻辑

1. 选择导出范围和格式
2. 预览数据
3. 点击导出 → 调用导出 API
4. 生成文件 → 下载到本地 / 分享

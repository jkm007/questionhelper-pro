# 错题列表

> **路径**：`pages/wrong/list`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 搜索栏
- 输入框 + 筛选按钮（漏斗图标）
- 点击筛选按钮 → 打开右侧筛选抽屉

### 活跃筛选标签
- 可移除的药丸标签（× 按钮）
- 显示当前筛选条件（分类、掌握程度、题型）

### 批量操作栏
- 全选复选框 + 已选数量
- 移除按钮（`#ff4d4f`，仅选中时显示）

### 错题卡片列表
每张卡片包含：
- 复选框（批量模式下显示）
- 标题（30rpx，`#333`）
- 类型标签（颜色编码）
- 错误次数（红色，如 "错误 3 次"）
- 下次复习时间（24rpx，`#999`）
- 掌握度进度条：
  - 红色 `#ff4d4f`（<30%）
  - 黄色 `#faad14`（30%-70%）
  - 绿色 `#52c41a`（≥70%）

### 底部栏
- **批量模式**切换按钮
- **查看分析**链接（`#4A90D9`）

### 筛选抽屉
- 右侧滑入（600rpx）
- 筛选项：
  - 分类选择
  - 掌握程度：药丸选项（未掌握 / 部分掌握 / 已掌握）
  - 题型选择
- 底部：重置 / 确认按钮

---

## 交互逻辑

1. 进入页面加载错题列表
2. 搜索 → 输入关键词筛选
3. 筛选 → 打开抽屉选择条件
4. 批量模式 → 显示复选框，可多选删除
5. 点击卡片 → `navigateTo` 错题详情页
6. 查看分析 → `navigateTo` 错题分析页
7. 下拉刷新 + 上拉加载

# 错题详情

> **路径**：`pages/wrong/detail`  
> **参数**：`?id=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 加载状态
- 加载骨架屏（`skeleton-title` + `skeleton-line`）

### 题目内容
- 类型标签 + 题目文字
- 选项列表（正确/错误高亮）

### 答案对比
双卡片并排：
- **我的答案**：红色卡片（`#fff1f0` 背景，`#ffccc7` 边框）
- **正确答案**：绿色卡片（`#f6ffed` 背景，`#b7eb8f` 边框）
- 中间箭头指示（→）

### 解析区域
- 解析文字（28rpx，`#333`）
- 知识点药丸标签（`#F5F7FA` 背景）

### 错题时间线
- 圆点 + 连接线 + 日期
- 展示历史错误记录（每次做错的时间）

### 掌握度指示器
- 圆环（120rpx，`#1677ff` 边框，8rpx 宽）显示百分比
- 状态文字：
  - <30%：未掌握
  - 30%-70%：部分掌握
  - ≥70%：已掌握
- 进度条（红/黄/绿）

### 学习笔记
- 文本域（500 字限制）
- 失焦时自动保存

### 底部栏
- **再做一次** 按钮（`#4A90D9`）
- **标记已掌握** 按钮（`#52c41a` 绿色）

---

## 交互逻辑

1. 根据 `id` 加载错题详情
2. 点击"再做一次"→ 进入练习模式，仅包含此题
3. 点击"标记已掌握"→ 调用 API 更新掌握度
4. 编辑笔记 → 失焦自动保存

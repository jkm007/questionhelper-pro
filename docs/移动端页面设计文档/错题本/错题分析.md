# 错题分析

> **路径**：`pages/wrong/analysis`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 汇总卡片
3 张卡片横排：
- **错题总数**（红色图标）
- **已掌握数**（绿色图标）
- **掌握率**（蓝色图标）

### 分类分布
- 标题：**"分类分布"**
- 水平柱状图（Canvas 绘制）
- 5 种颜色循环：`#1677ff` / `#52c41a` / `#faad14` / `#722ed1` / `#ff4d4f`
- 每个柱形右侧显示数量

### 薄弱知识点 TOP10
- 标题：**"薄弱知识点"**
- 排名列表：
  - 排名圆圈：第 1 名=`#ff4d4f` 红、第 2 名=`#fa8c16` 橙、第 3 名=`#faad14` 黄、其余=`#f5f5f5`
  - 知识点名称
  - 进度条 + 百分比

### 掌握趋势
- 标题：**"掌握趋势"**
- 30 天可滚动柱状图（Canvas 绘制）
- Y 轴标签（百分比）
- X 轴日期标签

### 复习建议
- 标题：**"复习建议"**
- 列表项：图标 + 标题 + 描述 + 操作箭头
- 类型样式：
  - 紧急：红色背景 `#fff1f0`
  - 复习：蓝色背景 `#e6f0ff`
  - 练习：绿色背景 `#f6ffed`

---

## 数据加载

调用 API：
- `getWrongAnalysis` — 获取错题分析数据
- `getWrongTrend` — 获取掌握趋势数据

---

## 交互逻辑

1. 进入页面加载分析数据
2. 柱状图支持横向滚动
3. 点击薄弱知识点 → 跳转到相关错题列表
4. 点击复习建议 → 跳转到对应练习模式

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 汇总卡片 | 白色，20rpx 圆角，阴影 |
| 柱状图颜色 | `#1677ff` / `#52c41a` / `#faad14` / `#722ed1` / `#ff4d4f` |

| 进度条 | 12rpx 高 |
| 复习建议卡片 | 16rpx 圆角，颜色编码背景 |

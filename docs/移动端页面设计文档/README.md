## 全局设计规范

### 两套主色方案

| 模块 | 主色 | 渐变 | 页面背景 |
|------|------|------|---------|
| 首页 / 登录 / 题库 / 班级 / 个人中心 / 错题 / 评论 / 通知 | `#4A90D9` 或 `#1677ff` | 无 | `#f5f6fa` 或 `#F5F7FA` |
| 练习 / 考试 | `#667eea` | `#667eea` → `#764ba2` | `#f5f5f5` |

### 通用组件

| 组件 | 用途 |
|------|------|
| `QuestionCard` | 题目卡片（类型标签 + 难度徽章 + 标题 + 分类 + 浏览/点赞数） |
| `ExamCard` | 考试卡片（标题 + 状态 + 时间 + 人数 + 得分） |
| `Empty` | 空状态占位（图标 + 文字 + 可选按钮） |
| `SearchBar` | 搜索栏（支持 readonly 跳转模式） |
| `ClassCard` | 班级卡片（封面 + 名称 + 创建者 + 人数） |
| `AnswerSheet` | 答题卡弹窗（已答/未答/标记统计 + 题号网格） |
| `OptionsList` | 选项列表（单选/多选 + 正确/错误高亮） |
| `Timer` | 计时器（格式化显示 + 紧急状态变色） |
| `ScoreChart` | 分数展示（数值 + 进度条 + 颜色分级） |
| `LoadMore` | 加载更多（加载中 / 已完成 / 点击加载） |

### 通用交互模式

- **下拉刷新 + 上拉加载**：`<scroll-view>` 的 `refresher-enabled` + `@scrolltolower`
- **确认弹窗**：`uni.showModal` 用于删除、退出等危险操作
- **导航**：`uni.navigateTo`（带查询参数）、`uni.switchTab`（Tab 页切换）
- **底部安全区**：`padding-bottom: env(safe-area-inset-bottom)`
- **空状态**：列表为空时显示图片/表情 + 描述文字
- **状态徽章**：小型彩色药丸标签，标识状态

### 单位与字体

- **单位**：`rpx`（响应式像素，750rpx = 屏幕宽度）
- **字体大小**：标题 32-36rpx，正文 28rpx，辅助文字 24rpx
- **圆角**：卡片 20rpx，按钮 16-44rpx，输入框 16rpx

# 练习答题

> **路径**：`pages/practice/session`  
> **参数**：`?mode=xxx&categoryId=xxx`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 自定义导航栏
- 返回按钮
- 模式名称（如 "Random Practice"）
- 收藏切换（★/☆）

### 计时栏
- 计时器（⏱ 格式化显示）
- 进度条（`#667eea` 渐变填充）
- 题号计数（如 "3/20"）

### 题目区域
- Swiper 横向滑动切换题目
- 每题包含：
  - 题号 + 题型标签
  - 题目内容
  - 选项列表
    - 选中：`#f0f2ff` 背景
    - 正确：`#e8f5e9` 背景
    - 错误：`#ffebee` 背景
  - Explanation 卡片（`#fff8e1` 背景，`#ff9800` 左边框）
    - 点击 "Show Answer" 后出现

### 底部操作栏
固定底部，3 个按钮：
- **Previous**：灰色（不可用时）/ `#666`（可用时）
- **Show Answer / Hide Answer**：橙色 `#ff9800`
- **Next / Submit All**：渐变紫色（`#667eea` → `#764ba2`）

---

## 交互逻辑

1. 进入页面加载题目列表
2. Swiper 滑动切换题目
3. 点击选项 → 立即判断对错 + 高亮显示
4. Show Answer → 显示正确答案和解析
5. Next → 切换到下一题
6. 最后一题 → Next 变为 Submit All
7. Submit All → 提交练习结果 → 跳转结果页
8. 收藏切换 → 调用收藏 API

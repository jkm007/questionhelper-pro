# 练习结果

> **路径**：`pages/practice/result`  
> **参数**：`?sessionId=xxx`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 渐变分数区
- 渐变背景（`#667eea` → `#764ba2`）
- 动画分数圆环（CSS 旋转）
- **Accuracy** 标签 + 百分比
- 励志文案（根据正确率）：
  - ≥90%：**"Excellent!"**
  - ≥80%：**"Great job!"**
  - ≥70%：**"Good work!"**
  - ≥60%：**"Not bad, keep practicing!"**
  - <60%：**"Keep trying, you will improve!"**

### 统计行
4 列统计：
- **Total**（总题数）
- **Correct**（正确数）
- **Wrong**（错误数）
- **Duration**（用时）

### 题目详情
- 标题：**"Question Details"**
- 可展开的逐题列表
- 每题显示：
  - 正确/错误指示器（✓ 绿色 / ✗ 红色）
  - 选项高亮（正确=绿，错误=红）
  - Explanation 卡片

### 底部栏
- **Retry** 按钮（`#667eea`）
- **Back to Practice** 按钮（灰色）

---

## 交互逻辑

1. 根据 `sessionId` 加载练习结果
2. 分数圆环动画：页面加载后播放 CSS 旋转动画
3. 点击题目 → 展开/收起详情
4. Retry → 重新开始相同模式的练习
5. Back to Practice → 返回练习首页

---

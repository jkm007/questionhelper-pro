# 练习首页

> **路径**：`pages/practice/index`  
> **Tab 页**：是（练习 Tab）  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 渐变头部
- 背景：`#667eea` → `#764ba2`
- 标题：**"Practice"**（白色，36rpx，粗体）
- 副标题：**"Improve your skills"**（白色，24rpx，半透明）

### 今日统计卡片
- 浮在头部下方（负边距重叠）
- 白色圆角卡片，阴影
- 3 列统计：

| 指标 | 图标 | 说明 |
|------|------|------|
| Questions | 📝 | 今日刷题数 |
| Accuracy | 📊 | 正确率 |
| Duration | ⏱ | 学习时长 |

### 快捷开始卡片
- 渐变背景（`#667eea` → `#764ba2`）
- ▶ 图标 + **"Quick Start"** + **"Start practice with recommended settings"**
- 点击进入随机练习模式

### 练习模式网格
2 列布局，7 种模式卡片：

| 模式 | 图标 | 描述 | 背景色 |
|------|------|------|--------|
| Random | 🔀 | Random questions | `#E3F2FD` |
| Sequential | 📋 | In order | `#E8F5E9` |
| Wrong Questions | ❌ | From mistakes | `#FFF3E0` |
| Favorites | ⭐ | Saved questions | `#FCE4EC` |
| Mock Exam | 📝 | Full simulation | `#E8EAF6` |
| Challenge | 🏆 | Gamified practice | `#F3E5F5` |
| Timed | ⏱ | Speed practice | `#E0F7FA` |

- 卡片：圆角 16rpx，内边距 24rpx
- 图标：48rpx
- 模式名：28rpx，`#333`，粗体
- 描述：22rpx，`#999`

### 最近记录
- 标题：**"Recent Records"**
- 展示最近的练习记录列表
- 每条记录：模式 + 题数 + 正确率 + 时间

---

## 数据加载

调用 API：
- `getTodayStats` — 获取今日统计
- `getRecentRecords` — 获取最近记录

---

## 交互逻辑

1. 点击快捷开始 → 进入随机练习模式
2. 点击练习模式卡片 → 进入对应模式的练习答题页
3. 点击最近记录 → 进入练习结果页

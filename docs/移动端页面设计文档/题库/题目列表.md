# 题目列表

> **路径**：`pages/question/list`  
> **Tab 页**：是（题库 Tab）  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 搜索栏
- 点击跳转搜索页（readonly 模式，内联实现）
- 背景：`#F5F7FA`，圆角 32rpx

### 筛选栏
横向滚动，药丸按钮组：
- **类型筛选**：全部 / 单选题 / 多选题 / 判断题
- **分类筛选**：分类（点击打开右侧分类抽屉）
- 选中状态：`#4A90D9` 背景 + 白色文字
- 未选中：白色背景 + `#666` 文字

### 题目列表
- 使用 `QuestionCard` 组件
- 卡片信息：类型标签 + 难度徽章 + 标题 + 分类 + 浏览数/点赞数
- 支持下拉刷新（`refresher-enabled`）
- 上拉加载更多（内联文字："加载中..." / "没有更多了" / "上拉加载更多"）

### 分类抽屉
- 右侧滑入（600rpx 宽）
- 半透明遮罩（`rgba(0,0,0,0.5)`）
- 树形结构：父分类 + 子分类
- 点击分类筛选题目列表

---

## 交互逻辑

1. 进入页面加载题目列表（分页，默认 20 条/页）
2. 切换筛选条件重新请求数据
3. 点击题目卡片 → `navigateTo` 题目详情页
4. 下拉刷新 → 重新加载第一页
5. 上拉到底部 → 加载下一页

# 题目详情

> **路径**：`pages/question/detail`  
> **参数**：`?id=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 加载状态
- 加载中显示"加载中..."文字（非骨架屏）

### 题目信息区
- 类型标签（颜色编码）：单选=蓝、多选=橙、判断=绿、填空=紫、简答=红
- 难度徽章：简单=绿、中等=黄、困难=红
- 标题（32rpx，`#333`，粗体）
- 分类（24rpx，`#999`）
- 浏览数 + 点赞数（24rpx，`#999`）

### 题目内容
- 富文本展示区域

### 选项列表
- 可选择的选项（单选/多选/判断题）
- 选中高亮：`#f0f2ff` 背景

### 折叠区域
- **答案**：点击展开/收起，展开后显示正确答案
- **解析**：点击展开/收起，展开后显示解析内容
- 折叠箭头动画

### 知识点标签
- 药丸形标签（`#F5F7FA` 背景，`#666` 文字）

### 创建者信息
- 头像（48rpx 圆形）+ 昵称（26rpx，`#333`）

### 底部操作栏
固定底部，5 个操作按钮（表情图标 40rpx）：

| 图标 | 功能 | 说明 |
|------|------|------|
| ⭐/☆ | 收藏 | 切换收藏状态 |
| ❤️/🤍 | 点赞 | 切换点赞状态 |
| 📤 | 分享 | 调用分享功能 |
| ⚠️ | 纠错 | 跳转纠错反馈页 |
| 📝 | 错题本 | 添加到错题本 |

---

## 交互逻辑

1. 根据 `id` 参数加载题目详情
2. 点击选项可选择答案（不提交，仅展示交互）
3. 折叠区域点击切换展开/收起
4. 收藏/点赞 → 调用 API + 更新图标状态
5. 分享 → 调用系统分享或复制链接
6. 纠错 → `navigateTo` 纠错反馈页（携带题目 ID）
7. 错题本 → 调用添加错题 API

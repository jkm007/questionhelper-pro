# 我的分类

> **路径**：`pages/question/my-category`  
> **页面背景**：`#f5f6fa`  
> **权限**：创作者/教师

---

## 页面结构

### 导航栏
- 标题：**"我的分类"**
- 返回按钮
- 右侧：添加分类按钮

### 分类树
- 树形结构展示我的私有分类
- 每个分类：名称 + 题目数量 + 操作（编辑/删除）
- 支持展开/收起子分类

### 添加/编辑分类弹窗
- 分类名称（必填）
- 父级分类（可选，选择器）
- 分类描述（选填）

---

## 交互逻辑

1. 加载我的私有分类树
2. 点击添加 → 弹出添加弹窗
3. 点击编辑 → 弹出编辑弹窗
4. 删除 → 确认弹窗（含关联题目数量提示）
5. 点击分类 → 查看该分类下的题目

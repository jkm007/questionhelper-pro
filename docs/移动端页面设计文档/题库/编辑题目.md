# 编辑题目

> **路径**：`pages/question/edit`  
> **参数**：`?id=xxx`（编辑模式，无 id 则为创建模式）  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 基本信息区

#### 题型选择
- 药丸按钮组：单选题 / 多选题 / 判断题 / 填空题 / 简答题
- 选中：`#4A90D9` 背景 + 白色文字

#### 难度选择
- 药丸按钮组：简单 / 中等 / 困难
- 颜色编码：简单=绿、中等=黄、困难=红

#### 分类选择器
- 点击弹出底部分类选择弹窗
- 显示当前选中分类

### 题目内容区
- 标题输入框（必填）
- 内容文本域（可选）

### 选项区（选择题类型）
- 动态添加/删除选项
- 选项标签：A-H，48rpx 圆形，`#4A90D9` 背景
- 正确答案切换按钮：单选=单选按钮，多选=复选框
- 最少 2 个选项，最多 8 个

### 答案与解析
- 答案文本域
- 解析文本域

### 知识点标签
- 可添加/删除的知识点药丸标签
- 点击弹出多选弹窗

### 提交按钮
- 页面内提交按钮（非底部固定）
- `#4A90D9` 全宽按钮

---

## 交互逻辑

1. 创建模式：表单为空，题型默认单选题
2. 编辑模式：加载已有题目数据填充表单
3. 切换题型 → 动态显示/隐藏选项区
4. 添加选项 → 追加输入框
5. 删除选项 → 确认弹窗后移除
6. 提交 → 表单验证 + 调用创建/更新 API
7. 成功 → Toast 提示 + 返回上一页

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 药丸按钮 | 56rpx 高，28rpx 圆角，24rpx 字号 |
| 选项标签 | 48rpx 圆形，`#4A90D9` 背景 |
| 输入框背景 | `#F5F7FA`，16rpx 圆角 |
| 文本域高度 | 200rpx |
| 提交按钮 | `#4A90D9`，88rpx 高，44rpx 圆角 |

> **注意**：移动端不支持复杂富文本编辑（如公式、代码块）。创作者在移动端仅可进行简单题目编辑（纯文本+图片），复杂创作请使用管理后台。

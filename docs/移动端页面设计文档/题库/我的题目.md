# 我的题目

> **路径**：`pages/question/my`  
> **页面背景**：`#f5f6fa`  
> **权限**：创作者/教师

---

## 页面结构

### 标签栏
- **全部** / **草稿** / **已发布** / **审核中** / **已拒绝**

### 题目列表
每项包含：
- 标题（28rpx，`#333`）
- 题型标签 + 难度徽章
- 状态徽章：
  - 草稿：灰色
  - 已发布：绿色
  - 审核中：橙色
  - 已拒绝：红色（显示拒绝原因）
- 浏览数 + 使用数（22rpx，`#999`）
- 操作：编辑 / 删除 / 提交审核

### 浮动按钮
- ＋ 创建题目 → `navigateTo` 编辑题目页

---

## 交互逻辑

1. 加载我的题目列表
2. 切换标签 → 筛选状态
3. 点击编辑 → `navigateTo` 编辑题目页
4. 点击提交审核 → 调用 API
5. 删除 → 确认弹窗
6. FAB → 创建新题目

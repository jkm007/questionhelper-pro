# 题目搜索

> **路径**：`pages/question/search`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 搜索头部
- 输入框（自动聚焦）+ 清除按钮（×）
- 取消链接（`#4A90D9`，右侧）
- 内联搜索栏实现（未使用 `SearchBar` 组件）

### 搜索前状态

#### 搜索历史
- 标题：**"搜索历史"** + 右侧 **"清空"**（`#999`）
- 药丸标签布局（自动换行）
- localStorage 存储，最多 20 条
- 点击标签 → 填入搜索框并执行搜索

#### 热门搜索
- 标题：**"热门搜索"**
- 编号列表（1-10）
- 前 3 名红色（`#F56C6C`），其余 `#333`
- 热门标识基于 `item.hot` 字段
- 点击 → 填入搜索框并执行搜索

### 搜索后状态

#### 搜索结果
- 使用 `QuestionCard` 展示结果
- 支持分页加载
- 空结果：`Empty` 组件提示"未找到相关题目"

---

## 交互逻辑

1. 输入框输入时实时搜索（防抖 500ms）或按回车搜索
2. 搜索结果展示后，搜索历史自动保存
3. 点击"清空"→ 清除搜索历史
4. 点击取消 → 返回上一页
5. 点击清除按钮 → 清空输入框，回到搜索前状态
6. 点击搜索结果 → `navigateTo` 题目详情

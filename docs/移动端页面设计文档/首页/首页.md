# 首页

> **路径**：`pages/index/index`  
> **Tab 页**：是（首页 Tab）  
> **页面背景**：`#f5f6fa`  
> **滚动方式**：`<scroll-view>` 整页滚动，支持下拉刷新

---

## 设计原则

底部 TabBar 已提供 **题库、练习、班级、我的** 四个一级入口，首页快捷入口应**避免重复**，聚焦以下功能：
- 高频但不在 TabBar 上的功能（错题本、考试、收藏、统计）
- 运营引导（轮播图、公告）
- 学习数据概览（今日学习、学习统计）

## 1. 自定义导航栏

**高度**：状态栏高度 + 88rpx

**背景**：白色 `#ffffff`，底部细线阴影 `0 1rpx 4rpx rgba(0,0,0,0.05)`

**布局**：
- 左侧：应用 Logo（48rpx）+ 应用名 **"题小助"**（34rpx，`#333`，font-weight: 700）
- 右侧：
  - 消息图标（✉️ 44rpx），右上角未读角标（红色圆点 `#ff4d4f`，16rpx；或数字角标，`#ff4d4f` 背景，白色文字 18rpx）
  - 签到图标（📅 44rpx），点击弹出签到弹窗

## 2. 搜索栏

**位置**：导航栏下方，内边距 24rpx

**样式**：
- 背景：`#F5F7FA`，圆角 32rpx

## 3. 轮播图（Banner）

**组件**：`<swiper>` + `<swiper-item>`

**尺寸**：
- 宽度：750rpx（满宽）
- 高度：320rpx
- 左右内边距：24rpx（图片实际宽度 702rpx，圆角 16rpx）

**轮播配置**：
- 自动播放：`autoplay="true"`
- 间隔：`interval="4000"`（4 秒）
- 切换时长：`duration="500"`（0.5 秒）
- 循环：`circular="true"`
- 指示点：底部居中圆点
  - 未选中：`rgba(255,255,255,0.5)`，12rpx
  - 选中：`#ffffff`，12rpx × 24rpx（椭圆）

**轮播内容**（3-5 张，后台可配置）：

| 序号 | 内容 | 背景 | 跳转 |
|------|------|------|------|
| 1 | 新用户专属：注册送 7 天 VIP | 蓝紫渐变 `#667eea` → `#764ba2` | 注册页 |
| 2 | 每日刷题挑战：完成 30 题赢积分 | 橙红渐变 `#f093fb` → `#f5576c` | 练习页 |
| 3 | 期末考试季：模拟真题免费练 | 蓝青渐变 `#4facfe` → `#00f2fe` | 考试列表 |
| 4 | 错题攻克计划：每天 5 道错题 | 绿青渐变 `#43e97b` → `#38f9d7` | 错题本 |

**每张 Banner 结构**：
```
┌─────────────────────────────────┐
│                                 │
│   标题文字（32rpx 白色 粗体）     │
│   副标题（24rpx 白色 半透明）     │
│   [立即参与]                     │  ← 白色药丸按钮（可选）
│                      🎯 插画    │
│                                 │
└─────────────────────────────────┘
```

- 左侧文字区（60%）：标题 + 副标题 + 可选按钮
- 右侧图标/插画区（40%）：装饰性图标

**交互**：
- 点击 Banner → `navigateTo` 对应页面
- 支持手动滑动切换

## 4. 快捷入口

**设计说明**：底部 TabBar 已覆盖"题库、练习、班级、我的"，快捷入口聚焦 TabBar 未覆盖的高频功能。

**布局**：4 宫格，等分宽度，白色圆角卡片

**样式**：
- 背景：白色圆角卡片（20rpx 圆角）
- 内边距：32rpx 0
- 图标：80rpx（圆形背景 + 图标）
- 文字：26rpx，`#333`，居中
- 图标与文字间距：16rpx

**入口配置**：

| 图标背景色 | 图标 | 文字 | 跳转 | 说明 |
|-----------|------|------|------|------|
| `#FCE4EC` | ❌ | 错题本 | `navigateTo` 错题列表 | 高频复习入口 |
| `#E8F5E9` | 📋 | 考试中心 | `navigateTo` 考试列表 | 不在 TabBar 上 |
| `#FFF3E0` | ⭐ | 我的收藏 | `navigateTo` 收藏页 | 快速查看收藏 |
| `#E3F2FD` | 📊 | 学习统计 | `navigateTo` 统计页 | 查看学习数据 |

**为什么选这 4 个**：
- ❌ **错题本**：复习高频入口，用户每天需要快速访问
- 📋 **考试中心**：考试不在 TabBar 上，但考试是核心功能之一
- ⭐ **我的收藏**：收藏的题目需要快速找回
- 📊 **学习统计**：激励用户查看学习进度，提升粘性

## 5. 公告滚动条

**位置**：快捷入口下方

**样式**：
- 背景：`#FFF8E1`（浅黄），圆角 12rpx
- 高度：64rpx
- 左侧：📢 图标（28rpx）
- 右侧：关闭按钮（✕ 24rpx，`#999`）

**滚动方式**：
- `<swiper>` 纵向自动轮播
- 间隔：`interval="3000"`（3 秒）
- 纵向：`vertical="true"`

**公告内容**：
- 文字：24rpx，`#333`
- 单行截断，支持 marquee 效果
- 点击公告 → `navigateTo` 通知详情页

**交互**：
- 点击关闭 → 隐藏公告栏（本次启动不再显示）
- 点击公告文字 → 跳转通知详情

## 首页与 TabBar 的功能分工

| 功能 | 入口位置 | 说明 |
|------|---------|------|
| 题库浏览 | **TabBar** → 题库 | 一级入口 |
| 刷题练习 | **TabBar** → 练习 | 一级入口 |
| 班级管理 | **TabBar** → 班级 | 一级入口 |
| 个人中心 | **TabBar** → 我的 | 一级入口 |
| 错题本 | **首页** → 快捷入口 | 高频复习，不在 TabBar |
| 考试中心 | **首页** → 快捷入口 | 核心功能，不在 TabBar |
| 我的收藏 | **首页** → 快捷入口 | 快速找回，不在 TabBar |
| 学习统计 | **首页** → 快捷入口 | 激励查看，不在 TabBar |
| 搜索 | **首页** → 搜索栏 | 全局搜索入口 |
| 消息通知 | **首页** → 导航栏 | 角标提醒 |
| 推荐题目 | **首页** → 内容区 | 内容发现 |
| 热门考试 | **首页** → 内容区 | 内容发现 |
| 热门分类 | **首页** → 内容区 | 内容发现 |

## 数据加载

`onMounted` 时并行调用：

| API | 说明 | 缓存策略 |
|-----|------|---------|
| `getBanners()` | 获取轮播图数据 | 缓存 10 分钟 |
| `getAnnouncements()` | 获取公告列表 | 缓存 5 分钟 |
| `getTodayStudy()` | 获取今日学习数据 | 不缓存 |
| `getRecommendQuestions()` | 获取推荐题目 | 缓存 5 分钟 |
| `getHotExams()` | 获取热门考试 | 缓存 5 分钟 |
| `getHotCategories()` | 获取热门分类 | 缓存 30 分钟 |
| `getUnreadCount()` | 获取未读通知数 | 不缓存 |

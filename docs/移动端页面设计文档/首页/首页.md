# 首页

> **路径**：`pages/index/index`  
> **Tab 页**：是（首页 Tab）  
> **页面背景**：`#f5f6fa`

## 1. 自定义导航栏

**高度**：状态栏高度 + 88rpx

**背景**：白色 `#ffffff`，底部细线阴影 `0 1rpx 4rpx rgba(0,0,0,0.05)`

**布局**：
- 左侧：应用 Logo（48rpx）+ 应用名 **"题小助"**（34rpx，`#333`，font-weight: 700）
- 右侧：
  - 通知图标（🔔 44rpx），右上角未读角标（红色圆点 `#ff4d4f`，16rpx；或数字角标，`#ff4d4f` 背景，白色文字 18rpx）
  - 签到图标（📅 44rpx），点击弹出签到弹窗

## 2. 搜索栏

**位置**：导航栏下方，内边距 24rpx

**样式**：
- 背景：`#F5F7FA`，圆角 32rpx
- 高度：72rpx
- 左侧：搜索图标（🔍 28rpx，`#c0c4cc`）

## 3. 轮播图（Banner）

**组件**：`<swiper>` + `<swiper-item>`

**尺寸**：
- 宽度：750rpx（满宽）
- 高度：320rpx
- 左右内边距：24rpx（图片实际宽度 702rpx，圆角 16rpx）

**轮播配置**：
- 自动播放：`autoplay="true"`
- 间隔：`interval="4000"`（4 秒）
- 切换时长：`duration="500"`（0.5 秒）
- 循环：`circular="true"`
- 指示点：底部居中圆点
  - 未选中：`rgba(255,255,255,0.5)`，12rpx
  - 选中：`#ffffff`，12rpx × 24rpx（椭圆）

**轮播内容**（3-5 张）：

| 序号 | 内容 | 背景 | 跳转 |
|------|------|------|------|
| 1 | 新用户专属：注册送 7 天 VIP | 蓝紫渐变 `#667eea` → `#764ba2` | 注册页 |
| 2 | 每日刷题挑战：完成 30 题赢积分 | 橙色渐变 `#f093fb` → `#f5576c` | 练习页 |
| 3 | 期末考试季：模拟真题免费练 | 绿色渐变 `#4facfe` → `#00f2fe` | 考试列表 |
| 4 | 班级功能上线：邀请同学一起学 | 粉色渐变 `#43e97b` → `#38f9d7` | 班级页 |

**每张 Banner 结构**：
- 左侧文字区（60%）：
  - 标题（32rpx，白色，粗体）
  - 副标题（24rpx，白色，半透明）
  - 按钮（可选）：白色药丸按钮，如 **"立即参与"**
- 右侧图标/插画区（40%）：
  - 装饰性图标或插画

**交互**：
- 点击 Banner → `navigateTo` 对应页面
- 支持手动滑动切换

## 数据加载

`onMounted` 时并行调用：

| API | 说明 | 缓存策略 |
|-----|------|---------|
| `getBanners()` | 获取轮播图数据 | 缓存 10 分钟 |
| `getAnnouncements()` | 获取公告列表 | 缓存 5 分钟 |
| `getTodayStudy()` | 获取今日学习数据 | 不缓存 |
| `getRecommendQuestions()` | 获取推荐题目 | 缓存 5 分钟 |
| `getHotExams()` | 获取热门考试 | 缓存 5 分钟 |
| `getHotCategories()` | 获取热门分类 | 缓存 30 分钟 |
| `getUnreadCount()` | 获取未读通知数 | 不缓存 |

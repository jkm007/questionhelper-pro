# 学习报告

> **路径**：`pages/profile/study-report`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 时间切换
- **本周** / **本月** 标签栏

### 学习概览卡片
- 学习天数 / 总练习数 / 总用时 / 平均正确率

### 数据对比
- 与上周/上月对比：
  - 练习量变化（↑12% / ↓5%）
  - 正确率变化（↑3% / ↓2%）
  - 学习时长变化

### 练习趋势
- Canvas 折线图（每日练习数）

### 正确率趋势
- Canvas 折线图（每日正确率）

### 薄弱知识点
- TOP 5 薄弱知识点列表
- 每个知识点：名称 + 正确率 + 建议

### 学习建议
- 根据数据生成的个性化建议

---

## 交互逻辑

1. 切换本周/本月 → 重新加载数据

# 学习计划

> **路径**：`pages/profile/study-plan`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 当前计划卡片
- 计划名称
- 每日目标：XX 题
- 本周进度：已完成 XX / 目标 XX 题
- 进度条（`#4A90D9` 渐变）
- 连续执行天数

### 计划列表
- 我的计划列表
- 每个计划：名称 + 每日目标 + 状态（进行中/已暂停/已完成）
- 操作：编辑 / 暂停 / 删除

### 创建计划弹窗
- 计划名称输入框
- 每日目标题数（10-100）
- 选择分类（可选，全部/指定分类）
- 选择难度（可选，全部/指定难度）
- 提醒时间（可选，时间选择器）

---

## 交互逻辑

1. 加载当前计划和历史计划
2. 点击创建 → 弹出创建计划弹窗
3. 创建成功 → 设为当前计划
4. 编辑 → 修改计划参数
5. 暂停/恢复 → 切换计划状态
6. 删除 → 确认弹窗

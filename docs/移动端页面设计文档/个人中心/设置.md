# 设置

> **路径**：`pages/profile/settings`  
> **UI 语言**：英文  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"Settings"**
- 返回按钮

### 设置分组

#### Notifications（通知设置）
| 菜单项 | 类型 | 说明 |
|--------|------|------|
| Push Notifications | 开关 | 推送通知 |
| Sound | 开关 | 提示音 |
| Vibration | 开关 | 震动 |

#### General（通用设置）
| 菜单项 | 类型 | 说明 |
|--------|------|------|
| Language | 选择器 | 弹出选择器：English / Chinese Simplified / Chinese Traditional / Japanese / Korean |

#### Storage（存储）
| 菜单项 | 类型 | 说明 |
|--------|------|------|
| Cache Size | 信息 | 显示缓存大小 |
| Clear Cache | 按钮 | 清除缓存 |

#### Legal（法律条款）
| 菜单项 | 类型 | 说明 |
|--------|------|------|
| Privacy Policy | 链接 | 跳转 WebView |
| Terms of Service | 链接 | 跳转 WebView |

#### About（关于）
| 菜单项 | 类型 | 说明 |
|--------|------|------|
| Version | 信息 | 显示版本号 |

### 账号注销
- 红色文字 **"Deactivate Account"**（`#ff4d4f`）
- 双重确认弹窗（两次 `showModal` 确认，无需输入密码）

---

## 交互逻辑

1. 切换开关 → 调用更新设置 API
2. 点击 Language → 弹出选择器 → 选择后更新
3. 点击 Clear Cache → 确认弹窗 → 清除本地缓存
4. 点击 Privacy Policy / Terms of Service → `navigateTo` WebView
5. 点击 Deactivate Account → 第一次确认 → 第二次确认 → 调用注销 API → 清除数据 → 跳转登录页

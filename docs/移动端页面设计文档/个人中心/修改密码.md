# 修改密码

> **路径**：`pages/profile/change-password`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 表单字段
- **原密码**：输入框，带显示/隐藏切换
- **新密码**：输入框，带显示/隐藏切换 + 密码强度指示器
- **确认新密码**：输入框，带显示/隐藏切换

### 密码强度指示器
- 弱（红色）：< 6 位或纯数字
- 中（橙色）：6-10 位，含字母
- 强（绿色）：> 10 位，含字母+数字+特殊字符

### 提交按钮
- **确认修改**：`#4a90d9` 全宽按钮

---

## 交互逻辑

1. 输入原密码 + 新密码 + 确认密码
2. 表单验证：原密码非空、新密码 ≥ 6 位、两次一致
3. 提交 → 调用修改密码 API
4. 成功 → Toast 提示"密码修改成功" + 清除 Token + 跳转登录页

# 申请创作者

> **路径**：`pages/profile/apply`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"申请创作者"**
- 返回按钮

### 状态显示
同实名认证的 3 种状态卡片：
- **已通过**：绿色上边框 + 创作者权限说明
- **审核中**：橙色上边框 + 提示文字
- **已拒绝**：红色上边框 + 拒绝原因 + 重新提交按钮

### 创作者权益列表
- 图标 + 标题 + 描述
- 权益项：
  - 创建和管理题目
  - 创建试卷和考试
  - 查看创作数据统计

### 申请表单

#### 申请理由
- 文本域（必填）
- 最少 10 字
- placeholder：请说明您申请创作者的理由

#### 作品集描述
- 文本域（必填）
- 最少 10 字
- placeholder：请描述您的教育相关经验或作品

#### 用户协议
- 复选框 + 文字：**"我已阅读并同意《创作者协议》"**

### 提交按钮
- `#4a90d9` 全宽按钮

---

## 交互逻辑

1. 加载申请状态
2. 已通过 → 显示权限说明，隐藏表单
3. 审核中 → 显示提示，隐藏表单
4. 已拒绝 → 显示原因 + 可重新提交
5. 未申请 → 显示表单
6. 提交 → 表单验证 + 调用申请 API
7. 成功 → Toast 提示 + 返回

---

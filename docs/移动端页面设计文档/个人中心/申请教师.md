# 申请教师

> **路径**：`pages/profile/apply-teacher`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 状态卡片（已申请时显示）
- **审核中**：橙色上边框 + 提示文字
- **已通过**：绿色上边框 + 教师权限说明
- **已拒绝**：红色上边框 + 拒绝原因 + 重新申请按钮

### 教师权益列表
- 📚 创建和管理班级
- 📋 发布考试和布置作业
- 📊 查看学生学习数据
- ✏️ 审批班级创作者申请

### 申请表单
- 申请理由（文本域，最少 10 字）
- 教学经验证明（文本域，最少 10 字）
- 资质证书上传（可选，最多 3 张）
- 用户协议复选框

### 提交按钮
- `#4a90d9` 全宽按钮

---

## 交互逻辑

1. 加载申请状态
2. 未申请 → 显示表单
3. 提交 → 调用申请 API
4. 成功 → Toast 提示 + 返回

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 状态卡片 | 白色，16rpx 圆角，颜色编码上边框 |
| 权益列表 | 白色卡片，图标 + 文字 |
| 文本域高度 | 200rpx |
| 证书上传 | 180rpx 方形 |
| 提交按钮 | `#4a90d9`，88rpx 高 |

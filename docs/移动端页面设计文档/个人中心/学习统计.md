# 学习统计（个人中心）

> **路径**：`pages/profile/statistics`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"学习统计"**
- 返回按钮

### 汇总卡片
3 张卡片横排：
- **Total Practice**（总练习数）
- **Total Exams**（总考试数）
- **Avg Accuracy**（平均正确率）

### 练习趋势图
- 标题：**"练习趋势"**
- Canvas 折线图（30 天）
- 网格 + 折线 + 数据点（`#4a90d9`）

### 考试分数趋势
- 标题：**"考试分数趋势"**
- Canvas 柱状图
- 网格 + 柱形（`#67c23a`）

### 分类正确率分布
- 标题：**"分类正确率"**
- 进度条列表
- 每个分类：名称 + 进度条（渐变 `#4a90d9` → `#67c23a`）+ 百分比

---

## 数据加载

调用 API：
- `getUserStatistics` — 获取学习统计数据

---

## 交互逻辑

1. 进入页面加载统计数据
2. 图表支持触摸交互（显示数据点详情）

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 汇总卡片 | 白色，20rpx 圆角，阴影 |
| 图表高度 | 400rpx |
| 折线颜色 | `#4a90d9` |
| 柱形颜色 | `#67c23a` |
| 进度条 | 12rpx 高，渐变填充 |

# 我的班级（个人中心）

> **路径**：`pages/profile/my-class`  
> **UI 语言**：英文  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"My Classes"**
- 返回按钮

### 班级列表
每项包含：
- 封面图（200rpx 宽）
- 班级名称（28rpx，`#333`，粗体）
- 角色徽章：
  - Owner：`#fff3e0` 橙底 + `#fa8c16` 文字
  - Admin：`#e6f0ff` 蓝底 + `#1677ff` 文字
  - Member：`#f5f5f5` 灰底 + `#999` 文字
- 人数（24rpx，`#999`）
- 描述（24rpx，`#666`，单行截断）

### 空状态
- 图标 + **"暂未加入任何班级"**
- **"Browse Classes"** 按钮（`#4a90d9`）

---

## 交互逻辑

1. 加载我的班级列表
2. 点击班级 → `navigateTo` 班级详情页
3. 点击"Browse Classes"→ `switchTab` 到班级 Tab
4. 下拉刷新 + 上拉加载

---

| 元素 | 样式 |

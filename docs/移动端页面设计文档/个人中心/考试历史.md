# 考试历史（个人中心）

> **路径**：`pages/profile/exam-history`  
> **UI 语言**：英文  
> **页面背景**：`#f5f5f5`

---

## 页面结构

### 导航栏
- 标题：**"Exam History"**
- 返回按钮

### 筛选栏
药丸标签：
- **All**（全部）
- **Completed**（已完成）
- **In Progress**（进行中）
- **Graded**（已评分）

### 考试卡片
每张卡片包含：
- 标题（30rpx，`#333`，粗体）
- 状态徽章：
  - completed：`#e6f7e6` 绿底 + `#4caf50` 文字
  - in_progress：`#fff3e0` 橙底 + `#ff9800` 文字
  - timeout：`#ffeaea` 红底 + `#f44336` 文字
  - graded：`#e6f0ff` 蓝底 + `#1677ff` 文字
- 分数（带图标 📊）
- 时长（带图标 ⏱）
- 日期（带图标 📅）
- **"查看详情"** 链接（`#4a90d9`）

---

## 交互逻辑

1. 加载考试历史列表
2. 切换筛选标签 → 筛选对应状态
3. 点击"查看详情"→ `navigateTo` 考试结果页
4. 下拉刷新 + 上拉加载

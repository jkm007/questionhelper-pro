# 我的创作

> **路径**：`pages/profile/my-creation`  
> **UI 语言**：英文  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"My Creations"**
- 返回按钮

### 标签栏
- **Questions**（题目）
- **Papers**（试卷）
- **Exams**（考试）
- 激活标签下划线（`#4a90d9`）

### 创作卡片
每张卡片包含：
- 标题（28rpx，`#333`，粗体）
- 状态徽章：
  - Draft（草稿）：灰色
  - Published（已发布）：绿色
  - Rejected（已拒绝）：红色
  - Reviewing（审核中）：橙色
- 浏览数 + 使用数（24rpx，`#999`）
- 创建时间（22rpx，`#999`）
- 操作按钮：
  - **Edit**（`#e6f0ff` 背景，`#1677ff` 文字）
  - **Delete**（`#ffeaea` 背景，`#ff4d4f` 文字）

### 空状态
- 图标 + **"暂无创作内容"**

---

## 交互逻辑

1. 加载我的创作列表（默认显示 Questions 标签）
2. 切换标签 → 加载对应类型创作
3. 点击 Edit → `navigateTo` 编辑页
4. 点击 Delete → 确认弹窗 → 调用删除 API
5. 下拉刷新 + 上拉加载

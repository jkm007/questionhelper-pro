# 个人信息

> **路径**：`pages/profile/info`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"个人信息"**
- 返回按钮

### 头像区
- 120rpx 圆形头像
- 右下角相机图标覆盖层（40rpx 圆形，`#4a90d9` 背景）
- 点击上传（`uni.uploadFile` 到 `/api/user/avatar`）

### 表单字段
白色圆角卡片，cell 列表样式：

| 字段 | 类型 | 说明 |
|------|------|------|
| 昵称 | 输入框 | 可编辑 |
| 性别 | 选择器 | `uni-popup` 弹出底部选择面板（男/女/保密） |
| 生日 | 日期选择器 | `uni-popup` 内嵌 `picker mode="date"` |
| 个人简介 | 文本域 | 200 字限制 |

### 保存按钮
- `#4a90d9` 全宽圆角按钮
- 底部固定（`position: fixed`）

---

## 交互逻辑

1. 加载当前用户信息填充表单
2. 点击头像 → 选择图片 → 上传 → 更新头像
3. 点击性别 → 弹出选择面板
4. 点击生日 → 弹出日期选择器
5. 点击保存 → 调用更新 API
6. 成功 → Toast 提示 + 返回上一页

---

| 元素 | 样式 |

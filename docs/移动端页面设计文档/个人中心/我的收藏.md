# 我的收藏

> **路径**：`pages/profile/favorites`  
> **UI 语言**：英文  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"My Favorites"**
- 返回按钮

### 标签栏
- **Questions**（题目）
- **Exams**（考试）
- 激活标签下划线（`#4a90d9`）

### 批量操作栏
- 全选复选框
- 已选数量
- **移除** 按钮（`#e74c3c`，仅选中时显示）

### 收藏列表
每项包含：
- 复选框（批量模式下显示）
- 内容：标题 + 类型徽章 + 收藏时间
- **移除** 按钮（右侧）

### 空状态
- 图标 + **"暂无收藏"** 文字

---

## 交互逻辑

1. 加载收藏列表（默认显示 Questions 标签）
2. 切换标签 → 加载对应类型收藏
3. 批量模式 → 显示复选框，可多选移除
4. 点击单项 → 跳转详情页
5. 移除 → 确认弹窗 → 调用移除 API
6. 下拉刷新 + 上拉加载

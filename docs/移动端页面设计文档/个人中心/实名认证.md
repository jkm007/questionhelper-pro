# 实名认证

> **路径**：`pages/profile/auth`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"实名认证"**
- 返回按钮

### 状态卡片（已认证时显示）
根据认证状态显示：
- **已通过**：绿色上边框（`#4caf50`）+ 姓名（脱敏）+ 身份证（脱敏）
- **审核中**：橙色上边框（`#ff9800`）+ 提示文字 **"审核中，请耐心等待"**
- **已拒绝**：红色上边框（`#e74c3c`）+ 拒绝原因 + **重新提交** 按钮

### 认证表单

#### 真实姓名
- 输入框（必填）
- placeholder：请输入真实姓名

#### 身份证号
- 输入框（必填）
- 正则验证：18 位身份证号码
- placeholder：请输入身份证号

#### 身份证正面照片
- 虚线边框占位（`aspect-ratio: 1.6`）
- 点击选择图片
- 选中后显示预览图

#### 身份证反面照片
- 同正面照片

### 提交按钮
- `#4a90d9` 全宽按钮
- 已通过时隐藏

---

## 交互逻辑

1. 加载认证状态
2. 已通过 → 显示脱敏信息，隐藏表单
3. 审核中 → 显示提示，隐藏表单
4. 已拒绝 → 显示原因 + 可重新提交
5. 未认证 → 显示表单
6. 上传身份证照片 → 调用文件上传 API
7. 提交 → 表单验证 + 调用认证 API
8. 成功 → Toast 提示 + 返回

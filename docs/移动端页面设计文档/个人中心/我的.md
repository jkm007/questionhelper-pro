# 我的（个人中心主页）

> **路径**：`pages/profile/index`  
> **Tab 页**：是（我的 Tab）  
> **UI 语言**：英文  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 用户卡片
- 渐变背景（`#4a90d9` → `#357abd`）
- 头像（120rpx 圆形，白色边框 4rpx）
- 昵称（32rpx，白色，粗体）+ 等级徽章
- 用户 ID（24rpx，白色，半透明）

### 菜单分组
3 组白色圆角卡片：

#### 第一组
| 菜单项 | 图标 |
|--------|------|
| My Info | 👤 |
| Exam History | 📋 |
| My Statistics | 📊 |
| My Favorites | ⭐ |
| My Classes | 👥 |
| My Creations | ✏️ |

#### 第二组
| 菜单项 | 图标 | 说明 |
|--------|------|------|
| Real-name Auth | 🆔 | 状态标签（已认证=`#e6f7e6` 绿底） |
| Apply Creator | 📝 | 状态标签（创作者=`#fff3e0` 橙底） |

#### 第三组
| 菜单项 | 图标 |
|--------|------|
| Settings | ⚙️ |
| About | ℹ️ |

- 每个菜单项：表情图标 + 英文标签 + 右箭头
- 高度：100rpx

### 退出登录
- 红色文字按钮（`#ff4d4f`）
- 点击弹出确认弹窗

---

## 交互逻辑

1. 加载用户信息
2. 点击菜单项 → `navigateTo` 对应页面
3. 点击"About"→ 弹出 `showModal` 对话框（非独立页面）
4. 点击"退出登录"→ 确认弹窗 → 清除 Token + 跳转登录页

# 忘记密码

> **路径**：`pages/login/forget`  
> **页面背景**：`#ffffff`

---

## 页面结构

### 导航栏
- 标题：**"忘记密码"**
- 返回按钮

### 步骤指示器
3 步横向流程：
1. 验证手机
2. 重置密码
3. 完成

- 圆点（32rpx）+ 连接线 + 文字标签
- 激活状态：`#4A90D9` 圆点 + 文字
- 已完成：`#67C23A` 圆点 + 对勾
- 未激活：`#ddd` 圆点 + 文字

### 步骤 1：验证手机
- 手机号输入框
- 短信验证码输入框 + 发送按钮（60 秒倒计时）
- **下一步**按钮（`#4A90D9`）

### 步骤 2：重置密码
- 新密码输入框（带显示/隐藏切换）
- 确认密码输入框（带显示/隐藏切换）
- **重置密码**按钮（`#4A90D9`）

### 步骤 3：完成
- 成功图标：绿色圆（120rpx）+ 白色对勾 ✓
- 文字：**"密码重置成功"**（32rpx，`#333`）
- **去登录**按钮（`#4A90D9`）

---

## 交互逻辑

1. 步骤 1 → 步骤 2：验证手机号和验证码正确
2. 步骤 2 → 步骤 3：密码校验（最少 6 位，两次一致）+ 调用重置密码 API
3. 步骤 3：点击"去登录"→ `redirectTo` 登录页
4. 物理返回键：回到上一步，步骤 1 时返回登录页

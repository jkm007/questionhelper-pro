> **Tab 页**：否  
> **页面背景**：`#ffffff`

## 页面结构

### Logo 区
- 应用图标（160rpx × 160rpx，居中）
- 应用名：**"题小助"**（40rpx，`#4A90D9`，font-weight: 700）
- 标语：**"学习好帮手"**（28rpx，`#999`）

### 登录表单
- **用户名/手机号输入框**
  - 背景：`#F5F7FA`，圆角 16rpx
  - 左侧图标：👤
  - placeholder：请输入用户名或手机号

- **密码输入框**
  - 背景：`#F5F7FA`，圆角 16rpx
  - 左侧图标：🔒

  - placeholder：请输入密码

- **图形验证码**
  - 水平排列：左侧验证码输入框 + 右侧验证码图片
  - 输入框背景：`#F5F7FA`，圆角 16rpx，左侧图标：🛡️，placeholder：请输入验证码
  - 验证码图片：高度与输入框一致（88rpx），宽度 220rpx，圆角 8rpx

- **记住我**
  - 位于验证码下方、登录按钮上方，左对齐
  - 复选框 + 文字"记住我"（24rpx，`#666`）
  - 默认不勾选；勾选后登录请求携带 `rememberMe: true`，Refresh Token 有效期延长至 30 天

- **登录按钮**
  - `#4A90D9` 全宽圆角按钮（88rpx 高，44rpx 圆角）
  - 文字：**"登录"**（白色，32rpx）

### 底部链接
- 链接行：**"注册"** | **"忘记密码"**（24rpx，`#4A90D9`）
- 分隔线 + 微信登录圆形按钮
  - 背景：`#07C160`
  - 文字：**"微信"**（白色，20rpx）

## 交互逻辑

1. 页面加载时自动调用 `/api/v1/auth/captcha` 获取图形验证码（captchaId + captchaBase64）
2. 输入验证：用户名、密码、验证码非空校验
3. 调用 `useUserStore().login()` 接口，传入 `username`、`password`、`captchaId`、`captchaCode`、`rememberMe`
4. 登录成功 → `uni.switchTab({ url: '/pages/index/index' })`
5. 登录失败（含验证码错误）→ `uni.showToast` 显示错误信息，并自动刷新验证码
6. 点击验证码图片 → 重新获取验证码
7. 点击密码输入框右侧图标 → 切换密码明文/密文显示
8. 点击"注册"→ `navigateTo` 注册页
9. 点击"忘记密码"→ `navigateTo` 忘记密码页
10. 点击微信登录 → 调用微信授权登录流程

## API 接口

### 获取图形验证码

- **接口**：`GET /api/v1/auth/captcha`
- **响应**：

```json
{
  "code": "00000",
  "data": {
    "captchaId": "string",
    "captchaBase64": "string"
  }
}
```

### 登录

- **接口**：`POST /api/v1/auth/login`
- **请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 用户名或手机号 |
| password | string | 是 | 密码 |
| captchaId | string | 是 | 图形验证码 ID（从获取验证码接口获得） |
| captchaCode | string | 是 | 用户输入的验证码 |
| rememberMe | bool | 否 | 记住我，Refresh Token 延长至 30 天 |

- **请求示例**：

```json
{
  "username": "admin",
  "password": "admin123",
  "captchaId": "xxxxx",
  "captchaCode": "Ab3d",
  "rememberMe": true
}
```

- **响应**：

```json
{
  "code": "00000",
  "data": {
    "accessToken": "string",
    "refreshToken": "string",
    "tokenType": "Bearer",
    "expiresIn": 7200,
    "user": {
      "id": 1,
      "username": "admin",
      "nickname": "管理员"
    }
  }
}
```

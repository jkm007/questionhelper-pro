# 网页容器

> **路径**：`pages/webview/index`  
> **参数**：`?url=xxx&title=xxx`  
> **页面背景**：`#ffffff`

---

## 页面结构

### 导航栏
- 标题：动态标题（从参数或网页获取）
- 返回按钮

### 加载进度条
- 固定顶部
- `#007aff` 蓝色
- `translateX` 动画（从左到右）
- 高度：4rpx

### 错误状态
- 🌐 地球图标（120rpx，`#c0c4cc`）
- 标题：**"页面加载失败"**（32rpx，`#333`）
- 错误信息（26rpx，`#999`）
- **Retry** 按钮（`#4A90D9`）
- **Back** 按钮（灰色）

### WebView 组件
- `<web-view :url="url">`
- 支持：
  - 消息接收（标题更新、关闭页面）
  - 加载完成/错误处理
  - URL 验证（仅允许 `http:` / `https:` 协议）

---

## 交互逻辑

1. 验证 URL 协议（仅允许 http/https）
2. 验证失败 → 显示错误状态
3. 加载中 → 显示进度条
4. 加载完成 → 隐藏进度条
5. 加载失败 → 显示错误状态
6. 网页发送消息 → 接收处理（标题更新、关闭页面）
7. 返回逻辑：优先 `navigateBack`，无历史页时跳转首页

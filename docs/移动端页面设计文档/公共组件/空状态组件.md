# 空状态组件（Empty）

> **组件路径**：`components/Empty/index.vue`  
> **类型**：通用占位组件  
> **触发时机**：列表数据为空时展示

---

## 组件结构

```
┌─────────────────────────────┐
│                             │
│          📭                 │  ← 图标/插画
│                             │
│     暂无数据                 │  ← 主标题
│     快去刷题吧~              │  ← 副标题（可选）
│                             │
│     [ 去刷题 ]              │  ← 操作按钮（可选）
│                             │
└─────────────────────────────┘
```

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `icon` | String | `'📭'` | 图标或图片 URL |
| `title` | String | `'暂无数据'` | 主标题 |
| `description` | String | `''` | 副标题 |
| `buttonText` | String | `''` | 按钮文字（不传则不显示按钮） |

## 使用场景

| 页面 | icon | title | description | buttonText |
|------|------|-------|-------------|------------|
| 错题列表 | ❌ | 暂无错题 | 太棒了，继续保持！ | 去刷题 |
| 我的收藏 | ⭐ | 暂无收藏 | 快去收藏感兴趣的题目吧 | 去题库 |
| 考试历史 | 📋 | 暂无考试记录 | 快去参加考试吧 | 查看考试 |
| 通知列表 | 🔔 | 暂无通知 | — | — |
| 班级列表（我的） | 👥 | 暂未加入任何班级 | 加入班级和同学一起学习 | 去发现班级 |
| 班级列表（发现） | 🌍 | 暂无发现班级 | 可通过班级码或二维码加入 | — |
| 搜索结果 | 🔍 | 未找到相关题目 | 换个关键词试试吧 | — |
| 评论列表 | 💬 | 暂无评论 | 快来发表第一条评论吧 | — |

## 样式规范

| 元素 | 样式 |
|------|------|
| 图标 | 160rpx，居中 |
| 主标题 | 32rpx，`#333` |
| 副标题 | 26rpx，`#999` |
| 按钮 | `#4A90D9`，64rpx 高，32rpx 圆角 |
| 整体内边距 | 120rpx 0 |

# 签到弹窗（CheckIn Modal）

> **组件路径**：`components/CheckInModal/index.vue`  
> **类型**：居中弹窗（Modal）  
> **触发时机**：首页导航栏点击签到图标（📅），或每日首次打开 App 自动弹出

---

## 弹窗结构

### 未签到状态

```
┌─────────────────────────────┐
│         每日签到              │
│                             │
│    ┌──┬──┬──┬──┬──┬──┬──┐  │
│    │一│二│三│四│五│六│日│  │  ← 本周日历
│    │✓│✓│✓│✓│✓│ │ │  │
│    └──┴──┴──┴──┴──┴──┴──┘  │
│                             │
│    已连续签到 5 天            │
│    再签 2 天可获得额外奖励     │
│                             │
│      [ 立即签到 ]            │  ← 签到按钮
│                             │
│    补签（本月剩余 2 次）       │  ← 补签入口
└─────────────────────────────┘
```

### 已签到状态

```
┌─────────────────────────────┐
│         ✓ 签到成功            │
│                             │

│                             │
│    今日奖励：+10 积分         │
│    连续 7 天奖励：+50 积分    │
│                             │
│      [ 知道了 ]              │
└─────────────────────────────┘
```

---

## 功能说明

### 日历视图
- 展示本周 7 天
- 已签到：绿色对勾（`#52c41a`）
- 今日未签到：蓝色高亮边框（`#4A90D9`）
- 未签到：灰色

### 连续签到
- 显示连续签到天数
- 连续 7 天：额外奖励 +50 积分
- 连续 30 天：额外奖励 +200 积分

### 补签
- 每月允许补签 2 次
- 补签消耗积分（如 50 积分/次）
- 点击补签 → 选择日期 → 确认弹窗

### 自动弹出规则
- 每日首次打开 App 且未签到时自动弹出
- 弹出后本次启动不再弹出
- 已签到不弹出

---

## 交互逻辑

1. 点击签到图标或首次打开 App → 弹出签到弹窗
2. 点击"立即签到"→ 调用签到 API → 更新为已签到状态 → 显示奖励
3. 点击"补签"→ 选择日期 → 确认 → 扣除积分 + 补签成功
4. 点击遮罩或"知道了"→ 关闭弹窗

# 图片预览（Image Preview）

> **组件方式**：调用 `uni.previewImage` API  
> **触发时机**：题目详情图片、评论图片、纠错佐证图片、身份证照片等  
> **支持平台**：H5 / 微信小程序 / APP

---

## 功能说明

### 基础功能
- 全屏展示图片
- 左右滑动切换多张图片
- 双指缩放（放大/缩小）
- 底部指示点（多图时显示）

### 高级功能
- 长按弹出操作菜单：
  - 保存图片到相册
  - 识别二维码（图片含二维码时）
  - 转发给朋友（APP/小程序）

### 调用方式

```typescript
uni.previewImage({
  current: currentIndex,     // 当前显示图片索引
  urls: imageUrls,           // 图片 URL 数组
  longPressActions: {        // 长按操作（APP/小程序）
    itemList: ['保存图片', '识别二维码', '转发给朋友'],
    success: (res) => { /* 处理选择 */ }
  }
})
```

---

## 使用场景

| 场景 | 触发方式 | 图片来源 |
|------|---------|---------|
| 题目详情 | 点击题目中的图片 | 题目内容 HTML 中的 `<img>` |
| 评论列表 | 点击评论图片网格 | 评论图片数组 |
| 题目纠错 | 点击佐证图片 | 纠错附件图片 |
| 实名认证 | 点击身份证照片 | 用户上传的照片 |
| 错题详情 | 点击错题图片 | 错题内容中的图片 |

## 样式规范

| 元素 | 样式 |
|------|------|
| 背景 | 黑色 `#000000` |
| 底部指示点 | 白色圆点，当前高亮 |
| 底部操作栏 | 半透明黑色背景 |

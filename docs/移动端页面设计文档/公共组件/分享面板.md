# 分享面板（SharePanel）

> **组件路径**：`components/SharePanel/index.vue`  
> **类型**：底部弹出面板（Bottom Sheet）  
> **触发时机**：题目详情、练习结果、考试结果等页面点击"分享"按钮

---

## 面板结构

```
┌─────────────────────────────┐
│           分享到              │  ← 标题
├─────────────────────────────┤
│  微信好友  朋友圈  复制链接   │  ← 分享渠道
│  保存海报  保存图片           │
├─────────────────────────────┤
│          取消                │  ← 取消按钮
└─────────────────────────────┘
```

### 分享渠道（平台差异）

| 渠道 | H5 | 微信小程序 | APP |
|------|-----|-----------|-----|
| 微信好友 | ❌ 隐藏 | ✅ `wx.shareMessage` | ✅ 系统分享 |
| 朋友圈 | ❌ 隐藏 | ✅ `wx.shareTimeline` | ✅ 系统分享 |
| 复制链接 | ✅ `uni.setClipboardData` | ✅ 复制小程序路径 | ✅ 复制链接 |
| 保存海报 | ✅ Canvas 生成 + 保存 | ✅ Canvas 生成 + 保存 | ✅ Canvas 生成 + 保存 |
| 保存图片 | ✅ 长按保存 | ✅ `wx.saveImageToPhotosAlbum` | ✅ 保存到相册 |

### 分享内容配置

| 分享类型 | 标题 | 图片 | 路径 |
|---------|------|------|------|
| 题目分享 | 题目标题 | 题目缩略图 | 题目详情页 |
| 练习结果 | "我在题小助完成了XX题练习" | 成绩海报 | 练习结果页 |
| 考试结果 | "我在XX考试中获得XX分" | 成绩海报 | 考试结果页 |
| 班级邀请 | "邀请你加入XX班级" | 班级封面 | 加入班级页（含班级码） |

---

## 交互逻辑

1. 点击分享按钮 → 从底部弹出面板（`0.3s ease` 动画）
2. 点击渠道 → 执行对应分享逻辑
3. 复制链接 → Toast 提示"链接已复制"
4. 保存海报 → 生成 Canvas 海报 → 保存到相册 → Toast 提示
5. 点击取消或遮罩 → 关闭面板

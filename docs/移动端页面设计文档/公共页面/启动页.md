# 启动页（Splash）

> **路径**：`pages/common/splash.vue`（或通过 `App.vue` 实现）  
> **触发时机**：应用冷启动  
> **显示时长**：1.5 - 2 秒（或等待初始化完成）  
> **页面背景**：白色 `#ffffff`

---

## 页面结构

### Logo 区
- 应用图标（`app-icon.svg`），居中显示
- 尺寸：240rpx × 240rpx
- 圆角：48rpx（与图标风格一致）

### 品牌区
- 应用名：**"题小助"**（48rpx，`#4A90D9`，font-weight: 700）
- 标语：**"学习好帮手"**（28rpx，`#999`）

### 加载指示器
- 底部圆形加载动画（`uni.showLoading` 或自定义 CSS 动画）
- 颜色：`#4A90D9`

### 版本号
- 最底部显示版本号（20rpx，`#ccc`）
- 格式：`v2.0.0`

---

## 交互逻辑

1. 应用启动后显示启动页
2. 并行执行初始化：检查登录状态、加载配置、预加载数据
3. 初始化完成或超时（2 秒）后跳转：
   - 已登录 → `uni.switchTab({ url: '/pages/index/index' })`
   - 未登录 → `uni.redirectTo({ url: '/pages/login/index' })`
4. 仅在冷启动时显示，热启动不显示

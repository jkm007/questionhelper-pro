# 用户协议 / 隐私政策

> **路径**：`pages/common/agreement.vue`  
> **参数**：`?type=privacy`（隐私政策）| `?type=terms`（用户协议）  
> **触发时机**：注册页点击协议链接，或设置页查看协议  
> **页面背景**：`#ffffff`

---

## 页面结构

### 导航栏
- 标题：**"用户协议"** 或 **"隐私政策"**（根据 `type` 参数切换）
- 返回按钮

### 内容区
- 富文本渲染（`rich-text` 组件或 `v-html`）
- 内容来源：后端 API 或本地 HTML 文件
- 样式：28rpx 正文，32rpx 小标题，行高 1.8

### 底部操作（仅从注册页进入时显示）
- **同意并继续**：`#4A90D9` 全宽按钮
- 点击后返回注册页并自动勾选协议复选框

---

## 交互逻辑

1. 根据 `type` 参数加载对应协议内容
2. 从注册页进入时显示底部"同意"按钮
3. 从设置页进入时不显示底部按钮（仅供查看）
4. 支持页面内滚动浏览

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 页面背景 | `#ffffff` |
| 标题 | 36rpx，`#333`，font-weight: 600 |
| 正文 | 28rpx，`#333`，行高 1.8 |
| 小标题 | 32rpx，`#333`，font-weight: 600 |
| 底部按钮 | `#4A90D9` 背景，白色文字，fixed 定位 |

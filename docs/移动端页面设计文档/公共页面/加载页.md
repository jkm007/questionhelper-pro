# 加载页

> **路径**：`pages/common/loading.vue`  
> **触发时机**：页面数据加载中（可选使用，部分页面使用内联加载状态）  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 全屏加载状态（用于页面级加载）
- 居中显示加载动画（CSS 旋转圆环或 Lottie 动画）
- 颜色：`#4A90D9`
- 尺寸：80rpx × 80rpx
- 下方文字：**"加载中..."**（28rpx，`#999`）

### 骨架屏模式（用于列表页加载）
- 模拟页面结构的灰色占位块
- 使用 CSS `@keyframes` 闪烁动画（`#f0f0f0` → `#e0e0e0`）
- 圆角与实际元素一致

### 内联加载状态（组件级）
- 用于列表底部"加载更多"
- 三种状态：
  - 加载中：旋转图标 + "加载中..."
  - 已完成："— 没有更多了 —"
  - 点击加载："上拉加载更多" 或 "点击加载更多"

---

## 交互逻辑

1. 全屏加载：页面 `onMounted` 时显示，数据返回后隐藏
2. 骨架屏：进入页面立即显示，首屏数据返回后过渡到真实内容
3. 内联加载：滚动到底部自动触发，或点击触发

---

| 元素 | 样式 |

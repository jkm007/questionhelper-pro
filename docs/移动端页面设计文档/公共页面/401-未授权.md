# 401 未授权页面

> **路径**：`pages/common/401.vue`  
> **触发时机**：Token 过期或无效，API 返回 401 且 Refresh Token 刷新失败  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 图标区
- 大号锁形图标（🔒 或 SVG），居中显示
- 尺寸：200rpx × 200rpx
- 颜色：`#999` 或 `#c0c4cc`

### 文案区
- 主标题：**"登录已过期"**（36rpx，`#333`，粗体）
- 副标题：**"请重新登录以继续使用"**（28rpx，`#999`）

### 操作按钮
- **重新登录**：`#4A90D9` 全宽圆角按钮（80rpx 高，44rpx 圆角）
  - 点击后清除本地 Token，跳转登录页
- **返回首页**：文字按钮（28rpx，`#4A90D9`）
  - 点击后 `uni.switchTab` 到首页

---

## 交互逻辑

1. 页面加载时自动清除本地存储的 access_token 和 refresh_token
2. 点击"重新登录"→ `uni.reLaunch({ url: '/pages/login/index' })`
3. 点击"返回首页"→ `uni.switchTab({ url: '/pages/index/index' })`
4. 物理返回键拦截：阻止返回上一页，直接跳转首页

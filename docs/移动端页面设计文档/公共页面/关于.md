# 关于页面

> **路径**：`pages/common/about.vue`  
> **触发时机**：个人中心 → 设置 → 关于  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 品牌区
- 应用图标（`app-icon.svg`），居中
- 尺寸：160rpx × 160rpx，圆角 32rpx
- 应用名：**"题小助"**（36rpx，`#333`，font-weight: 600）
- 版本号：**"v2.0.0"**（24rpx，`#999`）

### 信息列表
- 白色圆角卡片，cell 列表样式：

| 菜单项 | 说明 | 右侧 |
|--------|------|------|
| 当前版本 | 显示版本号 | `v2.0.0`（`#999`） |
| 检查更新 | 检测新版本 | 右箭头 |
| 用户协议 | 跳转用户协议页 | 右箭头 |
| 隐私政策 | 跳转隐私政策页 | 右箭头 |
| 意见反馈 | 跳转反馈页 | 右箭头 |
| 联系我们 | 显示联系方式 | 右箭头 |

### 公司信息
- 底部居中显示
- 版权信息：**"© 2026 题小助. All rights reserved."**（22rpx，`#ccc`）

---

## 交互逻辑

1. 点击"检查更新"→ 调用版本检查 API
   - 有新版本：弹窗提示更新
   - 已是最新：Toast 提示"已是最新版本"
2. 点击"用户协议"→ `navigateTo` 协议页（type=terms）
3. 点击"隐私政策"→ `navigateTo` 协议页（type=privacy）
4. 点击"意见反馈"→ `navigateTo` 反馈页
5. 点击"联系我们"→ 弹窗显示邮箱/电话

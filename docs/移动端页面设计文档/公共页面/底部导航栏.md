# 底部导航栏（TabBar）

> **组件路径**：`components/TabBar/index.vue`  
> **配置文件**：`pages.json` → `tabBar`  
> **显示页面**：首页、题库、练习、班级、个人中心  
> **实现方式**：自定义 TabBar（非原生 tabBar，支持更丰富的样式和动画）

## 布局结构

**容器**：
- 固定底部：`position: fixed; bottom: 0`
- 背景：白色 `#ffffff`
- 顶部边框：`1rpx solid #f0f0f0`
- 阴影：`0 -2rpx 12rpx rgba(0,0,0,0.04)`
- 安全区适配：`padding-bottom: env(safe-area-inset-bottom)`

**Tab 项**：
- 5 个 Tab 等分宽度
- 每个 Tab 高度：100rpx（不含安全区）
- 布局：垂直居中，图标在上，文字在下

## Tab 项结构

```
┌──────────────┐
│     🔔       │  ← 图标区（可选角标）
│              │
│    首页      │  ← 文字标签
└──────────────┘
```

### 图标区
- 尺寸：48rpx × 48rpx
- 默认状态：灰色图标（`#999`）
- 选中状态：彩色图标（使用 `-active.svg`）
- 支持角标（未读数）：
  - 红色圆点：`#ff4d4f`，16rpx
  - 数字角标：`#ff4d4f` 背景，白色文字 18rpx，最小宽度 32rpx，高度 32rpx，圆角 16rpx

### 文字标签
- 字号：20rpx
- 默认颜色：`#999`
- 选中颜色：与对应模块主色一致
- 与图标间距：6rpx

## 选中状态

### 默认状态
```
   🔘
   题库      #999 灰色
```

### 选中状态
```
   🔵
   题库      #4A90D9 蓝色
```

- 图标切换为 `-active.svg` 版本
- 文字颜色变为主色
- **无放大动画**（保持简洁）

## 页面切换行为

| 操作 | 行为 |
|------|------|
| 点击当前 Tab | 无响应（或滚动到顶部） |
| 点击其他 Tab | `uni.switchTab` 切换页面 |
| 页面显示 | 触发 `onShow` 生命周期 |
| 页面隐藏 | 触发 `onHide` 生命周期 |

## 实现要点

### 自定义 TabBar 配置（pages.json）

```json
{
  "tabBar": {
    "custom": true,
    "color": "#999999",
    "selectedColor": "#4A90D9",
    "backgroundColor": "#ffffff",
    "borderStyle": "white",
    "list": [
      {
        "pagePath": "pages/index/index",
        "text": "首页",
        "iconPath": "static/images/tabbar/home.svg",
        "selectedIconPath": "static/images/tabbar/home-active.svg"
      },
      {
        "pagePath": "pages/question/list",
        "text": "题库",
        "iconPath": "static/images/tabbar/question.svg",
        "selectedIconPath": "static/images/tabbar/question-active.svg"
      },
      {
        "pagePath": "pages/practice/index",
        "text": "练习",
        "iconPath": "static/images/tabbar/exam.svg",
        "selectedIconPath": "static/images/tabbar/exam-active.svg"
      },
      {
        "pagePath": "pages/class/list",
        "text": "班级",
        "iconPath": "static/images/tabbar/class.svg",
        "selectedIconPath": "static/images/tabbar/class-active.svg"
      },
      {
        "pagePath": "pages/profile/index",
        "text": "我的",
        "iconPath": "static/images/tabbar/my.svg",
        "selectedIconPath": "static/images/tabbar/my-active.svg"
      }
    ]
  }
}
```

### 组件结构（components/TabBar/index.vue）

```vue
<template>
  <view class="tabbar">
    <view
      v-for="(item, index) in tabList"
      :key="index"
      class="tabbar-item"
      :class="{ active: current === index }"
      @tap="switchTab(index)"
    >
      <!-- 角标 -->
      <view v-if="item.badge" class="badge">
        <text v-if="item.badge > 0">{{ item.badge > 99 ? '99+' : item.badge }}</text>
      </view>
      <view v-else-if="item.dot" class="dot" />

      <!-- 图标 -->
      <image
        class="tabbar-icon"
        :src="current === index ? item.selectedIconPath : item.iconPath"
      />

      <!-- 文字 -->
      <text class="tabbar-text">{{ item.text }}</text>
    </view>
  </view>
</template>
```

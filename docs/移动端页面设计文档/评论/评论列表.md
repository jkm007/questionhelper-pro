# 评论列表

> **路径**：`pages/comment/list`  
> **参数**：`?targetId=xxx&type=question|exam`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"评论"**
- 返回按钮

### 评论列表
每条评论包含：
- **头像**（64rpx 圆形）+ **昵称**（28rpx，`#333`）+ **时间**（22rpx，`#999`，相对时间）
- 评论内容（28rpx，`#333`）
- 图片网格（180rpx 方形，点击预览 `uni.previewImage`）
- 操作行：❤️ 点赞数 / 💬 回复数

### 嵌套回复
- 1 级嵌套
- 较小头像（48rpx）
- `#f8f8f8` 背景区分
- 回复内容 + 时间 + 操作

### 底部输入栏
固定底部：
- 输入框（圆角，`#F5F7FA` 背景）
- 发送按钮：
  - 无内容：`#ccc` 背景，禁用
  - 有内容：`#4a90d9` 背景，可点击

---

## 交互逻辑

1. 根据 `targetId` 和 `type` 加载评论列表
2. 输入评论 → 点击发送 → 调用评论 API + 刷新列表
3. 点击回复 → 输入框变为回复模式（显示"回复 @xxx"）
4. 点击图片 → 全屏预览 `uni.previewImage`
5. 点赞 → 调用点赞 API + 更新图标

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 评论头像 | 64rpx 圆形 |
| 回复头像 | 48rpx 圆形 |
| 回复背景 | `#f8f8f8` |
| 图片网格 | 180rpx 方形 |
| 输入栏高度 | 100rpx + 安全区 |
| 发送按钮 | `#4a90d9`（可用）/ `#ccc`（禁用） |

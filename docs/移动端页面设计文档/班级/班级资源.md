# 班级资源

> **路径**：`pages/class/resource`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"班级资源"**
- 返回按钮
- 右侧：导入按钮（教师/创作者可见）

### 标签栏
- **全部** / **题目** / **试卷** / **文件**

### 资源列表
每项包含：
- 类型图标（题目=📝、试卷=📋、文件=📎）
- 标题（28rpx，`#333`）
- 上传者 + 上传时间（22rpx，`#999`）
- 文件大小（文件类型显示）

### 导入功能（教师端）
- 从题库导入题目
- 上传文件资源

---

## 交互逻辑

1. 加载班级资源列表
2. 切换标签 → 筛选类型
3. 点击题目/试卷 → 跳转详情
4. 点击文件 → 下载或预览
5. 教师点击导入 → 选择导入方式

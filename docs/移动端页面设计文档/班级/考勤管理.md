# 考勤管理

> **路径**：`pages/class/attendance`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`  
> **权限**：教师/创建者

---

## 页面结构

### 导航栏
- 标题：**"考勤管理"**
- 返回按钮
- 右侧：发起考勤按钮

### 考勤记录列表
每条记录包含：
- 考勤标题 + 发起时间
- 状态：进行中 / 已结束
- 签到人数 / 总人数
- 操作：查看详情 / 导出

### 发起考勤弹窗
- 考勤标题
- 有效时间（分钟）
- 签到方式：定位签到 / 二维码签到 / 口令签到

### 考勤详情
- 签到人员列表：头像 + 昵称 + 签到时间
- 未签到人员列表
- 签到率统计

---

## 交互逻辑

1. 教师发起考勤 → 生成签到二维码/口令
2. 学生扫码/输入口令 → 签到成功
3. 查看详情 → 签到/未签到列表
4. 导出 → 导出 Excel 文件

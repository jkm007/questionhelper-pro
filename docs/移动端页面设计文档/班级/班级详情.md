# 班级详情

> **路径**：`pages/class/detail`  
> **参数**：`?id=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 封面头部
- 封面图（360rpx 高）
- 渐变遮罩：`transparent` → `rgba(0,0,0,0.6)`
- 班级名称（白色，36rpx，粗体）叠加在遮罩上

### 统计行
4 列统计（白色卡片）：
- 资源数
- 考试数
- 作业数
- 通知数
- 数字：`#1677ff`，标签：`#999`

### 操作按钮
- **加入** 按钮（`#52c41a` 绿色，未加入时显示）
- **退出** 按钮（`#ff4d4f` 红色，已加入时显示）
- **成员列表** 按钮（灰色边框）

### 功能入口卡片
5 张卡片，每张有图标字体背景 + 标题 + 箭头：

| 功能 | 背景色 | 图标 |
|------|--------|------|
| 班级资源 | `#e8f3ff` | 📚 |
| 班级考试 | `#fff0e6` | 📋 |
| 班级作业 | `#f6ffed` | 📝 |
| 班级通知 | `#f9f0ff` | 🔔 |
| 班级讨论 | `#fff1f0` | 💬 |

> 注：图标使用 iconfont 字符（`&#xe618;` 等），非 emoji 表情。

---

## 交互逻辑

1. 根据 `id` 加载班级详情
2. 点击"加入"→ 调用加入 API
3. 点击"退出"→ 确认弹窗 → 调用退出 API
4. 点击"成员列表"→ `navigateTo` 成员列表页
5. 点击功能入口 → `navigateTo` 对应页面

# 教师审批

> **路径**：`pages/class/approval`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`  
> **权限**：仅教师/创建者可见

---

## 页面结构

### 标签栏
- **待审批** / **已通过** / **已拒绝**

### 申请卡片
每张卡片包含：
- 申请人：头像 + 昵称
- 申请类型：班级创作者
- 申请理由（3 行截断）
- 申请时间
- 操作按钮（待审批时显示）：
  - **通过**（`#52c41a` 绿色）
  - **拒绝**（`#ff4d4f` 红色）

### 拒绝弹窗
- 拒绝原因输入框
- 确认拒绝按钮

---

## 交互逻辑

1. 加载申请列表（默认显示待审批）
2. 切换标签 → 筛选对应状态
3. 点击"通过"→ 确认弹窗 → 调用审批 API
4. 点击"拒绝"→ 弹出拒绝原因弹窗 → 调用审批 API
5. 审批完成 → 更新列表

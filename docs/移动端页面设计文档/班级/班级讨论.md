# 班级讨论

> **路径**：`pages/class/discussion`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"班级讨论"**
- 返回按钮
- 右侧：发帖按钮（✏️）

### 讨论列表
每张帖子卡片包含：
- 置顶标签（📌 红色，置顶帖显示）
- 标题（28rpx，`#333`，粗体）
- 内容预览（26rpx，`#666`，2 行截断）
- 发布者：头像（40rpx 圆形）+ 昵称（24rpx，`#666`）
- 底部信息行：
  - 💬 回复数（22rpx，`#999`）
  - 👀 浏览数（22rpx，`#999`）
  - 📅 时间（22rpx，`#999`，相对时间）

### 发帖弹窗
底部弹出面板（教师/创建者/班级创作者可见）：
- 标题输入框
- 内容文本域
- 图片上传（最多 9 张）
- 发布按钮

---

## 交互逻辑

1. 根据 `classId` 加载讨论列表（分页）
2. 点击帖子 → `navigateTo` 讨论详情页
3. 点击发帖 → 弹出发帖面板
4. 发布成功 → 刷新列表
5. 下拉刷新 + 上拉加载

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 帖子卡片 | 白色，16rpx 圆角 |
| 置顶标签 | `#ff4d4f`，20rpx |
| 内容预览 | 2 行截断 |
| 图片网格 | 180rpx 方形 |

# 班级列表

> **路径**：`pages/class/list`  
> **Tab 页**：是（班级 Tab）  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 标签栏
- **我的班级** / **发现班级**
- 激活标签下划线（`#1677ff`，3rpx 高，40rpx 宽）

### 搜索栏
- 圆角输入框 + 清除按钮
- 背景：`#F5F7FA`

### 班级列表
水平卡片布局（`ClassCard` 或内联实现）：
- 封面图（200rpx 宽）
- 内容区：名称 + 创建者 + 人数 + 描述
- 发现班级标签页有 **"加入"** 按钮（`#52c41a` 绿色）

### 空状态
- 图片 + 描述文字
- "我的班级"：**"暂未加入任何班级"**
- "发现班级"：**"暂无发现班级"**

### 浮动按钮（FAB）
- 右下角圆形按钮
- 背景：`#1677ff`
- 阴影：`0 8rpx 24rpx rgba(22,119,255,0.4)`
- 图标：＋（白色）
- 仅"我的班级"标签页显示
- 点击 → `navigateTo` 创建班级页

---

## 交互逻辑

1. 进入页面加载"我的班级"列表
2. 切换标签 → 加载对应列表
3. 搜索 → 输入关键词筛选
4. 点击卡片 → `navigateTo` 班级详情页
5. 点击"加入"→ 调用加入班级 API
6. 点击 FAB → `navigateTo` 创建班级页
7. 下拉刷新 + 上拉加载

# 班级列表

> **路径**：`pages/class/list`  
> **Tab 页**：是（班级 Tab）  
> **页面背景**：`#f5f6fa`  
> **核心逻辑**：用户必须先加入班级才能使用班级功能，页面根据是否已加入班级展示不同状态

---

## 页面状态

页面有两种状态，根据用户是否已加入班级切换：

| 状态 | 条件 | 展示内容 |
|------|------|---------|
| **空状态** | 未加入任何班级 | 引导加入班级（主视觉） |
| **班级列表** | 已加入 ≥1 个班级 | 我的班级列表 + 发现班级 |

---

## 加入班级流程

### 方式一：输入班级码

```
步骤 1: 点击"输入班级码加入"
         ↓
步骤 2: 跳转加入班级页（/pages/class/join）
         ↓
步骤 3: 输入班级码 → 点击搜索
         ↓
步骤 4: 显示班级信息卡片（封面 + 名称 + 创建者 + 人数）
         ↓
步骤 5: 点击"确认加入"
         ↓
步骤 6: 调用加入 API → 成功提示 → 跳转班级详情
```

### 方式二：扫描二维码

```
步骤 1: 点击"扫描班级二维码"
         ↓
步骤 2: 调用 uni.scanCode() 打开相机
         ↓
步骤 3: 扫描成功 → 解析班级码
         ↓
步骤 4: 自动搜索班级 → 显示确认弹窗
         ↓
步骤 5: 确认加入 → 调用加入 API → 跳转班级详情
```

### 方式三：发现公开班级

```
步骤 1: 点击"发现公开班级"
         ↓
步骤 2: 跳转发现班级列表（或切换到发现班级标签）
         ↓
步骤 3: 浏览班级列表 → 点击"加入班级"
         ↓
步骤 4: 调用加入 API → 成功提示 → 刷新列表
```

### 方式四：创建新班级

```
步骤 1: 点击"创建新班级"
         ↓
步骤 2: 跳转创建班级页（/pages/class/create）
         ↓
步骤 3: 填写班级信息 → 提交
         ↓
步骤 4: 创建成功 → 自动加入 → 跳转班级详情
```

## 交互逻辑

### 空状态
1. 点击"输入班级码加入"→ `navigateTo` 加入班级页
2. 点击"扫描班级二维码"→ 调用 `uni.scanCode`
3. 点击"发现公开班级"→ `navigateTo` 发现班级页（或切换标签）
4. 点击"创建新班级"→ `navigateTo` 创建班级页（需教师权限）

### 班级列表
1. 进入页面加载"我的班级"列表
2. 切换标签 → 加载对应列表
3. 搜索 → 输入关键词筛选
4. 点击卡片 → `navigateTo` 班级详情页
5. 发现班级点击"加入"→ 调用加入 API + 刷新列表
6. 点击 FAB → `navigateTo` 创建班级页
7. 下拉刷新 + 上拉加载

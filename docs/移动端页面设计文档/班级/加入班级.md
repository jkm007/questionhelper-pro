# 加入班级

> **路径**：`pages/class/join`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"加入班级"**
- 返回按钮

### 方式卡片
2 种方式，水平排列：

| 方式 | 图标 | 图标背景 |
|------|------|----------|
| 输入班级码 | 🔑 | `#e8f3ff` 圆形 |
| 扫描二维码 | 📷 | `#e8f3ff` 圆形 |

- 点击"扫描二维码"→ 调用 `uni.scanCode`

### 搜索区域
- 班级码输入框（placeholder：请输入班级码）
- **搜索** 按钮（`#1677ff`）

### 搜索结果卡片
- 封面图 + 班级名称 + 创建者 + 人数 + 描述
- **确认加入** 按钮（`#52c41a` 绿色）

### 最近加入记录
- 标题：**"最近加入"**
- 列表：头像 + 班级名 + 加入时间

---

## 交互逻辑

1. 输入班级码 → 点击搜索 → 调用搜索 API
2. 搜索到班级 → 显示结果卡片
3. 点击"确认加入"→ 调用加入 API
4. 扫描二维码 → 解析班级码 → 自动搜索并加入
5. 加入成功 → Toast 提示 + 跳转班级详情

---

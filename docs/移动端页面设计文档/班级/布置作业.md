# 布置作业（教师端）

> **路径**：`pages/class/homework-create`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`  
> **权限**：教师/创建者

---

## 页面结构

### 表单字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 作业标题 | 输入框 | 必填，50 字限制 |
| 作业描述 | 文本域 | 选填 |
| 选择题目 | 题目选择器 | 可从题库选择或手动添加 |
| 截止时间 | 日期时间选择器 | 必填 |
| 总分 | 自动计算 | 根据题目自动累加 |
| 允许迟交 | 开关 | 默认关闭 |
| 迟交扣分比例 | 数字输入 | 允许迟交时显示（%） |

### 题目选择器
- 点击"选择题目"→ 跳转题目选择页
- 支持按分类、题型筛选
- 已选题目列表（可拖拽排序、删除）

### 提交按钮
- **布置作业**：`#1677ff` 全宽按钮

---

## 交互逻辑

1. 填写作业信息
2. 选择题目 → 题目选择页 → 返回已选列表
3. 设置截止时间 → 验证时间在未来
4. 提交 → 调用布置作业 API
5. 成功 → Toast 提示 + 返回班级作业页

# 成员列表

> **路径**：`pages/class/member`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"成员列表"**
- 返回按钮

### 搜索栏
- 搜索成员（圆角输入框）

### 成员计数
- 显示总人数（如 "共 32 人"，24rpx，`#999`）

### 成员列表
每行包含：
- 头像（88rpx 圆形）
- 昵称（28rpx，`#333`）
- 角色徽章：
  - 创建者：橙色（`#fa8c16` 背景，白色文字）
  - 管理员：蓝色（`#1677ff` 背景，白色文字）
  - 成员：灰色（`#999` 背景，白色文字）
- 加入日期（22rpx，`#999`）

### 操作菜单（创建者可见）
长按或点击右侧更多按钮 → 底部弹出操作面板：
- 顶部圆角：`24rpx`
- 过渡动画：`0.3s ease`
- 操作项：
  - **设为管理员**
  - **设为普通成员**
  - **移除成员**（红色）

---

## 交互逻辑

1. 根据 `classId` 加载成员列表
2. 搜索 → 输入关键词筛选
3. 创建者长按成员 → 弹出操作菜单
4. 设为管理员/普通成员 → 调用 API 更新角色
5. 移除成员 → 确认弹窗 → 调用移除 API

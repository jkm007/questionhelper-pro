# 班级考试

> **路径**：`pages/class/exam`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"班级考试"**
- 返回按钮
- 右侧：发布考试按钮（教师/创建者可见）

### 状态标签栏
- **全部** / **未开始** / **进行中** / **已结束**
- 激活标签下划线（`#1677ff`）

### 考试卡片
每张卡片包含：
- 标题（30rpx，`#333`，粗体）
- 状态徽章：未开始=蓝、进行中=绿、已结束=灰
- 时间范围（📅 24rpx，`#666`）
- 题目数 + 时长（24rpx，`#666`）
- 分数（已参加时显示）
- 参与人数（👥 24rpx，`#666`）
- 操作按钮：
  - **进入考试**（`#1677ff`，未开始/进行中时显示）
  - **查看详情**（灰色，已结束时显示）

---

## 交互逻辑

1. 根据 `classId` 加载班级考试列表
2. 切换标签 → 筛选对应状态
3. 点击"进入考试"→ `navigateTo` 考试详情/答题页
4. 点击"查看详情"→ `navigateTo` 考试结果页
5. 教师点击发布考试 → `navigateTo` 发布考试页

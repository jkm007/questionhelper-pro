# 创建班级

> **路径**：`pages/class/create`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"创建班级"**
- 返回按钮

### 表单

#### 封面图上传
- 虚线边框占位（200rpx × 200rpx）
- 点击选择图片（`uni.chooseImage`）
- 选中后显示预览图 + 右上角删除按钮

#### 班级名称
- 输入框（必填）
- 30 字限制 + 右下角字数计数器（如 "12/30"）

#### 班级描述
- 文本域
- 200 字限制

#### 最大成员数
- 数字输入框
- 范围：10 - 1000

#### 允许搜索加入
- 开关组件（`switch`）
- 说明文字：**"开启后其他用户可通过搜索加入班级"**

#### 需要审批加入
- 开关组件（`switch`）
- 说明文字：**"开启后加入申请需要管理员审批"**

### 提交按钮
- 底部固定（`position: fixed`）
- `#1677ff` 药丸形按钮（44rpx 圆角）

---

## 交互逻辑

1. 表单验证：名称必填，描述选填
2. 上传封面图 → 调用文件上传 API
3. 提交 → 调用创建班级 API
4. 成功 → Toast 提示 + 返回班级列表
5. 物理返回键 → 确认弹窗（"确定放弃创建？"）

---

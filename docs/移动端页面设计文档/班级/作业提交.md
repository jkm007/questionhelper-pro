# 作业提交

> **路径**：`pages/class/homework-submit`  
> **参数**：`?homeworkId=xxx&classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 作业信息
- 作业标题 + 截止时间（临近 24h 红色警告）
- 总分 + 题目数

### 答题区域
- 逐题展示，每题包含：
  - 题号 + 题型 + 分值
  - 题目内容
  - 答题区（选择题选项 / 填空题输入框 / 简答题文本域）

### 附件上传（可选）
- 支持上传文件附件（图片、PDF 等）
- 最多 5 个文件

### 提交按钮
- **提交作业**：`#1677ff` 全宽按钮
- 提交前确认弹窗

---

## 交互逻辑

1. 加载作业题目
2. 逐题作答
3. 提交 → 确认弹窗 → 调用提交 API
4. 成功 → Toast 提示 + 返回作业列表
5. 超过截止时间 → 提示"已过截止时间，提交将被标记为迟交"

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 题目卡片 | 白色，16rpx 圆角 |
| 截止时间警告 | `#ff4d4f` 文字 |
| 提交按钮 | `#1677ff`，88rpx 高 |

# 班级创作者申请

> **路径**：`pages/class/creator-apply`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 状态卡片（已申请时显示）
- **审核中**：橙色上边框 + 提示文字
- **已通过**：绿色上边框 + 权限说明
- **已拒绝**：红色上边框 + 拒绝原因 + 重新申请按钮

### 申请表单
- 申请理由（文本域，最少 10 字）
- 教育经验描述（文本域，最少 10 字）
- 用户协议复选框

### 提交按钮
- `#1677ff` 全宽按钮

---

## 交互逻辑

1. 加载申请状态
2. 未申请 → 显示表单
3. 提交 → 调用 `POST /:id/creator-apply`
4. 成功 → Toast 提示 + 返回班级详情

---

## 样式规范

| 元素 | 样式 |
|------|------|
| 状态卡片 | 白色，16rpx 圆角，颜色编码上边框 |
| 文本域高度 | 200rpx |
| 提交按钮 | `#1677ff`，88rpx 高 |

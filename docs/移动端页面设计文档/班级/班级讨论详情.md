# 班级讨论详情

> **路径**：`pages/class/discussion-detail`  
> **参数**：`?id=xxx&classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 帖子内容区
- 标题（32rpx，`#333`，粗体）
- 内容（28rpx，`#333`，富文本）
- 图片网格（点击预览）
- 发布者信息：头像 + 昵称 + 角色徽章 + 时间
- 操作：👍 点赞 / 💬 回复 / ⋯ 更多（举报、编辑、删除）

### 回复列表
- 回复数量标题
- 每条回复：头像 + 昵称 + 角色徽章 + 时间 + 内容 + 图片 + 操作

### 底部输入栏
- 回复输入框
- 图片上传按钮
- 发送按钮

---

## 交互逻辑

1. 加载帖子详情 + 回复列表
2. 输入回复 → 发送 → 刷新回复列表
3. 点击图片 → 全屏预览
4. 长按/点击更多 → 操作菜单（编辑、删除、置顶、举报）
5. 点赞 → 调用 API + 更新状态

---

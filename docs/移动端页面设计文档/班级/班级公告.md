# 班级公告

> **路径**：`pages/class/notice`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"班级公告"**
- 返回按钮
- 右侧：发布按钮（教师/创建者可见）

### 公告列表
每张公告卡片包含：
- **未读指示器**：
  - 左边框：`#1677ff`，6rpx 宽
  - 红色圆点：`#ff4d4f`，14rpx
- 标题（28rpx，`#333`，粗体）
- 时间（22rpx，`#999`）
- 内容预览（26rpx，`#666`，3 行截断 `text-overflow: ellipsis`）
- 发布者（头像 + 昵称）

### 发布公告弹窗（教师/创建者）
- 底部弹出面板
- 标题输入框
- 内容文本域
- 发布按钮（`#1677ff`）

---

## 交互逻辑

1. 根据 `classId` 加载公告列表
2. 点击公告 → 展开详情（或 `navigateTo` 详情页）+ 标记已读
3. 点击发布 → 弹出发布公告面板
4. 提交发布 → 调用 API + 刷新列表

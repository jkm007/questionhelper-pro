# 班级作业

> **路径**：`pages/class/homework`  
> **参数**：`?classId=xxx`  
> **页面背景**：`#f5f6fa`

---

## 页面结构

### 导航栏
- 标题：**"班级作业"**
- 返回按钮
- 右侧：布置作业按钮（教师/创建者可见）

### 作业卡片
每张卡片包含：
- 标题（30rpx，`#333`，粗体）
- 状态徽章：进行中=蓝、已截止=灰
- 截止时间（24rpx）：
  - 临近 24 小时：红色 `#ff4d4f` 警告
  - 正常：`#666`
- 提交状态徽章：
  - 待提交：橙色（`#fa8c16`）
  - 已提交：蓝色（`#1677ff`）
  - 已批改：绿色（`#52c41a`）
- 分数（已批改时显示）
- 创建者（24rpx，`#999`）
- 操作按钮：
  - **提交**（`#1677ff`，待提交时显示）
  - **查看结果**（灰色，已批改时显示）

---

## 交互逻辑

1. 根据 `classId` 加载作业列表
2. 点击"提交"→ `navigateTo` 作业提交页
3. 点击"查看结果"→ `navigateTo` 作业结果页
4. 教师点击布置作业 → `navigateTo` 布置作业页

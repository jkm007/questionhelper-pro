## 一、功能概述

### 1.1 功能清单

| 功能 | 说明 |
|------|------|
| 用户注册 | 用户名/手机号/邮箱注册，密码强度检测 |
| 用户登录 | 账号密码/手机验证码/第三方OAuth登录，支持记住我（Refresh Token延长至30天） |
| Token管理 | Access Token + Refresh Token 双令牌，有效期从配置读取 |
| 密码找回 | 手机/邮箱验证码重置密码，独立reset token（15分钟有效期） |
| 退出登录 | Token加入黑名单，支持退出所有设备 |
| 图形验证码 | base64Captcha防机器人，支持数字/字母/算术多种类型 |
| 第三方登录 | 微信(小程序/公众号/扫码)/GitHub/Google |
| 第三方账号绑定 | 绑定/解绑第三方账号，查看绑定状态 |
| 登录设备管理 | 查看/踢出登录设备 |
| 异常登录检测 | IP地理位置突变、多地登录、非常用时间段检测，触发二次验证 |
| 账号注销 | 二次确认（验证码）、30天冷静期、数据导出（JSON格式） |
| 安全设置 | 登录保护、新设备验证、登录通知 |
| 密码强度指示 | 弱=纯数字/纯字母、中=字母+数字、强=字母+数字+特殊字符(12位以上) |
| 短信/邮箱验证 | 频率限制、每日上限、验证次数限制（最多5次错误） |
| 通知渠道 | 站内消息、短信/邮箱通知集成 |

### 1.2 技术栈

| 组件 | 技术选型 |
|------|---------|
| 后端框架 | Go 1.21+ + Gin 1.9+ |
| ORM | GORM 1.25+ |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis 7.x |
| JWT | golang-jwt/jwt/v5 |
| 密码加密 | bcrypt (cost=12, 可通过配置调整) |
| 验证码 | base64-captcha |
| 限流 | 基于Redis分布式令牌桶算法 |
| 日志 | Zap高性能日志库 |
| 配置管理 | Viper |
| IP地理位置 | ip2region |
| 敏感词 | DFA算法自定义实现 |

## 二、数据库设计

### 2.1 用户表 (users)

```sql
CREATE TABLE `users` (
  `id` BIGINT UNSIGNED NOT NULL AUTO_INCREMENT COMMENT '用户ID',
  `username` VARCHAR(50) NOT NULL COMMENT '用户名',
  `password` VARCHAR(100) NOT NULL COMMENT '密码(bcrypt加密)',
  `nickname` VARCHAR(50) DEFAULT '' COMMENT '昵称',
  `email` VARCHAR(100) DEFAULT '' COMMENT '邮箱',
  `phone` VARCHAR(20) DEFAULT '' COMMENT '手机号',
  `avatar` VARCHAR(255) DEFAULT '' COMMENT '头像URL',
  `gender` TINYINT DEFAULT 0 COMMENT '性别:0=未知,1=男,2=女',
  `birthday` DATE DEFAULT NULL COMMENT '生日',
  `bio` VARCHAR(500) DEFAULT '' COMMENT '个人简介',
  `status` TINYINT DEFAULT 1 COMMENT '状态:0=禁用,1=正常,2=注销中',
  `real_name` VARCHAR(50) DEFAULT '' COMMENT '真实姓名',
  `id_card` VARCHAR(100) DEFAULT '' COMMENT '身份证号(AES加密存储)',
  `is_real` TINYINT(1) DEFAULT 0 COMMENT '是否实名认证',
  `register_source` VARCHAR(20) DEFAULT 'web' COMMENT '注册来源:h5/miniapp/app/web',
  `last_login_at` DATETIME DEFAULT NULL COMMENT '最后登录时间',
  `last_login_ip` VARCHAR(50) DEFAULT '' COMMENT '最后登录IP',
  `last_login_device` VARCHAR(200) DEFAULT '' COMMENT '最后登录设备',
  `login_fail_count` INT DEFAULT 0 COMMENT '连续登录失败次数',
  `lock_until` DATETIME DEFAULT NULL COMMENT '锁定截止时间',
  `password_changed_at` DATETIME DEFAULT NULL COMMENT '密码最后修改时间',
  `logout_at` DATETIME DEFAULT NULL COMMENT '注销申请时间',
  `created_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `updated_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  `deleted_at` DATETIME DEFAULT NULL COMMENT '删除时间',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_username` (`username`),
  UNIQUE KEY `uk_phone` (`phone`),
  UNIQUE KEY `uk_email` (`email`),
  KEY `idx_status` (`status`),
  KEY `idx_deleted_at` (`deleted_at`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户表';
```

### 3.2 接口详情

#### 3.2.1 用户注册

**请求**：
```json
POST /api/v1/auth/register
{
  "username": "testuser",
  "password": "Test123456",
  "confirm_password": "Test123456",
  "nickname": "测试用户",
  "phone": "13800138000",
  "email": "test@example.com",
  "captcha_id": "xxx",
  "captcha_code": "1234",
  "agreement": true
}
```

**响应**：
```json
{
  "code": 0,
  "message": "注册成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "user": {
      "id": 1,
      "username": "testuser",
      "nickname": "测试用户",
      "avatar": ""
    }
  }
}
```

#### 3.2.2 用户登录

**请求**：
```json
POST /api/v1/auth/login
{
  "username": "testuser",
  "password": "Test123456",
  "captcha_id": "xxx",
  "captcha_code": "1234",
  "device_id": "device-uuid",
  "remember_me": true
}
```

> **说明**：`remember_me` 为 `true` 时，Refresh Token 有效期延长至30天（默认7天）。Access Token 有效期不变。

**响应**：
```json
{
```

```go
// Register 用户注册
func (s *AuthService) Register(req *dto.RegisterRequest) (*dto.LoginResponse, error) {
    // 检查用户名是否已存在
    existingUser, _ := s.userRepo.FindByUsername(req.Username)
    if existingUser != nil {
        return nil, errors.New("用户名已存在")
    }
    
    // 检查昵称敏感词
    if sensitive.HasSensitive(req.Nickname) {
        return nil, errors.New("昵称包含敏感词")
    }
    
    // 检查手机号是否已存在
    if req.Phone != "" {
        existingUser, _ = s.userRepo.FindByPhone(req.Phone)
        if existingUser != nil {
            return nil, errors.New("手机号已注册")
        }
    }
    
    // 检查邮箱是否已存在
    if req.Email != "" {
        existingUser, _ = s.userRepo.FindByEmail(req.Email)
        if existingUser != nil {
            return nil, errors.New("邮箱已注册")
        }
    }
    
    // 验证图形验证码
    if !s.verifyCaptcha(req.CaptchaID, req.CaptchaCode) {
        return nil, errors.New("验证码错误")
    }
    
    // 加密密码
    hashedPassword, err := encrypt.HashPassword(req.Password)
    if err != nil {
        return nil, err
    }
    
    // 创建用户
    user := &model.User{
        Username: req.Username,
        Password: hashedPassword,
        Nickname: req.Nickname,
        Phone:    req.Phone,
        Email:    req.Email,
        Status:   1,
        RegisterSource: req.Source,
    }
    
    if err := s.userRepo.Create(user); err != nil {
        return nil, err
    }
    
    // 创建默认隐私设置
    privacy := &model.UserPrivacy{UserID: user.ID}
    s.userRepo.CreatePrivacy(privacy)
    
    // 保存密码历史
    s.savePasswordHistory(user.ID, hashedPassword)
    
    // 生成Token
    return s.generateTokens(user, req.DeviceID)
}

// Login 用户登录
func (s *AuthService) Login(req *dto.LoginRequest) (*dto.LoginResponse, error) {
    // 查找用户（支持用户名/手机号/邮箱）
    user, err := s.findUserByUsername(req.Username)
    if err != nil {
        return nil, errors.New("用户不存在")
    }

    // 检查账号状态
    if user.Status == 0 {
        return nil, errors.New("账号已被禁用")
    }
    if user.Status == 2 {
        return nil, errors.New("账号正在注销中")
    }

    // 检查是否被锁定（从MySQL读取锁定状态）
    if user.LockUntil != nil && user.LockUntil.After(time.Now()) {
        remaining := time.Until(*user.LockUntil).Minutes()
        return nil, fmt.Errorf("账号已被锁定，请%.0f分钟后再试", remaining)
    }

    // 验证密码
    if !encrypt.CheckPassword(req.Password, user.Password) {
        // 增加失败次数（MySQL记录）
        s.userRepo.IncrementLoginFailCount(user.ID)

        // 检查是否需要锁定（第5次失败）
        if user.LoginFailCount >= 4 {
            lockUntil := time.Now().Add(15 * time.Minute)
            s.userRepo.LockUser(user.ID, &lockUntil)
            return nil, errors.New("连续登录失败5次，账号已锁定15分钟")
        }

        return nil, fmt.Errorf("密码错误，还可尝试%d次", 4-user.LoginFailCount)
    }

    // 重置失败次数（MySQL）
    s.userRepo.ResetLoginFailCount(user.ID)

    // 更新最后登录信息
    s.userRepo.UpdateLastLogin(user.ID, req.IP, req.DeviceInfo)

    // 记录登录日志
    s.recordLoginLog(user, req, true, "")

    // ---- 异常登录检测 ----
    if s.detectAbnormalLogin(user, req) {
        zap.L().Warn("检测到异常登录",
            zap.Uint("user_id", user.ID),
            zap.String("ip", req.IP),
        )
        return nil, errors.New("检测到异常登录，请通过短信/邮箱验证码完成二次验证")
    }

    // 检测新设备
    isNewDevice := s.checkNewDevice(user.ID, req.DeviceID)
    if isNewDevice {
        s.sendNewDeviceNotification(user, req)
    }

    // 生成Token（支持remember_me）
    return s.generateTokens(user, req.DeviceID, req.RememberMe)
}

// detectAbnormalLogin 异常登录检测
// 检测规则：IP地理位置突变、短时间内多地登录、非常用时间段登录
func (s *AuthService) detectAbnormalLogin(user *model.User, req *dto.LoginRequest) bool {
    // 1. IP地理位置突变检测
    if user.LastLoginIP != "" && user.LastLoginIP != req.IP {
        lastLocation := geoip.Parse(user.LastLoginIP)
        currentLocation := geoip.Parse(req.IP)
        if lastLocation.City != currentLocation.City {
            // 两次登录间隔小于1小时且城市不同，视为异常
            if user.LastLoginAt != nil && time.Since(*user.LastLoginAt) < time.Hour {
                return true
            }
        }
    }

    // 2. 短时间内多地登录检测（30分钟内在2个以上不同城市登录）
    recentLogs := s.getLoginLogsInWindow(user.ID, 30*time.Minute)
    cities := make(map[string]bool)
    for _, log := range recentLogs {
        loc := geoip.Parse(log.IP)
        cities[loc.City] = true
    }
    currentLoc := geoip.Parse(req.IP)
    cities[currentLoc.City] = true
    if len(cities) > 2 {
        return true
    }

    // 3. 非常用时间段检测（凌晨2点-5点）
    hour := time.Now().Hour()
    if hour >= 2 && hour < 5 {
        // 检查用户是否有该时段的历史登录记录
        if !s.hasHistoricalLoginAtHour(user.ID, hour) {
            return true
        }
    }

    return false
}

// generateTokens 生成Token
// rememberMe为true时Refresh Token有效期延长至30天
func (s *AuthService) generateTokens(user *model.User, deviceID string, rememberMe bool) (*dto.LoginResponse, error) {
    // 获取用户角色
    roles, _ := s.userRepo.GetUserRoles(user.ID)
    roleIDs := make([]uint, len(roles))
    roleCodes := make([]string, len(roles))
    for i, role := range roles {
        roleIDs[i] = role.ID
        roleCodes[i] = role.Code
    }

    // 生成唯一标识
    jti := uuid.New().String()

    // 从配置读取Token有效期（而非硬编码）
    accessTokenExpire := config.Auth.AccessTokenExpire   // 默认7200秒
    refreshTokenExpire := config.Auth.RefreshTokenExpire // 默认604800秒
    if rememberMe {
        refreshTokenExpire = config.Auth.RememberMeExpire // 30天
    }

    // 生成Access Token
    token, err := jwt.GenerateToken(user.ID, user.Username, roleIDs, jti, accessTokenExpire)
    if err != nil {
        return nil, err
    }

    // 生成Refresh Token
    refreshToken, err := jwt.GenerateToken(user.ID, user.Username, roleIDs, uuid.New().String(), refreshTokenExpire)
    if err != nil {
        return nil, err
    }

    // 记录登录设备
    s.recordDevice(user.ID, deviceID, jti, req)

    zap.L().Info("Token生成成功",
        zap.Uint("user_id", user.ID),
        zap.Bool("remember_me", rememberMe),
        zap.Int("refresh_expire", refreshTokenExpire),
    )

    return &dto.LoginResponse{
        Token:        token,
        RefreshToken: refreshToken,
        User: dto.UserInfo{
            ID:       user.ID,
            Username: user.Username,
            Nickname: user.Nickname,
            Avatar:   user.Avatar,
            Roles:    roleCodes,
        },
    }, nil
}

// DeactivateAccount 申请注销账号（第一步：验证密码，发送确认验证码）

    if err != nil {
        return errors.New("用户不存在")
    }

    // 验证密码
    if !encrypt.CheckPassword(password, user.Password) {
        return errors.New("密码错误")

    return nil
}

// ConfirmDeactivate 确认注销（第二步：验证验证码后进入冷静期）

    if !s.verifyDeactivateCode(userID, code) {
        return errors.New("验证码错误或已过期")

    if err != nil {
        return errors.New("用户不存在")
    }

    // 更新状态为注销中，进入30天冷静期
    now := time.Now()
    user.Status = 2
    user.LogoutAt = &now
    s.userRepo.Update(user)

    // 记录安全日志
    s.recordSecurityLog(userID, "account_deactivate", "申请注销账号（已二次确认）")

    // 发送通知
    s.sendDeactivateNotification(user)

    zap.L().Info("账号注销申请已确认，进入冷静期", zap.Uint("user_id", userID))
    return nil
}

    resetExpire := config.Auth.ResetTokenExpire // 默认900秒=15分钟

    // 存储reset token到Redis
    ctx := context.Background()
    s.cache.Set(ctx, key.ResetTokenKey(user.ID), resetToken, time.Duration(resetExpire)*time.Second)

    // 发送重置链接/验证码到用户手机或邮箱
    s.sendResetNotification(user, resetToken)

    zap.L().Info("密码重置令牌已生成", zap.Uint("user_id", user.ID), zap.Int("expire", resetExpire))
    return nil
}

// ResetPassword 重置密码（验证reset token后重置，重置后所有设备强制退出）
func (s *AuthService) ResetPassword(account, resetToken, newPassword string) error {
    user, err := s.findUserByUsername(account)
    if err != nil {
        return errors.New("用户不存在")
    }

    // 验证reset token
    ctx := context.Background()
    storedToken, err := s.cache.Get(ctx, key.ResetTokenKey(user.ID))
    if err != nil || storedToken != resetToken {
        return errors.New("重置令牌无效或已过期")
    }

    // 加密新密码（bcrypt cost从配置读取）
    hashedPassword, err := encrypt.HashPassword(newPassword)
    if err != nil {
        return err
    }

    // 更新密码
    user.Password = hashedPassword
    now := time.Now()
    user.PasswordChangedAt = &now
    s.userRepo.Update(user)

    // 保存密码历史
    s.savePasswordHistory(user.ID, hashedPassword)

    // 删除已使用的reset token
    s.cache.Del(ctx, key.ResetTokenKey(user.ID))

    // ---- 密码重置后所有设备强制退出 ----
    // 获取用户所有设备，将Token全部加入黑名单
    devices, _ := s.deviceRepo.FindByUserID(user.ID)
    for _, device := range devices {
        ttl := time.Duration(config.Auth.AccessTokenExpire) * time.Second
        s.cache.Set(ctx, key.TokenBlacklistKey(device.TokenJTI), "1", ttl)
    }
    // 删除所有设备记录
    s.deviceRepo.DeleteByUserID(user.ID)

    zap.L().Info("密码重置成功，所有设备已强制退出", zap.Uint("user_id", user.ID))
    return nil
}
```

#### 4.4.3 限流中间件 (rate_limit.go)

> **统一限流方案**：基于Redis实现分布式令牌桶算法，适用于多实例部署场景。

```go
import (

// RedisTokenBucket 基于Redis的分布式令牌桶
type RedisTokenBucket struct {
    keyPrefix string
    rate      float64 // 每秒产生的令牌数
    capacity  int64   // 桶容量
}

func NewRedisTokenBucket(keyPrefix string, rate float64, capacity int64) *RedisTokenBucket {
    return &RedisTokenBucket{
        keyPrefix: keyPrefix,
        rate:      rate,
        capacity:  capacity,
    }
}

// Allow 检查是否允许请求（Lua脚本保证原子性）
func (b *RedisTokenBucket) Allow(ctx context.Context, identifier string) bool {
    key := fmt.Sprintf("%s:%s", b.keyPrefix, identifier)
    now := time.Now().UnixMilli()

    // Lua脚本：令牌桶算法
    script := `
        local key = KEYS[1]
        local capacity = tonumber(ARGV[1])
        local rate = tonumber(ARGV[2])
        local now = tonumber(ARGV[3])
        local requested = tonumber(ARGV[4])

        local data = redis.call('HMGET', key, 'tokens', 'last_time')
        local tokens = tonumber(data[1]) or capacity
        local last_time = tonumber(data[2]) or now

        -- 计算时间差，补充令牌
        local delta = (now - last_time) / 1000.0
        tokens = math.min(capacity, tokens + delta * rate)

        local allowed = 0
        if tokens >= requested then
            tokens = tokens - requested
            allowed = 1
        end

        redis.call('HMSET', key, 'tokens', tostring(tokens), 'last_time', tostring(now))
        redis.call('EXPIRE', key, 3600)

        return allowed
    `

    result, err := database.RDB.Eval(ctx, script, []string{key},
        b.capacity, b.rate, now, 1).Int()
    if err != nil {
        // Redis异常时放行（降级策略）
        return true
    }
    return result == 1
}

// RateLimitMiddleware 通用限流中间件
func RateLimitMiddleware(keyPrefix string, ratePerSecond float64, capacity int64) gin.HandlerFunc {
    bucket := NewRedisTokenBucket(keyPrefix, ratePerSecond, capacity)

    return func(c *gin.Context) {
        ip := c.ClientIP()
        if !bucket.Allow(c.Request.Context(), ip) {
            response.Error(c, http.StatusTooManyRequests, "请求过于频繁，请稍后再试")
            c.Abort()
            return
        }
        c.Next()
    }
}

// LoginRateLimitMiddleware 登录限流中间件
```

```go
package middleware

import (
    "net/http"
    "strings"
    
    "github.com/gin-gonic/gin"
    "questionhelper-server/pkg/cache"
    "questionhelper-server/pkg/cache/key"
    "questionhelper-server/pkg/database"
    "questionhelper-server/pkg/jwt"
    "questionhelper-server/pkg/response"
)

func AuthMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        authHeader := c.GetHeader("Authorization")
        if authHeader == "" {
            response.Error(c, http.StatusUnauthorized, "未提供认证令牌")
            c.Abort()
            return
        }
        
        tokenString := strings.TrimPrefix(authHeader, "Bearer ")
        if tokenString == authHeader {
            response.Error(c, http.StatusUnauthorized, "认证格式错误")
            c.Abort()
            return
        }
        
        // 解析Token
        claims, err := jwt.ParseToken(tokenString)
        if err != nil {
            response.Error(c, http.StatusUnauthorized, "令牌无效或已过期")
            c.Abort()
            return
        }
        
        // 检查Token是否在黑名单中
        ctx := c.Request.Context()
        exists, _ := database.RDB.Exists(ctx, key.TokenBlacklistKey(claims.JTI)).Result()
        if exists > 0 {
            response.Error(c, http.StatusUnauthorized, "令牌已失效")
            c.Abort()
            return
        }
        
        // 设置上下文
        c.Set("user_id", claims.UserID)
        c.Set("username", claims.Username)
        c.Set("role_ids", claims.RoleIDs)
        c.Set("jti", claims.JTI)
        
        // 更新设备活跃时间
        go updateDeviceActivity(claims.UserID, claims.JTI)
        
        c.Next()
    }
}

func updateDeviceActivity(userID uint, jti string) {
    // 更新设备最后活跃时间
    deviceRepo := user.NewDeviceRepo()
    deviceRepo.UpdateLastActive(userID, jti)
```

### 5.3 Token 自动刷新

```typescript
// src/utils/request.ts

import axios from 'axios'
import { getToken, setToken, getRefreshToken, removeToken } from './auth'
import { refreshToken } from '@/api/auth/auth'

const service = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 15000
})

// 是否正在刷新Token
let isRefreshing = false
// 等待刷新的请求队列
let requestsQueue: Function[] = []

// 请求拦截器
service.interceptors.request.use(
  (config) => {
    const token = getToken()
    if (token) {
      config.headers['Authorization'] = `Bearer ${token}`
    }
    return config
  },
  (error) => {
    return Promise.reject(error)
  }
)

// 响应拦截器
service.interceptors.response.use(
  (response) => {
    return response.data
  },
  async (error) => {
    const originalRequest = error.config
    
    // Token过期，尝试刷新
    if (error.response?.status === 401 && !originalRequest._retry) {
      if (isRefreshing) {
        // 如果正在刷新，将请求加入队列
        return new Promise((resolve) => {
          requestsQueue.push((token: string) => {
            originalRequest.headers['Authorization'] = `Bearer ${token}`
            resolve(service(originalRequest))
          })
        })
      }
      
      isRefreshing = true
      originalRequest._retry = true
      
      try {
        const refreshTokenStr = getRefreshToken()
        const res = await refreshToken(refreshTokenStr)
        const { token, refresh_token } = res.data
        
        setToken(token)
        localStorage.setItem('refresh_token', refresh_token)
        
        // 执行队列中的请求
        requestsQueue.forEach(cb => cb(token))
        requestsQueue = []
        
        originalRequest.headers['Authorization'] = `Bearer ${token}`
        return service(originalRequest)
      } catch (refreshError) {
        // 刷新失败，跳转登录
        removeToken()
        window.location.href = '/login'
        return Promise.reject(refreshError)
      } finally {
        isRefreshing = false
      }
    }
    
    return Promise.reject(error)
  }
)

export default service
```

```typescript
const BASE_URL = 'http://localhost:8080/api/v1'

// 是否正在刷新Token
let isRefreshing = false
let requestsQueue: Function[] = []

export const request = <T = any>(options: RequestOptions): Promise<ApiResponse<T>> => {
  return new Promise((resolve, reject) => {
    const token = uni.getStorageSync('token')
    
    uni.request({
      url: BASE_URL + options.url,
      method: options.method || 'GET',
      data: options.data,
      header: {
        'Content-Type': 'application/json',
        'Authorization': token ? `Bearer ${token}` : '',
        ...options.header
      },
      success: (res) => {
        const data = res.data as ApiResponse<T>
        
        if (data.code === 0) {
          resolve(data)
        } else if (data.code === 10002) {
          // Token过期，尝试刷新
          refreshTokenAndRetry(options, resolve, reject)
        } else {
          uni.showToast({ title: data.message, icon: 'none' })
          reject(new Error(data.message))
        }
      },
      fail: reject
    })
  })
}

// 刷新Token并重试
const refreshTokenAndRetry = async (options, resolve, reject) => {
  if (isRefreshing) {
    // 如果正在刷新，加入队列
    requestsQueue.push(() => {
      request(options).then(resolve).catch(reject)
    })
    return
  }
  
  isRefreshing = true
  
  try {
    const refreshToken = uni.getStorageSync('refresh_token')
    const res = await refreshTokenApi(refreshToken)
    
    uni.setStorageSync('token', res.data.token)
    uni.setStorageSync('refresh_token', res.data.refresh_token)
    
    // 执行队列中的请求
    requestsQueue.forEach(cb => cb())
    requestsQueue = []
    
    // 重试原请求
    const retryRes = await request(options)
    resolve(retryRes)
  } catch (e) {
    uni.removeStorageSync('token')
    uni.removeStorageSync('refresh_token')
    uni.navigateTo({ url: '/pages/login/index' })

  } finally {
```

## 七、安全设计

### 7.1 密码安全

| 措施 | 说明 |
|------|------|
| 加密算法 | bcrypt (cost=12，可通过 `config.Auth.BcryptCost` 配置调整) |
| 密码策略 | 6-20位，必须包含字母和数字 |
| 密码历史 | 不能与最近3次历史密码相同 |
| 传输安全 | HTTPS加密传输 |
| 密码强度指示 | 前端显示密码强度等级，规则如下： |
| | **弱** = 纯数字 或 纯字母 |
| | **中** = 字母 + 数字 |
| | **强** = 字母 + 数字 + 特殊字符，且长度12位以上 |
| 密码重置 | 生成独立reset token，15分钟有效期 |
| 重置后强制退出 | 密码重置成功后，将所有设备Token加入黑名单，强制退出 |

### 7.2 Token安全

| 措施 | 说明 |
|------|------|
| Access Token | 2小时有效期（从 `config.Auth.AccessTokenExpire` 读取） |
| Refresh Token | 7天有效期（从 `config.Auth.RefreshTokenExpire` 读取） |
| 记住我 | 启用后Refresh Token有效期延长至30天（`config.Auth.RememberMeExpire`） |
| 密码重置Token | 独立生成，15分钟有效期（`config.Auth.ResetTokenExpire`） |
| Token黑名单 | Redis存储，退出时加入黑名单 |
| Token刷新 | 静默刷新，用户无感知 |
| JTI唯一标识 | 每个Token都有唯一标识，用于精确黑名单 |

### 7.3 登录安全

| 措施 | 说明 |
|------|------|
| 登录限制 | 连续5次失败锁定15分钟（MySQL记录失败次数和锁定状态） |
| 图形验证码 | 密码错误3次后启用，输入错误时自动刷新 |
| IP限制 | 同一IP每小时最多50次登录请求（基于Redis令牌桶） |
| 异地登录 | 检测并通知 |
| 新设备通知 | 新设备首次登录发送通知 |
| 异常登录检测 | IP地理位置突变检测（使用ip2region）：两次登录间隔<1小时且城市不同 |
| | 短时间内多地登录：30分钟内在2个以上不同城市登录 |
| | 非常用时间段：凌晨2-5点登录且无历史该时段记录 |
| | 检测到异常后要求二次验证（短信/邮箱验证码） |
| 登录失败记录 | 统一使用MySQL记录失败次数和锁定状态，不使用Redis冗余Key |

### 7.4 验证码安全

| 措施 | 说明 |
|------|------|
| 图形验证码 | base64Captcha生成，支持多种类型（见下文） |
| 验证码类型 | **数字验证码**：纯数字，安全级别低，适用于低风险场景 |
| | **字母验证码**：大小写字母，安全级别中 |
| | **算术验证码**：加减乘除运算题，安全级别高（推荐） |
| 安全级别配置 | 通过 `config.Captcha.Type` 配置（digit/letter/math） |
| 验证码自动刷新 | 验证码输入错误时前端自动请求新验证码 |
| 验证次数限制 | 验证码最多允许输入错误5次，超过后自动作废需重新获取 |
| 有效期 | 5分钟 |
| 使用即销毁 | 验证后立即删除 |
| Redis存储 | 避免内存泄漏 |
| IP发送限制 | 同一IP每小时最多发送50次验证码（基于Redis令牌桶） |
| 短信验证码 | 60秒频率限制，每天10次上限 |

### 7.5 第三方登录安全

| 措施 | 说明 |
|------|------|
| 一对一绑定 | 一个第三方账号只能绑定一个本地账号 |
| 绑定/解绑API | `POST /api/v1/user/oauth/bind/{provider}` 绑定 |
| | `DELETE /api/v1/user/oauth/unbind/{provider}` 解绑 |
| | `GET /api/v1/user/oauth/status` 查看绑定状态 |
| 解绑验证 | 解绑需验证密码或短信验证码，至少保留一种登录方式 |
| Token安全 | 第三方Token AES加密存储（密钥通过 `config.AES.Key` 配置） |

### 7.6 AES密钥管理（身份证号加密）

| 措施 | 说明 |
|------|------|
| 加密算法 | AES-256-GCM |
| 密钥存储 | 密钥通过环境变量注入，禁止硬编码在代码中 |
| 密钥配置 | `config.AES.Key`（32字节）、`config.AES.IV`（16字节） |
| 密钥轮换 | 支持密钥版本管理，旧密钥解密、新密钥加密 |
| 密钥备份 | 密钥需在安全位置（如KMS、Vault）备份，丢失将无法恢复身份证号 |

### 7.7 通知渠道设计

| 渠道 | 说明 |
|------|------|
| 站内消息 | `notifications` 表存储，支持已读/未读状态、消息类型分类 |
| 短信通知 | 集成阿里云/腾讯云短信SDK，用于登录通知、密码重置等关键操作 |
| 邮箱通知 | SMTP集成，用于注册确认、密码重置、安全告警等 |
| 通知触发场景 | 新设备登录、异地登录、密码修改、账号注销、绑定/解绑第三方账号 |

## 八、缓存设计

### 8.1 Redis Key 设计

| Key格式 | 数据结构 | TTL | 说明 |
|---------|---------|-----|------|
| `qh:token:bl:{jti}` | String | Token剩余有效期 | Token黑名单 |
| `qh:captcha:{key}` | String | 5分钟 | 图形验证码（含验证次数计数器） |
| `qh:captcha:attempts:{key}` | String | 5分钟 | 验证码错误次数（5次作废） |
| `qh:sms:{phone}` | String | 5分钟 | 短信验证码 |
| `qh:sms:limit:{phone}` | String | 60秒 | 短信发送频率限制（每手机） |
| `qh:sms:daily:{phone}` | String | 24小时 | 每日短信限制 |
| `qh:email:{email}` | String | 5分钟 | 邮箱验证码 |
| `qh:email:limit:{email}` | String | 60秒 | 邮箱发送频率限制 |
| `qh:login:limit:{ip}` | Hash | 1小时 | IP登录限制（令牌桶） |
| `qh:limit:sms:ip:{ip}` | Hash | 1小时 | IP维度验证码发送限制（令牌桶） |
| `qh:reset:token:{uid}` | String | 15分钟 | 密码重置令牌 |
| `qh:user:session:{uid}` | Hash | 7天 | 用户会话信息 |
| `qh:deactivate:code:{uid}` | String | 5分钟 | 注销确认验证码 |

> **说明**：`qh:login:fail:{uid}` 已废弃，登录失败次数和锁定状态统一使用MySQL `users` 表的 `login_fail_count` 和 `lock_until` 字段记录。

### 9.3 账号注销处理

```go
// internal/task/account_deactivate.go

package task

import (
    "time"

    "go.uber.org/zap"
    "questionhelper-server/internal/repository/user"
    "questionhelper-server/internal/service/auth"
)

// AccountDeactivateTask 账号注销处理任务
type AccountDeactivateTask struct {
    userRepo    *user.UserRepo
    authService *auth.AuthService
}

// Run 执行注销
func (t *AccountDeactivateTask) Run() {
    // 查找30天前申请注销的用户
    threshold := time.Now().AddDate(0, 0, -30)
    users, err := t.userRepo.FindDeactivateUsersBefore(threshold)
    if err != nil {
        zap.L().Error("查找注销用户失败", zap.Error(err))
        return
    }

    for _, u := range users {
        // 执行数据删除
        if err := t.authService.DeleteUserData(u.ID); err != nil {
            zap.L().Error("删除用户数据失败", zap.Uint("user_id", u.ID), zap.Error(err))
            continue
        }

        // 标记为已删除
        t.userRepo.MarkAsDeleted(u.ID)

        zap.L().Info("用户数据已永久删除", zap.Uint("user_id", u.ID))
    }
}
```

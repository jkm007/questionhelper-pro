# 错题本模块数据库设计

### 6.3 wrong_questions (错题表)

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | uint | PK | | 主键 |

| mastered | boolean | | false | 是否已掌握 |
| created_at | datetime | | | 创建时间 |
| updated_at | datetime | | | 更新时间 |
| deleted_at | datetime | index, null | | 软删除时间 |

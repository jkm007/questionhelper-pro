# 数据库概述

> 本文档为题小助 V2.0 数据库设计概述，包含 ER 关系总览和枚举值汇总。
> 详细表结构请参见各功能模块文档。

**数据库**：MySQL 8.0
**字符集**：utf8mb4 / utf8mb4_general_ci
**表总数**：164 张

---

### many2many 关系汇总

| 实体 A | 实体 B | 关联表 | 说明 |
|--------|--------|--------|------|
| users | roles | user_roles | 用户-角色 |
| users | tags | user_tags | 用户-标签 |
| roles | menus | role_menus | 角色-菜单 |
| roles | permissions | role_permissions | 角色-权限 |
| questions | knowledge_points | question_knowledge | 题目-知识点 |

### 自引用树结构

| 表 | 父键字段 | 说明 |
|----|----------|------|
| categories | parent_id | 题目分类树 |
| knowledge_points | parent_id | 知识点树 |
| menus | parent_id | 菜单树 |
| depts | parent_id | 部门树 |
| comments | parent_id | 评论回复树 |

# 文件管理模块数据库设计

> 本文档包含文件引用、访问日志、防盗链、水印配置、清理日志等相关数据表。
> 共 5 张表。

---

### 18.1 file_references (文件引用表)

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | uint | PK | | 主键 |
| file_id | uint | not null, index, FK→files.id | | 文件 ID |
| ref_type | varchar(50) | not null | | 引用类型: question/exam/user/class/comment |
| ref_id | uint | not null | | 引用记录 ID |
| ref_field | varchar(50) | | | 引用字段名 |
| created_at | datetime | | | 创建时间 |

### 18.2 file_access_logs (文件访问日志表)

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | uint | PK | | 主键 |
| file_id | uint | not null, index, FK→files.id | | 文件 ID |
| user_id | uint | null, FK→users.id | | 访问用户 |
| ip | varchar(45) | not null | | 访问 IP |
| user_agent | varchar(500) | | | User-Agent |
| referer | varchar(500) | | | 来源页面 |
| access_type | varchar(20) | not null | view | 访问类型: view/download/hotlink |
| response_code | int | null | | HTTP 响应码 |
| response_time | int | null | | 响应耗时 (ms) |
| country | varchar(50) | | | 国家 |
| region | varchar(50) | | | 省份 |
| city | varchar(50) | | | 城市 |
| created_at | datetime | | | 创建时间 |

### 18.3 hotlink_protection_rules (防盗链配置表)

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | uint | PK | | 主键 |
| domain | varchar(255) | unique, not null | | 域名 (支持通配符) |
| is_allowed | tinyint | not null | 1 | 是否允许: 0=黑名单 1=白名单 |
| description | varchar(255) | | | 规则描述 |
| status | tinyint | not null | 1 | 状态: 0=禁用 1=启用 |
| created_at | datetime | | | 创建时间 |
| updated_at | datetime | | | 更新时间 |

### 18.4 watermark_configs (水印配置表)

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | uint | PK | | 主键 |
| name | varchar(100) | not null | | 配置名称 |
| type | varchar(20) | not null | | 水印类型: text/image |
| content | varchar(500) | not null | | 文字内容或图片 URL |
| position | varchar(20) | not null | bottom-right | 位置 |
| opacity | double | not null | 0.5 | 透明度: 0.0-1.0 |
| font_size | int | | 24 | 字体大小 |
| font_color | varchar(20) | | #FFFFFF | 字体颜色 |
| image_width | int | null | | 水印图片宽度 |
| image_height | int | null | | 水印图片高度 |
| margin_x | int | not null | 20 | 水平边距 (px) |
| margin_y | int | not null | 20 | 垂直边距 (px) |
| is_default | tinyint | not null | 0 | 是否默认配置 |
| status | tinyint | not null | 1 | 状态: 0=禁用 1=启用 |
| created_at | datetime | | | 创建时间 |
| updated_at | datetime | | | 更新时间 |

### 18.5 file_cleanup_logs (孤立文件清理记录表)

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | uint | PK | | 主键 |
| batch_no | varchar(50) | not null | | 清理批次号 |
| file_count | int | not null | 0 | 清理文件数量 |
| total_size | bigint | not null | 0 | 清理文件总大小 (字节) |
| status | tinyint | not null | 0 | 状态: 0=进行中 1=完成 2=失败 |
| error_msg | varchar(1000) | | | 错误信息 |
| started_at | datetime | not null | | 开始时间 |
| finished_at | datetime | null | | 完成时间 |
| operator | varchar(50) | | system | 执行者: system/manual |

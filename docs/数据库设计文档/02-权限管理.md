# 权限管理模块数据库设计

> 本文档包含角色、菜单、权限、部门、数据权限等相关数据表。
> 共 8 张表。

## 三、权限管理模块

### 3.1 roles (角色表)

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | uint | PK | | 主键 |
| name | varchar(50) | not null | | 角色名称 |
| code | varchar(50) | unique, not null | | 角色编码 |
| description | varchar(200) | | | 角色描述 |
| is_default | boolean | | false | 是否默认角色 |
| is_system | boolean | | false | 是否系统角色 |
| sort | int | | 0 | 排序 |
| status | tinyint | | 1 | 状态: 0禁用 1启用 |
| created_at | datetime | | | 创建时间 |
| updated_at | datetime | | | 更新时间 |
| deleted_at | datetime | index, null | | 软删除时间 |

## 1. 品牌概述

### 1.1 品牌定位

| 维度 | 说明 |
|------|------|
| **品牌名称** | 题小助 |
| **英文名称** | QuestionHelper |
| **品牌口号** | 学习好帮手 |
| **品牌定位** | 面向大众的社交学习平台 |
| **核心价值** | 学习助手、互助成长、高效刷题 |
| **目标用户** | 学生、考生、学习爱好者 |

### 1.2 品牌调性

```
品牌调性关键词：

┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   亲切      │    │   可爱      │    │   专业      │
│   Friendly  │    │    Cute     │    │ Professional│
│             │    │             │    │             │
│  朋友般的   │    │  萌萌的     │    │  值得信赖   │
│  陪伴感     │    │  年轻活力   │    │  专业可靠   │
└─────────────┘    └─────────────┘    └─────────────┘
         │                │                │
         └────────────────┼────────────────┘
                          │
                          ▼
                   ┌─────────────┐
                   │   助手      │
                   │   Helper    │
                   │             │
                   │  帮助、辅助  │
                   │  学习伙伴    │
                   └─────────────┘
```

### 1.3 品牌个性

- **像小助手**：不是冷冰冰的工具，而是贴心的学习小帮手
- **有温度**：鼓励、支持、陪伴，让学习不孤单
- **可爱活力**：年轻、时尚、有趣，符合年轻人审美
- **可信赖**：专业、准确、高效，值得依赖

## 4. 字体规范

### 4.1 字体选择

```
字体系统：

中文：
├── 主字体：PingFang SC（苹方）
├── 备选：Microsoft YaHei（微软雅黑）
└── 备选：Noto Sans SC（思源黑体）

英文：
├── 主字体：SF Pro Display
├── 备选：Helvetica Neue
└── 备选：Arial

数字：
├── 主字体：DIN Alternate
└── 备选：Roboto Mono
```

### 4.2 字号规范

| 用途 | 字号 | 行高 | 字重 | 示例 |
|------|------|------|------|------|
| **大标题** | 32px | 40px | Bold | 题小助 |
| **标题** | 24px | 32px | Bold | 页面标题 |
| **副标题** | 18px | 26px | Medium | 模块标题 |
| **正文** | 14px | 22px | Regular | 正文内容 |
| **辅助文字** | 12px | 18px | Regular | 时间、备注 |
| **小字** | 10px | 14px | Regular | 标签、角标 |

### 4.3 字重规范

| 字重 | 值 | 用途 |
|------|-----|------|
| **Regular** | 400 | 正文、普通内容 |
| **Medium** | 500 | 标题、强调 |
| **Semibold** | 600 | 重要标题 |
| **Bold** | 700 | 大标题、按钮 |

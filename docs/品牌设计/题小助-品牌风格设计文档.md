## 1. 品牌概述

### 1.1 品牌定位

| 维度 | 说明 |
|------|------|
| **品牌名称** | 题小助 |
| **英文名称** | QuestionHelper |
| **品牌口号** | 学习好帮手 |
| **品牌定位** | 面向大众的社交学习平台 |
| **核心价值** | 学习助手、互助成长、高效刷题 |
| **目标用户** | 学生、考生、学习爱好者 |

### 1.2 品牌调性

```
品牌调性关键词：

┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   亲切      │    │   可爱      │    │   专业      │
│   Friendly  │    │    Cute     │    │ Professional│

└─────────────┘    └─────────────┘    └─────────────┘
         │                │                │
         └────────────────┼────────────────┘
                          │
                          ▼
                   ┌─────────────┐
                   │   助手      │
                   │   Helper    │
                   │             │
                   │  帮助、辅助  │
                   │  学习伙伴    │
                   └─────────────┘

### 1.3 品牌个性

- **像小助手**：不是冷冰冰的工具，而是贴心的学习小帮手
- **有温度**：鼓励、支持、陪伴，让学习不孤单
- **可爱活力**：年轻、时尚、有趣，符合年轻人审美
- **可信赖**：专业、准确、高效，值得依赖

### 1.4 品牌IP形象

![题小助 IP 形象](assets/app-icon-mascot-512.png)

形象特点：
- 圆润可爱：大圆脸 + 肤色渐变

### 2.5 Logo安全区域

Logo 周围需保留足够的净空区域（Clear Space），确保 Logo 在任何场景下不被其他元素挤压，维持品牌识别度。

#### 净空区域规范

```
Logo 安全区域示意图：

          ┌─── 净空 = Logo 高度的 50% ───┐
          │                               │
          ▼                               ▼
    ┌─────────────────────────────────────────┐
    │                                         │
    │     ╔═══════════════════════════╗       │
    │     ║                           ║       │
    │     ║   ┌──────┐                ║       │  ▲
    │     ║   │ ╭──╮ │   题小助      ║       │  │
    │     ║   │ │QH│ │                ║       │  │ 净空
    │     ║   │ ╰──╯ │   QHelper     ║       │  │ = 50%
    │     ║   └──────┘                ║       │  │ Logo
    │     ║                           ║       │  │ 高度
    │     ╚═══════════════════════════╝       │  ▼
    │                                         │
    │     ▲                           ▲       │
    │     └─────── 净空区域 ──────────┘       │
    │                                         │
    └─────────────────────────────────────────┘

    净空区域内不得放置：
    - 其他 Logo 或品牌元素
    - 文字、线条、装饰图形
    - 边框、背景色块
```

#### 最小尺寸限制

| Logo 类型 | 最小高度 | 说明 |
|-----------|----------|------|
| 完整 Logo（图形+文字） | **32px** | 低于此尺寸文字不可读 |
| 图标版 Logo（仅图形） | **24px** | 低于此尺寸细节丢失 |
| 极小场景（Favicon） | **16px** | 使用 favicon-16x16 专用版本 |

> **注意**：当 Logo 高度小于 32px 时，应切换为图标版（简化 Logo）；小于 16px 时使用专用 favicon 文件。

### 3.6 无障碍对比度

遵循 WCAG 2.1 标准，确保文字在各色彩组合下的可读性。

#### WCAG 对比度要求

| 场景 | 最低对比度 | 适用标准 |
|------|-----------|----------|
| 普通正文文字（< 18px） | **4.5:1** | WCAG AA |
| 大号文字（>= 18px 或 >= 14px 粗体） | **3:1** | WCAG AA |
| UI 组件和图形元素 | **3:1** | WCAG AA |
| 增强级正文（AAA） | **7:1** | WCAG AAA |

#### 关键色彩组合对比度

| 前景色 | 背景色 | 对比度 | 是否达标 | 说明 |
|--------|--------|--------|----------|------|
| #FFFFFF（白色） | #4A90D9（主色） | **3.3:1** | 大号文字/UI 达标 | 不适合普通正文 |
| #FFFFFF（白色） | #2E6FB5（主色-深） | **6.2:1** | AA 达标 | 推荐用于按钮正文 |
| #FFFFFF（白色） | #1A1A2E（暗色背景） | **13.8:1** | AAA 达标 | 暗色模式标题 |
| #E2E8F0（暗色文字） | #1A1A2E（暗色背景） | **11.5:1** | AAA 达标 | 暗色模式正文 |
| #303133（正文灰） | #FFFFFF（白色） | **11.7:1** | AAA 达标 | 亮色模式正文 |
| #909399（信息灰） | #FFFFFF（白色） | **4.6:1** | AA 达标 | 辅助信息文字 |
| #C0C4CC（占位灰） | #FFFFFF（白色） | **2.6:1** | 不达标 | 仅用于非必要提示 |
| #94A3B8（暗色次要） | #1E293B（暗色卡片） | **5.8:1** | AA 达标 | 暗色模式辅助信息 |
| #67C23A（成功绿） | #FFFFFF（白色） | **3.1:1** | UI 达标 | 配合图标使用 |
| #F56C6C（错误红） | #FFFFFF（白色） | **3.2:1** | UI 达标 | 配合图标使用 |

#### 设计建议

- 主色 #4A90D9 用于按钮时，按钮文字建议使用白色 + 较大字号（>= 14px 粗体），或改用 #2E6FB5 作为按钮背景
- 状态颜色（成功/警告/错误）应搭配图标使用，不要仅依赖颜色传达信息
- 暗色模式下主色提亮至 #5B9FE6，确保在 #1A1A2E 背景上达到 4.5:1 以上对比度

## 4. 字体规范

### 4.1 字体选择

```
字体系统：

中文：
├── 主字体：PingFang SC（苹方）
├── 备选：Microsoft YaHei（微软雅黑）
└── 备选：Noto Sans SC（思源黑体）

英文：
├── 主字体：SF Pro Display
├── 备选：Helvetica Neue
└── 备选：Arial

数字：
├── 主字体：DIN Alternate
└── 备选：Roboto Mono
```

> **字体兼容性说明**：SF Pro Display 为 Apple 平台专属字体，DIN Alternate 为商业授权字体，在 Web 环境中无法直接使用。建议使用以下开源替代方案：
>
> | 原始字体 | 替代方案 | 来源 | 说明 |
> |----------|----------|------|------|
> | SF Pro Display | **Inter** | Google Fonts | 风格最接近，可变字重，免费商用 |
> | SF Pro Display | **DM Sans** | Google Fonts | 几何无衬线，圆润友好 |
> | DIN Alternate | **Roboto Mono** | Google Fonts | 等宽字体，适合数字展示 |
> | DIN Alternate | **Space Mono** | Google Fonts | 等宽字体，科技感强 |
>
> **Google Fonts 引入方式**：
>
> ```html
> <link rel="preconnect" href="https://fonts.googleapis.com">
> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Roboto+Mono:wght@400;500;700&display=swap" rel="stylesheet">
> ```
>
> **CSS 字体栈（含平台原生备选）**：
>
> ```css
> --font-heading: "Inter", "SF Pro Display", -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif;
> --font-mono: "Roboto Mono", "DIN Alternate", "SF Mono", "Menlo", monospace;
> ```
